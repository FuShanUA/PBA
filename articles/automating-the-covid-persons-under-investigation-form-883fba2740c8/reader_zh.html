# 自动化 COVID 疑似病例（PUI）表单 在 COVID-19 疫情的最前线，临床医生和卫生管理人员肩负着为实验室确诊及 COVID-19 疑似病例（PUI）填写一份长达 100 个字段表单的重任。为了消除这种流程滞后并提升标准化水平，Palantir 开发了一款应用程序，通过安全自动地连接临床医生的电子病历（EMR）系统，实现表单信息的自动预填。随后，去标识化的病例报告数据将实时发送至参与协作的地方、州及联邦系统，而原始机构仍保留对其数据的完全控制权和所有权。 [视频: Palantir：自动化 COVID 疑似病例表单] 如果您所属的医疗机构希望获取访问权限，请观看视频后通过 covid19response@palantir.com 与我们取得联系。 ## 挑战：用数据抗击 COVID-19 大流行 像 COVID-19 这种现已被正式定性为“大流行”的疫情，在鲜为人知、缺乏认知的阶段最为致命。在混乱与错误信息的掩护下，病毒极易肆意蔓延。只有当病毒变得可追踪时，才能减缓其传播速度、实施隔离并最终阻断疫情。然而，可追踪性高度依赖于数据。 公共卫生部门要想有效抗击病毒，唯有详尽、实时地收集现有及潜在病例记录，并将这些信息与人群风险、资源储备等其他关键数据相结合。然而，现代医疗体系的基础设施极其复杂且碎片化，在这样的环境中进行数据采集面临巨大挑战。为了采取有效行动并协助控制疫情，医疗专业人员亟需一种高效且安全的方式来获取数据并开展协同工作。 目前，创建和共享这些信息的重担全部压在了临床医生和地方公共卫生机构身上。针对每一位患者，公共卫生机构都要求临床医生完成一份详尽的病例报告，不仅要记录患者的病毒暴露史，还需涵盖既往病史、症状及人口统计学信息。这些报告为了解 COVID-19 等病毒的特性及传播规律提供了“极其宝贵”的信号；然而，临床医生的时间本就有限，在危机期间更是如此。 使情况更为复杂的是，各辖区之间的报告要求存在显著差异。尽管 CDC 等联邦机构已发布了关于如何采集此类信息的“模板”与指导文件，但如何具体采集和监控 COVID-19 等疾病，最终仍取决于州和地方卫生官员的决策。部分州已投资自建数据录入系统，并将 CDC 模板的某个版本硬编码其中。其他地区则使用 CDC 的 Web 端表单，如 Epi Info、RedCAP 以及《公共卫生事件响应数据整理与集成平台》（DCIPHER）。 归根结底，向上级汇报的负担落在地方机构肩上，但缺乏标准化的汇报机制导致数据采集工作极不统一且支离破碎。医院无法了解自身系统之外的情况；地方机构对所在社区之外的病毒传播态势一无所知；联邦机构则不得不整合来源各异的汇聚数据，以统筹有效的应对措施。最糟糕的是，如此高程度的阻力可能会直接打消临床医生花时间填写病例报告的积极性。 ## 解决方案：基于 FHIR 的病例报告应用 我们的方法在此处进行了演示，其建立在整个医疗行业日益重视采用开放数据标准的趋势之上，例如“FHIR（快速医疗互操作性资源）”规范。“Argonaut 项目”等倡议，以及由“SMART Health IT”主导的 FHIR API 进一步优化，为科技公司、医院系统和政府卫生机构奠定了基础，使其能够创新并构建应用程序，从而有效利用“电子病历”（EMR）系统中蕴含的丰富数据。 考虑到临床医生通常已经能在其 EMR 系统中获取患者病例报告所需的诸多要素，我们的团队开发了一款基于 FHIR 的全新病例报告表单。医院系统可将其注册到自身的 EMR 系统中，从而利用 FHIR API 安全地查询数据。这样一来，临床医生便能将去标识化的患者数据自动预填至符合 CDC 标准的病例报告模板中，更轻松地履行公共卫生机构的法定报告义务。细粒度的访问控制机制可确保仅传输临床医生经医院 EMR 系统授权访问的数据，且这些数据不会被留存或存储于病例报告平台内部。在实际操作中，这不仅减轻了临床医生在诊疗环节所承受的行政负担，也节省了地方机构解析和打包相关信息的时间。此外，它还实现了数据的同构化，便于上级联邦机构收集和监控，向着标准化报告迈出了关键一步。 尽管 FHIR 规范降低了医院采用和使用新病例报告表单的门槛，但要让这些信息的真正价值释放出来，各机构需将此类数据与 COVID-19 防控工作相关的其他关键信息协同利用。通过将孤岛数据源整合至统一的态势视图中，地方和联邦卫生机构能够协调呼吸机等关键设备的供应链物流，精准识别新兴疫情热点中的脆弱人群，并缓解疫情对企业与员工造成的经济冲击。 这不仅是理论构想，更是已经落地的产品。基于 FHIR 的病例报告应用已准备就绪，可快速部署，帮助医院、地方、州及联邦机构协同作战，挽救生命。随着越来越多的医疗组织参与到符合标准的数据共享基础设施中，其价值将产生乘数效应，通过依托 EMR 数据的无源疫情监测系统，加速对新兴疾病的识别，最终让病毒变得更易追踪，从而更轻松地将其击溃。
