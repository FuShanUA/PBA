家庭信息安全

注：在本文中我推荐了多款产品。这些观点均基于我个人的试用、试错和经验得出，绝不代表我所隶属的任何组织或公司的立场（且所有链接均非分销链接）。

作为三个孩子的家长，我深知在日益充斥技术的世界里保护孩子有多么困难。作为 Palantir 的一名信息安全工程师，我也清楚恶意攻击者利用安全漏洞的速度有多快。幸运的是，我在 Palantir 的工作也教会了我如何抢在这些恶意攻击者前面。

我们团队的目标之一是“让世界更安全”，我们会偶尔举办讲座，帮助同事和社区成员抵御信息安全威胁。最近，我一直在做一场演讲，讲述我是如何将团队的一些原则融入育儿过程中的。这篇文章分享了我的信息安全策略建议，旨在帮助家庭管理设备使用，并为孩子的成功保驾护航。

多层防御方法

我最初萌生将信息安全引入家庭生活的念头，是因为我儿子某天深夜熬夜玩电子游戏《堡垒之夜》。第二天早上我问他这怎么可能发生时，他只是简单地回答：“它就让我玩了。”为了防止这种事再次发生，我决定为家长控制创建一套多层防御方法。

像 Palantir 这样的企业网络绝不会依赖任何单一的安全层。相反，他们会在每一层实施管控，并提供备份防御，以确保即使某一层被攻破，安全依然有保障。你的家庭网络也应采用同样的策略，构建能够强化并巩固整体安全态势的信息安全管控机制。

家庭网络中可以包含的层次有：物理层、网络层、操作系统层、游戏机层、手机层、智能家居层和教育层。下面我将针对如何强化每一层提出建议。

物理层

第一步是设置孩子的设备。如果你直接把一台新设备交给他们并由他们自己设置，你就已经输掉这场博弈了。

网络层

让我们从网络层开始。你需要一个拥有易用移动界面的网络，因为如果离开手机就无法操作，你也就不会去使用这些功能。你的网络还应该是无代理的，这意味着你不需要在电脑上安装任何东西——它应该开箱即用。你应该利用网络设置统一的屏幕时间管控，跟踪所有设备上的使用情况，因为你不希望孩子仅仅通过切换设备就能绕过屏幕时间限制。如果可能的话，添加一个主动式入侵检测系统（IDS）将有助于在恶意软件、侵扰性广告和网络钓鱼企图触及你的设备之前就将其拦截。

我首选的路由器是 Norton Core 路由器，遗憾的是它在今年夏天早些时候停产了。（目前你仍可以以大幅折扣购买 Norton Core，且服务仍然正常，不过它很快就会被具有类似功能的另一款型号所取代。）Norton Core 并不是市面上速度最快的路由器，但拥有我所发现的最好用、最易上手的家长控制功能。其他设备要么会停止执行家长控制，要么会对所有人（包括我！）一视同仁地执行限制，要么需要安装代理，而 Norton Core 允许你设置基于年龄的限制，以及对特定设备进行控制，累加在该特定设备上花费的任何时间。

操作系统层

每个操作系统都有不同的家长控制功能，但你应该寻找具备以下几个关键功能的系统。最重要的是，操作系统应支持儿童或受限用户模式，但你也需要屏幕时间控制、应用白名单和内容过滤功能。（一般来说，互联网过滤最好在网络层完成。）

对于年纪较小的孩子（七到九岁），我通常推荐 Chromebook，因为它的界面是锁定的，没有可安装的应用（相反，应用都在云端）。你可以将 Chromebook 设置为仅白名单模式，这样你的孩子只能访问预先批准的网站，而且你在 Chromebook 上设置的任何限制都会应用于孩子的其他 Android 设备。它还会保留浏览历史记录供家长审查。不过，目前它还没有屏幕时间控制功能。

对于需要支持编程的操作系统的较大孩子（10岁以上），我推荐 Windows 10。尽管该操作系统的可用控制功能少于 Chrome，但 Windows 会为家长提供每周报告，并允许你控制孩子的屏幕时间。Windows 还支持一些基于年龄的控制，例如应用下载时的家长通知。此外，你在 Windows 10 上设置的任何限制都会自动应用于 Xbox，尽管报告会区分在每台设备上花费的时间。

对于任何年龄的平板电脑用户，我通常推荐 Amazon Fire 平板电脑儿童版。Fire 平板预定义了基于年龄的网络类别和内容，具备“先学后玩”功能，强制孩子在访问游戏前阅读一段选定的时间，并允许多个孩子使用同一台设备且适用不同的限制。或许最关键的是，它还提供为期两年的防儿童破坏保修，涵盖用户失误造成的损坏——包括把平板扔进马桶冲走！——且绝不多问。

游戏机层

和我一样，我的孩子们都是电子游戏迷，所以我需要为我们的游戏机开发一套家长控制系统。与其他层一样，你希望能够设置屏幕时间控制和过滤互联网内容，但也有一些针对游戏机的特定准则需要遵循。特别是，你希望能够根据游戏评级阻止某些游戏，并限制或完全禁止购买。你还需要一台能报告浏览历史或允许你完全禁用网页浏览的游戏机。

手机层

关于孩子和手机的问题有很多争议。家长们通常可以——而且确实——在什么年龄合适、什么设备合适以及花多少钱合适等问题上持有不同意见。与电子游戏机一样，这个决定取决于你和你的家庭。就我个人而言，我和妻子给大儿子和大女儿配了手机，以便我们能掌握他们的位置。在长途汽车旅行时，他们有时会用手机来安抚我们两岁的小儿子，这算是一个额外的好处。

如果你还没准备好为孩子订购手机套餐，你可以尝试 Cricket Wireless 或 T-Mobile 旗下的 Metro 系列的预付费选项试用几个月。除此之外，iPhone 和 iPad 是不错的选择，因为你可以设定它们在下载应用时需要家长许可。它们还有相当不错的屏幕时间和内容控制功能。

智能家居层

随着智能家居的普及，你可能还需要考虑智能家居层。Ring 门铃和 Nest Hello 非常棒，因为它们可以让你知道孩子何时回家（或试图溜出去）。它还允许你的孩子在没有手机的情况下联系你。

Amazon Cloud Cam 和 Nest Cam 室内版是极佳的婴儿监视器。我试过的大多数婴儿监视器画质很差，或者更令人担忧的是，它们会将隐私信息发送回设备制造地中国。Cloud Cam 和 Nest Cam 具备夜视和移动检测功能，当婴儿移动时会向你发出警报。它们还提供双向音频，这意味着你可以将其用作对讲机与大一点的孩子沟通。

Amazon Echo 儿童版具有许多教育益处，包括拼写单词和阅读故事，并允许你将 Echo 的功能限制在某些特定用途上，比如罚站或刷牙的计时器。它还能防止你的孩子购买物品，并允许你限制他们可以控制的设备，例如，他们可以关掉自己卧室的灯，但不能关掉你的。同样，像 Philips Hue 这样的智能灯允许你控制孩子的灯光，这样你就可以在就寝时间自动关灯。

还有许多其他功能，比如智能锁和智能烟雾报警器，拥有它们也是件不错的事。

教育层

然而，我所做的最重要的事情或许就是与孩子们谈论他们的技术使用。如果没有这种教育，设置所有这些限制可能会造成一种敌对且不可信的局面。通过利用这个机会与孩子们展开对话，我已经能够向他们解释使用技术的危险——以及优势！

特别是，我建议当孩子到了能够进行在线通讯的年龄时，开始与他们探讨防范陌生人的话题。你还应该谈论密码，并强调为什么密码必须保持私密和保密。我告诉我的孩子们，密码就像内衣：不要把它们留在桌子上，要经常更换，不要与任何人分享。网络欺凌也是必须涉及的重要话题——太多人没有意识到在他们的在线互动的另一端是真实的人。我已确保让我的孩子们知道，在线行为会产生现实世界的后果。

我的家人还会一起查看他们的互联网使用报告。我并不隐瞒我知道他们在网上做什么。我们大约每月进行一次讨论，探讨他们是否在富有成效地使用互联网，或者是否应该花更多时间在教育活动中，比如阅读、艺术或 YouTube 上的教学视频。

结论

随着技术日益成为我们日常生活中不可或缺的一部分，教导孩子负责任地使用技术变得至关重要。毕竟，无论如何我们的孩子都会拥抱技术。面临的挑战是确保他们能够以适合其年龄的方式使用技术，并让他们了解使用中潜在的危险。多层防御的家长控制方法有助于确保我们的孩子能够体验现代技术的所有优势，而不会成为其潜在陷阱的受害者。
