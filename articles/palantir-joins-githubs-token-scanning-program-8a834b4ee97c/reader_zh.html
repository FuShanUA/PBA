我们很高兴地宣布，我们已作为其令牌扫描计划的一部分与 GitHub 建立了合作关系。该计划将帮助我们更好地保护客户，通过发现并撤销意外提交到 GitHub.com 的 Palantir 产品令牌。

将身份验证和安全令牌意外泄露到源代码存储库中是一个令人遗憾的现实，如果不被检测到，可能会导致重大的安全事件和违规行为。事实上，攻击者拥有各种工具来发现这些意外泄露——并且可以在几分钟内找到并滥用未撤销的令牌！为了帮助保护我们的客户免受此类攻击，我们已与 GitHub 携手合作，主动搜索并撤销与 Palantir 软件产品相关的密钥和令牌，在发布到 GitHub.com 的几分钟内。

虽然这个项目已经酝酿了一段时间，但作为我们对保护客户及其数据承诺的一部分，特别是在这段前所未有的紧急时期，我们加快了实施和部署的步伐。

我们要感谢 GitHub 建立了这个计划并在此过程中与我们合作。我们强烈鼓励其他组织更广泛地参与进来，并希望这成为所有 SaaS 提供商的标准安全功能。

这只是 Palantir 的信息安全团队不断提升我们组织和客户安全标准的众多方式之一。要了解有关 Palantir 信息安全团队的信息，请访问我们的网站。
