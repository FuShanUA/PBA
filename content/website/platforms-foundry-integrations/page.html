Palantir Foundry is built to integrate.

Since 2003, we’ve been investing in integrations infrastructure for connecting to widely used enterprise software, as well as bespoke and industry-specific systems.

Foundry provides secure, high-scale, integrations for data pipelining, AI/ML, analytics, application building, and more; enabling you to make the most of our platform alongside existing and future technology investments.

04 /

Security [+] Authentication

Foundry’s third-party application authentication and authorization features enable non-Foundry applications and scripts to interact securely with Foundry’s APIs.

OAuth2.0

Inbound — Third-party applications can use OAuth2.0 to authenticate with Foundry when building on top of the Foundry API.
Outbound — Foundry also provides first-class support for OAuth2.0 when connecting to third-party systems for a streamlined user experience.

Custom Authentication

For outbound requests from Foundry to external systems, a wide variety of other authentication methods are supported. Native connectors are tailored to the recommended authentication offered by the target system whether it uses API keys, service accounts, JWT auth, or anything else.
