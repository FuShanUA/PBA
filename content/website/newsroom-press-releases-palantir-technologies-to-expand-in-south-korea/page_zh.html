Palantir Technologies 将在韩国扩展业务

  • 现代重工集团与 Palantir Technologies 将为现代重工集团的核心业务建立大数据平台
  • Palantir Technologies 与现代重工计划成立合资企业，以实现大数据解决方案的商业化
  • 现代重工将部署 Palantir Foundry

丹佛和韩国首尔--(美国商业资讯)--现代重工集团与Palantir Technologies Inc. (Palantir, NYSE: PLTR) 是一家为现代企业构建操作系统的领先厂商，双方将合作为现代重工的核心业务（包括造船和海工工程）建立大数据平台。双方还将考虑成立一家合资企业，以在中长期内推广该大数据平台业务。

现代重工集团和 Palantir 于1月4日签署了关于建立大数据平台及合资企业的谅解备忘录。签字仪式在“CES 2022”开幕前一天以线上和线下混合形式举行，现代重工控股首席执行官 Kisun Chung、现代斗山 Infracore 首席执行官 Young-cheul Cho、现代电气首席执行官 Seok Cho，以及 Palantir Technologies 的高管出席了此次活动。

根据协议，现代重工集团和 Palantir 将为该集团旗下涉及造船与海工工程、能源和工业机械的核心子公司联合建立大数据平台。现代重工集团将提供子公司的流程专业知识和销售诀窍，而 Palantir 将提供软件和开发人员。

在各子公司的大数据平台建成后，双方计划成立一家专门开发和销售大数据平台服务的合资企业。基于积累的成果，该合资企业将把从平台建设到运营的大数据解决方案商业化，面向国内外企业产生销售收入。

现代重工集团旗下负责造船和海工工程的子公司现代重工正在推进“未来造船厂 (FOS)”项目，目标是到2030年转型为全球首个智能造船厂。该项目将实现从设计到生产的全流程实时连接，打造一个支持智能工作管理的造船厂，其中便应用了 Palantir 的 Foundry 数据平台。

该大数据平台也将被现代 Oilbank 等能源子公司采用。这家能源公司计划从今年起，在其位于韩国忠清南道大山郡的工厂建设该平台，为期五年，战略目标是将100多个生产管理系统整合为一个，以实现流程效率最大化。

工业机械子公司现代斗山 Infracore 已于2019年与 Palantir Technologies 共同开发了大数据协作平台“DI 360”，目前正用于管理其零部件供应链、处理现场质量索赔问题并挖掘销售机会。现代重工集团和 Palantir Technologies 还计划考虑为现代建筑设备等其他子公司建立平台。

现代重工控股首席执行官 Kisun Chung 表示：“我们期待与 Palantir Technologies 的合作能大幅提升集团核心业务的竞争力。这将是组织文化创新的重要转折点，改变我们利用数据工作的方式。”

Palantir Technologies Inc. 联合创始人兼首席执行官 Alexander C. Karp 表示：“现代重工集团塑造了20世纪最重要的产业之一，对我们的生活方式至关重要。我们深表感激能有这样一家公司合作，它不仅是全球工业巨头中的领导者，而且其持续增长与成功对全人类的福祉与安全都至关重要。”
