Data Integration + Harmonization

Collect, integrate, and harmonize data flexibly

Foundry integrates and harmonizes complex, multi-modal data, making it analysis-ready. Its secure open architecture, and APIs, and dynamic ontology enable:

  • Secure integration, transformation, and harmonization of data into one open environment

  • Bi-directional connectivity to any systems, including EHRs/EMRs

  • Interoperability of data (including HL7, FHIR, CDISC data) previously unusable side-by-side

  • Interoperability of data models (FHIR and OMOP) and terminologies (LOINC, SNOMED, MedDRA)

Data Sharing

Share data securely

With open APIs and a secure, scalable environment, Foundry can facilitate massive-scale secure data sharing across systems and agencies, enabling:

  • Seamless collaboration and secure data sharing between multiple partners

  • Collaboration among users with various skillsets within the bounds of data access controls

Data Access

Oversee data access dynamically

Foundry features empower data owners—from agencies to their partners—to centrally govern data according to standards, policies, and regulations.

  • Foundry's granular access controls, security markings, and multi-tier organizational structures propagate throughout the platform to ensure that users only have the level of access to data that they are authorized to have.

  • Foundry's controls are flexible—allowing for the rapid reconfiguration of permissions, roles, and more as needed.

Data Quality

Manage data quality

With Foundry, users can verify that data and analyses are accurate, high-quality, and up-to-date. Foundry makes managing data complexity possible through:

  • Data harmonization

  • Automatic data health checks

  • Data quality and privacy validation

  • Tooling for quality issue identification and resolution

  • Resource versioning

  • Data provenance tracking

  • Branching of both code and data to protect the main environment

These features facilitate model-ready data—allowing modeling users to inspect data, understand its validity, and identify biases.

Data-Driven Workflows

Facilitate research and operational workflows

Foundry offers tools that meet users where they are. Its applications draw from and write back to the data environment on a near-real-time basis to support connected workflows and knowledge management. Foundry offers:

  • Point-and-click tools for cohorting and analysis

  • Code-based tools for open languages

  • A model management system to test, deploy, and manage models—all on top of live production data

Interoperability

Prevent vendor lock-in

Foundry is open and interoperable, and prevents vendor-lock in.

  • A major U.S. government agency validated Foundry’s openness by testing the ability to extract and replicate Foundry data, metadata, and logic in a third-party platform.

  • The agency deemed Foundry open and interoperable, and its capabilities set the standard for interoperability by which all other IT investments at the agency are being evaluated.

Data Privacy + Security

Protect data privacy and security

Privacy and security are fundamental values of our work built directly into our software.

  • Foundry is a modern FedRAMP Moderate-authorized SaaS that can meet SOC 2 Type 2, FISMA High, HIPAA, and other standards.

  • We offer features to securely work with PII, PHI, and other sensitive data, including data protection tools for the preparation of public access datasets.

  • All data in Foundry remains in the control of and owned by the data owner—never by Palantir Technologies.
