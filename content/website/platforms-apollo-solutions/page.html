Deploy in Any Environment

Deploy in Multi-cloud, Multi-prem, Private SaaS, and Single Tenant

Apollo supports deploying into many environments via the following capabilities:

  • Easily connect and install into the n+1 environment

  • Efficiently deploy and manages release at scale:

    • Release channels and constraint based deployment vs linear, imperative CI/CD pipelines

    • Configuration management (overriding configuration values on a per-installation basis and managing and auditing who can make changes to configuration)

  • Efficiently manage software at scale after it has been deployed

    • A centralized view including a software catalog and live picture of your deployment landscape, including live health, metric, logging information displayed alongside recent changes

    • Bulk actions across many installations of your products in various environments

    • Can set RBAC policies for various environment types and groups for who can make changes to what

  • Maintain flexibility

    • Doesn’t lock you into any cloud provider

    • One set of advanced tools across all of your environments
