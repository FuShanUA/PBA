Use the model building and training tools you already have, or create your models within Code Workbooks.

No matter what, you’ll get access to a range of AI/ML capabilities that help close the loop between operations and analytics.

  • Evaluate, release & deploy models at scale

  • Build custom, model-driven applications

  • Capture user actions in-app as feedback

Foundry supports connections with your entire existing stack, allowing your team the maximum flexibility to build models externally and import them as libraries, artifacts, or APIs.

  • Sagemaker

  • Azure ML

  • DataRobot

  • Dataiku

Foundry provides a notebook-like environment for exploring datasets, creating visualizations, building and training models, with emphasis on data lineage and reproducibility of resulting models and datasets. With Foundry, you can indicate objectives, associate metadata, and configure your APIs for the objective. This enables you to create, manage, and test scenarios by invoking models interactively.

  • Build and train models with any package

  • Version models and trace lineage

  • Simulate changes end-to-end via branches

  • Re-train models automatically with new data
