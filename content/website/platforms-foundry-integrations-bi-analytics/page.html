Foundry + BI / Analytics Tools in Action

  • Foundry integration for Power BI®
  • Foundry integration for Tableau
Partner
Power BI®
Solution

Use the Foundry Power BI integration tool to ingest data from Foundry directly into your Power BI® reports. The Power BI® connector + ODBC driver allow direct connectivity to Foundry—leveraging your already integrated SAML IdP.

How it Works

The integration with Power BI® consists of two components. The connector provides a customized interface within Power BI® for browsing and loading Foundry datasets available to the user. Once a dataset is selected within Power BI®, the actual request for data is delegated to the ODBC driver component. The driver implements the ODBC specification for communicating with databases. Power BI® natively supports communicating with ODBC drivers, and sends a query to the driver for any datasets that the user has selected to load. The driver then sends these queries to Foundry, and the query is computed and returned.

Tableau
Solution

Use the Foundry Tableau integration tool to connect data from Foundry directly into your Tableau dashboards and easily create Tableau interactive dashboards backed by Foundry data.

How it Works

The integration leverages Foundry access controls, including granular permissions and restricted views. The connector supports Tableau Desktop & Server, Tableau Online (via Bridge), as well as both token-based and OAuth authentication options.
