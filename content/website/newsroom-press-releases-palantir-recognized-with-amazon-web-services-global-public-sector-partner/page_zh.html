Palantir 荣获亚马逊云科技全球公共部门合作伙伴奖

Palantir 因其在 AWS 全球公共部门合作伙伴中开创性的抗疫工作而受到表彰

丹佛--(美国商业资讯)--Palantir Technologies Inc. (NYSE:PLTR) 今日宣布，因其 Palantir 平台在抗击 COVID-19 中的卓越贡献，公司被评为 2021 年度全球 AWS 合作伙伴网络（APN）公共部门合作伙伴奖得主。AWS 全球公共部门合作伙伴奖旨在表彰渠道领域的领导者，他们在帮助客户利用 AWS 云技术推动创新和构建解决方案方面发挥了关键作用。

在抗击 COVID-19 的过程中，Palantir 平台发挥了举足轻重的作用，为客户提供了一个安全平台，用于整合、调和及分析 COVID-19 数据。在美国国立卫生研究院（NIH）的国家新冠队列协作（N3C）项目中，Palantir 的软件被用于加速研究并增进对这种疾病的理解。超过 1000 名研究人员利用 N3C 开展了数百个项目的协作，这些项目最终为制定有效的 COVID-19 应对措施提供了指导。美国政府各部门的其他机构也使用了 Palantir 的产品，以更好地应对这场大流行。

Palantir 副总裁 Josh Harris 表示：“我们的软件专为解决世界上最棘手的问题而生，我们很荣幸能与 AWS 携手，在这场史无前例的公共卫生危机中迎难而上。Palantir 平台能够极大增强各类医疗卫生机构抗击疫情的能力。通过整合分散的数据源并对其进行调和，Palantir 让科学家们得以加速患者研究，并优化针对 COVID-19 的应对措施。”

AWS 全球公共部门合作伙伴与项目副总裁 Sandy Carter 说：“每年，我们的合作伙伴不断利用云技术进行创新，助力其客户提升使命成功率，这总令我们印象深刻，今年也不例外。2021 年 AWS 公共部门合作伙伴奖得主展现了他们对影响全球客户生活的真诚承诺。”

Palantir 将与其他获奖者一起，在 theCUBE 于 2021 年 6 月 30 日举办的特别线上活动中接受表彰。若需进一步了解 Palantir 平台及其在美国国立卫生研究院的成功实施案例，请报名参加《theCUBE 上的 2021 年 AWS 公共部门合作伙伴奖》。

APN 致力于帮助合作伙伴构建、推广和销售其产品，从而帮助他们发展成功的云业务。2021 年度全球 APN 公共部门奖旨在表彰那些积极投身创新、秉持客户至上的理念并取得卓越成果的合作伙伴。获奖者的评选标准包括：践行亚马逊领导力准则、在 APN 中的参与度与成就，以及以客户为中心向公共部门客户交付创新解决方案的表现。

关于 Palantir Technologies

Palantir Technologies 是一家软件公司，专门为数据环境复杂且高度敏感的组织构建企业数据平台。从打造更安全的汽车与飞机，到发现新药和打击恐怖主义，Palantir 帮助公共部门、私营机构和非营利组织的客户彻底改变其使用数据的方式。更多信息请访问 https://www.palantir.com。
