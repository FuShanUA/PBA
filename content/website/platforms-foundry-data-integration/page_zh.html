Data Governance. Automated.

DYNAMIC DATA LINEAGE

Move beyond theoretical and best-guess data lineage crawlers. Foundry provides active metadata and full user attribution to automatically apply and maintain lineage across all data elements. Every version of every dataset in Foundry indicates which code, parent datasets, and runtime created it.

MULTI-MODAL SECURITY

Eliminate “bolt-on” and legacy cloud security approaches. Foundry implements role-, marking-, and purpose-based access controls as active and immutable metadata. Foundry’s security and lineage engines work in concert, providing detailed accounts of who had access to what, and why, across data versions.

EMBEDDED DATA HEALTH

Bring data quality to life through an integrated suite of capabilities for defining, monitoring, and responding to data issues – across all assets. Proactive health checks can be applied using point-and-click or code-based configurations — and work at the level of individual jobs or entire pipeline orchestrations with complex and granular success criteria. User feedback on data quality keeps automated and proactive health checks fresh.

Global Collaboration. Universal Impact.

TREAT DATA LIKE CODE

Bring the power of branching, merging, and granular change management to data pipelines. Branching lets you isolate proposed changes in a sandboxed data branch, which can be fully materialized, examined, and iterated on with atomic commits. Impact analysis on downstream datasets and change management procedures help you review and release with confidence.

DISCOVERABILITY AND CATALOGUING

Foundry Data Catalog offers a curated view of data, while search that traverses all key metadata lets you go deeper. All data, metadata, and associated analytics can be accessed through read/write REST APIs — providing deep integration with other enterprise systems.
