Palantir 获得太空系统司令部追加的 4300 万美元合同

丹佛——（商业资讯）——Palantir Technologies Inc. (NYSE:PLTR) 今日宣布，太空系统司令部 (SSC) 的跨任务地面与通信企业 (ECX) 向其授予了一份价值 4300 万美元的合同，以继续交付数据和决策平台，支持国家安全目标。这份 4300 万美元的合同是一项追加扩展，在今年 4 月和 8 月的先前授标基础上进一步扩大。该合同的累计总面值达 9150 万美元。

此次授标为 Brown Heron 项目提供了持续的能力保障，该项目为北美空防司令部/美国北方司令部 (NORAD/USNORTHCOM, N-NC) 以及跨任务地面与通信企业 (ECX) 下的太空指挥与控制部门实现全域态势感知与决策。被称为 Warp Core 的 Palantir 平台正用于处理海量数据并做出决策，以支持这些任务。

Warp Core 将 Palantir 作为一种可配置的企业数据管理与运营软件解决方案，能够集成和管理来自各种数据源的数据，并支持跨层级和职能社区的应用交付、深度分析和数据驱动决策，包括在多个网络上以多个安全级别进行操作的用户。随着应用和数据生态在规模和复杂性上持续快速扩张，Warp Core 简化并优化了数据管理和集成，从而助力提升战备状态并改善决策。Warp Core 的工作也不仅限于太空领域感知任务，SSC 及其合作伙伴最近还利用它协助美国盟友从阿富汗撤离。

Palantir Technologies 全球国防负责人 Doug Philippone 表示：“正如近几周所见，我国在太空领域及通过太空领域面临的威胁非常现实，我们必须继续推进国家的太空领域感知能力。我们非常自豪能够持续赢得这一创新组织的信任，并继续我们的合作关系。”

关于太空系统司令部

太空系统司令部位于加利福尼亚州埃尔塞贡多的洛杉矶空军基地，是采购和开发军事太空系统的卓越中心。SSC 的业务组合包括太空发射、全球定位系统、军事卫星通信、气象卫星控制网络、靶场系统、天基红外系统以及太空态势感知能力。
