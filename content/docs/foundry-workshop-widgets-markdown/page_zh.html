# Markdown **Markdown** 组件支持渲染带有 Markdown 格式的文本。该组件的高级功能可用于实现文本的归属标注、引用和用户标记等额外工作流。 模块构建者在配置 Markdown 组件时，可使用以下功能： - 基础 Markdown 文本格式，如粗体、斜体、删除线和高亮 - 更高级的 Markdown 格式，如标题、表格、区块样式、代码样式和 URL 下方截图展示了一个已配置的 Markdown 组件示例，其中包含对象引用，带有本体对象引用的文本锚点显示为下划线且可选中： - 从特定高亮锚点文本到本体对象的嵌入引用 - 从特定高亮锚点文本触发点击 Workshop 事件 以下示例展示了一个已配置的 Markdown 组件，该组件显示标注对象，并在嵌入对象上配置了以下用户交互： - 在文本的特定部分显示标注对象 - 从显示的标注对象触发点击 Workshop 操作和事件 - 在文本的特定部分创建新的标注对象 ## 配置选项 下方截图展示了一个未配置的 Markdown 组件的初始状态及其初始配置面板。 对于 Markdown 组件，核心配置选项如下： - **输入数据：文本/变量** - **文本：** 如果选择文本选项，构建者可以直接在配置面板中输入想要显示的 Markdown 输入文本。 - **变量：** 如果选择变量选项，则可以选择一个字符串变量作为要显示的 Markdown 输入文本。 - **显示选项：** - **标签类型配置** - **标准：** 默认启用此选项，以标准 Markdown 格式显示输入内容。 - **内联引用：** 如果启用，则允许在文本中嵌入本体对象引用和点击 Workshop 事件。这是 Markdown 语法的自定义扩展；更多信息请参阅下方的内联引用部分。 - **标注：** 如果启用，则允许显示嵌入在文本中的标注对象、对现有标注对象进行用户交互，以及使用该组件创建新的标注对象。更多信息请参阅下方的标注部分。 - **启用等宽字体：** 切换以启用/禁用在该组件中以等宽字体显示文本。 - **启用滚动：** 切换以启用/禁用该组件的滚动功能。 - **允许长单词换行：** 切换以启用/禁用超出组件宽度的长且不间断字符串（如长 URL）的换行。启用时（这是新组件的默认设置），此类字符串会换行到下一行，而不是溢出组件。 - **换行符断行：** 切换以控制 Markdown 源代码中的单行换行符如何渲染。启用时（这是新组件的默认设置），源代码中的单个换行符会在渲染输出中开启新的一行。禁用时，单个换行符会折叠为空格，遵循标准 Markdown 渲染规则。 - **文本对齐方式：** 设置渲染后的 Markdown 内容在组件内的对齐方式。选择**左对齐**（默认）、**居中对齐**或**右对齐**。

Markdown 表格语法定义的显式列对齐方式（例如 | :---: |）优先于组件级别的文本对齐设置。无论选择何种对齐方式，代码块始终保持左对齐和全宽。

- **用户文本选择：** - **输出用户选定的原始文本：** 将用户选定的文本作为原始 Markdown 字符串输出。 - **选定文本：** 输出的字符串变量，包含用户选定的文本作为原始 Markdown。 - **输出用户选定的索引：** 将用户选定文本的起始和结束索引作为数值变量输出。 - **选定起始索引：** 数值输出变量，包含用户选定文本的起始索引。 - **选定结束索引：** 数值输出变量，包含用户选定文本的结束索引。 ## 语法示例 以下是受支持的 Markdown 语法的一些示例。请注意，尽管 `==text==` 高亮语法和任务列表并非标准 Markdown 实现的规范，但这里依然支持。下表展示了受支持的 Markdown 语法及其对应的示例。 Markdown 支持 1 级到 6 级的子标题。

Markdown 组件中的对象引用

行内引用

作为一项高级功能，Markdown 组件允许构建者为部分 Markdown 文本添加标记（即“锚点”），随后利用这些锚点链接至特定的本体对象，并触发 Workshop 的点击事件。

创建此类锚点的格式如下：

:objectreference[$需显示的文本]{objectType="$对象类型ID" primaryKey="$对象主键"}

我们来走查一个示例，尝试在同一个句子中引用两个航班警报对象。首先，来看看我们希望最终在用户屏幕上呈现的效果。请注意：下方引用的每一个航班警报对象均可由用户单独选中，进而成为 Markdown 组件输出的已选对象集。

为实现上述效果，底层的 Markdown 输入内容如下：

纽瓦克机场在五月__*极少*__出现航班问题，但曾发生过两次高优先级延误：:objectreference[警报 A00150]{objectType="flight-alert" primaryKey="A00150"} 和 :objectreference[警报 A00182]{objectType="flight-alert" primaryKey="A00182"}

除了上文描述的 Markdown 输入语法外，构建者还可以为对象引用配置以下选项：

  • 已选对象集：使用对象引用的必选项。这是 Markdown 组件输出的对象集。当用户在 Markdown 组件中选中某个对象引用时，该对象将被输出至此对象集变量中。
  • 对象类型：使用对象引用时必填。构建者应选择在 Markdown 组件中将被引用的所有对象类型。添加对象类型后，构建者可以在内部配置面板中进一步配置条件格式规则。如果 Markdown 组件中引用了某对象类型但未在此列表中配置，该对象引用将不会显示在 Markdown 组件中。
    • 对象类型：指定要进一步配置颜色和事件选项的对象类型。
    • 高亮颜色：选择静态颜色、从具有本体格式化的属性继承颜色，或定义自定义规则来确定颜色。
    • 选中时覆盖事件：配置针对指定对象类型触发的 Workshop 事件。这些事件将覆盖选中时的任何其他事件。
  • 选中行为：控制 Markdown 组件内的选中行为。
    • 如果选择不高亮，在 Markdown 组件中选择对象引用将不会产生选中状态。
    • 如果选择高亮最后选中项，在 Markdown 组件中选择对象引用将导致最近选中的锚文本被高亮显示。
    • 如果选择高亮选中的引用，Markdown 组件内的高亮将基于所选对象集的内容。当 Markdown 中的对象引用与来自其他组件的对象呈 1:1 对应，且两个组件选定的对象集相同时，此选项效果最佳。
  • 选中事件：此选项使模块构建者能够配置在 Markdown 组件中选中对象引用时触发的 Workshop 事件（例如，导致显示包含更详细对象视图的抽屉面板）。

Markdown 中的对象引用也可以应用标准的 Markdown 格式。下面的截图展示了多种 Markdown 格式示例，例如嵌入了对象的标题和表格。

标注

可以使用标注选项，通过 Markdown 组件在文本上显示标注对象。此选项还允许用户通过在选定的文本部分上创建新的标注对象，或对显示的标注对象执行操作和事件，来与组件内的文本进行交互。

标注对象使用从零开始的数字索引捕获选定的文本，包含起始索引和不包含结束索引，以此来表示文本选择的起止位置。Markdown 组件标注目前不支持负数索引。

选择标注选项后，将提供以下配置：

  • 显示现有标注 配置独立的标注对象层，以在组件中显示各种标注。
    • 标注输入
      • 名称：在组件配置中设置标注层的名称。
      • 标注对象集：指定包含要显示的标注对象类型的对象集。
      • 起始索引：设置表示标注包含起始索引的数字对象属性。
      • 结束索引：设置表示标注不包含结束索引的数字对象属性。
    • 现有标注交互
      • 选中的标注：包含当前选中标注对象的对象集。
      • 选中事件：设置在选中标注对象时触发的事件。
      • 在提示框中显示的属性：指定在标注对象的悬停提示框中显示的对象属性。
      • 悬停交互：配置悬停时的交互（如操作和事件），这些交互将显示在标注对象的悬停提示框中。
        • 图标：设置悬停交互显示的图标。
        • 标题：设置悬停交互显示的标题。将鼠标悬停在交互图标上时将显示该标题。
        • 交互：配置可在标注对象上触发的操作或事件。
          • 悬停的对象：一种特殊的操作参数值，可用于引用悬停的标注对象。
    • 标注显示选项
      • 高亮颜色：设置用于显示标注的颜色。可以静态定义自定义颜色，也可以设置条件格式规则。
      • 标注格式化：设置标注的显示方式。选项包括颜色高亮、使用实线或虚线添加下划线，或同时对标注进行高亮和加下划线。
  • 通过操作或事件创建标注： 配置操作或事件，以便在组件内高亮显示文本时创建新的标注对象。
    • 图标：设置文本高亮时显示的图标。
    • 标题：设置文本高亮时显示的标题。将鼠标悬停在图标上时将显示该标题。
    • 交互： 配置可对高亮文本触发的操作或事件。
      • 高亮文本： 一种特殊的操作参数值，可用于引用组件内高亮显示的文本。

局限性

Markdown 组件不支持渲染 HTML。提供给 Markdown 组件的 HTML 将以纯文本形式显示。
