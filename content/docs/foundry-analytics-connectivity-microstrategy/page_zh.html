从 MicroStrategy 连接到 Foundry 数据集

MicroStrategy 分析平台包含一个经 MicroStrategy 认证的连接器，使用户能够轻松创建由 Foundry 数据集作为数据支撑的 MicroStrategy 报告和档案。该连接器兼容 Foundry 的访问控制机制，包括细粒度的权限管理。

支持的产品：

  • MicroStrategy Workstation 作为预览功能（2021 Update 8 版本）
  • MicroStrategy Workstation（2021 Update 9 或更高版本）
  • MicroStrategy Library（2021 Update 9 或更高版本）

身份验证方式

  • 用户生成的令牌

安装

第 1 部分：验证 MicroStrategy 中是否已安装连接器

如果您使用的是 MicroStrategy 2021 Update 8 版本，Palantir Foundry 连接器应已作为预览功能 ↗安装。请按以下步骤进行验证：

  1. 打开 MicroStrategy Workstation 并启用预览功能。
  2. 在左侧边栏的管理部分，选择数据源旁边的加号。
  3. 在支持的数据源列表中搜索“Palantir Foundry”。如果在列表中找到它，请继续执行第 2 部分以安装 JDBC 驱动程序。

如果您使用的是 2021 Update 9 或更高版本，Palantir Foundry 连接器应已安装在您当前的 MicroStrategy 版本中。请按以下步骤进行验证：

  1. 打开 MicroStrategy Workstation。
  2. 在左侧边栏的管理部分下，选择数据源。
  3. 在支持的数据源列表中搜索“Palantir Foundry”。如果在列表中找到它，则无需单独安装 JDBC 驱动程序；请直接继续阅读下方的使用部分。

如果在支持的数据源列表中未看到 Palantir Foundry，请联系您的 MicroStrategy 支持代表以了解后续步骤。您可能需要升级到最新的 MicroStrategy 版本和 MicroStrategy Workstation。

第 2 部分：安装 Palantir Foundry JDBC 驱动程序

如果您使用的是 2021 Update 8 版本，请先安装 Foundry 数据集的 JDBC 驱动程序，以完成 Foundry 与 MicroStrategy 集成的设置。请前往《下载：Foundry 数据集 JDBC 驱动程序》下载并安装该驱动程序。

如果您使用的是 2021 Update 9 或更高版本，Foundry JDBC 驱动程序已预装在 MicroStrategy Windows Workstation 和 MicroStrategy Intelligence Server 中。如果您想使用本地档案访问 Foundry 数据，只需为 MicroStrategy Mac Workstation 安装 JDBC 驱动程序即可。

如果在安装过程中遇到任何问题，请联系您的 Palantir 代表。

连接器现已安装完毕，您可以开始在 MicroStrategy 中构建由 Foundry 数据集作为支撑的报告和档案了。

使用

本指南将介绍如何通过 MicroStrategy 对 Foundry 进行身份验证、选择数据集，并开始构建您的第一个档案。

第 1 部分：创建 Foundry 数据源

在 MicroStrategy 中创建 Foundry 数据源有两种方式：

  • 方式 1：在 MicroStrategy Workstation 中创建 Foundry 数据源
  • 方式 2：使用 MicroStrategy 数据集创建 Foundry 数据源

方式 1：在 MicroStrategy Workstation 中创建 Foundry 数据源

  1. 如果您使用的是 MicroStrategy 2021 Update 8 版本，请在帮助菜单中启用预览功能。如果您使用的是 2021 Update 9 或更高版本，请直接进行下一步。



  2. 在 Workstation 侧边栏的管理部分下，点击数据源旁边的加号图标。
  3. 在支持的数据源列表中找到并选择“Palantir Foundry”。




  4. 指定数据源名称，保持数据库版本不变。接着，选择将要使用该连接的 MicroStrategy 项目。点击 Default Database Connection 下拉菜单，然后选择 Add New Database Connection。



  5. 按照提示填写 Foundry 连接的详细信息，包括连接属性。



选项 2：使用 MicroStrategy 数据集创建 Foundry 数据源

  1. 如果使用的是 2021 Update 9 版本或更高版本，可以通过 MicroStrategy 数据集创建 Foundry 数据源。在 Workstation 的 Help 菜单中，禁用 Enable New Data Import Experience 选项。



  2. 在 Analysis 部分下，点击 Datasets 旁边的加号图标，然后在 Create Dataset 对话框中选择 Data Import Cube。



  3. 在 Data Sources 窗口中搜索“Palantir Foundry”并从列表中选择它。



  4. 在 Select Import Options 窗口中选择任意一个选项，然后选择 Next。本指南以选择 Select Tables 选项为例。



  5. 在 Import from Table 窗口中，点击 Data Sources 旁边的加号图标以创建数据源。



  6. 按照下方第二部分所示的步骤提供连接属性。



第二部分：配置 Foundry 连接设置

要创建与 Foundry 的连接，必须提供以下连接属性：

  • Server： 通常用于访问 Foundry 的 URL，例如 <subdomain>.palantirfoundry.com。
  • Catalog： Foundry 项目的路径，例如 /MyOrg/MyProject。此字段仅在 2021 Update 9 版本或更高版本中必填。设置此属性可以解决 MicroStrategy 中的表浏览问题。如果使用的是 2021 Update 8 版本，请切换到 Advanced 选项卡，并在 Additional Connection String Parameters 中添加该参数。
  • Token： 在 Foundry 中生成的有效用户令牌。有关如何获取令牌的说明，请参阅《用户生成的令牌》文档。

MicroStrategy 连接器仅支持基于令牌的身份验证。

  • Additional Connection String Parameters（可选）： 切换到 Advanced 选项卡并指定其他任何连接参数。参数之间应使用 & 符号分隔，例如 OptionalParam1=<VALUE>&OptionalParam2=<VALUE>。有关 Foundry 数据集 JDBC 驱动程序的完整连接参数列表，请查阅《参数参考》文档。如果使用的是 2021 Update 9 版本或更高版本，Dialect 将设置为推荐值 SPARK，以实现 Foundry 与 MicroStrategy 的最佳集成，且此设置无法更改。



第三部分：连接到 Foundry 并选择数据集

  1. 使用左侧的导航栏选择 Datasets，然后选择 Data Import Cube 或 Intelligent Cube。
  2. 连接到 Foundry 并选择数据集：
    • 如果使用的是 2021 Update 8 版本，请在 Help 菜单下选择启用 Enable New Data Import Experience。选择在前述步骤中创建的 Foundry 数据源，连接到它，然后导入您的 Foundry 数据集。
    • 如果使用的是 2021 Update 9 版本或更高版本，请在数据源列表中搜索“Palantir Foundry”，将其选中，然后在 Select Import Options 窗口中选择任意选项。在 Import from Table 窗口中，选择之前创建的 Foundry 数据源，连接到它，并导入您的 Foundry 数据集。
  3. 继续在 MicroStrategy 中构建您的报告。
