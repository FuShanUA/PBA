表单 [停止维护]

停止维护

表单功能已进入开发生命周期的停止维护阶段，未来将被废弃。目前仍提供全面支持。我们建议使用本体来构建用户输入工作流，将相关数据结构表示为对象类型，并通过操作和函数配置回写交互：

  • 操作： 对添加、编辑和删除数据相关的权限实现更强大、更精细的控制，包括遵循受限视图以及配置复杂的条件权限。Workshop 和 Slate 原生支持操作，您可以利用全套应用构建工具在其中打造复杂的数据录入用户体验。操作还会为平台 API 自动生成 API 绑定，使您能够将数据写入外部数据系统或触发其他下游效应。
  • 函数： 操作可以由函数提供支持，从而允许实现表现力更强、更复杂的回写逻辑。

什么是表单？

表单是一个表单构建界面，与其他 Foundry 应用程序无缝集成。表单提供直观的操作体验，同时不牺牲复杂的可配置性。

借助表单，没有编程经验的用户也能创建和管理自定义的数据录入与存储解决方案。用户可以使用表单配置各种工作流，包括：

  • 定义动态字段，引导受访者提供数据。
  • 将表单作为区块嵌入 对象资源管理器 和 Slate。
  • 将收集到的信息存储在 Fusion 中。

通过学习如何创建表单或审阅和编辑表单回复，开始使用表单。
