添加应用资源

在创建新的用例之后，您就可以开始为该用例添加新的应用资源了。添加应用后，您可以直接从用例页面跳转至 Foundry 应用程序，例如 Workshop、Slate 和 Quiver。

  • Workshop：一款应用构建工具，支持在零代码环境中为业务操作人员创建交互式高质量应用程序。
  • Slate：一款应用构建工具，支持在代码优先的环境中通过拖拽界面构建动态响应式应用程序。
  • Quiver：一套分析与仪表盘工具，支持对 Foundry 支持的对象和时间序列数据进行大规模的可视化、分析与转换。

创建新应用

在“用例”应用程序中，有两种方式可以创建新应用：通过概览页面创建，或者通过左侧边栏在应用页面创建。

从概览页面创建新应用

在用例的概览页面中，点击应用部分右侧的+ 创建新应用。

如果您尚未为用例保存任何应用，也可以在该部分中央点击+ 添加新应用。

从左侧边栏创建新应用

在左侧边栏中选择应用以打开应用页面。接着，点击右上角的创建新应用。

在创建新应用弹窗中，选择您想要构建的应用程序，然后点击创建。

您也可以直接在应用页面点击Workshop、Slate或Quiver卡片。

系统会将您引导至所选的 Foundry 工作区，以开始构建您的应用程序。保存新应用资源后，该资源及其引用的所有 Ontology 资源都会自动保存到您的用例中。

如需了解有关在Workshop、Slate和Quiver中进行构建的更多信息，或者想进一步了解如何在用例与 Foundry 应用程序之间进行导航，请参阅相关文档。
