召回制品

您可以召回 Conda 制品，以阻止下游使用者基于已召回的版本编译代码。我们建议在启动召回流程前，先准备好已召回制品的补丁版本。

请按照以下步骤召回制品：

  1. 在您的制品仓库中搜索该 Conda 制品并选中它，以在摘要页面查看“版本历史”部分。

  2. 选择要召回的版本，然后选择召回。



  3. 将弹出召回制品对话框。在输入框中填写召回制品的原因。



  4. 再次查看“版本历史”，可以看到该制品现在已标记为已召回。



取消召回

您可以取消召回制品。

要取消召回制品，请选择已召回制品的版本，然后点击取消召回。

删除

Conda 制品支持召回，但无法删除制品仓库中的任何制品。如果确实需要删除某个制品，您必须删除该制品仓库。
