概念：列

要运行本文档中的示例，请添加：from pyspark.sql import functions as F。

列由 PySpark 的 pyspark.sql.Column 类管理。当直接引用现有列或从中派生表达式时，都会创建列实例。可通过以下任一方式引用列：

  • F.col("column_name")
  • F.column("column_name")

引用列并不等同于执行 select 操作，因为“选择”列指的是对要在结果数据集中显示的列进行子集化（及重新排序）。

目录

  • 获取模式
  • 选择：选取要保留的列
  • 创建、更新
  • 重命名、别名
  • 删除：移除列
  • 类型转换

创建与更新

DataFrame.withColumn(name, column)



已复制！
1new_df = old_df.withColumn("column_name", derived_column)
  • new_df：生成的数据框，包含 old_df 中的所有列，并新增了 new_column_name 列。
  • old_df：我们要为其添加新列的数据框。
  • column_name：要创建（如果 old_df 中不存在）或更新（如果 old_df 中已存在）的列名。
  • derived_column：用于派生列的表达式，会应用于 column_name（或你指定的任意列名）下的每一行。

对于已有的 DataFrame，你可以使用 withColumn 方法来创建新列，或者用新值或修改后的值更新现有列。这在实现以下目标时尤为实用：

  1. 基于现有值派生新值

    已复制！

  2. 将值从一种类型转换为另一种类型

    

    已复制！

Copied!

删除列

DataFrame.drop(*cols)

Column.cast(type)

以下是所有存在的数据类型：NullType, StringType, BinaryType, BooleanType, DateType, TimestampType, DecimalType, DoubleType, FloatType, ByteType, IntegerType, LongType, ShortType, ArrayType, MapType, StructType, StructField

通常情况下，你可以使用列上的 cast 方法将大多数数据类型从一种转换为另一种：


```python
from pyspark.sql.types import StringType
df.select(df.age.cast(StringType()).alias("age"))
# 假设 df.age 为 IntegerType()
```

或


已复制！
1df.select(df.age.cast("string").alias("age"))
2# 其效果等同于使用 StringType()。
age
"2"
"5"

类型转换实质上是生成了一个新派生的列，你可以直接对该列执行 select、withColumn、filter 等操作。在 PySpark 中同样适用“向下转型”与“向上转型”的概念，因此你可能会丢失先前数据类型中存储的更细粒度信息，或者引入无用的垃圾信息。

When, otherwise

F.when(condition, value).otherwise(value2)

参数：

  • condition - 布尔类型的 Column 表达式
  • value - 字面值或 Column 表达式

该函数会求值并返回一个与 value 或 value2 参数相同的列表达式。如果未调用 Column.otherwise()，对于不满足匹配条件的记录将返回 None (null) 的列表达式。

when 和 otherwise 操作符类似于 if-else 语句，用于计算新的列值。基本用法如下：



已复制！
1# CASE WHEN (age >= 21) THEN true ELSE false END
2at_least_21 = F.when(F.col("age") >= 21, True).otherwise(False)
3# CASE WHEN (last_name != "") THEN last_name ELSE null
4last_name = F.when(F.col("last_name") != "", F.col("last_name")).otherwise(None)
5df = df.select(at_least_21.alias("at_least_21"), last_name.alias("last_name"))

你还可以根据需要，无限次地串联 when 语句：

已复制！
1switch = F.when(F.col("age") >= 35, "A").when(F.col("age") >= 21, "B").otherwise("C")

这些条件判断的结果可以赋值给列，也可以直接用于过滤：


```python
df = df.withColumn("switch", switch) # switch=A, B, or C
df = df.where(~F.isnull(last_name)) # 过滤出 last_name（空字符串被评估为 null 值后）不为 null 的行
```
