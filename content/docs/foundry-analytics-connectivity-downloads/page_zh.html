本页面提供《Foundry 数据集的 ODBC 和 JDBC 驱动》与《Foundry 数据集的 Tableau 连接器》的下载链接。

可用下载

  • ODBC 驱动
  • JDBC 驱动
  • Tableau 连接器

Foundry 数据集 ODBC 驱动

如需安装该驱动，请下载下方由 Palantir 签名的安装程序文件，并按照屏幕上的提示进行操作。有关配置 Foundry 连接的详细说明，请参阅《ODBC 设置指南》。

此安装程序支持 Windows 64 位系统。目前暂不支持 Windows 32 位、Mac 和 Linux 系统。

下载：

  • Foundry ODBC 驱动 3.39.0 (Windows x64) ↗

Foundry 数据集 JDBC 驱动

如需安装该驱动，请下载下方的 .jar 文件，并按照特定应用程序文档中的说明将其放入计算机上的相应目录中。有关配置 Foundry 连接的详细说明，请参阅《JDBC 设置指南》。

需要 Java 11 及以上版本。

下载：

  • Foundry JDBC 驱动 3.39.0 (JRE 11+) ↗

Foundry 数据集 Tableau 连接器 (2021.1+)

Foundry 数据集 Tableau 连接器由 Palantir 签名，在此以 Tableau 连接器 (.taco) 包文件的形式提供。

该连接器也可在 Tableau Exchange ↗ 上获取。Tableau Exchange 上列出的版本与下方链接中的版本没有任何区别。

下载：

  • foundry-jdbc-2.6.0.taco ↗

Foundry 数据集 Tableau 连接器

如果您的 Tableau 版本介于 2019.4 和 2021.1 之间，请使用此版本的连接器。除仅支持令牌身份验证（不支持 OAuth）且必须通过 RID/路径而非搜索加载数据集外，该版本提供与上述版本相同的功能。

下载：

  • foundry-jdbc-1.2.0.taco ↗
