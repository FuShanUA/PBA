入门指南

在开始建模工作流之前，请确保您已对 Foundry 的核心概念、《代码仓库》、《Python 转换》、《本体》和《函数》有深入的了解。

如果您是初次接触 Foundry，《监督学习教程》提供了从新建项目到将模型部署至生产环境的完整指南。

请查阅《专属页面》，详细了解选择合适工具时需要考虑的各项因素。

常见模型创建工作流

  • 在 Jupyter® 代码工作区、代码仓库中，利用 Palantir 平台的数据训练模型；或者通过模型工作室进行无代码模型训练
  • 导入模型权重并将其注册为模型
  • 导入容器并将其注册为模型
  • 注册外部托管的模型

部署模型以进行推理和调用

  • 将模型作为 Python 转换的输入
  • 在管道构建器中使用模型
  • 在建模目标中创建批处理部署并评估模型性能
  • 配置实时部署，将模型作为 REST 端点提供服务

注册模型以供本体使用

  • 注册实时部署以作为函数使用
  • 在另一个函数中使用该模型函数，以利用模型预测结果编写自定义业务逻辑
