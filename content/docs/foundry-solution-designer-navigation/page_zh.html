导航

解决方案设计师的界面由以下几个关键区域组成，如下节所述，以帮助您在图表中导航：

  • 导航栏（屏幕顶部）：包含图表名称、自动保存指示器、保存按钮、操作菜单（用于导入或导出图表），以及撤销/重做控制。
  • 图表工具栏（屏幕左上角）：包含添加节点到您的图表中的按钮，例如通过概念、组件或资源，以及对这些节点进行分组或添加描述性文本框的按钮。如果启用了AIP功能，您还将看到使用AIP生成和AIP批评者的按钮。
  • 画布（中心）：主要的工作区，您可以在其中构建和可视化您的图表。节点会出现在这里，您可以拖动以平移或使用鼠标滚轮或触控板缩放。
  • 节点工具栏 （当在画布上查看一个节点时）：出现在任何选定节点的上方，提供用于配置和管理节点的操作。有关详细信息，请参阅 Diagrams。
  • 画布控件（屏幕左下角）：提供包含缩放、布局和视图控件的工具栏。有关详细信息，请参阅 Diagrams。
  • 侧边栏（屏幕左侧）： 包含用于查看 图表信息、与 AIP Architect 或 AIP Critic 交互、查看 参考图表 和配置 图设置 的按钮。

导航栏

解决方案设计器顶部的导航栏提供了访问图表级操作的功能，使您可以：

  • 从顶部左侧的文件路径编辑图表的名称。
  • 查看您的图表的自动保存状态，例如 Up to date with manually saved version 或 Saved 2m ago。
  • 通过撤销和重做按钮浏览您的编辑历史记录。
  • 使用保存按钮手动保存您的图表。
  • 通过操作菜单访问导入和导出选项，并上传图片，利用AI视觉功能将手绘或白板图表转换为可编辑的Solution Designer图表。

查看并恢复到以前的版本

Solution Designer会维护您图表的完整版本历史。每次自动保存和手动保存都会创建一个带有时间戳的新版本。

要查看所有图表版本，请在左侧自动保存状态指示器旁边的导航栏中选择版本图标，它显示为v{N}。这将启动版本 显示所有已保存版本的弹窗，每个版本会显示保存时间戳、创建该版本的用户以及所做的更改的预览或描述。

您可以通过选择... > 恢复到此版本来恢复任何先前版本。恢复旧版本会创建一个新版本，并不会从图表的历史记录中删除任何版本。

操作菜单

使用操作菜单导入、导出、共享或清除图表，如果图表包含资源（如数据集或对象类型），还可以在数据血缘中打开它。

  • 导入图表：从JSON文件、图片上传图表，或从数据血缘中导入。
  • 导出图表： 您可以将整个图表或所选节点保存或复制到JSON文件中，用于备份、共享或粘贴到其他工具中。此外，您还可以将整个图表或所选节点导出为高分辨率PNG图像，Solution Designer会以2倍缩放显示以确保清晰度。

要导出图表，您必须拥有该图表的编辑权限，因为只读查看者无法导出图表。

图表工具栏

图表工具栏位于屏幕左上角，包含用于向您的图表添加节点以及访问Solution Designer关键功能的按钮：

  • AIP Generate（如果启用了AIP）：启动AI驱动的工作流规划。参见 AIP Architect。
  • 概念： 添加抽象概念节点。参见 概念节点。
  • 组件： 打开组件选择器以添加平台组件（Cmd/Ctrl + B）。参见 组件节点。
  • 资源： 添加与Foundry资源关联的节点。请参见 资源节点。
  • 组： 添加空组以对节点进行分组。请参见 组。
  • 文本： 添加用于注释的文本节点。请参见 文本节点。
  • AIP 评论员（如果启用了AIP）：打开图表评审面板。请参见 AIP 评论员。
  • 平移模式切换： 在编辑和只读平移模式之间切换。

侧边栏

在屏幕左侧的侧边栏中选择一个图标以显示其可折叠且可调整大小的完整视图。不同的按钮会在侧边栏中打开特定的面板：

  • 图表信息： 提供一个文本输入框，您可以在此编写图表描述，并渲染文档和教程链接。
  • AIP 架构师： 使用AIP 架构师进行AI驱动的工作流规划。请参见 AIP Architect。
  • AIP Critic: 查看并与基于AI的图表审查面板进行交互。参见 AIP Critic。
  • 参考架构图: 加载并浏览 Solution Designer 的参考架构库。
  • 图示设置: 查看和配置图表的设置和显示选项。

参考架构库

参考架构图 面板显示参考架构库，您可以浏览并加载按类别组织的预构建模式和模板（行业模式、技术模式和工作流模式）。通过关键字搜索模式，预览它们，然后创建新图表、添加到现有图表中或覆盖当前图表。

配置您的图表设置

图设置面板允许您自定义图表的显示方式并配置解决方案设计器功能。

可用的设置包括边显示选项、画布视图控制（小地图、背景网格和平移模式）、图标显示模式以及AIP功能开关。设置会本地保存到您的图表中。

有关小地图和平移控制等特定画布功能的详细信息，请参阅 画布控制 和 平移和缩放 文档。

键盘快捷键

解决方案设计器支持键盘快捷键以简化您的工作流程。

快捷键操作
Cmd/Ctrl + B打开组件选择器
Shift + 点击或拖动多选节点
Backspace 或 Delete删除所选节点或连接
Cmd/Ctrl + C将所选节点复制到剪贴板
Cmd/Ctrl + V从剪贴板粘贴节点
Cmd/Ctrl + Z撤销上一个操作
Cmd/Ctrl + Y 或 Cmd/Ctrl + Shift + Z重做上一个操作

这些快捷方式允许您高效地构建和修改图表，而无需不断切换菜单。

URL参数

Solution Designer 支持特定行为的 URL 参数：

  • viewer=true: 以只读查看器模式打开图表，并自动启用平移模式。这对于与只能查看不能编辑的干系人共享图表非常有用。
  • focusRID=<resource-id>: 自动将特定节点居中并放大，该节点由其资源 RID 标识。

示例：创建一个只读图表的链接

https://{FOUNDRY_URL}/workspace/solution-designer/<diagram-rid>?viewer=true

这将以只读查看器模式打开图表，非常适合演示或审查。
