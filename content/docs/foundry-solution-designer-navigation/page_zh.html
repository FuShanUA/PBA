导航

Solution Designer 的界面由以下几个关键区域组成，以帮助您浏览图表：

  • 导航栏（屏幕顶部）：包含图表名称、自动保存指示器、保存按钮、操作菜单（用于导入或导出图表）以及撤销/重做控件。
  • 图表工具栏（屏幕左上方）：包含用于向图表添加节点的按钮，例如通过概念、组件或资源添加，以及将这些节点分组或添加描述性文本框的按钮。如果启用了 AIP 功能，您还会看到使用AIP 生成和AIP Critic的按钮。
  • 画布（中心）：用于构建和可视化图表的主要工作区。节点会显示在此处，您可以拖动进行平移，或使用鼠标滚轮或触控板进行缩放。
  • 节点工具栏（在画布上查看节点时）：显示在所选节点上方，包含用于配置和管理节点的操作。详情请参阅《图表》。
  • 画布控件（屏幕左下方）：提供一个包含缩放、布局和视图控件的工具栏。详情请参阅《图表》。
  • 侧边栏（屏幕左侧）：包含用于查看图表信息、与AIP Architect或AIP Critic交互、查看参考图表以及配置图形设置的按钮。

导航栏

Solution Designer 顶部的导航栏提供了对图表级操作的访问权限，使您能够：

  • 从左上角的文件路径编辑图表名称。
  • 查看图表的自动保存状态，例如与手动保存的版本保持一致或2分钟前已保存。
  • 通过撤销和重做按钮浏览编辑历史记录。
  • 使用保存按钮手动保存图表。
  • 访问导入和导出选项，并通过操作菜单利用 AI 视觉功能上传图像，将手绘或白板图表转换为可编辑的 Solution Designer 图表。

查看并恢复到先前版本

Solution Designer 会维护图表的完整版本历史记录。每次自动保存和手动保存都会创建一个带有时间戳的新版本。

要查看所有图表版本，请在自动保存状态指示器左侧的导航栏中选择显示为v{N}的版本图标。这将启动版本模态窗口，显示所有已保存版本的列表，每个版本都会显示保存时的时间戳、创建该版本的用户以及更改的预览或说明。

您可以通过选择... > 恢复到此版本来还原任何先前的版本。还原旧版本会创建一个新版本，不会从图表的历史记录中删除任何版本。

操作菜单

使用操作菜单可以导入、导出、共享或清除图表，如果图表包含资源（如数据集或对象类型），还可以在《数据血缘》中打开它。

  • 导入图表：从 JSON 文件、图像上传图表，或从数据血缘中导入。
  • 导出图表：您可以将整个图表或所选节点保存或复制到 JSON 文件中，以用于备份、共享或粘贴到其他工具中。此外，您还可以将整个图表或所选节点导出为高分辨率 PNG 图像，Solution Designer 会以 2 倍缩放比例渲染该图像以确保清晰度。

您必须拥有图表的编辑权限才能导出图表，只读查看者无法导出图表。

图表工具栏

图表工具栏显示在屏幕左上方，包含用于向图表添加节点和访问 Solution Designer 核心功能的按钮：

  • AIP 生成（如果启用了 AIP）：启动 AI 驱动的工作流规划。请参阅《AIP Architect》。
  • 概念：添加抽象概念节点。请参阅《概念节点》。
  • 组件：打开组件选择器以添加平台组件（Cmd/Ctrl + B）。参见《组件节点》。
  • 资源：添加链接到 Foundry 资源的节点。参见《资源节点》。
  • 分组：添加空分组以组织节点。参见《分组》。
  • 文本：添加文本节点以进行标注。参见《文本节点》。
  • AIP Critic（如果已启用 AIP）：打开图表审查面板。参见《AIP Critic》。
  • 平移模式切换：在编辑和只读平移模式之间切换。

侧边栏

选择屏幕左侧侧边栏中的图标，可显示其完整视图，该视图支持折叠和调整大小。不同的按钮会在侧边栏中打开特定的面板：

  • 图表信息：提供一个文本输入框，您可以在此编写图表描述，并渲染文档和教程链接。
  • AIP Architect：使用 AIP Architect 进行 AI 驱动的工作流规划。参见《AIP Architect》。
  • AIP Critic：查看并交互使用 AI 驱动的图表审查面板。参见《AIP Critic》。
  • 参考图表：加载并浏览 Solution Designer 的参考架构库。
  • 图表设置：查看并配置您的图表设置和显示选项。

参考架构库

参考图表面板会显示参考架构库，您可以在此浏览并加载按类别（行业模式、技术模式和工作流模式）组织的预置模式和模板。通过关键字搜索模式并进行预览，然后创建新图表、添加到现有图表或覆盖当前图表。

配置图表设置

图表设置面板支持您自定义图表的显示方式并配置 Solution Designer 的各项功能。

可用设置包括边线显示、画布视图控制（小地图、背景网格和平移模式）、图标显示模式以及 AIP 功能开关。这些设置会本地保存到您的图表中。

有关小地图和平移控制等特定画布功能的详情，请参见《画布控制》和《平移与缩放》文档。

键盘快捷键

Solution Designer 支持键盘快捷键以简化您的工作流。

快捷键操作
Cmd/Ctrl + B打开组件选择器
Shift + 点击或拖拽多选节点
Backspace 或 Delete删除选中的节点或连接
Cmd/Ctrl + C复制选中的节点到剪贴板
Cmd/Ctrl + V从剪贴板粘贴节点
Cmd/Ctrl + Z撤销上一步操作
Cmd/Ctrl + Y 或 Cmd/Ctrl + Shift + Z重做上一步操作

这些快捷键让您无需频繁切换菜单即可高效构建和修改图表。

URL 参数

Solution Designer 支持通过 URL 参数实现特定行为：

  • viewer=true：以只读查看模式打开图表，并自动启用平移模式。适用于向仅需查看而无需编辑的利益相关者分享图表。
  • focusRID=<resource-id>：自动居中并缩放至由资源 RID 标识的特定节点。

示例：创建只读图表链接

https://{FOUNDRY_URL}/workspace/solution-designer/<diagram-rid>?viewer=true

这会以只读查看模式打开图表，非常适合用于演示或评审。
