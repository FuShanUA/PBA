Oracle Eloqua

Oracle Eloqua 连接器是用于 Oracle Eloqua 的 Palantir 提供的驱动程序。

要创建新的 Oracle Eloqua 源，请遵循 Palantir 提供的驱动程序的标准设置流程，然后使用以下章节进行 Oracle Eloqua 特定的配置和网络设置。有关完整的属性参考，请参阅 官方 Oracle Eloqua 驱动程序文档 ↗。

配置

以下属性为必填或推荐项。

属性是否必填？描述默认值
AuthScheme ↗必填使用的身份验证方案。接受的条目为 Basic 和 OAuth。Basic
Schema ↗必填要使用的架构类型。OracleEloqua
Company ↗推荐Eloqua 账户的公司。此字段用于对用户进行身份验证。—
InitiateOAuth ↗推荐指定获取或刷新 OAuth 访问令牌的流程，以便在已验证、已授权的用户工作时维持用户的访问权限。REFRESH
OAuthClientId ↗推荐指定创建自定义 OAuth 应用程序时分配的客户端 ID。（也称为使用者密钥）。此 ID 将自定义应用程序注册到 OAuth 授权服务器。—
OAuthClientSecret ↗推荐指定创建自定义 OAuth 应用程序时分配的客户端密钥。（也称为使用者密钥）。此密钥将自定义应用程序注册到 OAuth 授权服务器。—
Password ↗推荐指定进行身份验证的用户账户的密码。—
User ↗推荐指定进行身份验证的 Oracle Eloqua 用户账户的用户 ID。—
