表格行生成器

此小部件仅适用于 文档模板。

表格行生成器 可用于动态地为传递给 Notepad 文档模板的对象集中的每个对象生成一行。表格行生成器与 部分生成器 非常相似，但不同于通用部分，表格行生成器返回的是行，并且只能在 Notepad 表格中使用。

可以通过表格右键菜单或工具栏中的表格 插入 操作将行生成器插入到常规的 Notepad 表格中。或者，选择 + 添加小部件 或在段落字段中输入 / 以打开 小部件插入菜单 ，您可以插入一个表格行生成器，其中预填充了一行常规行和一行生成器。

将嵌入式小部件与表格行生成器连接

要将嵌入式小部件的输入参数与其周围的表格行生成器连接起来，请将输入设置为Object from generator. Object from generator 输入项将自动可供选择，适用于接受对象模板输入的小部件。

使用表格行生成器对对象进行排序

要对从生成器返回的对象进行排序，请使用 排序 控件。从 Sort by 下拉菜单中选择一个对象属性以进行排序。请注意，可以排序的对象数量被限制为最多2000个。

模板配置

  • 生成器对象集 RID: 传递给表格行生成器的对象集。表格行生成器将为该集合中的每个对象生成一行表格。请注意，目前可以传递的对象数量被限制为最多100个对象。

嵌套表格行生成器

表格行生成器可以 嵌套在顶层部分生成器中 . 嵌套表格行生成器可以配置为在对象集模板输入上生成，或在顶层生成器对象的一级关联对象上生成。

表格行生成器对象限制

默认情况下，顶层表格行生成器的生成限制设置为最多100个对象。对于嵌套在部分生成器内的表格行生成器，限制设置为最多100个对象。请注意，从大量对象中生成行可能对模板生成性能和生成文档的文档加载性能产生负面影响。
