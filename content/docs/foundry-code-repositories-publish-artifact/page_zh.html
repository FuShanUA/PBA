发布构件

要将构件添加到构件仓库，请按照以下步骤生成令牌并查看发布说明：

  1. 在界面中导航至发布选项卡。

  2. 从下拉菜单中选择要发布的构件类型。



  3. 选择生成令牌。



  4. 在令牌字段中查看生成的令牌。令牌字段下方的文本会显示令牌的有效期。



  5. （可选）对于某些构件类型，你可以使用参数命令更新发布说明。在参数区域的字段中添加命令。



  6. 运行说明区域中列出的命令以发布你的构件。



  7. 出现提示时，提供详情区域中列出的附加信息。



既然你已经了解如何发布构件，接下来请学习如何撤回构件。

查找 Conda 构件的消费者

要确定哪些仓库使用了特定的 Conda 构件：

  1. 在构件仓库中导航至搜索选项卡。
  2. 将格式下拉菜单设置为Conda。
  3. 使用搜索栏按名称定位该构件。
  4. 从搜索结果中选择该构件以打开其版本历史记录。
  5. 找到你感兴趣的具体版本。
  6. 选择下载量列右侧的箭头。
  7. </> 消费者区域会列出所有使用该版本构件的仓库。

消费者追踪仅适用于 Conda 构件。
