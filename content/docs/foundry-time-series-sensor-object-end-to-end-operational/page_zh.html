在 Workshop 和 Quiver 中使用传感器对象类型的时间序列数据

要开始使用传感器对象类型的时间序列数据，请在 Workshop 中打开 飞行传感器数据 模块。您可以通过平台的 快速搜索 功能找到该模块。此示例模块将作为您使用传感器对象类型时间序列数据配置自己的 Workshop 模块时的参考。

本指南结束时，我们的模块将使您能够通过出发时间和机场筛选航班。之后，对象表小部件 将显示一个航班列表，您可以选择这些航班以查看关联的传感器对象时间序列数据。

第一部分：在Workshop模块中设置初始小部件

该模块的地图、对象表和过滤器小部件是标准的Workshop功能。请按照以下步骤配置这些小部件，然后再向模块中添加时间序列特定的小部件。

按路线ID过滤航班

过滤列表小部件应采用Flight对象集，并基于route id应用过滤条件。

  1. 在您的Workshop模块中，选择+ 添加小部件，然后选择过滤列表小部件。
  2. 在过滤列表小部件的右侧配置中，为对象集输入创建一个新变量，并将其命名为“按路线ID过滤航班”。起始对象集应使用Flights with linked flight sensors对象类型变量。

  1. 通过选择+ 添加筛选器来添加Route Id属性。
  2. 向下移动到筛选输出字段，找到已为您创建的筛选输出。将其命名为“按路线ID筛选的航班”之类的描述性名称；不需要其他配置。

配置对象表小部件以选择用于比较的航班

  1. 在您的Workshop模块中，选择+ 添加小部件，然后选择对象表小部件。
  2. 在右侧的配置面板中，在输入数据下拉菜单中，选择+ 新建对象集变量。
  3. 将变量命名为Flights filtered by route id。
  4. 选择起始对象集并选择现有的变量Flights with linked flight sensors。
  5. 选择筛选器使用变量，并选择Filter by route变量。

  1. 选择+ 添加列，然后从下拉菜单中选择出发时间戳。
  2. 向下滚动到选择部分，打开启用活动选择选项。这将配置表格所选对象的输出。
  3. 通过从活动对象下拉菜单中选择新对象集变量来创建一个新变量。将变量重命名为Selected flight from table 1. 不需要其他配置。

  1. 对第二个表格小部件重复此过程，该小部件输入你在步骤3-5中创建的Flights filtered by route id并创建一个名为Selected flight from table 2的输出变量。

第二部分：从根对象类型添加传感器对象类型的时序数据

创建传感器名称选择器

创建一个下拉菜单，包含你将在后续步骤中创建的Quiver仪表板所需的系列名称。

  1. 在你的Workshop模块中，选择 + 添加小部件，然后选择 String Selector 小部件（字符串选择器）。
  2. 在 选项生成 下的组件配置面板中，选择 + 添加选择器选项。
  3. 为 heading、vertical_speed、speed 和任何其他你想要显示的传感器名称添加选项。

  1. 在 选择 部分的下拉菜单中，选择 Selected series name 变量。
  2. 保持 下拉 选项作为 选择显示 配置的默认值。

创建航班ID变量

现在，您将创建两个变量，分别代表从左侧和右侧表格中选择的航班的航班ID。每个变量是一个字符串，由对象表小部件中配置的Flight Id属性支持，这些属性来自Selected flight from table 1和Selected flight from table 2变量。您将把这些变量传递给Quiver仪表板。

  1. 从Workshop模块左侧的变量导航到，然后选择+来添加一个新变量。
  2. 从下拉菜单中选择字符串。
  3. 从下一个下拉菜单中选择对象属性。

  1. 选择Selected Flight from Table变量作为单个选项的对象集，然后选择Flight Id作为Selected Flight Id变量的属性。

第三部分：创建一个Quiver仪表板

以下指导假定您对导航Quiver有一定的了解。如需了解更多关于Quiver通用功能的信息，请查阅 我们的文档。

本Workshop模块包含一个 嵌入式Quiver仪表板。按照以下说明设置仪表板中显示的时间序列图表和指标卡片。

1. 使用Flight Sensor对象类型创建一个Quiver分析

使用飞行传感器对象集

通过从顶部菜单栏选择对象并搜索Flight Sensor对象类型来创建一个 新的Quiver分析。选择添加对象集 将对象集表添加到画布。

为飞行ID添加两个字符串参数

  1. 从屏幕的左侧选择(x)以打开参数配置。
  2. 选择+以添加一个参数，然后从下拉菜单中选择字符串。该字符串参数将代表飞行ID，并且您将使用该Flight Sensor属性检索flight_id对象。
  3. 将字符串参数重命名为“飞行ID”以便于跟踪。
  4. 重复步骤1-3以添加第二个飞行ID，并将其标记为“飞行ID 2”。

为传感器名称添加一个字符串参数

  1. 导航到屏幕的左侧。
  2. 选择+以添加一个参数，然后从下拉菜单中选择字符串。该字符串参数将表示传感器名称，并且您将过滤从Workshop模块传入的与传感器名称匹配的Flight Sensor对象。
  3. 将字符串参数重命名为“传感器名称”以便于跟踪。

为所有参数添加对象集过滤器

  1. 将鼠标悬停在Flight Sensor对象集表上以显示搜索菜单，或选择搜索卡片以添加一个过滤对象集卡片。此卡片将根据所选的flight id来过滤传感器对象。有关更多信息，请参阅我们的 对象集过滤器文档。

  1. 通过选择添加过滤器以限制您的结果对象集，然后选择...其中航班ID是的选项来添加一个过滤器。

  2. 在下拉菜单中选择字符串变量，并选择您在上一步中创建的Flight Id 1字符串参数。

  4. 通过选择添加过滤器以限制您的结果对象集，然后选择...其中系列名称是的选项来添加另一个过滤器。

  5. 从下拉菜单中选择是（精确匹配），然后选择字符串变量。

  6. 选择您之前创建的Sensor name字符串参数。

添加一个按航班ID分组的时间序列图

  1. 将鼠标悬停在过滤器对象集卡片上以显示搜索菜单，或选择搜索卡片 要添加一个分组时间序列图。这将绘制对象上的所有时间序列属性并将其放在一个图表上。在这种情况下，您会期望这个分组时间序列图只包含一个传感器对象的一个时间序列。

  1. 将鼠标悬停在时间序列图配置上，找到配置图表图标并打开配置面板。
  2. 在批量时间序列选项下，找到时间序列列下拉菜单并选择默认时间序列属性。这将确保图表显示对象名称而不是系列ID列的名称。

  1. 重复这些步骤以对创建的Flight Id 2参数的过滤器卡片进行操作。然后，您应该会有两个独立的分组时间序列图表。

Overlay plots and add relative time shift

  1. Select the grouped time series plot to access the quick actions menu.
  2. Select Transform and search for a Relative time series card.

  1. Repeat for the second grouped time series plot.
  2. Then, in the second grouped time series plot, select the Configure plot option and navigate to the Display tab of the configuration panel.
  3. Under Colors, select a color to differentiate it from the previous plot. In this example, orange is selected to optimize for visual contrast with blue.

  1. Select one of the grouped time series plots and drag it onto the other plot to condense into one chart. You may notice plots seem to disappear. You will need to make the plots use the correct extents to appear together.

  1. Select the plot configuration for the Relative time series card. Under Relative Time Options, toggle on the Use source extent setting. Ensure that the Relative to dropdown menu is set to Start.

Add charts to new dashboard

  1. Choose to + Create new dashboard from the Dashboards tab to the left.
  2. Add the time series chart by selecting Add to new dashboard from the respective cards.

2. Configure the new dashboard

Navigate to the dashboard by selecting View dashboard, or access it from the Dashboard tab on the left side of your screen.

Resize and name the cards

Make sure the object selection card is at the top of the dashboard and rename the widgets to a useful name. For example, "Select an object" and "Flight sensor rolling 10 minute aggregate".

Add a string input to the dashboard

Select the Settings cog in the dashboard to open the dashboard configuration panel and add the following string inputs:

  • Input for Flight Id 1.
  • Input for Flight Id 2.
  • Input for Sensor name.

Publish dashboard

Rename your dashboard so that it is easily searchable from the Workshop module. In this example, the dashboard is named [Example] Sensor Object Time Series Data | Flight Sensor Reading Comparison.

Review our Quiver dashboard documentation for more information on how to create and customize a Quiver dashboard.
