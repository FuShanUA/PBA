在 Workshop 和 Quiver 中使用传感器对象类型的时序数据

要开始使用传感器对象类型的时序数据，请在 Workshop 中打开 飞行传感器数据 模块。你可以使用平台的快速搜索功能找到该模块。在你配置自己的使用传感器对象类型时序数据的 Workshop 模块时，此示例模块将作为参考。

在本指南结束时，我们的模块将允许你根据出发时间和机场筛选航班。然后，对象表格组件将显示航班列表，你可以选择这些航班以查看关联的传感器对象类型时序数据。

第一部分：在 Workshop 模块中设置初始组件

模块中的地图、对象表格和筛选组件是 Workshop 的标准功能。请按照以下步骤配置这些组件，然后再将特定于时序数据的组件添加到模块中。

按航线 ID 筛选航班

筛选列表组件应接收 Flight 对象集，并根据 route id 应用筛选。

  1. 在你的 Workshop 模块中，选择 + 添加组件，然后选择筛选列表组件。
  2. 在筛选列表组件右侧的配置面板中，为 对象集输入 创建一个新变量，并将其命名为“按航线 ID 筛选航班”。起始对象集 应使用 Flights with linked flight sensors 对象类型变量。

  1. 选择 + 添加筛选，添加 Route Id 属性。
  2. 向下滚动到 筛选输出 字段，找到已为你创建的筛选输出。为其指定一个描述性名称，例如“按航线 ID 筛选的航班”；无需其他配置。

配置对象表格组件以选择航班进行比较

  1. 在你的 Workshop 模块中，选择 + 添加组件，然后选择对象表格组件。
  2. 在右侧配置面板的 输入数据 下拉菜单中，选择 + 新建对象集变量。
  3. 将该变量命名为 Flights filtered by route id。
  4. 选择 起始对象集，并选择现有变量 Flights with linked flight sensors。
  5. 选择筛选条件 使用变量，并选择 Filter by route 变量。

  1. 选择 + 添加列，然后从下拉菜单中选择 出发时间戳。
  2. 滚动到 选择 部分，打开 启用活动选择 选项。这将配置表格中所选对象的输出。
  3. 在 活动对象 下拉菜单中，选择 新建对象集变量 以创建一个新变量。将该变量重命名为 Selected flight from table 1。无需其他配置。

  1. 对第二个表格组件重复此过程，接收你在第 3 至 5 步中创建的 Flights filtered by route id，并创建一个名为 Selected flight from table 2 的输出变量。

第二部分：从根对象类型添加传感器对象类型时序数据

创建传感器名称选择器

创建一个包含所需序列名称的下拉菜单，以便传递给你将在后续步骤中创建的 Quiver 仪表板。

  1. 在你的 Workshop 模块中，选择 + 添加组件，然后选择字符串选择器组件。
  2. 在组件配置面板的 选项生成 下，选择 + 添加选择器选项。
  3. 添加 heading、vertical_speed、speed 以及任何其他想要显示的传感器名称作为选项。

  1. 在 Selection 部分的下拉菜单中，选择 Selected series name 变量。
  2. 为 Selection Display 配置保留默认的 Dropdown 选项。

创建航班 ID 变量

现在，您将创建两个变量，分别代表从左侧和右侧表格中选择的航班 ID。每个变量都是一个字符串，其值由对象表格组件中配置的 Selected flight from table 1 和 Selected flight from table 2 变量的 Flight Id 属性提供。您需要将这些变量传递到 Quiver 仪表板中。

  1. 从 Workshop 模块的左侧导航至 Variables，然后选择 + 添加一个新变量。
  2. 在下拉菜单中，选择 String。
  3. 在下一个下拉菜单中，选择 Object property。

  1. 选择 Selected Flight from Table 变量作为包含单一选项的对象集，然后将 Flight Id 选作 Selected Flight Id 变量的属性。

第三部分：创建 Quiver 仪表板

以下指南假设您已具备 Quiver 的基本导航知识。要进一步了解 Quiver 的常规功能，请查阅《我们的文档》。

此 Workshop 模块包含一个《嵌入式 Quiver 仪表板》。请按照以下说明设置仪表板中显示的时间序列图表和指标卡。

1. 使用 Flight Sensor 对象类型创建 Quiver 分析

使用 Flight Sensor 对象集

通过从顶部菜单栏选择 Objects 并搜索 Flight Sensor 对象类型，创建一个《新的 Quiver 分析》并设置 Flight Sensor 对象类型。选择 Add object set 将对象集表格添加到画布中。

为航班 ID 添加两个字符串参数

  1. 在屏幕左侧选择 (x) 以打开 Parameters 配置。
  2. 选择 + 添加一个参数，然后从下拉菜单中选择 String。该字符串参数将代表航班 ID，您将使用该 flight_id 属性来检索 Flight Sensor 对象。
  3. 将该字符串参数重命名为“Flight Id”，以便于追踪。
  4. 针对第二个航班 ID 重复步骤 1-3，并将其命名为“Flight Id 2”。

为传感器名称添加一个字符串参数

  1. 导航至屏幕左侧。
  2. 选择 + 添加一个参数，然后从下拉菜单中选择 String。该字符串参数将代表传感器名称，您将使用它来筛选与从 Workshop 模块传入的传感器名称相匹配的 Flight Sensor 对象。
  3. 将该字符串参数重命名为“Sensor name”，以便于追踪。

为所有参数添加对象集筛选器

  1. 将鼠标悬停在 Flight Sensor 对象集表格上以显示 Search 菜单，或者选择 Search cards 来添加一个筛选对象集卡片。此卡片将根据选定的 flight id 筛选传感器对象。有关更多信息，请查阅《对象集筛选器文档》。

  1. 选择 Add a filter to limit your resulting object set 添加一个筛选器，然后选择 ...where flight id is 选项。

  2. 在下拉菜单中选择 String 变量，然后选择您在上一步中创建的 Flight Id 1 字符串参数。

  3. 选择添加过滤器以限制结果对象集来添加另一个过滤器，然后选择…其中序列名称为选项。

  4. 从下拉菜单中选择是（精确匹配），然后选择字符串变量。

  5. 选择您之前创建的 传感器名称 字符串参数。

添加飞行 ID 的分组时间序列图

  1. 将鼠标悬停在过滤器对象集卡片上以显示搜索菜单，或选择搜索卡片以添加分组时间序列图。这将绘制对象上的所有时间序列属性并将它们放在一个图上。在这种情况下，您会期望此分组时间序列图仅包含一个传感器对象的一个时间序列。

  1. 将鼠标悬停的时间序列图配置上以找到配置图表图标，并打开配置面板。
  2. 在批量时间序列选项下，找到时间序列列下拉菜单并选择默认时间序列属性。这将确保图表显示对象名称而不是序列 ID 列的名称。

  1. 针对为 飞行 ID 2 参数创建的过滤器卡片重复这些步骤。然后，您应该拥有两个独立的分组时间序列图表。

叠加图表并添加相对时间偏移

  1. 选择分组时间序列图以访问快捷操作菜单。
  2. 选择转换并搜索相对时间序列卡片。

  1. 对第二个分组时间序列图重复上述操作。
  2. 然后，在第二个分组时间序列图中，选择配置图表选项，并导航至配置面板的显示选项卡。
  3. 在颜色下，选择一种颜色以使其与之前的图表区分开来。在此示例中，选择了橙色以优化与蓝色的视觉对比度。

  1. 选择其中一个分组时间序列图并将其拖动到另一个图上，以合并为单个图表。您可能会注意到图表似乎消失了。您需要使图表使用正确的范围才能一起显示。

  1. 选择相对时间序列卡片的图表配置。在相对时间选项下，开启使用源范围设置。确保相对于下拉菜单设置为开始。

将图表添加到新仪表板

  1. 从左侧的仪表板选项卡中选择+ 创建新仪表板。
  2. 通过从各个卡片中选择添加到新仪表板来添加时间序列图表。

2. 配置新仪表板

通过选择查看仪表板导航至该仪表板，或者从屏幕左侧的仪表板选项卡访问它。

调整大小并命名卡片

确保对象选择卡片位于仪表板顶部，并将小组件重命名为有用的名称。例如，“选择一个对象”和“飞行传感器滚动 10 分钟聚合”。

向仪表板添加字符串输入

选择仪表板中的设置齿轮图标以打开仪表板配置面板，并添加以下字符串输入：

  • 飞行 ID 1 的输入。
  • `Flight Id 2` 的输入。 `Sensor name` 的输入。

    发布仪表板

    重命名仪表板，以便在 Workshop 模块中能够轻松搜索到它。在本例中，仪表板命名为 [示例] 传感器对象时间序列数据 | 飞行传感器读数对比。

    有关如何创建和自定义 Quiver 仪表板的更多信息，请查阅《Quiver 仪表板文档》。

    第四部分：在 Workshop 中嵌入仪表板

    1. 返回到本指南前面创建的 Workshop 模块。
    2. 选择添加 Quiver 仪表板小部件，然后选中新创建的仪表板。

    如有需要，你可以导航回仪表板并通过编辑 Workshop 模块、选择 Quiver 仪表板小部件的配置以及选择查看 Quiver 仪表板来启动分析。然后选择编辑以查看底层分析。

    有关如何在 Workshop 中自定义 Quiver 仪表板的更多信息，请查阅《我们的文档》。
