记事本中的 AIP 功能

Foundry 记事本提供了一系列 AIP 功能，旨在帮助您更快速、更高效地共享本体论洞察。您可以在记事本中使用 AIP 自动进行拼写检查、精简、修改或翻译文本，同时保留原有格式。您还可以使用函数，通过在 AIP Logic 中配置的自定义行为或其他已发布的函数来修改记事本内容。

记事本的 AIP 功能包括：

  • 自定义提示词：在输入提示词...字段中输入您自己的指令，对文本应用自定义转换。
  • 修复拼写和语法：支持英语、法语、德语、日语、韩语、西班牙语和乌克兰语。
  • 更改写作风格：在“专业”（适用于报告）或“自信”（适用于公告）风格之间进行选择。
  • 精简：使文本块更加简洁。
  • 翻译：将文本从一种支持的语言翻译为另一种语言。
  • 预览多个命令：预览同一文本块上多个 AIP 操作的结果，例如先纠正拼写，然后精简文本，最后翻译成法语，然后再替换原始文本。
  • 函数：使用接受字符串输入的平台内函数来替换文档或模板中的文本。这些函数可与其他 AIP 操作结合使用。如果未启用 AIP，您仍然可以使用函数编辑记事本内容。有关更多详细信息，请参阅自定义函数。

激活 AIP 功能

要在记事本中使用 AIP，请选择工具栏中的使用 AIP 编辑按钮。

注意：AIP 功能的可用性可能会发生变化，且不同客户之间的可用功能可能有所差异。
