下拉组件

下表提供了下拉组件可用属性的使用详情。表格后附有若干示例。

属性

属性描述类型必填修改来源
disabled指定用户是否可以与该组件交互。默认为 false。其值通常为生成布尔值的模板表达式（例如 “{{w1.on}}”）。boolean是直接编辑
fuzzySearchEnabled开启搜索的模糊匹配功能。boolean是直接编辑
displayValues在下拉菜单中显示的值。若未指定，则使用 ‘values’。any[]否直接编辑
groups对应的 ‘values’ 条目所属的分组。any[]否直接编辑
maintainValidSelection指示当组件失效时，只要当前选定的值仍然有效，是否应保持其为选定值。boolean是直接编辑
searchText用于过滤 ‘values’ 数组的搜索文本。详见 serverSearchEnabled。string是用户交互
selectedDisplayValue下拉菜单的显示值。string是用户交互
selectedValue用户当前选定的下拉菜单值。any是用户交互
serverSearchEnabled指示用户输入的搜索文本是否应触发新的查询。若否则在本地搜索 ‘values’ 数组以查找匹配项。boolean是直接编辑
values用于设置 ‘selectedValue’ 的值集合。如果使用了 displayValues，应与其建立一一映射关系。any[]是直接编辑

示例

动态下拉菜单


```json
{
  "disabled": false,
  "displayValues": "{{query1.memberName}}",
  "searchText": "John",
  "selectedValue": "65849",
  "serverSearchEnabled": true,
  "values": "{{query1.memberId}}"
}
```


静态下拉菜单



```json
{
  "disabled": false,
  "searchText": "",
  "selectedValue": "giraffe",
  "serverSearchEnabled": false,
  "values": [
    "giraffe",
    "rhinoceros",
    "elephant"
  ]
}
```

滑动条组件


下表提供了滑动条组件可用属性的使用详情。表格后附有若干示例。


属性



















































































































属性描述类型必填修改方
disabled指定用户能否与该控件交互。默认值为 false。其值通常为生成布尔值的模板表达式（例如 “{{w1.on}}”）。boolean是直接编辑
formatMode指定如何格式化滑块标签。“Grouped” 会使用指定的分隔符将数字进行分组。例如：如果分隔符为逗号，1000000 将变为 1,000,000。“Numeric” 将使用提供的 Numeral.js ↗ 格式来格式化数值。“Time” 将使用提供的 moment.js ↗ 格式来格式化数值。string是直接编辑
formatter当选择 “Numeric” 或 “Time” 时，此属性包含格式字符串。string否直接编辑
from启用范围模式时，此属性包含范围下限的值。否则，它包含与单个滑块手柄当前位置关联的值。number是用户交互
groupingDelimiter当选择 “Grouped” 格式时，此属性包含用于分隔数字组的分隔符字符串。string否直接编辑
max可选择的最大值。number是直接编辑
min可选择的最小值。number是直接编辑
numLabels要显示的轴标签数量。如果未提供值，则默认为 6。number是直接编辑
postfix要追加到所有值之后的文本。例如：” €” 会将 “100” 转换为 “100 €”string否直接编辑
prefix要添加到所有值之前的文本。例如：”$” 会将 “100” 转换为 “$100”string否直接编辑
rangeEnabled指定是启用数值范围选择还是仅启用单点选择。boolean是直接编辑
step可选择值的步长。number否直接编辑
to启用范围模式时，此属性包含范围上限的值。number否用户交互
updateOnSlide指定滑块值何时更新。“Release” 表示在手柄释放后更新值。“Slide” 表示在拖动滑块时更新值。由于对性能的影响，应避免使用 “Slide”，除非是轻量级操作（如 UI 更新）。“Slide” 更新会被限制为 100 毫秒的节流间隔。boolean是直接编辑


示例


滑块




已复制！
1{
2  "disabled": false,
3  "from": 25,
4  "groupingDelimiter": " ",
5  "groupingEnabled": true,
6  "max": "{{query1.maxValue}}",
7  "min": 1,
8  "prefix": "$",
9  "rangeEnabled": true,
10  "to": 50
11}



默认值




已复制！
1{
2  "disabled": false,
3  "from": 25,
4  "groupingEnabled": false,
5  "max": 100,
