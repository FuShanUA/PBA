对象下拉框

对象下拉框微件用于从对象列表中选择单个对象。模块构建者可以使用对象下拉框微件执行以下操作：

  • 显示一种或多种对象类型的数据。
  • 选择要显示的属性类型。
  • 设置排序条件。
  • 显示在 Ontology Manager 中配置的条件格式和数值格式选项。
  • 按对象隐藏空值属性，以实现更紧凑的显示效果。
  • 设置可搜索的属性。
  • 允许在列表中不进行任何选择。

对象下拉框微件提供以下核心配置选项：

  • 输入数据
    • 标签：为对象下拉框微件设置可选标签。此文本显示在微件顶部。
    • 输入对象集：此输入变量决定了微件中要显示的数据。模块构建者可以复用在模块其他位置创建的现有对象集变量，也可以内联定义一个新的对象集变量。下方显示的许多其他配置选项只有在填充此对象集参数后才可进行配置。
    • 选定对象：这是微件的一个输出变量，用于输出当前选定对象的单个对象集。此对象集可用于当前模块内的下游微件。
    • 允许不选择：启用后，将允许微件不选定任何对象。
  • 属性配置
    • 添加属性：选择要在对象标题下方显示的属性。
    • 隐藏空值属性：启用后，列表中将按对象隐藏空值属性。
    • 排序项依据：指定对象在下拉微件中的排序顺序。如果对象集中存在多种对象类型，则只能对共享属性进行排序。
    • 搜索项依据：指定执行搜索的对象属性。
      • 屏幕显示属性：启用后，将以下拉微件中显示的所有字符串属性为范围执行搜索。
      • 特定属性：启用后，将以指定的字符串属性为范围执行搜索。
      • 所有可搜索属性：启用后，将以对象集中的所有可搜索属性为范围执行搜索。
