REST 连接器设置

接下来的页面将探讨如何实现 Power BI® 连接器，以便从 Power Query 接口访问 Foundry 资源。如果您正在查找有关用于数据集成的 Microsoft Power BI® XMLA 连接器的信息，请参阅我们的《数据连接文档》。

您无需安装 Palantir Foundry ODBC 驱动程序，即可从 Power BI® 访问 Palantir Foundry 数据集。与 Power BI® 默认提供的内置 Palantir Foundry 连接器相比，此连接器仅支持较小的数据集规模。其仅适用于无法安装 ODBC 驱动程序的情况。Palantir Foundry REST 连接器仅支持以导入模式获取数据集，不支持直接查询模式。

步骤 1：在自定义连接器目录中安装连接器

您可以部署非原生提供的自定义连接器，将数据摄入到 Power BI® 中。在文件目录中找到 Power BI® 安装路径下的自定义连接器文件夹。该目录应作为 Power BI® 安装过程的一部分创建。下载 Palantir Foundry REST 连接器并将其移动至该目录。

下载：

  • 《Palantir REST Power BI® 连接器》 ↗

步骤 2：配置 Power BI® 以使用自定义连接器

前往选项 > 安全 > 数据扩展，更改 Power BI® 桌面版设置以允许未经验证的扩展。选择（不推荐）允许任何扩展加载而不验证或警告选项。

步骤 3：获取数据

重启 Power BI® 应用程序以使配置生效。自定义连接器将在启动时加载，此时应可使用。您可以按照《Power BI®：入门指南》中的说明，开始以 Foundry 数据为支撑构建您的第一个报告。请注意，在不使用 ODBC 连接器方法时，该连接器在 Power BI® 中将显示为“Palantir Foundry (REST)”。
