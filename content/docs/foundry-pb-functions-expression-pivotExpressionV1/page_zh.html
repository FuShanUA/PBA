透视

支持范围：流处理

在透视上下文中应用聚合表达式。该聚合将作为一组独立的聚合操作运行，每个操作的作用域限定于透视表达式的各个不同值。输出结果为一个映射，其键为透视值，值为聚合表达式的值。

表达式类别： 聚合

声明的参数

  • 聚合表达式： 要应用的聚合表达式。
    Expression<V>
  • 透视表达式： 要应用的透视表达式。
    Expression<K>

类型变量边界： K 接受 ComparableType**V 接受 AnyType

输出类型： Map<K, V>

示例

示例 1：基础用例

参数值：

  • 聚合表达式：
    sum(
     expression: value,
    )
  • 透视表达式： pivot

给定输入表：

pivotvalue
a1
b2
a3

输出结果： {
 a -> 4,
 b -> 2,
}

示例 2：基础用例

参数值：

  • 聚合表达式：
    sum(
     expression: value,
    )
  • 透视表达式：
    cleanString(
     cleanActions: {trim},
     expression: pivot,
    )

给定输入表：

pivotvalue
a1
b2
a3

输出结果： {
 a -> 4,
 b -> 2,
}
