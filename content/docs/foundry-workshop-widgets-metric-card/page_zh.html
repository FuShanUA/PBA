指标卡片

指标卡片微件以可配置的卡片式界面显示 Workshop 变量值。通常，指标卡片用于在 Workshop 模块中突出显示关键数据。配置指标卡片微件的模块构建者可以执行以下操作：

  • 将一组指标显示在一起。
  • 使用条件格式引起对指标的注意，并帮助用户解读指标值。
  • 设置指标的布局样式，使其以卡片、标签或列表形式显示。
  • 显示时间序列数据，包括展示时间序列历史记录的迷你图。

下面的截图展示了如何在 Workshop 模块中使用指标卡片微件来显示不同国家的新冠疫苗接种统计数据。接下来的示例也将沿用此上下文。

配置选项

下图展示了未配置的指标卡片微件的初始状态，随后是已配置为显示新冠疫苗接种指标的微件状态。

首先，我们将介绍该微件的配置选项，它包含一组指标，如箭头右侧第一个屏幕所示。接着，我们将描述每个单独指标可用的配置选项，如箭头右侧第二个屏幕所示。

微件配置

对于指标卡片微件，核心配置选项如下：

  • 标签

    • 为指标卡片微件设置可选标签。该文本显示在微件顶部。

  • 指标

    • 添加指标按钮向指标卡片微件添加新指标。
    • 列表中指标上的向上和向下方向箭头用于更改各个指标在微件中的显示顺序。

  • 选择布局样式

    • 卡片

      • 将微件中的指标显示为卡片，如上例和下方截图所示。请注意，时间序列可视化仅在此布局样式下受支持。



      • 卡片布局样式还允许用户选择：

        • 卡片排列的方向：水平或垂直。
        • 每张卡片中用于排列数据的模板：堆叠或并排。
    • 标签

      • 将微件中的指标显示为标签，如下方截图所示。



      • 标签布局样式允许用户选择卡片排列的方向：水平或垂直。

    • 列表

      • 将微件中的指标显示为列表，如下方截图所示。



      • 列表布局样式允许用户选择每张卡片中用于排列数据的模板：堆叠或并排。

  • 选择指标大小

    • 设置微件中每个指标的显示大小。选项包括紧凑、常规和大。

  • 条件可见性

    • 允许根据变量值或其他条件显示或隐藏指标卡片微件。这样可以根据当前应用程序状态，动态控制向用户显示的指标。
    • 通过指定计算结果为 true 或 false 的条件来配置条件可见性。当条件计算结果为 false 时，整个指标卡片微件将隐藏。

指标配置

可以使用以下选项配置每个单独的指标：

  • 标签
    • 为指标设置可选标签。该文本显示在单个指标的顶部。
  • 描述
    • 为指标设置可选的描述文本。当用户将鼠标悬停在i工具提示上时，该描述文本将作为工具提示显示，如下方截图所示。

  • 值类型

    • 指定要显示的值是字符串还是数字。
    • 此部分还配置以下选项：
      • 值：用于填充指标的值必须有相应类型的 Workshop 变量作为支撑。因此，如果值类型设置为字符串，用户必须选择一个字符串变量来填充指标。同理，如果值类型设置为数字，用户必须选择一个数字变量。有关 Workshop 变量的更多信息，请参阅《变量》。
      • 数字格式：此可选配置仅在数字值类型下可用。用户可以指定一种值格式化方案来显示变量的数值。有关值格式化的更多信息，请参阅《Workshop 中的格式化》。
      • 条件格式：此可选配置允许用户对显示的指标值应用基于规则的格式化，如下方示例所示：如果指标值小于或等于零则以红色显示，否则以绿色显示。有关更多信息，请参阅《Workshop 中的格式化》。



  • 显示次要指标？这是一个可选配置，用于在主指标下方同一指标显示区域内显示第二个指标。将此开关设置为 Yes 会打开次要指标的值类型配置，其配置方式与主指标类似。

  • 显示可视化？这是一个可选配置，用于显示描绘指标时间序列历史的迷你图。将此开关设置为 Yes 会打开包含以下选项的配置界面：

    • 位置：指定迷你图应 Side-by-side（并列）还是 Stacked（堆叠在下方）与指标值一同显示。

    • 时间序列集：要进行可视化的时间序列。使用 Time series set 变量进行指定，如下方截图所示。有关 Workshop 变量（包括时间序列集变量）的更多信息，请参阅《变量》。



    • 时间范围：指定应显示数据的时间范围。预设选项包括 All time、Last hour、Last day 和 Last week，但选择 Custom range 会打开一个详细范围选择器，您可以在其中指定 Exact 或 Relative 范围。有关指定时间范围的更多信息，请参阅《Workshop 中的时间序列属性》。

    • 基线：开启此参数后，会沿迷你图渲染一条基线，以辅助对趋势进行直观解读。有关指定基线的更多信息，请参阅《Workshop 中的时间序列属性》。

  • 交互式指标：这是一个可选配置，用于在选中卡片时触发《命令》、《操作》或《事件》。默认为 无交互。
