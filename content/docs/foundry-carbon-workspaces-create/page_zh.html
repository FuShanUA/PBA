创建工作区

每个 Carbon 工作区作为独立的文件资源存储在 Foundry 的项目（Project）中。每个工作区文件都有其专属的访问权限。

要创建新的 Carbon 工作区，需先点击 Carbon 菜单栏右侧的编辑按钮进入编辑模式。

对于没有默认 Carbon 工作区的用户，编辑按钮将不可见。但若其具备创建 Carbon 工作区所需的权限，可通过在 URL 中访问 workspace/carbon/edit 进入编辑器侧边栏。

在右侧弹出的编辑器侧边栏中，选择底部的创建工作区按钮。

接着为工作区命名，并选择资源的存储位置。

提示

建议将工作区与计划纳入其中的其他资源放在同一文件夹下。例如，将工作区置于包含所有 Workshop 模块的文件夹中，以便在此特定工作区中展示这些模块。这样，在文件夹层级设置的权限即可统一应用于所有资源。

将工作区保存至项目后，系统会跳转至新建的 Carbon 工作区。

可通过核对左上角的工作区名称是否一致来进行验证。随后即可继续配置工作区。

每个新工作区创建时都会自带一个默认主页，包含默认图标、基于 Object Explorer 的搜索栏，以及两列内容：分别展示核心对象类型（B列，中间列）和 Object Explorer 中保存的探索记录（C列，右侧列）。
