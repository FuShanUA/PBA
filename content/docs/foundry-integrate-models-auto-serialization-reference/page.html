API: palantir_models.serializers reference

The palantir_models.serializers library provides many default serialization methods for saving and loading models that are trained inside Foundry; most models should be able to use one of the default model serializers.

How to write a model serializer

In some cases it can be useful to create a reusable auto_serializer, for example, if your organization has a model format that is reused and often integrated into Foundry as models then creating a reusable auto_serializer can standardize and reduce duplicated code across different models and teams.

To create an auto_serializer, you should extend the palantir_models.models._serialization.ModelSerializer base class and implement the __init__, serialize, and deserialize methods.

  • The __init__ method is the constructor that a consumer of your serializer will interact with.
  • The serialize method will be provided with the object that you should serialize to the palantir_models.models._state_accessors.ModelStateWriter.
  • The deserialize method will be called at model load time and allows you to reinstantiate and return the serialized model from the provided palantir_models.models._state_accessors.ModelStateReader.

Your auto_serializer should be published as a shared Python library.

Implementation of provided serializers

For reference, we provide implementations of existing default serializers in palantir_models.serializers.

3    Serializer for huggingface transformers AutoModel classes, using the
4    from_pretrained and save_pretrained methods.
5    Allows configuring a specific subclass (e.g. AutoModelForSequenceClassification or

8    """
9
10    DIR_NAME = "model"
11

17        self.load_kwargs = load_kwargs
18
19    def serialize(self, writer: ModelStateWriter, obj):

22
23    def deserialize(self, reader: ModelStateReader):
24        model_dir = reader.dir(self.DIR_NAME)
25        return self.model_class.from_pretrained(model_dir, **self.load_kwargs)

11class TensorflowFormat(enum.Enum):
12    DIR = 0
13    H5 = 1
14    KERAS = 2
15
16    def get_save_path(self, dir_path):
17        if self == TensorflowFormat.DIR:
18            return dir_path
19        if self == TensorflowFormat.H5:
20            return os.path.join(dir_path, "model.h5")
21        if self == TensorflowFormat.KERAS:
22            return os.path.join(dir_path, "model.keras")

24
25class TensorflowKerasSerializer(ModelSerializer):
26    """Serializer for tensorflow keras models"""
27
28    DIR_NAME: str = "tensorflow_saved_model_dir"
29    __tensorflow: ModuleType
30
31    def __init__(self, format=TensorflowFormat.DIR, custom_objects: Optional[Dict[str, Any]] = None):
32        self.__tensorflow = importlib.import_module("tensorflow")
33        self.__custom_objects = custom_objects
34        self.__format = format
35
36    def serialize(self, writer: ModelStateWriter, obj: "tensorflow.keras.Model"):
37        dir_path = writer.mkdir(self.DIR_NAME)
38        self._save_model(dir_path, obj)
39
40    def deserialize(self, reader: ModelStateReader) -> "tensorflow.keras.Model":
41        dir_path = reader.dir(self.DIR_NAME)
42        obj = self._load_model(dir_path)
43        obj.compile()
44        return obj
45
46    def _save_model(self, dir_path: str, obj: "tensorflow.keras.Model"):
47        try:
48            save_path = self.__format.get_save_path(dir_path)
49            obj.save(save_path)
50        except ValueError as exc:
51            if "Invalid filepath extension for saving" in str(exc):
52                raise ValueError(
53                    "Serialization failed due to invalid save format. Specify the correct file format in the TensorflowKerasSerializer constructor:\n"
54                    "\tTensorflowKerasSerializer(format=TensorflowFormat.DIR) to use directory saving. This save format is considered legacy in Tensorflow\n"
55                    "\tTensorflowKerasSerializer(format=TensorflowFormat.H5) to use .h5 file format saving\n"
56                    "\tTensorflowKerasSerializer(format=TensorflowFormat.KERAS) to use .keras file format saving\n"
57                ) from exc
58            raise exc
59
60    def _load_model(self, dir_path: str) -> "tensorflow.keras.Model":
61        save_path = self.__format.get_save_path(dir_path)
62        return self.__tensorflow.keras.models.load_model(save_path, custom_objects=self.__custom_objects, compile=False)
