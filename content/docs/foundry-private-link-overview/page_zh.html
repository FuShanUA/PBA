概述

您的Foundry注册信息与Palantir Foundry之间的流量可以通过公共互联网或通过托管您的Foundry实例的云提供商的私有网络进行路由。私有链接是云提供商提供的服务，用于创建私有VPC（虚拟私有云）端点，允许您的云VPC和Palantir Foundry VPC之间直接、安全地连接，而无需经过公共互联网。

从Palantir Foundry的角度来看，此流量有两个方向：

  • 入站 从您的云VPC到Palantir Foundry。
  • 出站 从Palantir Foundry到您的云VPC。

有关如何为每个支持的云提供商设置私有链接的详细说明，请按照以下链接操作：

  • Amazon AWS PrivateLink
  • Microsoft Azure Private Link
  • Google Cloud Platform Private Service Connect
