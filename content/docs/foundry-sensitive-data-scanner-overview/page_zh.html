敏感数据扫描器

敏感数据扫描器（SDS，以前称为“Foundry 推理”）帮助组织在 Foundry 中发现并保护敏感数据。治理用户可以使用 SDS 指定敏感数据的模式以进行识别，并在找到此类敏感数据时采取适当的自动化操作。

根据这些说明，敏感数据扫描器可以在所有指定的数据集上运行一次性扫描或定期重复扫描。敏感数据扫描器为治理用户提供了强大的数据洞察力和控制力，并简化了在全球范围内遵守数据保护法规的过程。

我们建议新用户查看 核心概念 以及如何 创建敏感数据扫描 的说明。查阅 SDS 最佳实践指南 以减少扫描计算时间、提高工作流效率，并根据资源权限管理用户交互。

有关Foundry中其他可帮助您的组织遵守GDPR、CCPA、LGPD及其他隐私法规的功能，请参阅我们的页面：数据保护与治理。

敏感数据扫描器可能不会在您的注册中默认启用。要启用此功能，请按照以下说明操作：

  1. 导航到控制面板。
  2. 从左侧边栏中选择您的组织。
  3. 选择应用访问。
  4. 向下滚动，在安全与治理部分找到敏感数据扫描器，然后选择管理选项。
