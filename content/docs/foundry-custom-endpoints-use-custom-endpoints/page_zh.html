在您的应用程序中使用自定义端点

目前，自定义端点仅受 开发者控制台中的非限制应用程序 支持。受限应用程序目前不支持自定义端点。

在您将端点集创建、配置、发布并部署到子域名后，下一步是将现有端点与一个 非限制的自定义应用程序 集成，以便第三方应用程序可以进行身份验证并执行端点。在将您的端点集注册到您的应用程序之前，您应该 创建、设置并部署一个开发者控制台应用程序。

每个自定义端点都由一个本体论函数或操作支持。为了使您的开发者控制台应用程序有权执行您的自定义端点，请确定支持的本体论资源，并将它们添加到 资源访问限制 中，您的 OAuth 客户端需要权限才能访问这些资源。

一旦您的应用程序可以访问所需的本体论资源，您的自定义端点应该可以通过应用程序语言的 API 请求访问。

导航到端点设置的概览选项卡，并检查已部署的端点以查看自定义端点配置的文档。
