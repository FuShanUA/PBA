Compass 文件列表器

Compass 文件列表器是一项自动化功能，用于将给定输入文件夹中各资源的 rids（资源标识符）列出到代码仓库中。运行时，会在输出仓库中发起新的拉取请求，首次创建该文件或覆盖现有文件。生成的文件默认存储在以下路径：compass-lister/rids.json。请查阅《创建连接流》获取分步指南。

配置选项

  • 可以在配置块中设置 generated_file_path，以覆盖输出仓库中的基础路径。例如，若将其设置为 transforms-python/generated，输出结果将写入 transforms-python/generated/rids.json。

  • 若在配置块中将 merge_when_ready 设为 true，则允许生成的拉取请求自动合并。请查看输出仓库的设置，了解允许合并拉取请求的具体条件。

配置示例

{
  "generated_file_path": "transforms-python/generated",
  "merge_when_ready": true
}
