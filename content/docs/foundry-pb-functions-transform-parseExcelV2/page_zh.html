从 Excel 文件中提取行

支持在：批处理中

读取 Microsoft Excel 文件的数据集，并将每个文件解析为行。支持的文件格式：.xls, .xlt, .xltm, .xltx, .xlsx, .xlsm。

单个 Excel 文件的处理不会分布在多个 Spark 执行器上，因此如果输入数据集预计只包含一个文件，我们建议在构建设置中启用本地 Spark 的使用。

特别大的 Excel 文件可能需要大量的内存来处理，如果您观察到由于内存不足而失败的构建，请考虑使用自定义构建设置并增加执行器内存（或在使用本地 Spark 时增加驱动程序内存）。对于此类大文件，可能无法预览输出，但给定适当的构建设置，仍可以成功部署。

转换类别: 文件
