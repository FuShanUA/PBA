Spark UI

Spark 自带 Web UI ↗，作为对 Foundry 《Spark 详情》页面的补充，提供了更多信息，包括：

  • Executor 生命周期信息，例如 executor 的启动与关闭。
  • 更大规模的作业与 Executor 指标样本，包括内存使用峰值。
  • 执行期间使用的所有 Spark 配置。

查看 Spark UI

要查看 Transforms 作业的 Spark UI，需将该作业作为《调试作业》重新运行。界面上会出现 Spark UI 按钮，点击即可打开 Spark 的 Web UI。

Spark 事件会有 1 到 2 分钟的延迟，随后才会显示在 Spark UI 中。

Spark UI 在 Foundry 中的用法

Spark 的 Web UI 虽然细节丰富，但并未针对 Foundry 定制信息的展示方式。下面我们提供一些在 Spark Web UI 中浏览 Foundry 作业的建议。

SQL 执行

Spark 可以将 SQL 查询拆分为一个主查询和一个或多个子查询。在某些情况下，子查询往往比主查询更具参考价值。Foundry 中的许多数据集写入操作即是如此。

在 Spark UI 中查看《写入数据集...》的 SQL 执行情况时，你可以在子执行 ID（Sub Execution IDs）下找到该写入操作的查询图链接。

上下文预热

Spark UI 的作业标签页显示，Transforms 作业会触发一个初始的 count 作业。该 count 作业的作用是提前请求分配 executor，与此同时运行时会执行额外的设置工作（包括安装依赖项）。这能够提升在 Transform 准备就绪执行时 executor 已可用的概率。
