派生属性 [Beta]

Beta

派生属性正处于 Beta 开发阶段，可能在您的授权中不可用。在积极开发期间，功能可能会发生变化。

派生属性是基于对象上其他属性或链接的值在运行时计算出的属性。这包括对关联对象的属性进行聚合或选择。然后，派生属性可用于进一步的操作，例如在同一请求内的过滤、排序或聚合。派生属性使用所有参与计算的对象的安全性，因此它们不会暴露用户本无法看到的信息。

可用性

派生属性可在以下工作流程中使用：

  • 本体论 SDK: 可以在 TypeScript OSDK 中使用 withProperties 操作，将派生属性返回或用于其他过滤器、聚合或排序。TypeScript OSDK 必須運行 2.2.0-beta.x 或更高版本的 @osdk/client 套件。有關更多詳細信息，請查看 開發者控制台 中的 API 文檔。

已知限制

作為一個處於積極開發中的測試功能，目前派生屬性有一些能力限制。隨著更多功能的實現，這些能力將逐步可用。當前的限制如下：

  • OSv1 支持： 包含派生屬性的查詢可能不包含任何使用 OSv1 索引的物件類型。
  • 文本搜索： 派生屬性不能用於文本搜索或關鍵字過濾。
  • 在 OSDK 中的結構（Structs）： 在當前版本的 Typescript OSDK 中，包含派生屬性的查詢可能不包含任何 結構（struct） 屬性類型。您可以使用 Ontology SDK 中的 $select 操作來排除結構屬性。
