创建连接流

在逻辑流应用中，您可以：

  • 新建连接流（右上角）
  • 查看现有的连接流，按自动化任务分组

要打开该应用，请访问 /workspace/logic-flows。

要在《Compass 文件列表器》中创建连接流，请按照以下步骤操作：

  1. 创建新连接流时，系统会弹出一个对话框，让您设置名称、项目以及所需的参数和配置。这些信息会在输入时自动验证，您也可以点击验证手动校验。
    对于 Compass 文件列表器自动化任务，您需要设置输入文件夹和输出代码库。



逻辑流具有项目级作用域，这意味着您必须指定创建连接流资源所在的项目。连接流应与用于连接该流的任何参数保存在同一个项目中。

  1. 保存连接流后，它将出现在连接流列表中。



  2. 要触发该任务，请点击特定连接流所在行的构建。系统会提示一条消息，引导您前往构建应用视图，在那里可以监控任务状态。



  3. 要查看任务结果，请在目标代码库视图中检查拉取请求。



  4. 返回逻辑流应用，您可以查看您的连接流、其参数和配置，并从中创建调度计划。
