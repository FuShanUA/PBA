HyperAuto (SDDI)

HyperAuto V2 介绍

HyperAuto V2 带来了全新的用户体验和更高的性能，并支持实时流式处理，能在极短时间内提供更实用的应用就绪数据。

目前，HyperAuto V2 仅支持 SAP 数据源。

Palantir HyperAuto——也称为软件定义数据集成（SDDI）——是一套功能集合，旨在为组织中最常见且最关键的任务系统提供开箱即用的端到端数据集成能力。作为数字化转型的快速通道，HyperAuto 使您能够利用 ERP、CRM 及其他组织关键数据，自主创建有价值的工作流。

借助 HyperAuto，您可以在几分钟内同步、集成和结构化数据，并在主要数据系统之上即时构建新的工作流。HyperAuto 支持的广泛使用的系统包括：

  • SAP (V2)
  • Salesforce (V1)
  • Oracle NetSuite (V1)

HyperAuto V1

HyperAuto V1 支持多种 ERP 和 CRM 源系统，并包含全面的功能，例如通过派生要素实现的内置智能以及自动化的本体创建。在 HyperAuto V2 支持所有源类型和必要功能之前，V1 将继续可用并受到支持。

HyperAuto V1 的更新将提前通知。

有关更多信息，请参阅《HyperAuto V1 文档》。
