Model Studio

Model Studio 是 Foundry 的无代码模型开发工具。通过 Model Studio，用户可以选择训练任务、选择输入数据集，并可选择配置参数来训练和部署生产级机器学习模型，而无需编写代码。

特性

  • 简化的点击式界面 用于配置模型训练任务；无需编程。
  • 内置的生产级模型训练器 针对常见用例（如时间序列预测、回归和分类）进行了优化。
  • 智能默认值和引导式工作流程 可帮助您快速上手，即使您是机器学习的新手。
  • 深入的实验跟踪 与集成性能指标相结合，使您可以自信地监控和优化您的模型。
  • 完整数据血缘关系和安全访问控制 基于 Palantir 平台构建，确保每一步都透明且安全。

工作流

使用 Model Studio 构建模型需要以下步骤：

  1. 选择您的模型训练器： 有内置的训练器可用于常见的建模任务，例如 时间序列预测、回归 和 分类。
  2. 提供您的数据： 选择要作为训练和可选测试数据的数据集。
  3. 配置您的模型： 每个训练器都有参数。您可以选择微调这些参数或使用默认值。
  4. 启动构建： 启动转换作业，并在运行时创建的 实验 中可视化训练指标。

模型训练完成后，该模型可以用于下游应用中的推理，如 Python 转换、Pipeline Builder、函数，并通过 Marketplace 发布。

Model Studio 的训练任务还可以与 构建计划 集成，以便在新数据可用时自动重新训练模型。
