创建并发布应用

要创建新的 Slate 应用，请导航至项目视图并选择 +New > Slate application。你也可以从侧边导航栏打开 Slate 并选择 + New application。

创建集成应用

要创建集成应用，请在弹出的 Create new application 窗口中选择 Integrated application 选项。添加位置、标题和永久链接，以便通过唯一的 URL 标识该应用。

接下来，了解如何将应用结构化为页面，以实现可扩展性、可维护性和性能的优化。

创建公开应用

创建和编辑新的公开应用需要具备相应的权限，具体如权限中所述。要创建新的公开应用，请选择 Public application 选项。公开应用只能在项目中创建，无法存储在个人私有文件夹中。公开应用不支持使用模板。

公开应用必须先发布，未认证用户才能查看。

发布公开应用

要将新建的公开应用投入运行，必须通过版本对话框或 Save 旁边的下拉菜单进行发布（前提是所有先前的编辑均已保存）。与纯内部应用不同，公开应用无法自动发布，每次都需要确认要发布的版本，以防止误发布。

同一时间只能发布一个版本。未认证用户也无法访问标签，系统会将这些用户重定向到已发布的版本；如果不存在已发布的版本，则会显示错误消息。

取消发布公开应用

已发布的公开应用不再需要时，可随时取消发布。取消发布的应用可供具备所需资源权限的已认证用户访问，但未认证用户无法访问。

要取消发布，请在编辑模式下打开版本对话框，并在列表中导航至以地球图标表示的已发布版本。打开上下文菜单并选择 Unpublish，根据屏幕上的提示操作。
