地图

地图应用程序提供了强大的地理空间和时间分析与可视化功能，使您能够将来自 Foundry 的数据整合到一个连贯的地理空间体验中：

  • 探索地理空间对象之间的联系，遍历物理网络。
  • 通过边界框和多边形相交查询来搜索点和多边形数据。
  • 从各种来源（包括大规模矢量数据和卫星图像，以及路径、事件等时间数据）可视化上下文地理空间数据。
  • 通过绘制形状和执行地理空间操作进行交互。
  • 使用地图模板构建地理空间应用程序。

地图上的地理空间数据

地图应用程序使用 Web Mercator 投影 ↗ (EPSG:3857) 渲染地图，并期望 WGS 84 度数（EPSG:4326）中的纬度/经度坐标。有关在 Foundry 中转换地理空间数据的更多信息，请参见 Foundry 中的地理空间数据。
