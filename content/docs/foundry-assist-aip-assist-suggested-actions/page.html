Suggested actions in AIP Assist

AIP Assist can suggest navigational and in-application actions to better guide and support users in Foundry. This is especially useful for first-time users who learn about actions or receive application and feature suggestions following an interaction with AIP Assist.

Integration with the Palantir Developer Community forum

AIP Assist integrates with the Palantir Developer Community ↗ by directing users to create a forum post if AIP Assist does not have the necessary information. This action opens a dialog with a suggested title and description based on the conversation, which can be used in the forum post.

The suggested action to direct users to the Palantir Developer Community forum can be disabled in Control Panel by enrollment administrators on the AIP Assist page.
