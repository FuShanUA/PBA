Connectivity: SQL & BI

Foundry's open architecture makes both data and transformation logic accessible in non-proprietary formats. By storing data in open formats, exposing APIs, enabling easy connection and export of data, and providing access to transformation code, Foundry can easily interoperate with third-party software and open source tools.

Foundry's open integration ecosystem can be further enriched via installation of custom connectors, tailored to support specific integrations with popular business intelligence software, such as Power BI® and Tableau. Foundry also provides access to ODBC and JDBC drivers, which can be used for integrations with additional third-party tools.

Use this guide to set up your Foundry business intelligence tool integrations, authenticate via Foundry, and get started working with Foundry data in third-party tools.

Currently available connectors

  • Power BI® Connector
  • Tableau Connector
  • Microsoft Report Builder Connector
  • Excel
  • Qlik Sense
  • MicroStrategy
  • Foundry ODBC & JDBC Drivers
