参数

参数是可在管道的多个转换中使用的全局值。更新参数值能够一次性修改多个转换。

创建和编辑参数

在参数视图中可进行参数的创建、编辑和删除操作，该视图可通过图形顶部的参数按钮进入。创建新参数需设置参数名称和参数类型，并定义相应的值。已有的参数也可在参数视图中进行编辑或删除。

若参数被用于多个转换，编辑该参数可使用户一次性修改这些转换；修改参数逻辑会导致所有引用该参数的转换同步发生逻辑变更。

在构建管道时，可以创建不带值的参数作为占位符；当此类参数用于某个转换时，会导致该转换报错，并且在补充参数值之前，管道无法部署。

参数类型

  • 常量值：常量值参数需指定列类型和数值。在任何允许使用该设定列类型的转换中，此类参数可用作输入值。
  • 正则表达式：正则表达式参数需提供正则表达式，可用作启用了正则表达式的转换的输入。
  • 结构定位器：结构定位器参数为字符串，可用于指向结构中的某个元素。
  • 列：列参数用作列占位符，可在转换的列选择器中作为输入。列参数可通过名称指向已有的列，但在使用该参数的各个转换中，系统会首先验证该列是否存在。
  • 表达式：表达式类型参数为空白的 Pipeline Builder 表达式，可用作表达式占位符。
