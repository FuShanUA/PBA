Share in Contour Dashboard Mode

The easiest way to share results of your analysis is by creating a Dashboard. Dashboards are a separate view within the analysis where creators can present results annotated with text.

Dashboards are easy to build during the course of analysis using the dashboard preview panel. Key functionality includes chart-to-chart filtering, inline parameter references, a full-screen presentation view, and PDF exports.

Learn more about Contour Dashboard Mode.

Share in Notepad

Notepad is Foundry’s primary reporting tool and is recommended for the majority of reporting use cases. With Notepad, you can create both live-updating and static (“point-in-time”) documents, annotate documents, share documents in-platform, and export documents as PDFs.

In Notepad, you can present results from Contour alongside content from a wide range of Foundry sources, including Quiver, Code Workbook, and Object Explorer.

To add a Contour board to a Notepad, open the ... menu on the top right of the board in the analysis, and select Copy for Notepad. This copies the board to your clipboard and enables you to paste the board into Notepad. You can also copy a Contour board from a Contour dashboard by hovering over the board and selecting the Copy button.

Learn more about Notepad.
