AIP 文档智能

AIP 文档智能是 Foundry 提供的所有文档提取工作流的入口。您可以使用 AIP 文档智能打开一组企业文档，快速执行不同的最先进的文档提取策略，并检索这些策略的质量、速度和令牌成本评估结果。只需单击一下，您就可以将配置好的提取策略作为 Python 转换部署到一个媒体集的批量管道中。

功能

AIP 文档智能具备多种提取功能，并提供一个简化的界面，以便高效地进行即用型操作：

  • 直观的用户界面：轻松搜索并选择Foundry媒体集，使用各种提取策略进行处理。
  • 快速确认循环：通过将Markdown输出映射到原始PDF上的边界框，可以轻松确认执行策略的结果。
  • 传统提取：利用已有的策略从文档中提取Markdown：
    • 原始文本：通过读取文档元数据提取文本。仅适用于电子生成的PDF。
    • OCR：使用传统的OCR（光学字符识别）提取文本，不保留布局信息。
    • 感知布局的OCR：使用带有边界框的高级OCR来保留文档布局和结构。
  • 生成式AI提取：执行针对提取Markdown而微调的视觉语言模型（VLM）策略。默认提示语在我们的内部评估集中表现出最佳性能。您还可以通过配置接口使用自定义逻辑修改提示语。
  • VLM提取的预处理：对于更复杂的用例，您可以将文档预处理与传统方法（如感知布局的OCR）结合使用。然后，您可以将预处理结果传递给VLM以获得更好的性能。了解更多关于文档预处理的信息。
  • 执行指标： 观察提取质量、执行时间和输入/输出标记消耗的评分标准。
  • 评估： 使用VLM作为评判者，根据可能包含在文档中的各种因素对提取结果进行评分，包括列表、表格和代码块的质量。评估策略经过调整，以匹配我们在内部评估集中的真实结果。了解更多关于 提取评估 的信息。
  • 部署： 在确认理想的提取策略后，只需单击一下即可将该策略轻松部署到Python转换仓库中。转换将使用精确的提取策略进行配置，包括模型选择和任何提示修改。了解更多关于 提取策略部署 的信息。

开始使用

AIP 文档智能遵循一个测试工作流程，用户导入一组媒体，选择一个配置，并通过观察结果和评估来迭代他们的策略，直到满意后再进行部署。按照以下步骤开始：

  1. 上传媒体集：从应用程序首页，选择从您的可用文件中选择一个媒体集，或上传一个新的媒体集。

  2. 选择配置：媒体集打开后，打开配置选项卡以设置您想要使用的提取方法。在传统方法或生成式AI方法之间进行选择，如需可启用预处理，并自定义与VLM一起使用的提示信息。选择保存以便在未来记住您的配置选择。



  3. 对媒体集执行策略：配置好提取方法后，在配置选项卡的右上角选择运行。

  4. 预览提取结果：运行提取后，导航到提取结果选项卡以查看基于所选策略的输出。

  5. 评估提取结果（可选）：从结果选项卡中，选择评估结果，使用LLM来评估提取结果。继续测试步骤3到步骤5，直到您对提取结果和评估感到满意。



  6. 可视化分块（可选）：从提取结果选项卡中，选择一个提取结果旁边的分块按钮，以预览文本将如何被分割。您可以调整分块参数并查看结果。

  7. 部署提取策略：打开部署选项卡，从下拉菜单中选择您保存的配置，并选择创建转换仓库。您将被提示为新的Python转换仓库选择名称和位置。可选择启用分块和嵌入。仓库创建完成后，指定输出数据集并开始构建。请参阅仓库的README.md以获取详细说明。
