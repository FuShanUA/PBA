导航

本页介绍 Pipeline Builder 的界面、导航控件以及可用工具。

在 Pipeline Builder 中，用户可以通过顶部工具栏访问三种不同的视图：编辑、提案和历史记录。

编辑

编辑视图是开始构建管道工作流的地方。该视图分为四个部分：顶部工具栏、详情侧边栏、图表和预览面板。

顶部工具栏

顶部工具栏可用于管理视图、监控构建检查状态以及编辑构建设置。

  • 撤销/重做：撤销或重做工作流中的近期操作。
  • 分支下拉菜单：查看当前工作所在分支的名称，或切换到其他分支。还可以选择创建新分支。《了解有关分支的更多信息》》。
  • 保存：保存进度。当更改保存到分支后，会出现已填充的已保存状态。

对管道分支的更改不会自动保存。若要将更改正确保存到管道分支，必须在离开图表选项卡之前手动保存更改。

  • 提案：保存更改后，选择提案以请求合并到主分支。不支持从主分支向外合并。

  • 部署：管道通过验证检查后，选择部署以构建管道输出。

  • 构建设置：更改构建的计算配置文件。

    • 默认：默认的自动缩放配置文件，使用最少的执行器核心和内存。
    • 中等：缓慢扩容且快速缩容的计算资源。
    • 大型：缓慢扩容且快速缩容的计算资源。使用较大型配置文件的构建可能会更快完成，但会产生更高的计算成本。

  • 构建和检查状态：查看构建同步的状态以及检查的通过与失败情况。

  • 共享：打开详情侧边栏以访问管道的共享选项。



详情侧边栏

详情侧边栏可用于查看管道的元数据、访问要求和角色。点击右上角的图标可展开和折叠侧边栏。

  • 描述：编写或编辑管道的描述。

  • 浏览量：过去 30 天内管道被查看的总次数。

  • 关注者：关注管道工作流进度的用户总数。点击此选项可关注当前正在查看的管道。

  • 评论：对管道工作流发表的评论总数。点击此选项可发表评论。

  • 协作者：在管道上进行协作的用户。点击查看可了解用户被添加为工作流协作者的时间。

  • 访问权限：显示管道所属的组织以及用户在管道文件上的角色。点击查看可查看和管理访问设置。

    • 访问要求：显示访问管道所需的组织和标记要求。

      • 组织：用户必须属于才能访问该管道的组织。
      • 标记：用户必须具有访问权限才能访问该管道的标记。
        • 添加：打开一个面板以搜索并向管道工作流添加标记要求。

    • 角色：显示用户在管道上的角色。

      • 管理：打开一个面板以管理角色并共享管道工作流。
        • 链接共享：切换以开启链接共享，并向通过可共享链接访问管道的用户授予角色。
        • 添加用户或群组：搜索用户或群组以向其授予管道工作流的访问角色。

    • 检查访问权限：检查特定用户或群组的访问要求。

      • 检查：打开一个面板，用于搜索并检查用户或群组的访问权限。
        • 检查访问权限：使用搜索字段查找用户或群组。
          • 选择用户或群组：检查用户是否满足访问要求，并查看其在管道工作流中分配的角色。



    • 杂项：显示管道的附加元数据。

      • 位置：管道在 Foundry 文件结构中的位置。
      • 修改时间：管道上次修改的日期和时间。

    图谱

    图谱是用户添加数据、创建参数和描述转换的地方。它是管道构建器应用的主视图。

    • 平移模式：允许通过点击并拖动在图谱中进行导航。

    • 拖拽选择模式：在图谱上点击并拖动以选择多个节点。要在平移模式下快速切换至此模式，请在点击并拖动节点时按住 shift 键。

    • 全选：选中图谱中的所有节点。

    • 移除：从管道中移除图谱中选中的节点。

    • 布局：均匀分布并整理图谱中的节点。

    • 添加数据集：打开一个窗口，搜索并向图谱中添加数据集。

    • 参数：配置并添加可复用的转换参数。

      • 添加参数：为新参数命名并赋值。

    • 转换：在图谱中选择数据集节点进行转换、连接或合并。在图谱上选择数据集也会弹出一个菜单，允许您添加或编辑转换。

    • 编辑：在图谱中选择转换节点以编辑转换设置。



    • 图例：显示应用于数据集和转换的颜色指示。

      • 添加颜色：从颜色选择器中选择一种新颜色，将其分配给图谱中的节点。



    • 输出侧边栏：在侧边栏中，查看管道数据集的构建要求、编辑模式并添加输出。点击构建器图谱最右侧的“展开”图标以展开输出侧边栏。



      • 数据集：显示管道中描述的数据集输出、健康状态检查以及任何模式错误。
        • 编辑模式：识别模式错误，移除或添加列，并使用输入模式解决错误。
        • 删除数据集输出：从管道中移除选定的数据集输出。
        • 添加输出：为管道构建创建新的数据集输出。
      • 对象类型：显示指导管道集成的对象类型输出。
        • 添加输出：打开一个窗口，创建新的对象类型输出、从数据集模式生成对象类型，或添加现有对象类型。
      • 链接类型：显示指导管道集成的链接类型输出。
        • 添加输出：打开一个面板，从两个对象类型创建对象链接。

    • 缩放设置：选择放大、缩小或缩放以适应图谱。

    • 复制节点：您可以在 MacOS 上使用 Cmd + C 和 Cmd + V，或在 Windows 上使用 Ctrl + C 和 Ctrl + V，在管道构建器部署之间以及同一资源内复制和粘贴节点。此功能可用于复制部分管道或整个管道。

    预览面板

    预览面板允许您查看单个选中节点的数据样本。点击图谱左下角的图标可展开或折叠预览面板。

    • 名称：显示所选节点的名称。您可以在此字段中编辑节点名称。
    • 搜索：在数据中搜索列名。

    您也可以右键点击数据集节点并点击 打开，以访问完整的数据集预览页面。

    提议

    在提议视图中，您可以查看此流水线中所有开启、已合并或已关闭的提议。从下拉菜单中选择筛选条件，即可查看该状态类别下的所有提议。

    每条提议都会显示创建日期和时间、创建者用户名，以及表示开启、已合并或已关闭状态的图标。

    历史记录

    通过历史记录视图，您可以查看流水线工作流中任意分支的近期活动。从下拉菜单中选择一个分支即可查看其历史记录。

    点击 操作 可创建新分支、查看详情，或查看每个已保存版本的具体改动。

    创建分支

    此选项允许您基于所选分支创建一个新分支。

    查看详情

    查看详情 选项会打开一个只读版的图表，展示保存活动时流水线的状态。此选项有助于跟踪工作流随时间的演变情况，同时避免流水线回退到早期状态的风险。

    查看变更

    与 查看详情 选项类似，您可以通过 查看变更 了解特定时间点的流水线修改情况。不过在该视图中，图表会以分屏视图的形式，展示前一次保存与已应用变更之间的差异或 diff。

    查看提议

    查看提议功能允许构建者审查并编辑现有提议。

    如需了解更多信息，请参阅《查看流水线变更》。
