Connectors

The Data Connection application is intended to provide connectivity between Foundry and your systems. This page provides an overview of the available connectors.

Filesystems and blob stores

Including:

  • Amazon S3
  • OneLake and Azure Blob Filesystem (ABFS)
  • Directory (Agent-level filesystem)
  • FTP/FTPS
  • Google Cloud Storage
  • HDFS
  • Server Message Block (SMB)
  • SFTP
  • SharePoint Online
