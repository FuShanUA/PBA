foundryts.functions.statistics

foundryts.functions.statistics(start=None, end=None, window=None, **kwargs)

返回一个函数，该函数会将单个系列划分为窗口，并在每个窗口上计算统计信息。

使用 window 参数将系列划分为窗口。每个窗口包含在Dataframe schema（数据框模式）中列出的统计信息。统计信息是基于周期性（每个窗口的宽度）或固定数量的窗口计算的，其中宽度是通过四舍五入总点数除以桶的数量来计算的。用于滚动窗口的选项由 window 或 window_count 参数决定。

  • 参数：

    • start（起始）Union[int（整数）,datetime（日期时间）, str],可选) – 时间戳（包含）, 从提供的序列开始划分窗口。 （默认是整个序列）

    • end (Union[int,datetime,str],可选) – 时间戳（包含）, 从提供的序列结束划分窗口。 （默认是整个序列）

    • window (Union[int,datetime,str],可选) –

      每个窗口的 timedelta 宽度，每个窗口的大小用于将序列划分为若干个窗口。 （默认是整个序列）

    • **kwargs – 确定窗口行为和输出类型的标志。

  • 关键字参数：
    • include_std_dev (bool,False) – 如果设置为 True，输出将包含标准差。
    • window_count (int,可选) – 要计算统计信息的窗口数量（而不是每个窗口的大小）。
  • 返回值： 返回一个接受单个系列作为输入的函数，并将其划分为窗口，每个窗口提供每个窗口的统计信息。
  • 返回类型： (FunctionNode) -> SummarizerNode
