SQL

SQL 语言拥有丰富的在线外部文档。使用 SQL 编写数据转换的主要优势如下：

  • SQL 是性能最佳的语言（包含绝大部分 Spark 优化）。
  • 转换 SQL 为您提供了一个 SQL 临时查询区，允许您运行示例 SQL 查询以检查 SQL 语法。

了解有关《SQL 转换》的更多信息。

Python

Python 语言拥有丰富的在线外部文档。您选择使用 Python 编写数据转换，可能是为了充分利用 Python 特定于语言的功能与库。Python API 的抽象层级低于 SQL 等其他语言。使用 Python 的主要优势如下：

  • transforms Python 库是一个提供文件读写等功能的 API。在数据转换管道初期需要进行数据解析与清理时，基于文件的数据转换会非常有用。
  • 对 pandas、NumPy 及其他机器学习库等外部库提供一流的支持。
  • 您可以访问完整的 Spark Python (PySpark) API，其中包含了其他语言不支持的 Spark 附加功能。

了解有关《Python 转换》的更多信息。
