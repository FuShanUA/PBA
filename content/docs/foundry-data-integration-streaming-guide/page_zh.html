流式处理资源指南

本页面列出了在实施端到端流式处理工作流时可能需要参考的资源。

数据连接支持将来自各种流式处理平台的数据同步至 Foundry 流式处理数据集中，随后可在《流式处理管道》中使用这些数据。流式同步支持数据以低延迟、高吞吐量的方式流入 Foundry，从而支撑实时决策流程。

将数据从流同步至 Foundry 有两种方式：

  • 数据连接支持从流式处理平台拉取记录至 Foundry。与批量同步类似，系统会从流中读取数据，并仅通过采用《智能体架构》的单向连接将其同步至 Foundry。
  • 如有需要，Foundry 支持通过流代理将记录从流直接推送至 Foundry 流中。

Foundry 可连接众多流式处理数据源。提供专用连接器的数据源包括：

  • Apache Kafka
  • Amazon Kinesis
  • Amazon SQS
  • Aveva PI
  • Google Pub/Sub

对于没有专用连接器的流式处理数据源，可使用《外部转换》进行连接。此类数据源包括：

  • ActiveMQ
  • Amazon SNS
  • IBM MQ
  • RabbitMQ
  • MQTT [Beta]
  • Solace
