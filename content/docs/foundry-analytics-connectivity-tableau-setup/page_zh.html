设置

您可以在 Tableau 中访问 Palantir Foundry 数据集，并利用它们构建交互式仪表板。要在 Tableau Desktop 中使用 Foundry，您的计算机上必须同时安装 Foundry 数据集 JDBC 驱动程序和 Tableau 连接器文件。

请按照以下指南完成安装。

步骤 1：安装 Foundry 数据集的 JDBC 驱动程序

前往《下载：Foundry 数据集 JDBC 驱动程序》，下载 .jar 文件，并将其放置在操作系统对应的正确目录下。如果该目录尚不存在，请先行创建。

  • 如果使用 Windows：C:\Program Files\Tableau\Drivers
  • 如果使用 Mac：~/Library/Tableau/Drivers

步骤 2：安装 Tableau 连接器文件

前往《下载：Foundry 数据集 Tableau 连接器》，下载 .taco 文件，并将其放置在计算机的 My Tableau Repository\Connectors 目录下。

  • 如果使用 Windows：C:\Users\[Windows 用户名]\Documents\My Tableau Repository\Connectors
  • 如果使用 Mac：~/Documents/My Tableau Repository/Connectors

如果在上述位置找不到 My Tableau Repository 文件夹，或者在打开 Tableau 后看不到 Foundry by Palantir 连接器，说明您的连接器文件夹位于其他位置。请打开 Tableau，选择 文件 -> 存储库位置 以找到正确路径。通常情况下，您无需自行创建该文件夹。

步骤 3：开始构建交互式仪表板

安装完 JDBC 驱动程序和 Tableau 连接器文件后，您可以按照《Tableau：入门指南》中的说明，开始构建由 Foundry 数据提供支持的首个交互式仪表板。
