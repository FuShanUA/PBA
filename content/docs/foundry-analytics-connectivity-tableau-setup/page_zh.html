设置

您可以通过 Tableau 访问 Palantir Foundry 数据集，并使用它们构建交互式仪表板。要在 Tableau Desktop 中使用 Foundry，您的计算机上必须安装 Foundry 数据集的 JDBC 驱动程序和 Tableau 连接器文件。

请按照以下指南完成此安装。

步骤 1：安装 Foundry 数据集的 JDBC 驱动程序

转到 下载：Foundry 数据集 JDBC 驱动程序，下载 .jar 文件，并将其放置在适用于您操作系统的正确目录中。如果该目录尚不存在，请创建它。

  • 如果使用 Windows：C:\Program Files\Tableau\Drivers
  • 如果使用 Mac：~/Library/Tableau/Drivers

第2步：安装Tableau连接器文件

导航至 下载：Foundry 数据集 Tableau 连接器，下载 .taco 文件，并将其放置在您计算机上的 My Tableau Repository\Connectors 目录中。

  • 如果使用 Windows：C:\Users\[Windows User]\Documents\My Tableau Repository\Connectors
  • 如果使用 Mac：~/Documents/My Tableau Repository/Connectors

如果您在上述位置找不到 My Tableau Repository 文件夹，或者在打开 Tableau 后无法看到 Foundry by Palantir 连接器，您的连接器文件夹位于其他位置。打开 Tableau 并选择 File -> Repository Location 以找到正确的路径。您永远不需要自己创建该文件夹。

第3步：开始构建交互式仪表板

现在您已经安装了JDBC驱动和Tableau连接器文件，您可以按照 Tableau：入门指南 中的说明，开始使用Foundry数据构建您的第一个交互式仪表板。
