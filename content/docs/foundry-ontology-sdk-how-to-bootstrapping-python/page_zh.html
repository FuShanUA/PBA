初始化一个新的 OSDK Python 应用

本页将介绍创建使用 OSDK 的 Python 应用的整个过程。以下示例适用于多种 Python 框架，例如 Flask ↗、Streamlit ↗ 和 Jupyter ↗。

1: 前提条件

创建 Developer Console 应用

请按照《创建新的 Developer Console 应用》页面中列出的步骤操作。

设置令牌

在本地环境中导出您的令牌。以下示例使用的是示例个人访问令牌，您也可以在 Developer Console 中生成有效期更长的令牌。请勿将此令牌提交到源代码管理中，因为它是您的个人访问令牌。

Copied!
1export FOUNDRY_TOKEN=<YOUR-TOKEN-FROM-GETTING-STARTED-PAGE>

检查 Python 版本

Python SDK 要求 Python 版本 >=3.9 且 <3.13。要检查您正在使用的 Python 版本，请输入以下命令：

Copied!
1python3 --version

可选：设置证书

如果您的组织要求网络流量使用证书，您可能需要告知 Python 该证书的存放位置。



已复制！
1export SSL_CERT_FILE="/path/to/my.crt"
2export REQUESTS_CA_BUNDLE="/path/to/my.crt"

2: 安装最新版本的 SDK

运行以下命令安装最新版本的 SDK，将 < > 替换为应用程序专属的值，该值可在应用程序的概览页面找到。

已复制！
1pip install <YOUR-PACKAGE-NAME> --upgrade --extra-index-url "https://:$FOUNDRY_TOKEN@<INDEX-URL>"

开发前端应用程序

在应用程序中初始化 Foundry 客户端，即可开始开发。


```python
import os
from  import FoundryClient
from .core.api import UserTokenAuth

auth = UserTokenAuth(hostname="", token=os.environ["FOUNDRY_TOKEN"])

client = FoundryClient(auth=auth, hostname="")

object = client.ontology.objects.
print(object.take(1))
```
