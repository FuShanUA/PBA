批准变更

在工作流项目中拥有 Edit 访问权限的 Pipeline Builder 用户，可以批准对 Main 管道分支的变更提议。要批准提议，首先打开 Builder 的 Proposal 视图，并从 open 提议列表中选中它。在这里，你可以查看详情，包括合并行为（例如，从 test/branch 合并到 Main）、变更描述以及提议中发现的任何错误。

查看变更

切换到 Changes 标签页，将提议的变更与 Main 分支工作流进行对比。在左侧面板中，你可以按变更类型进行筛选。在下面的示例中，提议的变更包括对 Transforms 的四处编辑和对 Outputs 的三处编辑。

每项提议的编辑都会按提议类型打上标签：ADD、MOD 或 DEL。

  • ADD：该编辑是向工作流中新增的内容。
  • MOD：该编辑是对现有工作流节点的修改。
  • DEL：该编辑从工作流中移除节点。

点击单个编辑项，可并排查看 Main 分支与提议分支之间的差异。在下面的示例中，提议的 Join person data 修改向现有的连接中添加了 CITY 列。

查看和比较统计信息

你可以查看并比较分支上任何转换的列统计信息。双击进入转换，并在右上角或底部面板的标签页中选择 Statistics。

选择 View column stats 图标，查看并比较所选列的统计信息。

在这里，你可以查看各种指标，例如：

  • 空值数量。
  • 唯一值数量。
  • 对于字符串：最小和最大长度，以及有多少字符串需要去除首尾空格。
  • 对于数值：最小值、最大值、中位数、标准差、负数和正数值。

分支与主分支之间统计信息的差异将以颜色编码显示在指标右侧。

切换回 Overview 标签页以继续审查提议。

跟踪并标记变更为已审查

在 Changes 标签页中，你可以将每个节点或设置的变更标记为“已审查”。Pipeline Builder 会跟踪并显示你已审查的变更数量以及变更总数，让你一目了然地掌握审查进度。

从侧面板或直接在图表中选择一个节点。然后，你可以直接在图表上节点下方选择 Mark as reviewed，或者选中节点或设置并勾选 Mark as reviewed。

在界面右上角，一个进度条显示了你在提议的所有变更中已审查的数量。

请注意，如果对之前已标记为“已审查”的内容进行了变更，其“已审查”标记将被自动取消。

将变更标记为“已审查”是特定于用户的。当你将变更标记为已审查时，它仅适用于你的用户账户。其他用户可以独立地将相同的变更标记为已审查，而不会影响你的审查状态。

评论

你可以在提案的主概览选项卡中添加和查看评论。如需评论，请滚动至概览页面底部直接输入评论，或者在变更选项卡中对特定节点进行评论。默认情况下，主概览页面会隐藏已解决的评论，但你可以根据需要切换查看。

要对特定节点进行评论，请导航至右上角的变更选项卡。在左侧面板中，打开评论部分以查看所有评论。若要为节点添加评论，请选中该节点并选择评论图标，或者直接点击节点本身。

查看特定节点时，选择右上角的评论图标即可添加评论。你可以查看该节点上的评论总数以及已解决的数量。左侧面板将继续显示流水线的所有评论列表。

所有评论均支持 Markdown 格式，包括粗体、斜体、链接和代码块。提交评论后，你可以对其进行编辑、复制、解决或删除。你还可以在评论中使用 @ 键来提及特定用户。

解决评论

你可以在评论侧边栏或节点内的弹窗中解决评论。评论被解决后，默认会处于隐藏状态。你可以在评论面板中展开未解决部分，或者在评论弹窗中启用显示已解决选项来查看它们。

要在节点内解决评论，请导航至该评论，选择三个点图标，然后选择解决。

你也可以在评论侧边栏中解决评论。选择评论旁边的三个点图标，然后选择解决。

要解决与某个节点关联的所有评论，请使用侧边栏查看该节点所有未解决的评论。选择三个点图标，然后选择全部解决。

你也可以在已解决评论区下方选择取消解决或全部取消解决，将已解决的评论恢复为未解决状态。

解决错误

在某些情况下，提案会显示在合并前必须解决的 Schema 或编辑错误。

要解决阻碍成功构建和合并的 Schema 错误，请选择修复 Schema。

这将带你返回到存在 Schema 错误的图表中。

双击图表中存在错误的步骤，并解决出现问题的条件。

点击提议以自动保存并提交已解决的分支。

解决合并冲突

在某些情况下，提案会显示在合并前必须解决的合并冲突。合并冲突是指自你创建分支以来，对基础分支所做的更改。必须解决这些合并冲突，以确保将正确的更改合并到流水线的基础分支中。要解决这些冲突，你需要对分支进行变基。在流水线构建器中执行变基操作，可以将本地分支中未发布的更改与远程分支上最新发布的更改进行合并。

要通过变基开始解决冲突，请在提案中选择解决。

这会将你带到变基分支视图的图形界面中。在这里，你可以从顶部栏或左侧边栏查看并解决冲突列表，还能深入排查单个冲突与变更。

在变基分支中，你可以直接在图上查看冲突和变更。将鼠标悬停在受影响节点右上角的图标上，即可获取更多细节，包括详细说明和可能的解决方案。在下方示例中，双击该图标即可解决合并冲突。

解决所有冲突后，你可以点击图形界面右上角的提交按钮，将更改提交至你的分支和提案中。
