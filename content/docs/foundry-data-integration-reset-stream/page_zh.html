重置流

重置接收流会清除流中的现有记录，并提供了更改流模式、吞吐量和配置值的机会。这在流管道开发期间非常有用，因为您无需替换 Pipeline Builder 管道内现有的流引用，即可清除并更新接收流。

请注意，重置操作仅适用于接收流。重置后，必须重放该接收流的下游消费管道。

  • 《基于推送的接收》需要更新 POST URL 以引用新的流 viewRid 值。
  • 《基于 Magritte 的接收》需要重新构建智能体。

重置流可能会对数据造成不可逆的影响。我们不建议重置生产环境的接收流，因为现有的记录将会丢失。

要重置流，请按照以下说明操作：

  1. 打开一个接收流。
  2. 选择 Details 选项卡。
  3. 在 Stream Settings 部分，选择 Reset stream。
  1. 您将被重定向到流重置页面。在此页面上，您可以选择更新模式、吞吐量或配置值。如果只想清除流中的数据，请保持这些部分不变。
  1. 选择 Reset stream 以启动流重置。
