创建接口

请按照以下步骤在 本体论管理器 中创建一个新的接口。

  1. 首先，通过查看左侧面板顶部的 本体论 下拉菜单，确认您正在使用您选择的本体论。

  2. 要创建新的接口，您可以执行以下任一操作：

    • 在页面右上角，选择 新建 > 接口。
    • 从左侧面板中，在 资源 部分下选择 接口 > + 新建接口。然后，从 接口 页面中，选择屏幕右上角的 新建接口。

  3. 助手的第一页提供有关接口的信息。选择 下一步。

  4. 输入接口的显示名称和API名称。您还可以可选地提供接口的描述，并选择适当的图标。

  5.     对于 必需的 属性，任何实现该接口的对象类型都必须提供从本地属性到接口属性的映射。对于 可选的 属性，在实现过程中可以跳过映射。可选属性在构建 Marketplace 包时非常有用，这样可以在不引入难以解决的升级阻塞的情况下迭代您的接口。

  6. 选择一个项目以将此接口保存到该位置，然后选择 创建。

  7. 回到本体论管理器中，选择右上角的 保存 以 将更改应用到您的本体论。

创建接口链接类型（可选）

  1. 在左侧面板中选择 链接类型约束。
  2. 然后在右上角选择 创建新的链接类型约束。

如果您的建模用例需要接口链接类型，则任何实现该接口的对象类型都必须添加一个新的或现有的链接类型，以满足接口链接类型约束。

创建接口操作类型约束（可选）

如果您希望接口在实现对象类型中定义预期的操作能力，您可以可选地向接口添加 接口操作类型约束。

  1. 在左侧面板中选择 操作类型约束。
  2. 选择 创建新的。
  3.
