Notepad

Notepad 是一款具备对象感知能力的协同富文本编辑器。除了能够添加文本和图像等内容并进行格式化排版外，你还可以集成图表等小组件，或者嵌入来自其他 Foundry 应用（如 Contour、Quiver、Code Workbook 和 Object Explorer）的对象。此外，你还可以创建模板，将其作为生成新 Notepad 文档的蓝图。

功能特性

Notepad 的核心功能允许你：

  • 从其他 Foundry 应用中嵌入图表、表格和图形。
  • 导出和打印所有 Notepad 文档，并支持对导出展示效果进行细粒度控制（如嵌入外观、分页等）。
  • 维护结构化链接，将其指向嵌入的对象和资源，并自动丰富底层本体。
  • 冻结内容，以捕获特定时间点的上下文。
  • 将文档模板化，基于新输入轻松创建报告版本。

使用场景

Notepad 支持以下工作流：

  • 临时笔记记录（例如，在调查工作流中结合嵌入内容记录笔记）。
  • 平台内文档编写（例如，为数据管道、数据集或应用编写文档）。
  • 周期性报告（例如，使用模板生成月度状态报告）。
  • 模板化导出（例如，根据选定的对象按需创建文档模板的新副本）。

Notepad 不太适用于以下场景：

  • 创建仪表板：Notepad 不支持基于用户输入或图表选择进行动态筛选。如果你想创建仪表板来展示分析结果，对于对象或时间序列数据，建议考虑使用《Quiver Dashboards》；对于表格数据，建议考虑使用《Contour Dashboards》。此外，也可以使用《Workshop》来创建仪表板或更高级的运营应用。
  • 复杂的、基于页面的富文本编辑：Notepad 不具备传统桌面文本编辑器那样的可配置性。Notepad 旨在成为一款易于上手的文本编辑器，其核心优势在于与 Foundry 平台其余部分的无缝集成。
