入门

以下教程将引导您使用 Insight 分析本体论（Ontology）数据，从查找和打开资源到保存和共享结果。

创建新分析

要创建新的 Insight 分析，请从主页右上角选择 + 新分析。

您也可以从其他应用程序中创建新的 Insight 分析：

  • 从 对象浏览器 中，选择任意对象类型或对象集，以查看在 Insight 中分析数据的选项。
  • 从其他应用程序中，可以从 操作（Actions） 菜单中选择 在 Insight 中打开 选项。

查找数据

当你创建一个新的洞察分析时，你必须首先选择本体论（Ontology）数据。Insight与Foundry本体论（Ontology）一起工作，使你可以搜索对象类型、对象集合和接口（interfaces）。使用选择起始数据窗口根据名称和描述查找数据。

如需更高级的搜索，请点击搜索栏右侧的筛选图标。可用的筛选包括以下内容：

  • 状态（Statuses）
  • 空间（Spaces）
  • 对象类型组（Object type groups）
  • 接口（Interfaces）
  • 共享属性（Shared properties）

筛选功能允许进行更复杂的搜索，通过组合多个筛选条件来缩小搜索范围，以找到特定的对象类型。

例如，您可以选择 Endorsed 状态筛选器，并仅选择某些组进行搜索，从而显著缩小结果的范围。

预览并选择数据

通过在搜索结果中选择一个卡片，可以预览对象类型以获取更详细的信息。您可以选择多个对象类型以打开多个预览。预览提供属性、组和关联对象类型等信息。也可以单独预览对象。

选择一个或多个对象类型并将其添加到分析中。选择 开始分析 在右下角，或选择任意对象类型卡片上的打开以开始创建您的洞察分析。
