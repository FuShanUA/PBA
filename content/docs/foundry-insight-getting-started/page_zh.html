Getting started

The following tutorial walks you through using Insight to analyze Ontology data, from finding and opening resources to saving and sharing results.

Create a new analysis

To create a new Insight analysis, select + New analysis from the top right of the home page.

You can also create a new Insight analysis from other applications:

  • From Object Explorer, select any object type or object set to view an option to analyze the data in Insight.
  • From other applications, an Open in Insight option is available from the Actions menu.

Find data

当您创建一个新的洞察分析时，必须首先选择本体论数据。Insight 与 Foundry 本体论一起使用，使您可以搜索对象类型、对象集合和 接口。使用 选择起始数据 窗口，可以根据名称和描述查找数据。

如需更高级的搜索，请点击搜索栏右侧的筛选图标。可用的筛选包括以下内容：

  • 状态
  • 空间
  • 对象类型组
  • 接口
  • 共享属性

筛选功能允许更复杂的搜索，通过组合不同的筛选条件来缩小搜索范围，以找到特定的对象类型。

例如，您可以选择 认可 状态筛选器，并仅选择某些组进行搜索，从而显著缩小结果范围。

预览并选择数据

通过在搜索结果中选择一个卡片，可以预览对象类型以获取更详细的信息。您可以选择多个对象类型以打开多个预览。预览会提供诸如属性、组和关联的对象类型等信息。也可以预览单个对象。

选择一个或多个对象类型，并将它们添加到分析中。在右下角选择开始分析，或者选择任何对象类型卡片上的打开以开始创建您的Insight分析。
