控制台

在工作簿右侧，控制台提供了一个 REPL（读取-求值-输出循环），支持对图中的任何转换进行快速、临时分析。分支上启用的每种语言都有对应的控制台。

使用控制台时：

  • 按 Enter 键执行当前输入的代码。
  • 按 Shift+Enter 键换行，编写多行语句。
  • 按上、下方向键浏览之前执行过的命令。

在控制台下方，可通过变量窗格为输入到控制台的给定转换设置输入类型。

若控制台命令返回数据框（dataframe），可点击+ 添加至图按钮，将控制台命令转化为一个转换。这样一来，可以先在控制台中调试逻辑，再将其提升为可重复执行的转换。

在 Windows 系统上按 Ctrl+Shift+Enter，或在 macOS 系统上按 Cmd+Shift+Enter，即可使用键盘快捷键将代码从转换直接发送至控制台。
