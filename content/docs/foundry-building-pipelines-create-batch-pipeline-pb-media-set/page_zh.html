使用流水线构建器创建媒体集批处理流水线

在本教程中，您将使用流水线构建器创建一个带有媒体集的标准批处理流水线，以从 PDF 中提取文本。

本示例使用 Palantir 发布的公开文档 PDF 文件。

本教程结束时，您将得到一个如下所示的流水线：

该流水线将生成一个包含提取出的 PDF 文本的全新对象输出，可供进一步探索使用。

第 1 部分：初始设置

首先，创建一条新流水线。

  1. 登录 Foundry 后，从左侧导航栏访问流水线构建器。如果应用列表中没有流水线构建器，请选择查看全部，然后在构建和监控流水线部分下找到流水线构建器。

  2. 接下来，在流水线构建器主页的右上角，选择新建流水线创建一条新流水线，然后选择批处理流水线。在选择批处理计算资源下，选择标准或更快。本示例选择标准。

    并非所有的 Foundry 环境都支持创建流式流水线。如果您的业务场景需要此功能，请联系 Palantir 支持团队获取更多信息。

  3. 选择保存流水线的位置。请注意，流水线无法保存在个人文件夹中。

  4. 选择创建流水线。

第 2 部分：添加媒体集

现在，您可以向流水线工作流中添加数据集了。在本教程中，您将使用 Palantir 公开文档的 PDF 文件。

  1. 在流水线构建器页面，于主页上选择添加 Foundry 数据。



    您也可以在顶部面板选择添加数据操作。



    或者，您也可以从计算机中拖放文件，将其作为媒体集使用。

  2. 如果您选择了添加数据或添加 Foundry 数据，系统将提供选项供您选择所需的媒体集。



  3. 选中所有媒体集后，选择添加数据。

  4. 导入媒体集后，您将能够看到该媒体集及其缩略图预览。



第 3 部分：媒体集转换

添加原始媒体集后，您可以执行一些基础转换操作。在此工作流中，您将从这些 PDF 文件中提取文本。

从 PDF 中提取文本

您可以使用媒体引用直接转换媒体或从中提取信息。在本示例中，您将从年度信件媒体集媒体集中提取文本。

  1. 在图表中选择年度信件媒体集节点。

  2. 选择转换。



  3. 从下拉菜单中搜索并选择从 PDF 提取文本转换操作以打开面板。



  4. 根据需要选择提取方法，并填写其余参数。

  • 原始文本：计算机生成的 PDF
  • OCR：复印件
  • 布局感知: 文本与边界框



  1. 选择 应用 将转换添加至管道。

  2. 将鼠标悬停在提取出的文本上时，输出结果应如下所示：



    现在，您可以针对提取出的文本列执行可用的字符串转换操作。

  3. 点击右上角的 关闭 返回管道图。



(可选) 语义搜索工作流

如有需要，您可以利用提取出的文本，继续进行语义搜索工作流。

第 4 部分：添加输出

既然您已完成从 PDF 中提取文本并可能执行了额外的字符串转换操作，接下来可以添加输出了。在本教程中，您将添加一个对象输出。

  1. 在已完成各项转换的 Transforms 节点中，选择 添加输出。



  2. 选择 新建对象类型。



  3. 为对象类型命名，并通过选择 请选择本体 来设置本体。



  4. 选择 编辑 并编辑任意列映射。确保为主键选择一个有效的列。



第 5 部分：构建管道

  1. 要构建管道，请务必选择 保存，然后选择 部署 > 部署管道。



  2. 在 部署管道 侧边栏选项下，您应看到 正在初始化部署。



  3. 选择 查看部署历史 以追踪部署进度。您将跳转至管道的 历史 标签页，在此查看部署的状态与历史记录：





(可选) 第 6 部分：本体之上

部署完成且对象初始化后，您应能直接对对象输出执行操作。选择 创建 Workshop 模块，基于管道输出生成一个 Workshop 模块。

完成这最后一步，您就已生成了管道输出以及一个 Workshop 模块。

常见问题

能否在提取数据前先对媒体进行转换？

可以，您可以在提取数据前先对媒体进行转换，这适用于多种提取操作，包括文本提取以及使用大语言模型 (LLM)。举个例子，假设媒体集中包含横向图像，需要先顺时针旋转 90 度才能执行基于 OCR 的文本提取。您可以添加一个表达式来旋转图像，随后再添加一个表达式利用 OCR 提取文本。

为了将管道结果安全写入数据集输出，您必须从输出模式中移除媒体引用列，因为转换后的媒体引用不是有效的列输出。

为此，请使用输出对话框移除该列。
