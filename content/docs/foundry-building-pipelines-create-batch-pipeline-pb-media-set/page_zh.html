使用 Pipeline Builder 创建一个媒体集批量管道

在本教程中，您将使用 Pipeline Builder 创建一个标准的批量管道，通过媒体集从 PDF 中提取文本。

此示例使用了 Palantir 发布的公开文档的 PDF 文件。

本教程结束后，您将拥有一个如下所示的管道：

该管道将生成一个包含提取出的 PDF 文本的新对象输出，可用于进一步探索。

第一部分：初始设置

首先，创建一个新的管道。

  1. 登录到 Foundry 后，访问 Pipeline Builder 从左侧导航栏中。如果 Pipeline Builder 不在应用程序列表中，请选择查看全部，然后在构建和监控管道部分下找到Pipeline Builder。

  2. 接下来，在 Pipeline Builder 首页的右上角，通过选择新建管道来创建一个新管道，然后选择批量管道。在选择批量计算下，选择标准或更快。对于此示例，选择标准。

    在所有 Foundry 环境中都无法创建流式管道。如果您的使用场景需要，请联系 Palantir 支持获取更多信息。

  3. 选择保存管道的位置。请注意，管道不能保存在个人文件夹中。

  4. 选择 创建管道。

第2部分：添加媒体集

现在，您可以将数据集添加到您的管道工作流中。对于本教程，您将使用Palantir提供的公开文档的PDF文件。

  1. 从管道构建器页面，选择主页上的 添加Foundry数据。



    您也可以在顶部面板上选择 添加数据 操作。



    或者，您可以从计算机中拖放文件以用作您的媒体集。

  2. 如果您选择了 添加数据 或 添加Foundry数据 ，您将可以选择所需的媒体集。



  3. 当所有媒体集都已选择后，选择添加数据。

  4. 当您导入了媒体集后，您将能够看到带有缩略图预览的媒体集。



第三部分：媒体集转换

在添加原始媒体集后，您可以执行一些基本的转换。对于这个工作流，您将从这些PDF文件中提取文本。

从PDF提取文本

您可以直接对媒体进行转换或使用 媒体引用 从媒体中提取信息。在此示例中，您将从 Media set of Annual Letters 媒体集提取文本。

  1. 在您的图中选择 Media set of Annual Letters 节点。

  2. 选择 转换。



  3. 从下拉列表中搜索并选择 从PDF提取文本 转换以打开面板。



  4. 根据您的需求选择提取方法并填写其余参数。

  • Raw text: 计算机生成的PDF
  • OCR: 复印件
  • Layout aware: 文本和边界框



  1. 选择 应用 以将转换添加到您的管道中。

  2. 当您悬停在提取的文本上时，您的输出应该看起来像这样：



    现在，您可以在提取的文本列上运行可用的字符串转换。

  3. 在右上角选择关闭返回到您的管道图。



(可选) 语义搜索工作流

如果需要，您可以使用提取的文本继续进行语义搜索工作流。

第4部分：添加一个输出

现在您已经完成了从PDF中提取文本，并且可能运行了额外的字符串转换，您可以添加一个输出。对于本教程，您将添加一个对象输出。

  1. 在已完成转换的 Transforms 节点中，选择 添加输出。



  2. 选择 新对象类型。



  3. 为你的对象类型命名，并通过选择 请选择一个本体论 来设置本体论。



  4. 选择 编辑 并编辑任何列映射。确保你为主键选择了一个有效的列。



第5部分：构建管道

  1. 要构建你的管道，请确保选择 保存，然后 部署 > 部署管道。



  2. 您应该在 Deploy Pipeline 侧边栏选项下看到 Intializing deployment。



  3. 选择 查看部署历史 以跟踪您的部署进度。您将被引导至管道中的 History 标签页，您可以查看部署的状态和历史记录：





(可选) 第6部分：本体论的北方

一旦部署完成并且您的对象已初始化，您应该能够直接对您的对象输出进行操作。选择 创建Workshop模块 来使用您的管道输出生成一个Workshop模块。

通过这最后一步，您已经生成了管道输出和一个 Workshop 模块。

常见问题

在从媒体中提取数据之前可以转换媒体吗？

是的，您可以在从媒体中提取数据之前对其进行转换，并且这与许多提取操作兼容，包括提取文本和使用大型语言模型（LLMs）。例如，考虑一个包含横向图像的媒体集，需要在进行基于OCR的文本提取之前将其顺时针旋转90度。您可以添加一个表达式来旋转图像，然后添加另一个表达式来使用OCR提取文本。

为了安全地将管道结果写入数据集输出，您必须从输出模式中删除媒体引用列，因为转换后的媒体引用不是有效的列输出。

为此，请使用输出对话框删除该列。
