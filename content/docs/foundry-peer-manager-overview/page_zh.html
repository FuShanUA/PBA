对等管理器

对等管理器应用程序使您可以查看和监控与已建立的对等连接相关的作业，这些连接能够将 对象 在 Foundry 注册之间导入或导出。

功能

  • 创建对等连接： 创建或导入 对等连接邀请代码，从您的注册到另一个注册创建对等连接。
  • 查看对等连接： 选择一个连接以启动应用程序的 概览 窗口，在这里，对等管理器会显示与您的 空间 相关的对等连接的数据类型、来源和方向性。
  • 配置本体论对等： 通过 本体论对等 在对等连接之间同步本体论数据。
  • 配置文件对等： 通过 文件对等 在对等连接之间同步 Gotham 文件。
  • 大规模管理数据共享：使用 对等配置文件 来定义可重复使用的数据共享配置，并一次性在多个对等连接中一致地应用这些配置。
  • 监控对等作业：通过查看单个对等作业的状态来跟踪您的对等连接的健康状况。
