保存为数据集

在 Contour 路径底部点击 Save as Dataset，即可将分析结果作为数据集保存至 Foundry 中。

如果该路径已关联数据集，该数据集将显示在 Contour 路径底部。点击 Update 可使用新逻辑或新数据更新该数据集。需要注意的是，如果在 Contour 分析中修改了逻辑，通过数据集预览或数据血缘关系工具构建的结果数据集并不会同步更新所用的构建逻辑。必须使用 Update 按钮才能使逻辑变更生效。若通过 Dataset 应用或数据血缘关系进行构建，数据更新将自动同步。

在 Contour 中构建的所有数据集将始终使用输入数据集的最新版本。这意味着，即使分析使用的是旧版本的数据（例如通过版本选择器指定的版本），基于该分析构建的数据集仍会使用最新版本。

受限视图

无法将包含受限视图的路径作为输入保存为数据集。

在项目范围内的分析中，所有输入和输出必须处于该范围内，才能将路径的结果保存为数据集。了解有关项目引用的更多信息。
