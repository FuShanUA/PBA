对象浏览器

对象浏览器 是一个用于回答有关本体论中任何事物问题的搜索和分析工具。

  • 用户可以通过运行从简单的关键词搜索到全面的属性过滤器的各种查询，轻松找到感兴趣的对象，所有操作都可以通过直观的点击界面完成。然后，他们可以使用探索视图来探索结果对象集，或者以结果表格的形式查看，或者选择特定对象以查看其 对象视图。
  • 此外，用户可以比较和对比对象集，在对象集上执行批量操作（如写回），将对象集在其他兼容应用程序（如 Quiver）中打开，或将其导出。用户还可以保存他们的探索，并在任何时候重新访问以查看最新的结果。
  • 探索视图是一组预设和可配置的可视化内容（如图表或地图），用户可以进一步利用这些内容深入查看对象的具体子集。
  • 对象浏览器需要最少的配置，专为非技术人员设计。
