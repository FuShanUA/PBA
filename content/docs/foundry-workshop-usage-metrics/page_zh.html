使用指标

Workshop 的使用指标为模块构建者提供对其应用程序使用情况的可见性。在 Workshop 编辑器左侧边栏的指标选项卡中，您可以查看操作提交次数和布局视图次数，以了解您的模块中哪些部分最活跃以及使用趋势随时间的变化。所有指标都是汇总计数，不能归因于任何特定用户。

操作指标

操作指标显示模块中每个操作成功提交的次数。概览卡片会显示所选时间段内模块中操作提交的总数，以及与前一相同时间段相比的百分比变化。

概览下方是各个操作及其提交次数，以及表示相对使用情况的比例条形图。选择一个操作，可以查看模块中哪些小部件使用了该操作。

所有模块默认都提供操作指标，无需任何额外配置。

布局视图指标

布局视图指标记录模块中每个页面、标签页和覆盖层被用户查看的次数。概览卡片显示所有布局的总浏览量，列表视图则按每个布局项分解浏览量。

在列表中选择一个页面或覆盖层会导航到编辑器中的相应布局。

启用布局视图跟踪

布局视图指标需要构建者进行选择。要启用跟踪，请执行以下步骤：

  1. 以编辑模式打开模块。
  2. 打开模块设置。
  3. 导航到指标选项卡。
  4. 打开使用指标跟踪。

启用后，可能需要最多24小时，视图指标才会出现在指标面板中。布局视图数据是按日聚合处理的，因此新视图每天反映一次，而不是实时反映。

只有在主分支的查看模式下查看模块时，才会记录布局视图。在编辑模式或草稿分支中的视图不会被跟踪。

时间范围

操作指标和布局视图指标都支持可配置的时间范围。使用指标面板顶部的时间选择器来选择7天、30天或90天。默认值为30天。

每个概览卡片显示与前一相同时间段的百分比变化。例如，当查看最近30天时，百分比变化将与之前的30天进行比较。

Embedded modules

For modules that use embedded modules, overlay views originating from an embedded module are attributed to the embedded module rather than to the parent module.
