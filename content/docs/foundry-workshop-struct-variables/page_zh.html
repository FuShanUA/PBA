结构变量

一个结构变量是一个复合变量，包含其他Workshop支持的变量类型的字段。嵌套的结构目前不被支持。

创建结构变量

可以在Workshop中静态初始化结构变量，使用对象的结构属性，或使用返回CustomType的函数。请参阅 自定义类型文档 了解更多信息。如果某个字段的类型在Workshop中不受支持，它将被忽略，并从初始化的变量中省略。

创建结构数组变量

同样，包含多个结构体的结构体数组变量可以使用对象的结构体数组属性或函数进行初始化。

遍历结构体数组

结构体数组变量可用于循环布局，以便在需要较少加载调用以显示对象数据的应用程序中实现更高效的循环设置，或者在构建者希望显示通过函数输出的非本体论数据的工作流中。有关遍历数组的更多信息，请参阅 循环布局文档。

从结构体中提取字段

小部件和变量转换操作不能整体使用结构体，因此必须提取单个结构体字段以供使用。下图显示了如何使用 提取结构体字段的变量转换 从 person 结构体变量中提取字符串类型 name 字段，然后在 指标卡片小部件 中使用。

将结构体作为函数输入

结构体和结构体数组变量也可以作为函数的输入。在配置使用结构体作为输入的函数时，可以通过将鼠标悬停在 Preview schema 标签上来预览结构体输入的所需字段。这使构建者能够通过引用结构体变量的原始 Current value 来验证预期的输入模式是否与所选结构体变量匹配。
