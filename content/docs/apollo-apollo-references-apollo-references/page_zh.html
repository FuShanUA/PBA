Apollo 技术参考

概述

本节提供有关使用 Apollo 构建和部署软件的技术参考资料。

有四个主要的参考部分：

  • Apollo 产品规范 是一种声明式方法，用于定义应用程序的构建时和运行时属性，这样 Apollo 就知道如何部署和管理它。
  • Apollo API 文档 说明了如何通过 Apollo UI 直接与 Apollo API 进行交互的说明。
  • Apollo CLI 文档 提供了通过其提供的 CLI 与 Apollo 交互的语法和选项。
  • Cel 策略规范 记录了在编写通用表达语言 (CEL) 策略以配置您的空间的审批要求时可用的更改类型、响应类型和函数。
