Cast

Supported in: Batch, Faster, Streaming

Cast expression to given type.

Expression categories: Cast, Popular

Declared arguments

  • Expression: Expression to cast.
    Expression<AnyType>
  • Type: Type to cast to.
    Type<C>

Type variable bounds: C accepts AnyType

Output type: C
