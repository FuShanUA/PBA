AIP 特性

Palantir 平台上的应用程序都配备了 AIP 驱动的功能，如本页面所述。这些功能由平台支持的 大型语言模型 (LLMs) 提供支持。

AIP 应用和构建者功能

AIP 使开发人员和构建者能够在 Palantir 平台上使用 LLM 原生工具（如 AIP Chatbot Studio（前身为 AIP Agent Studio）和 AIP Logic），或 AIP 加速的平台应用（如 Pipeline Builder 和 Workshop）创建 LLM 支持的工作流程、代理和应用程序。

Palantir 提供的 LLM 也可在核心 Foundry 功能中使用，例如 Functions、Transforms，以及通过 Code Workspaces 的 Jupyter® 笔记本。

此外，Palantir 原有的模型集成功能允许用户 连接自定义的大规模语言模型，并独立地从头开始构建使用案例。

更多详细信息请参阅下面的 应用参考。

AIP 应用参考

下表描述了 AIP 应用，并提供了何时可能需要使用它们的信息。 您还可以查看 Foundry 套件应用的参考页面。

应用描述
AIP AssistAIP Assist 是一个由大语言模型（LLM）驱动的支持工具，通过提供实时、安全的自然语言协助，帮助用户在 Palantir 平台上进行导航。用户可以通过向 AIP Assist 提问并接收上下文感知的响应来加速工作流程并提高生产力。
AIP LogicAIP Logic 是一个无代码开发环境，用于创建、测试和部署基于 AI 的函数 - 为您提供一种点击即可使用的方式，利用由您的 本体论 数据支持的大语言模型（LLM）的力量。AIP Logic 提供了直观的界面，用于工程提示、设置自动化，并集成结构化或非结构化的 本体论 数据。您可以使用 AIP Logic 轻松地简化和自动化复杂流程，例如排程或优化问题，同时保持强大的安全控制。
AIP Chatbot StudioAIP Chatbot Studio 可让您创建能够利用 本体论 中的企业特定数据以及各种工具来完成任务和实现目标的交互式聊天机器人。您可以将由大语言模型驱动的 AIP 聊天机器人部署到自动化手动操作、编辑 本体论 数据、简化工作流程并增强应用程序交互。
AIP EvalsAIP Evals 是生产环境中稳定、可靠 AIP 工作流的基础；通过使用 AIP Evals 测试和评估基于大语言模型的功能和提示，您可以建立对基于大语言模型的工作流的信心。通过在 AIP Evals 中设置测试用例和评估标准，您可以系统地调试、迭代和改进您的实现，比较不同的模型，并检查运行之间的差异。
AIP ThreadsAIP Threads 为使用大语言模型执行任务和临时分析提供了便捷方式，无需技术设置即可与文档和 AIP 聊天机器人进行交互 - 只需将文档拖放到 AIP Threads 或从一系列现有资源和聊天机器人中选择，然后用您的请求提示大语言模型。
Palantir MCPPalantir MCP 允许外部 AI IDE 和代理连接到 Palantir 平台，并了解您的 本体论 和 Foundry 工具。使用 Palantir MCP 让外部 AI 系统更高效地查询数据、访问文档并构建应用程序。

AIP 与开发者工具链

Palantir 的 开发工具链 提供了构建 AI 应用的组件，这些应用可直接与您的 本体论 数据、逻辑和操作进行交互。 本体论 SDK 让您可以用 Python、Java 或 TypeScript 编写 AIP 驱动的应用程序，并内置了对 AIP 逻辑函数的访问权限。 Palantir MCP 将外部 AI IDE 和代理连接到 Palantir 平台，使其了解您的本体论和 Foundry 应用程序，以便它们可以查询数据、访问文档并更高效地构建应用程序。 这些工具共同使构建能够利用您组织的数据的 AI 解决方案变得简单，而无需将不同的系统拼接在一起。

AIP 在平台应用中的功能

AIP 的功能也已嵌入到核心 Foundry 应用程序中，以帮助用户加快工作流程并在平台上释放更多价值。以下是一些 AIP 功能的精选示例，并非完整列表。最新的 AIP 更新可以在文档的 公告 部分找到。平台管理员可以通过 AIP 设置 下的控制面板来管理这些功能的使用。

AIP Assist 侧边栏

从任何平台应用程序中，您都可以打开 AIP Assist 侧边栏以获得支持；AIP Assist 是上下文感知的，因此查询的答案会根据当前活动的平台应用程序而变化。您可以从工作区导航栏打开 AIP Assist 或使用快捷键（Cmd + Shift + U（macOS）或 Ctrl + Shift + U（Windows））进行访问。

了解更多关于 AIP Assist 的信息。

管道构建器

使用 Pipeline Builder 中的 AIP 功能，可以帮助您更好地理解、构建和管理您的管道。Pipeline Builder 具有一套核心的 Assist 特性，并提供了额外的 AIP 功能，用于自定义工作流。

在具有适当 权限 的情况下，您可以在 Pipeline Builder 中使用 AIP 功能来执行自定义工作流，例如以下功能：

  • 使用 LLM 节点 提供了一种便捷的方法，可在大规模数据上执行大语言模型（LLMs）。

  • 您还可以在输入数据集的几行数据上进行 试运行，在对整个数据集运行模型之前迭代您的提示。

  • Pipeline Builder 中提供的 文本转嵌入 表达式允许您使用文本嵌入 ada-002 模型将文本字符串转换为语义向量表示，从而基于单词或短语的意义实现高级文本分析和操作。

Pipeline Builder 的用户还可以使用 核心 Assist 功能，其中一些功能包括：

  • 解释： 了解您在 pipeline 开发中的步骤，并建议相关名称和描述。

  • 正则表达式助手： 为您的需求生成定制的正则表达式，适用于所有技能水平。

  • 转换助手： 创建和编辑正则表达式，并轻松将字符串转换为特定的时间戳格式。

自动化

自动化应用程序 使用户能够构建自动化流程，持续监控定义的条件，并在这些条件满足时自动执行相应的操作。

Automate 与 AIP Logic 集成，使您能够直接从您的 AIP Logic 文件创建自动化流程。此功能通过自动化本体论编辑的申请或暂存以供人工审查，从而改善您的本体论管理体验。工作流用户可以检查每个建议操作背后的逻辑，并在批准后自动应用更改。

便签

AIP 为 便签 带来了基于 LLM 的功能，您可以使用 AIP 自动拼写检查、缩短、修改或翻译文本，而不会影响文档的现有格式。

Scheduler（调度器）

您可以在 调度器应用程序中使用 AIP ，当使用特定时间触发器创建数据集构建计划时，生成一个计划配置。在 新计划视图 侧边栏内输入一个计划触发提示，以生成复杂触发器的适当 cron 格式。
