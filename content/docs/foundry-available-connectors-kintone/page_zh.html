Kintone

Kintone 连接器是面向 Kintone 的 Palantir 提供的驱动程序。

要创建新的 Kintone 源，请遵循 Palantir 提供的驱动程序的标准设置流程，然后使用以下章节进行 Kintone 专属配置和网络设置。有关完整的属性参考，请查阅 官方 Kintone 驱动程序文档 ↗。

配置

以下属性为必填或推荐配置项。

属性是否必填？描述默认值
AuthScheme ↗必填使用 用户名/密码、API 令牌 还是 OAuth 连接 Kintone。Password
Schema ↗必填指定要使用的 Kintone API 版本。REST
URL ↗必填Kintone URL。例如：https://SUBDOMAIN\_NAME.cybozu.com 。https://subdomain.example.com
Password ↗推荐指定进行身份验证的用户账户的密码。—
User ↗推荐指定进行身份验证的 Kintone 用户账户的用户 ID。—
