foundryts.nodes.SummarizerNode

class foundryts.nodes.SummarizerNode(children)

Lazy query container for summarizing one or more FunctionNode to a final result.

A SummarizerNode is the final evaluated form of a raw or transformed time series and cannot be transformed by FoundryTS any further. It is typical to evaluate a SummarizerNode to a dataframe using either SummarizerNode.to_pandas() or SummarizerNode.to_dataframe() and performing transformations and analysis using the respective dataframe libraries.

columns()

Returns a tuple of strings representing the column names of the pandas.DataFrame that would be produced by evaluating this node to a pandas dataframe.

Note

Keys of nested objects will be flattened into a tuple with nested keys joined with ..

  • Returns: Tuple containing names of the columns in the resulting dataframe which the current node gets evaluated to.
  • Return type: Tuple[str]

Examples

Copied!
1>>> node = foundryts.functions.series("series1")
2>>> node.columns()
3("timestamp", "value")
4>>> stats_node = series.statistics(start=0, end=100, window_size=None)
5>>> stats_node.columns()
6("count", "smallest_point.timestamp", "start_timestamp", "latest_point.timestamp", "mean",
7"earliest_point.timestamp", "largest_point.timestamp", "end_timestamp")

property series_ids

to_dataframe(fts=None)

Evaluates this node to a pyspark.sql.DataFrame.

PySpark DataFrames enable distributed data processing and parallelized transformations. They can be useful when working with dataframes with a large number of rows, for example loading all the points in a raw series or the result of a FunctionNode, or evaluating the results of multiple SummarizerNode or FunctionNode together.

  • Parameters: fts (foundryts.FoundryTS , optional) – FoundryTS session used to execute the query (a new session will be created if not provided).
  • Returns: Output of the node evaluated to a PySpark dataframe.
  • Return type: pyspark.sql.DataFrame

Examples

1>>> series_node = F.points(
2...     (100, 0.0), (200, float("inf")), (300, 3.14159), (2147483647, 1.0), name="series"
3... )

to_pandas(fts=None)

Evaluates this node to a pandas.DataFrame.

The DataFrame result of a SummarizerNode will be the flattened into a pandas.DataFrame with a single row (or one row per element if the summarizer returns multiple results) and one column per leaf level value in the resulting object. Nested keys will be dot-separated.

  • Parameters: fts (foundryts.FoundryTS , optional) – FoundryTS session used to execute the query (a new session will be created if not provided).
  • Returns: Output of the node evaluated to a Pandas dataframe.
  • Return type: pd.DataFrame
