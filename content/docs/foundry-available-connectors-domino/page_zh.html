Domino

Domino 连接器是 Palantir 提供的用于 Domino 的驱动程序。

要创建新的 Domino 源，请遵循《Palantir 提供驱动程序的标准设置流程》，然后使用以下章节进行 Domino 特定的配置和网络设置。有关完整的属性参考，请参阅《官方 Domino 驱动程序文档 ↗》。

配置

以下属性为必填或建议填写。

属性是否必填？描述默认值
DatabaseScope ↗必填Domino Web UI “作用域”页面中的 Domino 作用域名称。提供程序会公开受指定作用域管控的架构的表单和视图。—
TableTypes ↗必填选择提供程序是公开 Domino 表单还是 Domino 视图。Forms
URL ↗必填将其设置为托管 Domino 实例的服务器 URL。—
AuthScheme ↗建议连接到 Domino 时使用的身份验证类型。OAuthJWT
InitiateOAuth ↗建议指定获取或刷新 OAuth 访问令牌的流程，以便在经过身份验证和授权的用户工作时保持其访问权限。REFRESH
OAuthClientId ↗建议指定创建自定义 OAuth 应用程序时分配的客户端 ID。（也称为使用者密钥。）此 ID 用于向 OAuth 授权服务器注册自定义应用程序。—
OAuthClientSecret ↗建议指定创建自定义 OAuth 应用程序时分配的客户端密钥。（也称为使用者密钥）。此密钥用于向 OAuth 授权服务器注册自定义应用程序。—
Password ↗建议指定进行身份验证的用户帐户的密码。—
User ↗建议指定进行身份验证的 Domino 用户帐户的用户 ID。—
