创建端点集

按照以下说明创建一个新的端点集：

  1. 从工作区导航栏中选择文件，并找到您所需的项目或文件夹。
  2. 在右上角选择新建 > 端点集，以在当前项目或文件夹中创建一个新的端点集。新的端点集将继承其父项目或文件夹的权限。



  3. 在保存新端点集对话框中，为您的端点集输入一个名称并保存。这会创建一个新的空端点集资源。

要请求新的端点托管子域名，请按照以下说明操作：

  1. 在您选择的端点集中导航到 概览 > 端点域名，并指定一个子域名名称。
  2. 在右上角选择 请求端点托管域名，并确认说明以创建子域名注册请求。



  3. 等待您的注册管理员批准您的子域名注册请求。

在注册并获得您的子域名批准后，您应该会在已部署的子域名旁边看到一个 域名就绪 标签。在等待批准期间，您可以继续配置和测试您的端点定义。请查看 自定义端点定义 文档获取更多详细信息。
