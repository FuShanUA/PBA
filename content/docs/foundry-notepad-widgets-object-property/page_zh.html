对象属性

对象属性部分支持在行内引用对象属性。点击 + 添加组件，或在段落字段中输入 / 即可打开组件插入菜单，以此添加对象属性组件。随后会弹出一个搜索栏，你可以在此查找特定对象并将其选中添加。

默认情况下，系统会显示对象的 title 属性，但你可以在组件属性菜单中更改所显示的属性。

组件属性

  • 对象选择： 要显示的对象。
  • 属性： 要显示的属性。
  • 显示图标： 在属性内容旁显示对象图标。
  • 悬停时显示对象属性： 鼠标悬停在对象属性上时，会弹出对象信息浮层。该浮层包含主要对象属性以及指向对象视图的链接。
  • 显示属性名称： 在属性内容旁显示属性名称。
  • 应用本体格式： 应用在《本体管理器》中定义的条件格式。
  • 禁用链接： 开启后，将移除对象属性中指向《对象资源管理器》的链接。
  • 为空值显示自定义内容： 开启后，允许设定一个自定义内容，以便在本体属性为空时进行显示。

模板配置

  • 对象选择： 要显示的对象。

本体动作

将鼠标悬停在属性上可显示菜单，利用该菜单可编辑已配置行内编辑的文本属性、通过动作修改对象，以及在《对象资源管理器》中打开该对象。
