foundryts.objects.FoundryObject

class foundryts.objects.FoundryObject(object_type_id, object_primary_key)

本体中的对象。

  • 参数：
    • object_type_id (str) – 本体中对象类型的 ID。
    • object_primary_key (str) – 对象的主键，可在定义该对象的数据集或 《↗ 对象浏览器》中找到。
注意

创建 FoundryObject 时，应仅使用 Object.id()，该方法为访问本体中的对象提供了安全保障。请参考以下示例。

示例

Copied!
1>>> aircraft_object = Object("aircraft").id("aircraft-1l)

property(property_id, dataframe_identifier=None)

使用 FoundryObject 的时间序列属性创建时间序列引用。

该引用可用于 FoundryTS 支持的所有转换和分析。

  • 参数：
    • property_id (str) – 用于引用时间序列属性的属性 ID，可从 《↗ 本体管理器中的属性编辑器视图》中提取。
    • dataframe_identifier (str , 可选) – 当在 foundryts.NodeCollection 中计算多个时间序列时，结果数据集中的序列标识符。访问复杂的时间序列属性类型（例如 ↗ 模板）时需要此参数 （默认为平台中的序列 ID）。
  • 返回： 作为 FunctionNode 的时间序列引用，可用于转换和分析。
  • 返回类型： FunctionNode
注意

请确保为 property_id 使用属性 ID，因为平台上有三种可用的属性引用：属性 ID、属性 RID 和 API 名称。
