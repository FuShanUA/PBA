将字符串转换为时间戳

支持于：批处理、Faster、流处理

根据 Java DateTimeFormatter 的规范，将格式化字符串转换为时间戳并返回。默认格式为 yyyy-MM-dd'T'HH:mm:ss.SSSXXX 和 yyyy-MM-dd。格式将按顺序进行匹配，返回首个匹配成功的格式结果。

表达式类别： 转换、日期时间

声明参数

  • 字符串： 要转换为时间戳的字符串列。
    Expression<String>
  • 可选 格式： 默认格式为 ISO8601 yyyy-MM-dd'T'HH:mm:ss.SSSXXX 和 yyyy-MM-dd。
    List<Literal<String>>
  • 可选 时区： 用于解析不包含时区的格式。如果格式本身也包含时区，此参数将覆盖原有设定 - 详情请参见示例。
    TimeZone

输出类型： Timestamp

示例

示例 1：基础案例

参数值：

  • 字符串： timestamp
  • 格式： [yyyy-MM-dd'T'HH:mm.SSSSSSX]
  • 时区： null
timestamp输出
