近似百分位

支持于：批处理

返回表达式的近似百分位，即按从小到大排序后的表达式值中最小的值，使得不超过该百分比值比例的表达式值小于或等于该值。

表达式类别： 聚合

声明的参数

  • 表达式： 输入表达式。
    Expression<Numeric>
  • 百分位数： 要计算的百分位数。如果给定单个值，输出将为双精度浮点数；如果提供多个值，输出将是表示各个百分位数的双精度浮点数数组。取值范围必须在 0 到 1 之间。
    List<Literal<Double>>
  • 可选 准确度： 准确度参数（默认值：10000）是一个正整数，通过消耗内存来控制近似计算的准确度。准确度值越高，精度越好，1.0/准确度 是近似值的相对误差。
    Literal<Integer>

输出类型： Array<Numeric> | Byte | Decimal | Double | Float | Integer | Long | Short

示例

示例 1：基础用例

参数值：

  • 表达式： values
  • 百分位数： [0.5]
  • 准确度： null

给定输入表：

values
2
4
3

输出： 3

示例 2：基础用例

参数值：

  • 表达式： values
  • 百分位数： [0.33, 0.5, 0.66]
  • 准确度： null

给定输入表：

values
2
4
3
5
1

输出： [ 2, 3, 4 ]

示例 3：空值用例

参数值：

  • 表达式： values
  • 百分位数： [0.5]
  • 准确度： null

给定输入表：

values
null
null
null

输出： null

示例 4：空值用例

参数值：

  • 表达式： values
  • 百分位数： [0.5]
  • 准确度： null

给定输入表：

values
null
1
3

输出： 1
