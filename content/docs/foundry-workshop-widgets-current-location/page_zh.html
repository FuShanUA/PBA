Widget: Current Location Manager

Current Location Manager 部件会提示用户共享其位置，并将检测到的位置以字符串变量的形式发布，供您在应用中使用。此外，该部件还能利用 Mapbox 反向地理编码 API，向最终用户展示可读的位置预览。

若用户拒绝共享当前位置，系统将显示警告，且不会设置当前位置变量值：

请注意，仅当包含该部件的页面处于打开状态时，Current Location Manager 部件才会提示用户共享位置并更新检测到的位置。如果用户在应用内跳转至其他页面，位置检测与更新将暂停。建议将此部件放置在工具栏或其他显眼位置，以确保在需要时始终可见。在包含 Current Location Manager 的页面上层打开抽屉组件不会阻止其更新；只有跳转至其他页面才会导致更新暂停。

配置选项

以下是 Current Location Manager 的核心配置选项：

  • 标签： 显示在位置预览左侧的文本标签。可留空以移除该标签。
  • 位置输出变量： 用于存储用户当前位置的字符串变量，格式为 lat,long。
  • 隐藏位置显示： 启用后将隐藏标签与位置预览，但仍会提示用户共享位置。同时，系统不会尝试对用户位置进行反向地理编码。适用于无需向用户展示检测位置预览的场景。
