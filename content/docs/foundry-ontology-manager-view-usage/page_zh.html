本体论指标

本体论管理器可以配置为显示对象类型和链接类型的使用指标。

关键术语

  • 读取: 当应用程序加载指定对象类型的对象时，会记录一次读取。这包括在Workshop中显示表格中的对象、返回给定对象类型的所有对象、对对象类型的一个属性进行聚合等。请注意，一次读取代表从 对象存储V1（Phonograph） 或对象集服务（OSS）发出的一次加载请求。同时加载或聚合多个对象只会被记录为一次读取。另外请注意，任何在本体论管理器中发生的对象类型或链接类型的使用都不会被包含在内。
  • 写入: 当应用程序通过 操作、函数、Foundry 表单、直接对象浏览器编辑或 API 调用对这类对象进行编辑时，会记录一次写入。请注意，一次写入代表发送到 对象存储 V1 (Phonograph) 的一个编辑请求。同时编辑的多个对象只会被记录为一次写入。
  • 交互: 该类型对象在过去 30 天内的读取和写入总数。
  • 活跃用户: 在过去 30 天内触发了所记录的读取和写入的唯一用户 ID 数量。

查看使用情况

在本体论管理器中有两个位置可以查看对象类型和链接类型的使用情况：

  • 在 概览 选项卡上的使用情况图表：过去 30 天使用情况的高级摘要，使本体论用户能够快速了解对该资源进行重大更改的影响。

警告

如果您在使用图表中看到“过去30天内无使用情况”，而您本应看到使用统计信息，则可能是内部表未正确配置。请联系您的Palantir代表以获取更多信息。

  • 一个专用的使用情况选项卡：资源的详细使用指标。用户可以查看过去30天内，每个对象类型被谁使用、何时使用以及在哪些Foundry应用程序中使用。该功能旨在帮助Ontology用户更安全地进行Ontology更改，通过提供对更改影响的更清晰理解。还可以通过点击查看更多来访问概览选项卡中的使用图表。

启用Ontology使用

Usage on the Overview tab and detailed usage metrics in the Usage tab are configured from the Ontology settings tab in Control Panel using the Ontology metrics toggle. This toggle can only be enabled or disabled by Ontology administrators and changes may take up to 60 minutes to take effect in Ontology Manager.

共享本体论使用情况

如果您的组织与其他组织共享一个本体论，那么 Usage 标签页将对所有拥有本体论指标的组织的用户可用。显示的使用指标仅包括拥有对象类型访问权限的用户的使用情况以及来自已启用本体论指标的组织的用户。有关更多信息，请参阅 启用本体论使用 中概述的步骤。
