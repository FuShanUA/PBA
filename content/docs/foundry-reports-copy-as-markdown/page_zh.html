将报告文本复制为 Markdown

报告文本可快速导出为 Markdown 格式。

Markdown 是一种用于设置文本样式（如粗体、斜体、下划线）和定义文档结构（标题、表格、列表等）的标记语言。作为一种开源标准，Markdown 与 Palantir Foundry 内外的众多应用均兼容。

由于 Markdown 语法基于纯文本，报告应用通过将文本复制到剪贴板的方式实现 Markdown 的“导出”，供您粘贴至其他位置。在报告的 Markdown 表示形式中，仅包含文本和表格小组件（参见《向报告添加表格》）。

若要将报告导出为 Markdown：

  1. 点击应用右上角的操作。
  2. 点击将文本复制为 Markdown。
