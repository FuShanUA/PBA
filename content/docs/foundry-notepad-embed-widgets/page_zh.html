嵌入微件

Notepad 中的微件允许您向 Notepad 文档添加内容，例如图像、图表、表格、样式元素、链接，以及与其他 Foundry 应用程序的连接。

支持的集成

Notepad 支持从其他多个 Foundry 应用程序嵌入微件，例如图表或表格。以下集成受支持：

  • Code Workbook 图表
  • 命令
  • Contour 图表
  • 函数
  • 地图
  • 对象卡片
  • 对象媒体预览
  • 对象属性 Markdown 编辑器
  • Quiver 图表
  • Quiver 仪表板
  • Solution Designer 图表
  • Vertex 图

此外，Notepad 还提供多个原生微件：

  • 锚点链接
  • 当前日期
  • 水平分割线
  • 图像
  • LaTeX
  • 分页符
  • 资源链接
  • 表格
  • 用户提及
  • 值嵌入

微件参考提供了对配置选项和功能的详细说明。Notepad 目前不支持嵌入带有 Workshop 模块或 Slate 应用程序视图的微件。

添加微件

您可以在文档中或直接从源应用程序添加微件。

从文档添加

  1. 在段落中使用 / 键盘快捷键，或点击 + 微件 打开插入菜单。
  2. 选择嵌入类型。
  3. 点击 插入 添加新微件。

来自其他 Foundry 应用程序的集成可能需要额外配置才能显示数据或可视化效果。请使用右侧的 微件属性 面板进行配置。

从资源添加

所有 可嵌入的 Foundry 应用程序 中都提供了 为 Notepad 复制 按钮。点击此按钮会将内容复制到剪贴板，随后您可以将其粘贴到文档中。

微件链接

文档中引用的所有对象、对象集和 Foundry 资源都会自动链接到该文档。使用右上角 操作 菜单中的 查看文档链接 按钮可查看文档链接。

指向对象的链接将用于 Workshop 的 Notepad：链接文档 微件中，以显示引用特定对象的所有文档。有关更多详细信息，请参阅 显示链接到对象的文档。
