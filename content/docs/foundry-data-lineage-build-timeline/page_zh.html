查看构建时间线

使用数据血缘关系中的构建时间线工具，查看数据集的构建历史。

在数据血缘关系中，点击窗口左下角的构建时间线。此操作会展开视图面板，以甘特图形式展示所选时间段内的构建情况。你可以选择时间线中需要查看的天数或小时数，范围从一小时到十天不等。你还可以选择根据调度或作业状态，以不同颜色展示构建情况。

要查看特定数据集的构建时间线，请在图表中选中该数据集。使用拖拽选择模式工具，或在点击时按住Ctrl / Command 键，即可选中多个数据集。

要查看构建时间线中某项作业的详情，请点击甘特图中的对应作业。你将看到有关作业状态、起止时间和运行时长的信息。

点击作业信息窗口内的链接，可查看有关构建、作业和调度的更多详情。
