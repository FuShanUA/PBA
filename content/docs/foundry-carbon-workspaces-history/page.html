View version history

Workspace version history allows you to preview, optionally make edits, and republish previous workspace versions. This enables you to quickly return to a known working configuration if a workspace configuration update leads to errors.

Versions dialog

The versions dialog is where version metadata may be viewed, with the option to select a row to set a version to preview.

Version button

The versions dialog can be launched by selecting the Version button, located at the top of the configuration editor. The version number of the version you are currently editing is shown as the text in this button.

Unsaved edits indicator

An asterisk (*) will appear after the version number if there are unsaved edits.

Previewing version

A Previewing tag is displayed next to the versions button when a version is selected from the versions dialog.
