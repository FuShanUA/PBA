集成

Automate 与以下 Foundry 应用程序原生集成：

  • AIP Logic
  • Notepad
  • Object Explorer
  • Ontology Manager
  • 时间序列 (Quiver)

AIP Logic

将您的 AIP Logic 自动化，以便自动应用或暂存本体编辑供人工审查。自动化可以在现有对象或创建新对象时触发。

了解如何将您的 Logic 函数自动化。

Notepad

Notepad 文档和 Notepad 模板可以附加到自动化的 通知效果中，从而实现利用 Notepad 中的最新数据自动分发报告和摘要。

要直接从 Notepad 创建自动化，请在右上角选择 操作 下拉菜单，然后选择 添加自动化。

您将被引导至 Notepad 文档的自动化概览页面。选择 新建自动化 以启动自动化创建向导，该向导打开时会将此 Notepad 文档作为附件预填充。

配置并保存自动化后，您将返回到文档内的自动化概览页面，该页面现在会显示您新创建的自动化。

Object Explorer

按照以下步骤在 Object Explorer 中创建自动化：

  1. 在探索视图中选择 保存。
  2. 选择项目或文件夹目标位置。



3. 保存后，在页面右上角选择 监控，以查看使用此探索作为输入的所有自动化。新创建的探索将显示一个空列表。



4. 选择 添加新自动化，打开用于创建新自动化的简化设置视图。

Ontology Manager

Ontology Manager 允许用户定义其 Foundry 本体，包括对象类型和操作类型。Automate 通过以下两种方式与 Ontology Manager 集成：

  1. 对象类型视图中的 自动化 选项卡显示与所选对象类型关联的所有自动化，包括在创建或更新该类型的对象时触发的自动化。选择 添加自动化 可启动自动化创建向导，并将该对象类型预填充为条件。



  2. 类似地，操作类型视图中的 自动化 选项卡显示在其 操作效果中使用所选操作类型的所有自动化。选择 添加自动化 可启动创建向导，并将该操作类型预填充为效果。

您可以在 Quiver 中将时间序列搜索逻辑自动化，以便将时间序列数据中感兴趣的时间段作为事件保存在本体中。
