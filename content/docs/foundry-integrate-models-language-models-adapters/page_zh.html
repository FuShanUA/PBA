此页面记录了之前用于 导入开源模型 功能的 模型适配器。这些适配器为与 Hugging Face 语言模型交互提供了默认参数和结构，包括根据部署基础设施中的可用性自动路由到 CPU 和 GPU 设备。以下的 适配器源代码 和 导入指南 可以作为在您自己的代码仓库中编写替换适配器的起点，根据需要进行扩展。下面的适配器包括自定义 load 和 save 实现。在编写新适配器时，建议使用 Hugging Face 自动序列化程序。

当前正在使用的模型适配器可在 Modeling Objectives 应用程序中的模型提交页面上查看。

Seq2SeqLMModelAdapter

用法

此模型适配器支持 序列到序列语言模型 ↗。该适配器基于提供的输入文本生成文本。

示例： 所期望的输入文本及其结构取决于所选择的模型。我们强烈建议阅读模型详情，以确保正确的提示工程。例如，flan-t5-large ↗ 模型可以执行从翻译到摘要再到问答的各种提示。

Model API

  • Input “text”: 语言模型生成输出预测的文本输入。
  • Output “prediction”: 模型生成的文本。

NerAdapter

用法

此模型适配器支持命名实体识别管道 ↗。该模型适配器从文本中提取实体，并以列表形式返回。

示例：当通过实时部署将文本“我的名字是Max，我住在德国”发送到模型时，模型会识别出两个实体：Max和Germany。

模型API

  • 输入 “text”: 语言模型生成输出预测的文本。
  • 输出<list> “prediction”: 模型的输出是一个字典列表，每个字典提供有关提取实体的详细信息。具体来说，字典包含：
    • entity: 被识别的实体类型。
    • score: 特定实体的数值分数。
    • index: 实体的标记索引（例如，索引5表示识别出的实体是输入文本中的第五个标记）。
    • word: 识别出的实体的字符串表示形式。
    • start: 识别出的实体在输入字符串中的起始索引。
    • end: 识别出的实体在输入字符串中的结束索引。

EmbeddingAdapter

用法

此模型适配器基于模型的 注意力掩码 ↗ 计算给定文本的嵌入。该适配器执行“平均池化”和归一化操作，这与 sentence-transformers ↗ 的默认设置一致。

示例：

模型API

  • 输入“text”（文本）: 嵌入模型的输入文本。
  • 输出<list> “embedding”（嵌入）: 表示输入文本的n维向量。

文本分类适配器

用法

该模型适配器对预定义类集的输入文本进行分类。文本分类的常见例子是情感分析或语言检测。

示例：

模型API

  • 输入 “text”: 语言模型生成输出预测的输入文本。
  • 输出 “prediction”: 最可能的类别。
  • 输出<list> “logits”: 每个类别的logit值。
  • 输出<list> “classes”: 模型可以预测的类别。顺序与logits的顺序相同（例如，第一个logit条目对应于classes列表中的第一个条目）。

ZeroShotClassificationAdapter

用法

这个模型适配器根据预测时提供的类别列表对输入文本进行分类。这种行为使模型适配器能够在不需要微调语言模型以适应特定用例的情况下对文本进行分类。

示例：当发送文本“我喜欢开发新的语言模型”以及候选标签“旅行”、“体育”、“工作”和“娱乐”时，模型将对这些标签进行评分，并将“工作”作为最可能的分类。

Model API

  • Input “text”: 语言模型用于生成输出预测的输入文本。
  • Input<list> “candidate_labels”: 模型进行评分的输入文本的潜在标签列表。
  • Output “prediction”:
    • sequence: 用于零样本分类的文本。
    • labels: The candidate labels ordered based on the models score, in descending order.
    • scores: The float number representing the classification score for the candidate labels. The order of the scores matches the order of the labels.

6class DeviceMixin:
7    _moved_model_to_device: bool = False
8
9    def _move_model_to_device_once(self, device: torch.device) -> None:
10        """Expected to be called in the ModelAdapter.predict() method.
11
12        This cannot be done during load() method because the model is loaded in a separate process from inference,
13        causing a CUDA shared memory re-initialization error during inference.
14
15        Lazy initialization of the CUDA context to ensure model init happens in the same process as inference, to
16        avoid multiprocessing hassles.
17        """
18        if not self._moved_model_to_device:
19            self.model.to(device)
20            self._moved_model_to_device = True
21
22    def _move_pipeline_to_device_once(self, device: torch.device) -> None:
23        if str(device) == "cpu":  # noop
24            self._moved_model_to_device = True
25        elif not self._moved_model_to_device:
26            with TemporaryDirectory() as temp_dir:
27                self.pipeline.save_pretrained(temp_dir)
28                self.pipeline = pipeline(self.PIPELINE_TYPE, temp_dir, device=device)
29            self.loaded_model_to_device = True
30
31    @staticmethod
32    def _get_cuda_or_cpu() -> torch.device:
33        """
34        If available, returns cuda device. Otherwise, default to cpu
35        """
36        return torch.device("cuda:0" if torch.cuda.is_available() else "cpu")

130    def _compute(
131        self,
132        predictions,
133        references,
134        rouge_types=None,
135        use_aggregator=True,
136        use_stemmer=False,
137        tokenizer=None,
138    ):
139        if rouge_types is None:
140            rouge_types = ["rouge1", "rouge2", "rougeL", "rougeLsum"]
141
142        multi_ref = isinstance(references[0], list)
143
144        if tokenizer is not None:
145            tokenizer = Tokenizer(tokenizer)
146
147        scorer = rouge_scorer.RougeScorer(
148            rouge_types=rouge_types, use_stemmer=use_stemmer, tokenizer=tokenizer
149        )
150        if use_aggregator:
151            aggregator = scoring.BootstrapAggregator()
152        else:
153            scores = []
154
155        for ref, pred in zip(references, predictions):
156            if multi_ref:
157                score = scorer.score_multi(ref, pred)
158            else:
159                score = scorer.score(ref, pred)
160            if use_aggregator:
161                aggregator.add_scores(score)
162            else:
163                scores.append(score)
164
165        if use_aggregator:
166            result = aggregator.aggregate()
167            for key in result:
168                result[key] = result[key].mid.fmeasure
169
170        else:
171            result = {}
172            for key in scores[0]:
173                result[key] = list(score[key].fmeasure for score in scores)
174
175        return result

11class EmbeddingAdapter(pm.ModelAdapter, DeviceMixin):
12    """
13    Expected input columns/fields:
14    - "text": str
15    """
16
17    INPUT_COLUMN: str = "text"
18    OUTPUT_COLUMN: str = "embedding"
19
20    SENTENCE_BATCH_SIZE = 128
21
22    pretrained_model_tmp_dir: tempfile.TemporaryDirectory = None
23
24    def __init__(self, tokenizer: AutoTokenizer = None, model: AutoModel = None):
25        self.model = model
26        self.tokenizer = tokenizer
27
28    @classmethod
29    def load(cls, state_reader):
30        pretrained_model_tmp_dir = tempfile.TemporaryDirectory()
31        state_reader.extract(pretrained_model_tmp_dir.name)
32        tokenizer = AutoTokenizer.from_pretrained(pretrained_model_tmp_dir.name)
33        model = AutoModel.from_pretrained(pretrained_model_tmp_dir.name)
34        adapter = cls(tokenizer=tokenizer, model=model)
35        adapter.pretrained_model_tmp_dir = pretrained_model_tmp_dir
36        return adapter
37
38    def save(self, state_writer):
39        with tempfile.TemporaryDirectory() as model_temp_dir:
40            self.tokenizer.save_pretrained(model_temp_dir, from_pt=True)
41            self.model.save_pretrained(model_temp_dir, from_pt=True)
42            state_writer.put_directory(model_temp_dir)
43
44    @classmethod
45    def api(cls):
46        inputs = {
47            "inference_data": pm.Pandas(columns=[(cls.INPUT_COLUMN, str)])
48        }
49        outputs = {
50            "output_data": pm.Pandas(columns=[(cls.INPUT_COLUMN, str), (cls.OUTPUT_COLUMN, list[float])])
51        }
52        return inputs, outputs
53
54    def _gen_batches(self, df: pd.DataFrame) -> pd.DataFrame:
55        for i in range(0, len(df), self.SENTENCE_BATCH_SIZE):
56            yield df[i:i + self.SENTENCE_BATCH_SIZE]
57
58    def predict(self, inference_data: pd.DataFrame) -> pd.DataFrame:
59        device = self._get_cuda_or_cpu()
60        self._move_model_to_device_once(device)
61
62        # Sentences we want sentence embeddings for
63        sentences = inference_data[self.INPUT_COLUMN].tolist()
64
65        batch_embeddings_list = []
66
67        for sentence_batch in self._gen_batches(sentences):
68            encoded_input = self.tokenizer(
69                sentence_batch, padding=True, truncation=True, return_tensors="pt"
70            ).to(device)
71
72            # Compute token embeddings
73            with torch.no_grad():
74                model_output = self.model(**encoded_input)
75
76            # Perform pooling
77            batch_embeddings = mean_pooling(
78                model_output, encoded_input["attention_mask"]
79            )
80
81            batch_embeddings_list.append(batch_embeddings)
82
83        sentence_embeddings = torch.cat(batch_embeddings_list)
84
85        # Normalize embeddings
86        sentence_embeddings = F.normalize(sentence_embeddings, p=2, dim=1)
87
88        inference_data[self.OUTPUT_COLUMN] = sentence_embeddings.tolist()
89
90        return inference_data
91
92
93def mean_pooling(
94    model_output: torch.Tensor, attention_mask: torch.Tensor
95) -> torch.Tensor:
96    # First element of model_output contains all token embeddings
97    token_embeddings = model_output[0]
98    input_mask_expanded = (
99        attention_mask.unsqueeze(-1).expand(token_embeddings.size()).float()
100    )
101    return torch.sum(token_embeddings * input_mask_expanded, 1) / torch.clamp(
102        input_mask_expanded.sum(1), min=1e-9
103    )

138        self,
139        train_data: pd.DataFrame,
140        eval_data: pd.DataFrame = None,
141        input_column: str = "text_input",
142        target_column: str = "text_target",
143        max_input_length: int = 512,
144        max_target_length: int = 64,
145        train_args=dict(DEFAULT_TRAINING_ARGUMENTS),

159        self.model = self.model.to(device)
160
161        # Imports only needed for training
162        import evaluate
163        import nltk
164        from datasets import Dataset as hfDataset
165        from huggingface_adapters.utils import rouge
166        from transformers import (
167            DataCollatorForSeq2Seq,
168            Seq2SeqTrainer,
169            Seq2SeqTrainingArguments,
170        )
171
172        set_hf_evaluate_caches()
173
174        # Prepare data
175        model = self.model
176        tokenizer = self.tokenizer
177
178        def preprocess_data(examples):
179            model_inputs = tokenizer(
180                examples[input_column], max_length=max_input_length, truncation=True
181            )
182
183            # Setup the tokenizer for targets
184            with tokenizer.as_target_tokenizer():
185                labels = tokenizer(
186                    examples[target_column],
187                    max_length=max_target_length,
188                    truncation=True,
189                )
190
191            model_inputs["labels"] = labels["input_ids"]
192            return model_inputs
193
194        train_data_hf = hfDataset.from_pandas(train_data)
195        train_data_tokenized = train_data_hf.map(preprocess_data, batched=True)
196
197        eval_data_hf = hfDataset.from_pandas(eval_data)
198        eval_data_tokenized = eval_data_hf.map(preprocess_data, batched=True)
199
200        # Prepare training config
201        model_temp_dir = tempfile.mkdtemp()
202        args = Seq2SeqTrainingArguments(model_temp_dir, **train_args)
203
204        path_rouge = os.path.abspath(rouge.__file__)
205        metric = evaluate.load(path_rouge, keep_in_memory=True)
206        data_collator = DataCollatorForSeq2Seq(tokenizer)
207
208        def compute_metrics(eval_pred):
209            predictions, labels = eval_pred
210            decoded_preds = tokenizer.batch_decode(
211                predictions, skip_special_tokens=True
212            )
213
214            # Replace -100 in the labels as we can't decode them.
215            labels = np.where(labels != -100, labels, tokenizer.pad_token_id)
216            decoded_labels = tokenizer.batch_decode(labels, skip_special_tokens=True)
217
218            # Rouge expects a newline after each sentence
219            decoded_preds = [
220                "\n".join(nltk.sent_tokenize(pred.strip())) for pred in decoded_preds
221            ]
222            decoded_labels = [
223                "\n".join(nltk.sent_tokenize(label.strip())) for label in decoded_labels
224            ]
225
226            # Compute ROUGE scores
227            result = metric.compute(
228                predictions=decoded_preds, references=decoded_labels, use_stemmer=True
229            )
230            return result
231
232        # Training
233        trainer = Seq2SeqTrainer(
234            model=model,
235            args=args,
236            train_dataset=train_data_tokenized,
237            eval_dataset=eval_data_tokenized,
238            data_collator=data_collator,
239            tokenizer=tokenizer,
240            compute_metrics=compute_metrics,
241        )
242
243        training_result = trainer.train()
244
245        return training_result

132def reorder_prediction_to_candidate_labels(
133    pred_scores: list[float], pred_labels: list[str], candidate_labels: list[str]
134) -> tuple[list[float], list[str]]:
135    """Reorder scores to preserve the order of candidate labels"""
136    order_dct = {key: idx for idx, key in enumerate(candidate_labels)}
137    ordered_labels = sorted(
138        [(order_dct[key], idx) for idx, key in enumerate(pred_labels)],
139        key=lambda x: x[0],
140    )
141    order_idx = [o[1] for o in ordered_labels]
142    ordered_scores = [pred_scores[idx] for idx in order_idx]
143    ordered_labels = candidate_labels
144    return [ordered_scores, ordered_labels]
