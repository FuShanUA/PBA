Power BI®

以下页面介绍了如何实现 Power BI® 连接器，以便从 Power Query 界面访问 Foundry 资源。如需了解用于数据集成的 Microsoft Power BI® XMLA 连接器相关信息，请查阅我们的数据连接性文档。

什么是 Power BI® 的 Foundry 集成？

  • 包含一个经 Microsoft 认证的 Power Query 连接器，支持您基于 Foundry 数据轻松创建 Power BI® 报表。
  • 支持导入和 DirectQuery 两种模式。
  • 针对 Power BI® Desktop，提供与 Foundry 访问控制的兼容性，包括细粒度权限。
  • 针对 Power BI® Desktop，支持 OAuth 身份验证以及基于令牌的身份验证。

受支持的 Power BI® 产品

  • Power BI® Desktop（2020 年 6 月版或更高版本）
  • 通过 Power BI® 网关的 Power BI® Service（2020 年 6 月版或更高版本）

请注意，Power BI® 自定义连接器目前不支持 Power BI® Report Server。不过，您可以改用标准的 ODBC 驱动程序进行连接。

Power Query 兼容性

作为经认证的 Power Query 连接器，该集成使得各类 Microsoft 产品均能访问 Foundry 数据：

  • Microsoft Fabric（Dataflow Gen2）
  • Power Platform（包括 Dataverse 和 Power Apps）
  • Power BI®（数据流和语义模型）

若要详细了解兼容性，请查阅 Microsoft Power Query 文档中的 Palantir Foundry 页面。
