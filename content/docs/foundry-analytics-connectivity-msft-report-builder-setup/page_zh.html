安装设置

现在，您可以从报表生成器中访问 Palantir Foundry 数据集，并利用它们来构建报表和可视化。若要在报表生成器中使用 Foundry，您必须在本地计算机上安装 Foundry ODBC 驱动程序。请按照以下指南完成针对报表生成器的安装操作。

第 1 步：安装 ODBC 驱动程序

要开始设置 Foundry 报表生成器集成，您需要安装一个名为 ODBC 驱动程序的组件。请前往《下载页面：ODBC 驱动程序》下载并安装该驱动程序。

如果您在安装过程中遇到任何问题，请联系您的 Palantir 代表。

第 2 步：开始构建报表

既然您已安装了 ODBC 驱动程序，接下来可以按照《Microsoft 报表生成器：入门指南》中的说明，开始构建由 Foundry 数据提供支持的首个报表。
