Apollo API 探索器

某些 Apollo API 可以通过 GraphQL 直接访问。 GraphQL ↗ 是一种查询语言，允许您精确获取所需的数据。 这个功能直接在您的 Apollo Hub 中通过 API 探索器 应用程序提供。 文档探索器列出了通过 GraphQL 在您的 Hub 中可访问的所有信息。

文档

使用文档探索器查看 Apollo GraphQL 模式中所有可用的查询和变更。 这提供了一个可搜索的界面，用于浏览从根级查询和变更类型分叉出的嵌套模式。
