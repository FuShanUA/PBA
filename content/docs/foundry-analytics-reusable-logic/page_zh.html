可复用逻辑

在分析应用中，您可以选择保存部分逻辑以供其他用户复用。这有助于减少重复劳动、缩短价值实现时间，并让用户能够利用领域专家或偏技术型协作者创建的逻辑。

Contour

在 Contour 中，您可以保存表达式，以便在不同的分析和路径间轻松复用逻辑，并与他人共享逻辑。保存的表达式可用于创建或替换列，或执行聚合运算。了解有关 Contour 的更多信息。

Quiver

Quiver 中的可视化函数包含一个或多个用于加载、合并和转换数据的 Quiver 卡片。可视化函数会自动将一组逻辑步骤应用于 Quiver 分析中的数据输入，并可供他人在其分析中使用。了解有关 Quiver 的更多信息。

Code Workbook

通过将代码封装在简单的表单界面背后，Code Workbook 模板为技术背景深厚的用户提供了一种轻量级协作方式，使他们能够与不擅长编写代码的用户协同工作。用户选择的值会代入模板中，随后可以像工作簿中的其他任何转换一样运行。任何能用代码创建的转换，例如 Matplotlib 或 Plotly 可视化，都可以通过代码模板构建。了解有关 Code Workbook 的更多信息。
