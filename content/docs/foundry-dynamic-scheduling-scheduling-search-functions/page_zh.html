搜索功能

搜索功能使用户能够快速找到并评估其排程工作流程中出现的特定独特问题的可能解决方案。

每个搜索功能都由一个 TypeScript 函数支持。灵活的接口允许构建者编写可以返回圆盘、时间槽或两者的组合的功能。此外，搜索功能可以是基于行的或基于圆盘的。

对于基于行的搜索功能，用户必须右键单击空白区域，并将行对象和时间作为函数输入。对于基于圆盘的搜索功能，用户必须右键单击一个圆盘，以将对象和时间作为函数输入。

函数界面

以下类型表示从行或 puck 触发时编写搜索功能所需的信息，包括关于搜索组的详细信息。


type IPuckSearch = (puck: ObjectReference) => ISearchResult
type IRowSearch = (rowId: string, selectedTime: Timestamp) => ISearchResult


/*
   搜索函数可以返回时间槽、磁 puck 或两者。SLOT 和 PUCK 用于 IHighlight 中的类型。
*/

export enum HighlightType {
    SLOT = "SLOT",
    PUCK = "PUCK",
}


/*
   用于 IHighlight 中将返回一组时间槽的函数。
*/

export interface IDomain {
    start: Long;
    end: Long;
}

/*
   用于确定运行搜索函数后 UI 中突出显示的内容
*/

export interface IHighlight {
    type: string;
    // 用于 "SLOT"
    domain?: IDomain;
    containerId?: string;
    // 用于 "PUCK"
    schedulableObjectPrimaryKey?: string;
    schedulableObjectTypeId?: string;
    // 可选
    comment?: string;
}

/*
   定义新创建的搜索组的标题和函数返回的行。
*/
export interface IRowGroup {
    title: string;
    containerIds: string[];
    highlights: IHighlight[];
}

/*
   函数的整体返回类型。
*/

export interface ISearchResult {
    rowGroup?: IRowGroup;
    sourcePuckIds?: string[];
    error?: string;
}
