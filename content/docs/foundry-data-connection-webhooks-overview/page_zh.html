Webhooks

你可以使用 Data Connection 配置 webhooks，将 Foundry 与外部系统和工作流连接起来。

本节包含有关出站 webhooks（Foundry 向其他系统发起请求）的信息。如果你需要接收入站 webhooks（其他系统向 Foundry 发送请求），请考虑改用监听器。

每个 webhook 都提供了一种向 Foundry 外部系统发起请求的方法。例如，你可以创建一个 webhook，在用户点击 Foundry 应用的按钮时，对外部服务器执行 HTTP 请求，从而将该应用与现有的工作流和源系统打通。

每个 webhook 都会关联到 Data Connection 中的一个源。源用于存储连接外部系统所需的凭据。根据 webhook 关联的源类型，你可以使用特定的任务类型。例如，在使用 REST 时，你可以灵活配置要对外部服务发起的 HTTP 调用。

你可以灵活配置 webhooks，使其接受特定输入，并捕获外部系统请求的输出。此外，你还可以为 webhook 执行设置时间、并发和速率限制。有关详细的配置选项，请参阅 webhooks 参考文档。
