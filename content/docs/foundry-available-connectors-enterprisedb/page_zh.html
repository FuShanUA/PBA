EnterpriseDB

EnterpriseDB 连接器是用于 EnterpriseDB 的 Palantir 提供的驱动程序。

要创建新的 EnterpriseDB 源，请遵循 Palantir 提供的驱动程序的标准设置流程，然后使用以下各节进行 EnterpriseDB 特定的配置和网络设置。有关完整的属性参考，请参阅 《EnterpriseDB 官方驱动程序文档》 ↗。

配置

以下属性为必填或推荐属性。

属性是否必填？描述默认值
Password ↗必填指定进行身份验证的用户账户的密码。—
Server ↗必填服务器的主机名或 IP 地址。localhost
User ↗必填指定进行身份验证的 EnterpriseDB 用户账户的用户 ID。—
UseSSL ↗必填是否启用 SSL。TRUE
Database ↗推荐EnterpriseDB 数据库的名称。—
Port ↗推荐EnterpriseDB 服务器的端口号。5444
