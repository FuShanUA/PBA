图元素参考

节点类型

节点                                        类型描述
数据源这是显示在《数据连接》中的数据源名称。《了解有关不同源类型的更多信息》。
数据集Foundry 数据集及其相互间的血缘关系。数据集节点的颜色取决于用户选择。虚线边框表示非结构化数据集。
对象类型本体对象类型。节点的图标和颜色取决于各对象类型的定义。点击对象类型名称旁边的“链接”图标时，数据血缘关系将展示该对象类型与其他对象类型之间的关系。
制品数据血缘关系会展示不同的 Foundry 制品，例如：《Contour》分析、《报告》等。节点的颜色取决于制品类型，该类型标注在节点顶部。

节点指示器

节点指示器显示在数据集节点上方，提供有关资源的附加信息。

指示器类型描述
待处理问题此指示器表示图中该节点当前存在关联的未解决问题。将鼠标悬停在该信号上可查看未解决问题的数量。
定义对象类型此指示器出现在用于定义本体对象类型的数据集上。将鼠标悬停在右箭头上可展开这些关联的对象类型。《了解有关对象类型的更多信息》。
同步带有此指示器的数据集已同步至其他数据库或系统。你可以通过选中节点并打开“属性”面板，或者在数据集预览中打开“详情”页签（右键点击节点并选择打开）来查看这些同步。
已废弃此指示器出现在代表已删除数据集或制品的节点上。已删除的节点也会呈半褪色状态，且其名称会带有删除线。
