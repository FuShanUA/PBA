导向与导航

您可以将 Palantir 平台视为组织数据的操作系统。本页面提供有关如何导航并查找所需工具和资源的信息，帮助您在该操作系统中高效工作。

本页面介绍了工作区导航侧边栏和搜索功能。前者是浏览平台的基础起点，后者用于定位所需的资源或数据。您还将了解在平台工作期间可获取的支持选项。最后，您可以通过Palantir 账户设置来配置公开用户和平台设置。

工作区导航侧边栏

侧边栏是平台中始终相伴的助手，也是导航的起点。点击右上角的图标或使用键盘快捷键 Cmd+O（macOS）或 Ctrl+O（Windows），即可展开或折叠侧边栏。

侧边栏包含五个主要部分，可带您前往平台内的不同功能和工具：

① 主页：返回组织的落地页
① 搜索...：打开快速搜索对话框
① 通知：查看平台和应用通知
① 新功能：在平台内阅读产品公告和发布说明

② 最近：快速跳转至最近访问的资源
② 文件：前往由 Compass 驱动的项目落地页
② 应用：通过该门户查找并访问所有平台应用

③ 应用（已收藏）：查找您之前标星的应用。整理并访问您最常用的应用

④ 文件（已收藏）：查找您之前标星的文件。整理并访问您最常用的资源和对象

⑤ AIP 助手：由大语言模型驱动的求助助手
⑤ 支持：访问 Palantir 文档、培训资源和帮助
⑤ 账户：查看账户详情并审阅权限和分组
⑤ 其他工作区：访问自定义工作区和控制面板（可用性取决于权限配置）

AIP 助手

在寻求 Palantir 平台相关帮助时，我们建议您首选 AIP 助手。这是 Palantir 推出的一款由大语言模型驱动的助手。AIP 助手支持以多种语言回答有关平台的问题，并能提供包括教程在内的平台使用指南。

您可以从侧边栏右下角访问 AIP 助手，或使用键盘快捷键 Cmd+U（macOS）或 Ctrl+U（Windows）。

AIP 助手具备“上下文感知”能力，能够检测您当前正在使用的应用程序，但它无法访问您正在处理的任何数据。

主页

Palantir 新部署环境会附带一个默认主页，帮助用户熟悉环境并了解该平台。管理员或构建者也可以为平台上的各类用户群体创建自定义落地页。部分环境可能采用完全定制的主页，而另一些则可能使用标准组件来提供对平台常用部分的访问。

尽管大多数主页侧重于导航，但您也可能在落地页上看到平台公告、组织内常用工作流的起点，或自定义文档的链接。

搜索

搜索（又称快速搜索）是一款用于导航和发现平台中元素的工具。搜索由两部分组成：

  1. 直达模式：提供简短的个性化结果列表，引导用户直接访问主要类型的可用内容：平台应用、自定义应用、对象、数据集及其他资源。
  2. 完整结果模式：旨在帮助用户查找内容并探索平台中的已有资源。用户可以通过高级筛选器、丰富的元数据和排序算法，搜索平台应用、对象、数据集及其他文件，从而突出显示最相关的结果。

要打开快速搜索，请在导航侧边栏中选择搜索...，或使用 ⌘ + J（macOS）或 Ctrl+J（Windows）。

Notifications

The Notifications panel collects notices from across the platform, ordered from newest to oldest. Most notifications include links to navigate directly to the relevant resources. Some, such as access requests, can be responded to inline without needing to navigate to another page.

By default, notifications are delivered both within the platform in the Notifications panel and by email. Under the See all link, notifications are grouped by type for easier navigation. The notification Settings give you granular control over global and per-notification-type delivery preferences. If notifications are unread, the bell icon will have a small, yellow badge. If a notification occurs while you’re using the platform, a small pop-up in the lower left corner will briefly show the message.

Recent

The Recent panel simply lists the last 20 resources you have opened or interacted with. Between Favorites and Recent, it’s possible to quickly navigate between primary resources in use for any project without needing to return to Search or browse through the Project folder structure.

Files

Files, powered by the Compass application takes you to the landing page for the Project folder structure, where you can access top-level Portfolios, Projects, Your files, and Shared with you shortcuts. You will learn more about Projects in the next step of this guide.

Applications portal

The Applications portal is a tool for finding and accessing all apps in the platform, including both platform apps and custom promoted applications.

Favorites

The Favorites section of the sidebar keeps links to specific applications, resources, and individual objects for quick navigation. Use Ctrl + Click on Windows or Cmd + Click on macOS to open a favorited resource in a new browser tab.

When you first use the platform, your Favorites section is empty and only Applications will show in the sidebar. Hover over the Applications header and select View all to see all the different applications in the platform.

In this view, you will see suggested applications and resources based on your recent usage. We will cover which applications we recommend based on your role later in this guide. To learn more, consider looking at the application reference.

Promoted applications

From the Applications portal, you can pin useful applications to mark them as "promoted" for easier access on screen. From the portal, find your desired application and select the star icon on the portal (see left), or find the Promoted apps heading on the left sidebar and use the ✔ Add option. Promoted apps appear under a dedicated section for quick access (see right).

Favorites

You can add and remove favorites with the star icon while navigating the folder structure or from within an open resource in a Palantir platform application.

Think of favorites as shortcuts that you can add and remove to keep frequently used resources close at hand.

Favorite objects, object types, and object type groups

对象浏览器 是一款用于探索平台中对象和链接的应用程序。进入单个对象视图后，点击其标题旁的星标即可将其收藏。该对象会随之显示在侧边栏中。

支持

Palantir 提供了丰富的功能、特性和组件，以满足各类业务场景和工作流需求。为帮助您充分了解平台如何契合自身需求，支持面板是获取更多信息和解答疑问的起点。通过该面板，您可以访问：

  • 向 AIP 助手提问： 向 AIP 提出问题。
  • 培训： 通过培训应用学习平台知识，并结合 Palantir 学习门户 ↗ 使用。
  • 探索示例： 从现有的用例解决方案中获取灵感。
  • 平台文档： 查阅相关文档。
  • 报告问题： 向问题应用提交平台问题。
  • 查看支持工单： 查看现有工单。
  • 平台更新： 查看平台内公告和发布说明。
  • 状态页面： 访问您 Palantir 平台的状态页面（已在启用的环境中）。
  • 关于： 查看许可证信息。

Palantir 客户成功服务团队 ↗ 提供支持、培训和咨询服务，助力您及所在团队充分发挥平台价值。若您的协议包含相关服务，可以使用 报告问题 选项在 问题 应用中创建支持工单，联系该团队解决问题。

查阅有关获取帮助的文档。

账户

在账户面板中选择 账户 可调出账户概览。在这里，点击您的姓名可以查看 用户 ID 和权限 用户组；您还可以前往 设置 修改用户和平台配置、更改平台语言或退出登录等。

您可以上传头像并补充额外信息，完善个人资料。这些信息将保存在平台上，供组织内的其他用户查看。当其他用户在项目或资源处悬停于您的姓名上时，便会看到这些资料。添加这些区分度信息对于那些依赖任务分配或资源共享的业务流程很有帮助，尤其是当您与同事同名时。

在 设置 中，通知 部分是修改通知推送设置的控制中心。

其他工作区

主平台工作区由我们一直在探索的工作区侧边栏定义。您可以通过选择 打开其他工作区 来访问其余工作区。

控制面板

管理员可以访问 控制面板 工作区来管理注册租户。有关更多资源，请查阅 管理文档。

Carbon 工作区

应用构建者可以使用 Carbon 为最终用户创建精选的定制工作区。通过组合多种应用，可以提供量身定制的体验。
