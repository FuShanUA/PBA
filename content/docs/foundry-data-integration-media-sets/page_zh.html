媒体集（非结构化数据）

媒体集是具备通用模式（例如相同格式）的媒体文件集合。媒体集专为处理海量非结构化数据而设计，支持处理音频、图像、视频和文档等媒体项。借助媒体集，用户可灵活访问存储资源、优化计算性能，并执行特定于模式的转换操作，从而提升媒体工作流与处理管线的能力。

媒体集的常见工作流包括：

  • 通过 Pipeline Builder 从 PDF 中提取文本，实现内容分析
  • 在 Map 应用中利用栅格切片（TIFF、NITF）进行地理空间分析
  • 使用 Pipeline Builder 处理医学影像文件（DICOM 格式）

如需进一步了解媒体集以及如何在 Foundry 中处理媒体内容，请访问《媒体集文档》。
