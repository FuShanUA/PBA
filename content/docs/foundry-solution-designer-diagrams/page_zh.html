图表

本页面说明如何在 Solution Designer 中与图表进行交互，包括添加节点、创建连接、组织图表以及自定义视图。

向图表添加节点

Solution Designer 提供了多种方式，用于向图表添加代表平台组件、资源或抽象概念的节点。

添加组件节点

组件节点代表特定的 Palantir 平台应用和工具，例如 Pipeline Builder、Workshop 或 Contour。

添加组件节点的步骤：

  1. 在图表工具栏中选择组件按钮，或按下 Cmd/Ctrl + B。
  2. 组件选择对话框随即打开，按分类显示所有可用的平台组件。
  3. 在左侧浏览分类，或使用顶部的搜索栏查找特定组件。
  4. 从列表中选择组件，将其添加到画布中。

添加组件节点后，请配置其他详细信息，这些信息将呈现在图表上，并为用户提供有关该节点在所描绘工作流中用途的更多信息。

添加资源节点

资源节点代表与图表关联的特定 Foundry 资源，例如数据集、对象类型、函数或操作。

添加资源节点的步骤：

  1. 在图表工具栏中选择资源按钮。
  2. 下拉菜单将显示四个选项：
    • 新建文件节点：浏览要添加的 Foundry 资源
    • 新建对象节点：从本体中选择对象类型
    • 新建接口节点：从本体中选择接口
    • 新建操作节点：从本体中选择操作类型
  3. 在弹出的对话框中搜索并选择资源，Solution Designer 会将该节点添加到画布中，并附带指向所选资源的链接。

资源节点会自动显示资源名称，并提供链接以便在其原生应用中打开该资源。

添加概念节点

概念节点代表抽象的高层工作流概念（如“数据获取”或“分析”），后续可将其替换为具体的实现。这些节点主要与 AIP Architect 配合使用，用于规划工作流。

添加概念节点时，在图表工具栏中选择概念按钮，然后选择概念类型，例如分析或数据处理。

可以使用 AIP Architect 将抽象概念节点转换为具体的实现模式。

添加分组

分组功能可将相关节点组织到逻辑容器中。

添加空分组时，在图表工具栏中选择分组按钮，然后将节点拖入分组即可将其添加为成员。

也可以通过选中多个节点并使用分组操作，从现有节点创建分组。了解有关从现有节点创建分组的更多信息。。

添加文本节点

文本节点可用于直接在图表上添加注释、说明或文档。

添加文本节点时，在图表工具栏中选择添加文本节点按钮，然后在画布上选择该文本节点以编辑其内容。

配置节点

将节点添加到图表后，可以配置其属性并将其链接到 Foundry 资源。

节点工具栏

选中节点后，其正上方会出现一个小型工具栏。该工具栏是与节点进行交互和配置的主要方式。工具栏包含多个水平排列的按钮：

  • 主操作按钮：对于已关联资源的节点，该按钮将在其原生应用（如 Pipeline Builder 或 Workshop）中打开资源；对于未关联资源的节点，该按钮允许你关联资源。
  • 关联资源：通过 RID 将 Foundry 资源与此节点关联。选择 关联资源 打开资源选择器对话框，然后选择数据集、流水线、应用或其他 Foundry 资源。关联后，节点将显示资源名称，并提供直达该资源的导航链接。
  • 操作菜单：选择 ... 访问更多节点操作：
    • 重叠顺序：调整重叠节点的图层叠放方式。
    • 分组：以所选节点为起始创建新分组。
    • 编辑背景颜色：自定义节点的背景颜色。
    • 移除关联资源：取消 Foundry 资源与该节点的关联。
    • 编辑标签：添加或移除标签，按标记、组织或团队对节点进行分类。
    • 删除：将节点从图中移除。

自定义节点外观

你可以自定义节点在图中的显示方式：

  • 背景颜色：右键点击节点并选择 更改背景颜色，或使用节点工具栏中的 操作 菜单。
  • 图标：更改自定义节点的图标及其颜色。
  • 自定义标题：在节点处于选中状态时直接点击标题即可进行编辑。
  • 调整节点大小：将鼠标悬停在节点的边角或边缘，拖动调整控制柄即可改变其大小。

节点文档卡片

每个节点的右上角都会显示一张文档卡片，展示其所代表的平台组件信息。卡片包含以下内容：

  • 组件名称与图标：用于标识该平台组件。
  • 简短描述：概述组件的功能。
  • 完整描述：可展开查看有关组件能力和用例的更多详情，点击 显示更多 即可查看。
  • 文档链接：在新标签页中打开该组件的官方 Foundry 文档。
  • 询问 Assist：如果启用了 AIP，将打开一个对话框，你可以向 AIP Assist 询问有关该组件的问题。

若要最小化文档卡片，请点击卡片右上角的最小化按钮。此时会保留一个小的信息图标，点击该图标即可重新展开卡片。

创建连接

节点之间的连接（即边）代表架构中的数据流、依赖关系或关联。

连接节点

创建连接的步骤如下：

  1. 将鼠标悬停在要作为起点的节点上，左右两侧边缘会显示出圆形控制柄。
  2. 按住控制柄，从源节点的 右侧控制柄（输出端）开始拖拽。
  3. 将光标拖至目标节点的 左侧控制柄（输入端）。
  4. 松开鼠标按钮即可建立连接。

编辑或删除连接

删除连接的步骤如下：

  1. 在画布上点击连接线，选中后连接线会高亮显示。
  2. 按下 Backspace 或 Delete 键。

你也可以右键点击连接，然后从上下文菜单中选择 删除边。

连接标签

你可以为连接添加标签，用以描述节点间的关联关系或数据流向。

添加或编辑连接标签的步骤如下：

  1. 在画布上选中该连接。
  2. 点击连接线上出现的标签区域。
  3. 在 点击编辑 文本输入框中输入标签内容。
  4. 点击画布上标签以外的任意区域以应用更改，随后选择 保存。

你可以使用画布工具栏统一切换所有边标签的显示状态。

节点分组

对相关节点进行分组有助于梳理复杂的图面结构，并为架构中的不同组成部分划定清晰的逻辑边界。

从现有节点创建组

若要对现有节点进行分组：

  1. 按住 Cmd/Ctrl 依次点选多个节点，或者按住 Shift 拖动鼠标框选节点。
  2. 选中节点后，点击图表工具栏中的 组 按钮，或者右键点击并选择 组。

Solution Designer 会将选中的节点归入同一个容器中。

管理组成员关系

创建组后，请按照以下操作管理组成员关系。

  • 添加节点： 从画布上选中任意节点并拖动到组内。当节点进入组边界内部时，松开鼠标即可将其添加为组成员。
  • 移除节点： 选中组内的节点并将其拖出组边界。该节点会从组中移除，但仍保留在画布上。
  • 移动组： 选中并拖动组容器本身。所有成员节点将随之一同移动，并保持相对位置不变。

画布控件

画布工具栏位于画布左下角，提供用于导航和操作图表的工具。该工具栏包含多个垂直排列的图标按钮：

  • 放大： 提高缩放比例以查看更多细节。
  • 缩小： 降低缩放比例以查看图表全貌。
  • 适应所有节点： 自动居中并缩放视图，以展示整个图表。
  • 锁定图表： 开启只读模式，防止意外修改。锁定状态下仍可平移和缩放。在演示时锁定图表非常实用。
  • 隐藏/显示背景点： 切换背景点阵网格的显示状态。
  • 自动布局节点： 自动将节点排列为间距均匀且边交叉最少的网格状结构。适合用于整理杂乱的图表。
  • 切换边流动动画： 为画布上的边流动标签添加动画效果。
  • 隐藏/显示边标签： 隐藏或显示所有连接线上的标签。
  • 隐藏/显示边有效性： 隐藏或显示边有效性的着色标记。
  • 最小化/最大化节点显示： 最小化节点上展示的信息。在大型或复杂图表中，这有助于节省空间。

小地图

小地图可以俯瞰整个图表，并显示当前的视口位置。

启用小地图的方法如下：

  1. 打开 设置 面板（参见 设置）。
  2. 将 小地图 切换为开启状态。
  3. 小地图将显示在画布右下角。

点击小地图可跳转至图表的不同区域。

平移和缩放

在画布上导航的方式如下：

  • 平移： 在画布空白处点选并拖动，或者在设置中开启 平移模式，即可在不误选节点的情况下进行平移。
  • 缩放： 使用鼠标滚轮、触控板捏合手势，或点击画布工具栏中的缩放按钮。
  • 适应视图： 点击 适应所有节点 按钮，可自动居中并缩放以展示所有节点。

高级节点操作

侧边菜单

在画布上选中某个节点后，屏幕右侧会弹出一个侧边菜单面板，提供该节点的更多操作选项。

节点遍历

对于所有节点，侧边菜单均包含一个 节点遍历 区域，支持通过连接的节点浏览图表：

  • 查看连接的节点： 查看与当前选中节点相连的所有节点列表。
  • 跳转至连接的节点： 在列表中选择任意相连的节点，即可将其选中并居中显示在画布上。
  • 理解数据流向： 顺着节点连接关系，快速追踪数据在架构中的流转路径。

节点遍历功能可帮助您在复杂的图表中导航，无需手动搜索即可理解组件之间的关系。

对象遍历

对于链接到本体对象类型的节点，侧边菜单包含一个对象遍历部分，显示该对象类型可用的本体关系：

  • 查看关系：查看本体中定义的所有链接类型和相关的对象类型。
  • 创建遍历：选择特定关系，自动在画布上生成相关对象类型节点，此操作模拟了节点工具栏中创建遍历操作的功能。
  • 探索本体结构：了解该对象类型如何连接到数据模型的其他部分。

上下文菜单

虽然画布和节点工具栏是与图表交互的主要方式，但您也可以使用上下文菜单作为替代方案：

  • 右键点击节点：显示诸如更改背景颜色、与其他选中节点编组以及删除等选项。
  • 右键点击画布：显示诸如添加评论、添加空组以及自动布局节点等选项。

上下文菜单提供了对常用操作的快速访问，但通过专用工具栏执行大多数操作会更加便捷。
