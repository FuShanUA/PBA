Quiver 控制板

在记事本文档中嵌入 Quiver 控制板，允许您将控制板导出为 PDF 或打印它。 嵌入控制板还允许您锁定某一时间点的数据。

您可以通过点击 + 小部件 在记事本文档中嵌入一个 Quiver 控制板。

小部件属性

  • 控制板：选择要嵌入的控制板。
  • 版本：选择控制板的版本，或者切换 自动更新 以始终显示最新版本。

模板配置

  • 仪表板输入: 如果您的仪表板有定义输入，这些输入将在此部分显示。您可以使用下面的映射表将输入映射到Notepad文档中的变量。
Quiver 输入类型Notepad 输入类型

时间时间戳
时间范围字符串, ISO format - ISO format
时间序列不支持
