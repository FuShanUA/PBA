# 添加内容 ## 在报告中添加文本 ### 如果报告为空 1. （如需）切换至编辑模式。 2. （如需）在报告主体中，点击“开始输入，或”消息。 3. 开始输入。 ### 如果报告中已有内容 1. （如需）切换至编辑模式。 2. 将光标移动到任何小部件或小部件行的上方或下方。 3. 点击悬停时出现的 `+` 图标（小部件创建器）元素。 4. 开始输入。 ## 在报告中编辑文本 在编辑模式下，可以自由设置报告文本的格式。具体而言，可以使用 **块级样式** 和 **内联样式** 来设置文本样式；前者影响整个文本块的格式，后者影响特定字词或文本片段的格式。 ### 内联样式 使用内联样式可以强调较大文本块中的片段。 #### 添加内联样式 应用内联样式有两种方式：使用键盘快捷键，或者选中文本后点击格式工具栏中的相应按钮。 报告支持以下内联样式：
样式键盘快捷键按钮示例
加粗Cmd + B
斜体Cmd + I
下划线Cmd + U
删除线Cmd + Shift + X
代码片段Cmd + Shift + K
链接Cmd + K
高亮无
#### 移除内联样式 要移除特定的内联样式，请点击工具栏中对应的按钮，或再次使用相应的键盘快捷键。 ### 块级样式 块级样式影响整个文本块的格式，例如段落、标题、列表、引用块或代码块。 #### 段落 默认情况下，文本块会被格式化为段落。如果后面还有其他段落，当前段落除了具有底部边距外，没有其他特殊样式。要将文本块设置或重设为段落样式： 1. 选中部分文本。 2. 点击内联工具栏中的块级样式按钮。 3. 在弹出的菜单中点击 **段落**。 #### 标题 在报告中最多可以添加六级标题。标题能为报告增添视觉结构，它们也会显示在报告 **大纲** 中并对内容进行缩进。要创建标题： 1. 在行首输入以下符号之一： - `#` 用于一级标题（最显著） - `##` 用于二级标题 - `###` 用于三级标题 - `####` 用于四级标题 - `######` 用于五级标题 - `#######` 用于六级标题（最不显著） 2. 按下空格键。 或者 1. 选中部分文本。 2. Click the block-style button in the inline toolbar.
  • Click Header one or Header two in the menu that appears.

    Lists

    Numbered List

    To add an ordered/numbered list to a report:

    1. Type 1. at the beginning of a line.
    2. Use the Spacebar key.

    or

    1. Select some text.
    2. Click the block-style button in the inline toolbar.
    3. Click Numbered list in the menu that appears.

    Numbered lists will always start at 1.

    Bulleted List

    To add a bulleted (or “unordered”) list to a report:

    1. Type - or * at the beginning of a line.
    2. Use the Spacebar key.

    or

    1. Select some text.
    2. Click the block-style button in the inline toolbar.
    3. Click Bulleted list in the menu that appears.

    Bulleted lists support up to 11 levels of indentation:

    • To indent a bullet, use the Tab key.
    • To de-indent a bullet, use the Shift+Tab keyboard shortcut.
    Checklist

    Use a checklist to track to-do's, follow-ups, and other action items. To create a checklist:

    1. Type [] at the beginning of a line.
    2. Use the Spacebar key.

    or

    1. Select some text.
    2. Click the block-style button in the inline toolbar.
    3. Click Checklist in the menu that appears.

    Blockquotes

    Use a blockquote to present quotations, excerpts, and other callouts. To create a blockquote:

    1. Type > at the beginning of a line.
    2. Use the Spacebar key.

    or

    1. Select some text.
    2. Click the block-style button in the inline toolbar.
    3. Click Blockquote in the menu that appears.

    Code block

    Use a code block to present multiple lines of code or other fixed-width text. To create a code block:

    1. Type ``` (three back-ticks) at the beginning of a line.
    2. Use the Enter key.

    or

    1. Select some text.
    2. Click the block-style button in the inline toolbar.
    3. Click Code block in the menu that appears.

    Other styling options

    Horizontal line

    To insert a horizontal line (also known as a horizontal rule or horizontal divider) into a report:

    1. Type --- on a blank line.
    2. Use the Enter key.

    Inline parameters

    You may wish to display the current value of a parameter within a block of text (learn more about parameters). To do this:

    1. Select some text.
    2. Click the Parameter button in the formatting toolbar to create a new string parameter.
    3. Merge the new parameter into the desired Contour parameter (learn more about merging parameters).

    The current value of the merged parameter will now appear inline when in Viewing mode (in Editing mode, only the name of the parameter will appear inline).

    Add a table to a report

    Reports supports several different types of tables. Table widgets are the most basic: you can create them inside of a report to format manually inputted information into a simple row-and-column layout.

    Table widgets are different from data-backed tables from other Palantir Foundry applications. For instance, a report can also contain:

    • Contour Table boards to see a preview of a particular dataset, or
    • Fusion spreadsheets to show richly formatted tables optionally backed by a dataset.

    Steps

    You can add a Table widget to a report at any time. Once you've added a table, you can type freely into its cells and also change its size (for details, see Resize a table in a report).

    If the report is empty

    1. (If needed) Switch to Editing mode.

    2. Click the table icon next to the “Start typing, or” message.



    3. Drag out the initial number of rows and columns the table should have.



    4. Click to create the table.



    If the report already has content

    1. (If needed) Switch to Editing mode.

    2. Move your cursor above or below any row of widgets, and click the Widget Creator element that appears on hover.



    3. Click the table icon next to the “Start typing, or” message.



    4. Drag out the initial number of rows and columns the table should have.



    5. Click to confirm.



    Resize a table in a report

    After adding a Table widget, you can easily add or remove rows and columns within it.

    Table widgets always have a single header row. You can insert rows above any row except the header row.

    Insert rows and columns

    Insert a single row or column

    Rows and columns can be inserted below or to the right of the currently focused cell.

    1. (If needed) Switch to Editing mode.

    2. (If needed) Click a cell to clear the current selection.

    3. Right-click the cell.

    4. Click any of the available insert options to insert a row or column where appropriate:

      • Insert column left
      • Insert column right
      • Insert row above
      • Insert row below

    Insert multiple rows or columns

    You can insert multiple rows or columns by selecting a larger region first. In particular, if there are N rows and M columns in your current selection, then you can insert N rows or M columns at once.

    1. (If needed) Switch to Editing mode.

    2. Select a region of N rows and/or M columns.

    3. Right-click any cell in the selected region.

    4. Click any of the available insert options to insert a row or column where appropriate:

      • Insert M columns left
      • Insert M columns right
      • Insert N rows above
      • Insert N rows below

    Delete rows and columns

    Delete a single row or column

    1. (If needed) Switch to Editing mode.
    2. (If needed) Click a cell to clear the current selection.
    3. Right-click the cell.
    4. Click Delete row or Delete column.

    Delete multiple rows or columns

    你可以先选中更大的区域，以此来一次性删除多行或多列。具体而言，如果当前选区包含 N 行和 M 列，你就可以同时删除 N 行或 M 列。

    表格控件始终只有一个表头行。表头行无法删除。

    1. （如需）切换至编辑模式。
    2. 选中 N 行和/或 M 列的区域。
    3. 右键点击所选区域内的任意单元格。
    4. 点击删除 N 行或删除 M 列。

    更改行和列的尺寸

    更改行高

    目前不支持更改行高。表格控件在每个单元格中仅显示单行文本。

    更改列宽

    1. （如需）切换至编辑模式。
    2. 将鼠标光标移至列表头单元格（即该列最顶部的单元格）的右边缘。
    3. 点击并拖动以调整列宽。
    4. 松开鼠标按钮以保存设置。

    向报告中添加新图片或视频

    向报告中添加图片、视频及其他资源共有三种方式：

    1. 直接将媒体上传至报告。媒体将存储在报告资源中，显示时不含任何额外的背景、页脚或边框。仅图片和视频可通过此方式上传。
    2. 链接至文件目录中的资源。可将 Foundry 中已有的资源添加到报告中，显示时会带有指向原始资源的页脚链接。有关此方法的更多信息，请参阅下文的《在报告中链接至 Foundry 资源》。
    3. 将资源上传至报告内的 Foundry 中，然后建立链接。这是一种向报告添加媒体的陈旧过时方式。其功能与链接至资源完全相同，但已被上文的1所取代。

    上传至报告

    你可以将图片和视频直接上传至报告。采用此方式时，图片和视频将与报告“打包”在一起，因此它们不会在 Foundry 中作为独立文件出现。

    如需直接将图片或视频上传至报告：

    1. （如需）切换至编辑模式。

    2. 点击“开始输入，或”提示信息旁边的图片图标。



    3. 点击拖放区选择文件，或将计算机中的文件拖入拖放区。



    上传至 Foundry 并从报告建立链接 [已弃用]

    此方法已被弃用，建议使用上文《上传至报告》中所述的方法。

    你也可以从报告中将图片和视频作为独立资源上传至 Foundry，然后建立指向该资源的链接（如下文《在报告中链接至 Foundry 资源》所述）。

    操作步骤如下：

    1. （如需）切换至编辑模式。

    2. 点击“开始输入，或”提示信息旁边的上传图标。



    3. 在弹出的对话框中选择要上传的文件。



    4. 点击下一步将图片上传至 Foundry。随后，该资源应会自动显示在报告中（若未显示，可按照《在报告中链接至 Foundry 资源》的说明建立链接）。



    支持的文件类型

    图片

    支持直接上传的图片文件类型如下：

    • BMP
    • GIF
    • JPG/JPEG
    • PNG

    视频

    支持直接上传的视频文件类型如下：

    • MOV
    • MP4

    在报告中链接 Foundry 资源

    您可以在 Foundry 的文件目录中链接图片、视频及其他类型的文件。链接的图片和视频将直接显示在报告中，而数据集、PDF 或 DOCX 文档等其他资源类型则会以简略形式呈现（包含可点击的资源标题及对应的资源图标）。

    复制报告内的内容

    您可以复制报告中包含的任何组件（例如文本块、图片或 Contour 表格）。

    若要复制组件：

    1. （如需）切换至编辑模式。
    2. 点击您想要复制的组件右上角的齿轮图标。



    复制后的组件将直接添加至原组件的下方。

    从其他报告添加内容

    只要您对两份报告均具备编辑权限，即可将任何组件（例如文本块、图片、Contour 表格）从一份报告复制到另一份报告中。

    若要将组件从一份报告复制到另一份报告：

    1. （如需）切换至编辑模式。

    2. 点击您想要复制的组件右上角的齿轮图标。

    3. 在弹出的菜单中点击添加到其他报告。



    4. 在弹出的菜单中选择一份报告。



      当组件成功复制到您所选的报告后，您将看到一条确认提示。



    被复制的组件将被添加至另一份报告的末尾。
