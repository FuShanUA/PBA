快速入门

本指南演示了如何访问 AIP Chatbot Studio，介绍了 AIP Chatbot Studio 的界面，说明了如何设置一个配备了您所选信息和工具的基础 AIP Chatbot，以及如何在生产环境中部署和监控该 AIP Chatbot。

访问 AIP Chatbot Studio

可通过平台工作区的导航栏或使用快速搜索快捷键 CMD + J (macOS) 或 CTRL + J (Windows) 访问 AIP Chatbot Studio。或者，您也可以从文件中创建新的 AIP Chatbot，只需选择 + 新建，然后选择 AIP Chatbot，如下所示。

打开 AIP Chatbot Studio 后，您可以创建新的 AIP Chatbot 文件。

创建 AIP Chatbot

AIP Chatbot 属于 Palantir 文件系统资源，具备精细的访问控制，可以像创建其他任何文件系统资源一样进行创建，如上一节上图所示。

您也可以在 AIP Chatbot Studio 中选择 新建 AIP Chatbot 选项。

或者，在 AIP Threads 中创建 AIP Chatbot。

设置 AIP Chatbot

为您的 AIP Chatbot 添加名称、描述以及用作头像的照片。这使您能够对聊天机器人进行白标化处理，以适配您的应用场景。如果未提供头像，系统将默认使用灰色的机器人图标。

接下来，您需要为 AIP Chatbot 配置企业特定的信息和工具，具体内容将在以下章节详述。

信息与工具类型

  • 检索上下文：简单且快速，适用于大多数用例。
  • 应用状态：用于在 Workshop 中为聊天机器人提供上下文。
  • 工具：用于需要执行复杂动作的聊天机器人。

正是这些配置使得大语言模型能够服务于您的企业、工作流和具体任务。

选择大语言模型 (LLM)

您可用的模型是在您的环境中启用的模型的子集。

修改系统提示词

系统提示词应概述 AIP Chatbot 在当前应用上下文中的功能。按下键盘上的 / 键，您可以引用已配置的工具和应用状态，并指导 AIP Chatbot 如何协调使用它们。请务必描述底层的业务逻辑，以及在上下文中使用正确工具的适当时机。

设置温度

用户可以修改模型温度，以在专注的、确定性的输出（默认值为 0）和随机性输出（最大值为 1）之间取得平衡。

添加对话引导语

您还可以设置输入占位符和建议提示词，根据您预期的工作流对聊天机器人进行定制。

保存、查看并发布 AIP 聊天机器人

完成 AIP 聊天机器人的配置后，可通过界面右上角的保存按钮保存进度。点击保存选项旁的下拉箭头图标，即可为保存的版本添加描述。

若要从终端用户交互视角查看 AIP 聊天机器人的实际运行效果，请使用查看功能并选择相应版本。

准备好部署 AIP 聊天机器人时，选择发布即可将其投入生产环境使用。点击发布选项旁的配置图标，还可将聊天机器人作为函数发布。这样便能通过 AIP Automate 和 AIP Evals 运行该聊天机器人。

跟踪 AIP 聊天机器人的反馈与使用情况

通过监控和使用情况选项卡，可监视聊天机器人的性能表现及使用状况，查看相关指标和用户反馈。用户在对话中给聊天机器人点赞或点踩，系统便会接收到反馈数据。

可在 AIP Threads、Workshop、查看模式，或结合开发者控制台和平台 API 的 OSDK 中使用。
