Microsoft SharePoint Excel

Microsoft SharePoint Excel 连接器是 Palantir 提供的驱动程序，用于 Microsoft SharePoint Excel。

要创建一个新的 Microsoft SharePoint Excel 源，请按照 Palantir 提供的驱动程序的标准设置流程，然后使用以下部分进行 Microsoft SharePoint Excel 特定的配置和网络设置。有关完整的属性参考，请参阅 官方 Microsoft SharePoint Excel 驱动程序文档 ↗。

配置

以下属性为必填或建议填写。

属性是否必需?描述默认值
AuthScheme ↗必填用于身份验证的方案。接受的条目包括 NTLM、Basic、Digest、Forms、None、Negotiate、ADFS。NTLM
File ↗必填要连接的 Excel 文件的名称。Book1.xlsx
SharePointVersion ↗必填您要连接的 SharePoint 服务器的版本。SharePoint 2013
URL ↗必填站点或站点集合的基础 URL。https://myorg.sharepoint.com
Folder ↗推荐包含由 File 属性指定的工作簿的文件夹。—
Library ↗推荐要连接的文档库。—
Password ↗推荐用于对服务器进行身份验证的账户的密码。—
User ↗推荐用于对服务器进行身份验证的账户的用户名。—
UseRESTAPI ↗推荐是否使用 REST API 来检索数据。TRUE

网络

下表列出了源需要能够访问的域名，以便成功运行。

对于每个领域，添加相应的 出站策略。如果源在本地托管且无法直接从 Foundry 访问，则应使用 代理程序出站策略；代理主机本身也必须能够访问列出的领域。有关详细信息，请参阅 使用代理作为代理。
