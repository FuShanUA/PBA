制品仓库

制品仓库允许用户发布并管理制品，包括 Conda ↗、Docker ↗ 和 Maven ↗。

所有未编写为库或无法通过外部 URL 访问的 Conda、Docker 或 Maven 制品，均应通过制品仓库进行上传。例如，您可能已在本地计算机上编写了一个 Conda 包，并希望在代码仓库中访问它。只需将该 Conda 包发布至制品仓库，您便可在代码仓库的库搜索面板中访问它。

制品仓库的核心功能包括：

  • 发布制品：生成令牌并将制品推送到制品仓库。
  • 搜索制品：从制品仓库界面查找制品。
  • 撤回 Conda 制品：撤回 Conda 制品，以防止下游使用方使用特定版本编译代码。

如需了解更多关于制品仓库界面的信息，以及如何创建制品仓库，请参阅相关文档。
