自定义函数

当工作流需要富文本格式的输入，或者工作流输出富文本格式的内容时，自定义函数会特别有用。函数的输入可以是富文本或纯文本，你也可以自行选择输出应为富文本还是纯文本。需要注意的是，如果需要富文本输入，你的函数应当能够接受 Markdown 格式。

此功能兼容接受单个字符串作为输入、且输出类型为字符串、布尔值、日期、双精度浮点数、单精度浮点数、整数或长整数的函数。

在 Notepad 中使用自定义函数

要在 Notepad 中使用自定义函数，必须先创建一个函数。函数可以使用 TypeScript 或 Python 编写；有关用例、支持的功能以及创建函数的指南，请参阅《函数文档》。创建自定义函数后，请按照以下步骤在 Notepad 中使用它：

  1. 在 Notepad 中，打开或创建一个文档或文档模板。在文档中填充内容，并通过以下两种方式之一选择函数输入：
    • 手动高亮选中相关内容，然后在工具栏中选择 使用 AIP 编辑。
    • 将光标置于相关内容上，然后在工具栏中选择 使用 AIP 编辑。系统会自动为你进行块级选择。

如果你的租户未启用 AIP，Notepad 中的自定义函数依然可用。请在工具栏中选择 使用函数编辑，而不是 使用 AIP 编辑，并忽略第二步。

  1. 完成选择后，会弹出一个菜单。选择 函数 选项，即可使用自定义函数对所选内容进行编辑。
  2. 在函数弹窗中，选择一个函数以及该函数的输入类型。在 函数输入 下拉菜单中，选择纯文本或 Markdown 格式。
  1. 可选择预览输入内容，然后运行该函数。
  2. 查看函数输出，并选择输出应为纯文本还是使用 Markdown 格式化。默认情况下，输出类型与输入类型保持一致。
  1. 选择 替换，将输出内容插入到你的文档中。
