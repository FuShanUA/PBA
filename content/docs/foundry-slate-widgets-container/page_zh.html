容器

容器小部件类别包括以下小部件：

  • 基本
  • 对话框
  • 弹性布局
  • 重复
  • 水平分割
  • 垂直分割
  • 标签页

下表提供了有关容器小部件可用属性的使用详情。表格后跟随几个示例。

属性

属性描述类型必需更改者
flex指定容器是否应自动对齐并分配子小部件之间的空间。布尔值（boolean）是直接编辑
flexOptions配置子小部件对齐方式的选项。布尔值（boolean）否 直接编辑
lazyRenderingEnabled指定是否仅渲染可见的子控件。布尔值（boolean）是直接编辑
selectedTabIndex当前显示选项卡的索引。数字（number）是直接编辑
tabContents一个内部数组，用于存储每个选项卡中的控件。{children: IWidgetModel[]}[]是用户交互
tabTitles每个选项卡的标题。ITabTitle[]是直接编辑
titlesEnabled指定是否在容器顶部显示可点击的选项卡标题。布尔值（boolean）是直接编辑
scrollingEnabled指定是否允许滚动溢出的内容。布尔值（boolean）是直接编辑
splitDirection指定容器被分割的方向。数字（number）是直接编辑
splitMovableInConsumerMode指定在消费者模式下分割容器是否可调整大小布尔值（boolean） 是直接编辑
splitPanel一个内部值，用于说明容器是否是拆分容器中的面板。boolean（布尔）是用户交互
repeat指定内容重复的次数number（数字）否直接编辑
repeating指定容器是否为重复容器。boolean（布尔）是直接编辑
previewRepeating指定是否预览重复容器。boolean（布尔）否直接编辑

ITabTitle

属性描述类型必需更改者
title（标题）此选项卡的标题。string（字符串）是直接编辑

IFlexOptions

属性（Attribute）描述（Description）类型（Type）是否必需（Required）修改者（Changed By）
justifyContent指定子小部件在主轴上的布局方式。默认值为水平排列容器的Left，垂直排列容器的Top。字符串（string）是直接编辑（Direct Edit）
flexDirection指定子小部件对齐的主轴。默认值为水平方向。字符串（string）是直接编辑（Direct Edit）
flexWrap指定子小部件是否可以溢出到多行。默认值为未选中。字符串（string）是直接编辑（Direct Edit）

使用方法

Drag widgets into and out of containers to add and remove them from the currently selected tab. The tabContents property is unsupposed in the raw tab and should not be edited manually, nor will it respond to changes from user interaction.

Properties

AttributeDescriptionTypeRequiredChanged By

canEscapeKeyCloseCloses the dialog when the Escape key is used.booleanYesDirect Edit

Actions

Action NameDescription

Event NameDescription

Common CSS

You can use the below CSS to set the background color and borders of a container:

Copied!
1sl-app-container {
2  background: lightgrey;
3  border-left:none;
4  border-right:none;
5}

Though the sl-app-container selector and background property will correctly set the background color and borders of a container widget, use the div.canvas-body selector and background-color property in your local stylesheet to adjust the colors and borders of the Slate application canvas. In the example below, we customized the Slate canvas, container widget, and bar chart colors using the Styles editor:

Tabbed containers

Tabs styling:

Copied!
1sl-app-container table.container-titles tbody tr td.tab-title {
2  color: #2786c1;
3  text-transform:uppercase;
4  border-left:none;
5  border-right:none;
6}

Hover styling:
