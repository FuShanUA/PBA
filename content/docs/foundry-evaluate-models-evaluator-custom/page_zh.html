自定义评估库

评估库是一个在Foundry中发布的Python包，用于生成模型评估器。评估库可以以可重用的方式测量不同< a href="/docs/foundry/model-integration/objectives/">建模目标中的模型性能、模型公平性、模型鲁棒性和其他指标。

除了Foundry针对二分类和回归模型的默认模型评估器之外，Foundry还允许您创建一个自定义模型评估器，该评估器可以在建模目标中原生使用。

建模目标内的自定义评估器

自定义评估器、其配置选项以及生成的指标将按照评估器实现顶部文档字符串中指定的名称和描述，在建模目标应用程序中显示。

一旦自定义评估器被发布，任何具有查看已发布库访问权限的用户都可以在建模目标应用程序中使用它。这使您能够为组织内的标准化指标计算编写可重用逻辑。

自定义评估器可在建模目标的评估库配置中选择；该库可根据评估器定义的参数进行配置。

创建自定义评估器

要创建自定义评估器：

  1. 创建一个代码仓库从Model Evaluator Template Library。
  2. 实现您的自定义评估器。
  3. 添加参数到您的自定义评估器。
  4. 提交并 发布一个新的标签 带上您的更改。

创建代码仓库

代码仓库应用有许多模板实现；在这里，我们将使用模型评估器模板库。导航到一个Foundry项目，选择+ 新建 > 仓库类型 > 模型集成 > 语言模板，选择模型评估器模板库，最后选择初始化仓库。

评估器模板结构

模型评估器模板库在文件src/evaluator/custom_evaluator.py中有一个示例实现。任何对Evaluator Python 接口的实现都会在您通过发布该仓库的新版本并使用新的仓库版本标签时自动注册并可用。

包含自定义评估逻辑的代码仓库可以发布多个评估器。任何其他评估器实现文件都需要作为模型评估器模块列表中的引用添加到评估器模板的 build.gradle 中。

实现自定义评估器

要实现自定义评估器，您需要创建 Evaluator 接口的实现，并且可以选择性地提供配置字段以供 Modeling Objectives 应用程序进行解释。

在评估器模板库中，将您的评估器添加到文件 src/evaluator/custom_evaluator.py 中。

要在 Modeling Objectives 应用程序中使用你新配置的自定义评估器，你首先需要 发布该仓库的新版本，并为其提供一个新的仓库版本标签。

评估器文档

自定义评估器及其配置选项和产生的指标将在 Modeling Objectives 应用程序中以实现代码顶部的 docstring 中指定的名称和描述显示。

所需值如下：

  • display-name: 评估器的显示名称
  • description: 评估器的描述

你可以选择性地添加以下内容中的零个或多个：

  • param: 你的自定义评估器的配置参数
  • metric: 由你的评估器生成的指标

示例评估器实现

这是一个示例评估器，用于计算输入数据集的行数。

此示例评估器将在建模目标应用程序中显示为：

  • 标题 Row Count Evaluator。
  • 描述 This evaluator calculates the row count of the input DataFrame.。
  • 生成的指标 Row Count，其描述为 The row count。
  • 零个配置参数。
Copied!
1from pyspark.sql import DataFrame
2from pyspark.sql import functions as F
3from foundry_ml_metrics.evaluation import ComputedMetricValue, Evaluator
4
5class CustomEvaluator(Evaluator):
6    """
7        :display-name: Row Count Evaluator
8        :description: This evaluator calculates the row count of the input DataFrame.
9
10        :metric Row Count: The row count
11    """
12
13    def apply_spark(self, df: DataFrame) -> List[ComputedMetricValue]:
14        row_count = df.count()
15
16        return [
17            ComputedMetricValue(
18                metric_name='Row Count',
19                metric_value=row_count
20            )
21        ]

参数化一个评估器

通过提供配置参数，可以在建模目标应用程序中使评估器可配置。 配置参数将在运行时由建模目标应用程序使用用户输入的值进行填充。 该评估器的用户在配置建模目标中的自动评估时，将有机会配置这些参数的值。

允许的配置字段是：

  • int: 一个整数
  • float: 一个浮点数（浮点）
  • bool: 一个布尔值（True 或 False）
  • str: 一个字符串（字符串）
  • Field[float]: 输入 DataFrame 中的一个浮点列
  • Field[int]: 输入DataFrame中的整数列
  • Field[str]: 输入DataFrame中的字符串列

可以通过将参数包裹在Optional（来自内置的typing包）中，使参数变为可选。

例如：

  • 一个可选的str将是Optional[str]
  • 一个可选的Field[str]将是Optional[Field[str]]

带有配置字段的示例评估器

这是一个示例评估器，用于计算输入数据集的行数以及输入DataFrame经过筛选后，输入列column的值为value时的行数。

这个示例评估器将在建模目标应用程序中显示为：

  • 标题Configurable Row Count Evaluator。
  • 描述This evaluator calculates the row count of the input DataFrame, filtered to the specified value.
  • 生成的一个指标Row Count，其描述为The unfiltered row count。
  • 生成的一个指标Filtered Row Count，其描述为The filtered row count
  • 两个配置参数：
    • 一个评估数据集中的列，必须是一个整数，名称为column，描述为Filtered column。
    • 一个整数值，名称为value，描述为Filtered value。
Copied!
1from pyspark.sql import DataFrame
2from pyspark.sql import functions as F
3from foundry_ml_metrics.evaluation import ComputedMetricValue, Evaluator, Field
4
5class CustomEvaluator(Evaluator):
6    """
7        :display-name: Configurable Row Count Evaluator
8        :description: This evaluator calculates the row count of the input DataFrame, filtered to the specified value.
9
10        :param column: Filtered column
11        :param value: Filtered value
12
13        :metric Row Count: The unfiltered row count
14        :metric Filtered Row Count: The filtered row count
15    """
16
17    column: Field[int]
18    value: int
19
20    def __init__(self, column: Field[int], value: int):
21        self.column = column
22        self.value = value
23
24    def apply_spark(self, df: DataFrame) -> List[ComputedMetricValue]:
25        column_name = self.column.name
26        column_value = self.value
27
28        row_count = df.count()
29
30        filtered_row_count = df.filter(
31            F.col(column_name) == column_value
32        ).count()
33
34        return [
35            ComputedMetricValue(
36                metric_name='Row Count',
37                metric_value=row_count
38            ),
39            ComputedMetricValue(
40                metric_name='Filtered Row Count',
41                metric_value=filtered_row_count
42            )
43        ]

MetricValue（指标值）

MetricValue 可以是以下任何一种：

  • 一个数值，可以是以下类型之一：
    • int（整数）
    • np.int8（np.int8）
    • np.int16（np.int16）
    • np.int32（np.int32）
    • np.int64（np.int64）
    • np.uint8（np.uint8）
    • np.uint16（np.uint16）
    • np.uint32（np.uint32）
    • np.uint64（np.uint64）
    • float（浮点）
    • np.float32（np.float32）
    • np.float64（np.float64）
  • 一个图形，可以是以下类型之一：
    • matplotlib.Figure（matplotlib.Figure）
    • matplotlib.pyplot.Figure（matplotlib.pyplot.Figure）
  • 任何实现以下方法之一的类：
    • get_figure(self) -> Figure：注意许多 seaborn 图表实现了这个函数。
    • save(self, path: str)：注意许多 seaborn 图表实现了这个函数。
    • savefig(self, path: str)
  • 一个 BarChart（柱状图）
  • 一个 LineChart（折线图）
