自定义评估库

评估库是 Foundry 中已发布的 Python 包，用于生成模型评估器。评估库可用于以可复用的方式，在不同的建模目标中衡量模型性能、模型公平性、模型鲁棒性及其他指标。

除了 Foundry 为二分类和回归模型提供的默认模型评估器外，Foundry 还允许您创建可在建模目标中原生使用的自定义模型评估器。

建模目标中的自定义评估器

自定义评估器及其配置选项和生成的指标将显示在“建模目标”应用中，其名称和描述是在评估器实现顶部的文档字符串中指定的。

自定义评估器发布后，对已发布库拥有查看权限的任何用户都可以在“建模目标”应用中使用它。这使您能够编写可复用的逻辑，用于在整个组织内计算标准化指标。

在建模目标的评估库配置中可以选择该自定义评估器；该库可根据评估器定义的参数进行配置。

创建自定义评估器

要创建自定义评估器：

  1. 从 模型评估器模板库 中创建一个代码仓库。
  2. 实现您的自定义评估器。
  3. 向您的自定义评估器添加参数。
  4. 提交并使用您的更改发布新标签。

创建代码仓库

代码仓库应用程序提供了多种模板实现；在此，我们将使用模型评估器模板库。进入 Foundry 项目，依次选择 + 新建 > 仓库类型 > 模型集成 > 语言模板，选择 模型评估器模板库，最后选择 初始化仓库。

评估器模板结构

模型评估器模板库在 src/evaluator/custom_evaluator.py 文件中提供了示例实现。当你 发布带有新版本标签的仓库新版本时，任何实现了 Evaluator Python 接口的代码都会自动完成注册并可供使用。

包含自定义评估器逻辑的仓库可以发布多个评估器。任何额外的评估器实现文件都需要作为引用，添加到评估器模板的 build.gradle 模型评估器模块列表中。

实现自定义评估器

要实现自定义评估器，你需要创建一个 Evaluator 接口的实现，并选择性地提供配置字段，供“建模目标”应用程序解析。

在评估器模板库中，将你的评估器添加到 src/evaluator/custom_evaluator.py 文件中。

评估器文档

自定义评估器及其配置选项和生成的指标，将按照实现代码顶部文档字符串中指定的名称和描述，显示在“建模目标”应用中。

必填值包括：

  • display-name：评估器的显示名称
  • description：评估器的描述

您可以选择性地添加零个或多个以下内容：

  • param：自定义评估器的配置参数
  • metric：评估器生成的指标

评估器实现示例

以下示例评估器用于计算输入数据集的行数。

该示例评估器将在“建模目标”应用中显示以下信息：

  • 标题 Row Count Evaluator。
  • 描述 This evaluator calculates the row count of the input DataFrame.。
  • 生成的指标 Row Count，其描述为 The row count。
  • 零个配置参数。
```python from pyspark.sql import DataFrame from pyspark.sql import functions as F from foundry_ml_metrics.evaluation import ComputedMetricValue, Evaluator class CustomEvaluator(Evaluator): """ :display-name: 行数评估器 :description: 此评估器计算输入 DataFrame 的行数。 :metric Row Count: 行数 """ def apply_spark(self, df: DataFrame) -> List[ComputedMetricValue]: row_count = df.count() return [ ComputedMetricValue( metric_name='Row Count', metric_value=row_count ) ] ```

参数化评估器

在“建模目标”应用中，可以通过提供配置参数使评估器变得可配置。

运行时，“建模目标”应用会将用户输入的值填充到这些配置参数中。

当用户在建模目标中配置自动化评估时，将有机会设置这些参数的值。

允许的配置字段包括：

  • int：整数
  • float：浮点数
  • bool：布尔值（True 或 False）
  • str：字符串值
  • Field[float]：输入 DataFrame 中的浮点数列
  • Field[int]：输入 DataFrame 中的整数列
  • Field[str]：输入 DataFrame 中的字符串列

可以通过将参数包裹在 Optional（来自内置的 typing 包）中，将其设为可选参数。

例如：

  • 可选的 str 表示为 Optional[str]
  • 可选的 Field[str] 表示为 Optional[Field[str]]

带配置字段的评估器示例

这是一个示例评估器，用于计算输入数据集的行数，以及输入 DataFrame 经过过滤（使输入列 column 的值为 value）后的行数。

该示例评估器将在“建模目标”应用中显示如下信息：

  • 标题为 Configurable Row Count Evaluator。
  • 描述为 This evaluator calculates the row count of the input DataFrame, filtered to the specified value.。
  • 生成的指标 Row Count，描述为 The unfiltered row count。
  • 生成的指标 Filtered Row Count，描述为 The filtered row count
  • 两个配置参数：
    • 评估数据集中必须为整数的列，名称为 column，描述为 Filtered column。
    • 整数值，名称为 value，描述为 Filtered value。
```python from pyspark.sql import DataFrame from pyspark.sql import functions as F from foundry_ml_metrics.evaluation import ComputedMetricValue, Evaluator, Field class CustomEvaluator(Evaluator): """ :display-name: 可配置行数评估器 :description: 该评估器计算输入 DataFrame 的行数，并按指定值进行过滤。 :param column: 过滤列 :param value: 过滤值 :metric Row Count: 未过滤的行数 :metric Filtered Row Count: 过滤后的行数 """ column: Field[int] value: int def __init__(self, column: Field[int], value: int): self.column = column self.value = value def apply_spark(self, df: DataFrame) -> List[ComputedMetricValue]: column_name = self.column.name column_value = self.value row_count = df.count() filtered_row_count = df.filter( F.col(column_name) == column_value ).count() return [ ComputedMetricValue( metric_name='Row Count', metric_value=row_count ), ComputedMetricValue( metric_name='Filtered Row Count', metric_value=filtered_row_count ) ] ```
