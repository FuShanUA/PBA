安装远程智能体

访问低于 SAP_BASIS 7.4 SP5 的旧版 SAP 系统

远程智能体旨在通过主要的 Palantir Foundry Connector 2.0 for SAP Applications（“连接器”）访问远程 SAP 系统。如果 SAP_BASIS（技术组件版本，适用于 SAP NetWeaver 和 SAP S/4HANA 系统）版本低于 7.4 SP05，或者您希望将系统环境中的两个或多个 SAP 系统作为单个 Foundry 源查看，则应安装远程智能体。远程智能体支持的最低 SAP_BASIS 版本为 7.00 SP32。对于更低的产品版本或 SAP_BASIS 7.00 的支持包，请参阅《安装适用于 ERP 4.6C/4.7 (620/640) 的远程智能体》。

要在远程 SAP 系统中安装远程智能体，请按照以下步骤操作：

  1. 下载安装包。
  2. 使用授权运行 SAINT 的用户登录 SAP 系统 client 000。
  3. 运行 SAINT 事务。
  4. 将 FOUNDRY-SAPCONN-INST-SP00SPXX.SAR 导入 SAP 服务器：选择 安装包 > 加载包 > 从前端。

对于某些 SAINT/SPAM 版本，SPAM 或 SAINT 设置可能会影响安装过程。请取消勾选“导入期间的检查”部分下的“签名检查无法执行，未实施 SAP note 2520826”选项。另请注意，对于某些 SPAM 版本，此复选框可能完全没有描述——您仍需取消勾选该框。

  1. 选择 开始 进行安装。
  2. 系统将列出可用包。

如果未列出，请点击过滤器图标以停用过滤器。

  1. 继续下一步。
  2. 对于连接器安装，选择 PALANTIR 和 PALAGENT。
  3. 选择 继续 进入支持包选择界面。
  4. 从列表中选择最高可用支持包，并确保两个组件选择了相同的 SP 级别。
  5. 确认安装队列并点击 继续。
  6. 在准备阶段选择 在对话模式下继续，在其他阶段选择 立即在后台继续，然后开始安装。

安装过程中可能会提示一些警告；请按照警告消息中的说明进行操作。在大多数情况下，连接器和远程智能体安装的警告消息可以忽略。特别是标题为“打开数据提取请求”的警告消息可以忽略。

  1. 选择 完成 以完成安装
  2. 运行 PFCG 事务代码，并为以下角色执行授权配置文件生成和用户比较：
    • /PALANTIR/CONTENT_RBEX_ALL
    • /PALANTIR/CONTENT_RFUNCT_ALL
    • /PALANTIR/CONTENT_RINFOPRV_ALL
    • /PALANTIR/CONTENT_RTABLE_ALL
    • /PALANTIR/CONTENT_RTCODE_ALL
    • /PALANTIR/SERVICE_USER

配置

连接器和连接器远程智能体通过 SAP 远程函数调用（RFC）进行通信。因此，需要建立两个 RFC 连接：一个从连接器到远程智能体，另一个从远程智能体到连接器。下一节将详细介绍如何创建这些 RFC 连接。

RFC 配置

RFC 配置包含四个步骤：

  1. 创建到 SAP 系统的 RFC 连接。
  2. 创建从连接器到远程 SAP 系统的 RFC（SOURCE）。
  3. 创建从远程 SAP 系统到连接器的 RFC（TARGET）。
  4. 通过 Web 浏览器测试连接器远程智能体。

创建到源/目标系统的 RFC 目标连接

本节需要建立两个 RFC 连接：一个从连接器到远程智能体，另一个从远程智能体到连接器。请按照《创建 RFC 连接》文档创建 RFC 连接。

对于从远程 SAP 系统到连接器的目标 RFC 定义，请重复相同步骤。您可以将其命名为 SAP_TARGET，而不是 SAP_SOURCE。（这些名称可自由定义。）

配置远程智能体并注册连接器

  1. 登录主连接器系统。
  2. 运行事务 /n/PALANTIR/PARAM_A1。
  3. 输入以下参数值：
    • 智能体 ID： 智能体标识符（也称为 CONTEXT）
    • 是否为 4.7 或更旧版本： 智能体版本标志
    • 智能体描述：智能体描述（供参考使用）
  4. 运行事务 /n/PALANTIR/PARAM_A2 并输入以下参数：
    • 智能体 ID：智能体 ID（如上一步骤所定义）
    • 参数 ID：参数 ID（参数的一种分类）
    • 参数名称：参数名称（供智能体使用）
    • 参数值：参数值（供智能体使用）
参数 ID参数名称参数值描述
RFCSOURCE从连接器到远程 SAP 系统的 RFC 连接。
RFCTARGET从远程 SAP 系统到连接器的 RFC 连接。
