面向 SAP 应用的 Palantir Foundry Connector 2.0 的备份与恢复

面向 SAP 应用的 Palantir Foundry Connector 2.0 的认证确保在系统迁移或卸载时能够进行备份和恢复。以下章节介绍了如何在需要时备份和恢复 Connector。

前提条件

  • 面向 SAP 应用的 Palantir Foundry Connector 2.0 SP13 或更高版本

备份重要表

若需将表备份为 .zip 文件并下载到文件系统以备后用，请运行事务码 /n/palantir/backup。这将备份所有 Connector 表及其内容。

Connector 备份中包含以下表

  • /PALANTIR/AGT_01：远程智能体配置头表。包含智能体 ID 和描述。
  • /PALANTIR/AGT_02：远程智能体配置项表。包含所有智能体特定参数，如指向智能体的 RFC 连接、资源检查设置、页面大小等。
  • /PALANTIR/CFG_01：Connector 参数。包含 Connector 特定参数，如抽取器默认值、对象类型设置、资源检查等。
  • /PALANTIR/CFG_02：敏感数据配置表。包含有关数据脱敏、哈希和加密的对象级和字段级设置。
  • /PALANTIR/CFG_03：预过滤配置表。包含 Connector 的对象级过滤器，独立于 Foundry 同步上的过滤器。
  • /PALANTIR/ENC_01：加密密钥
  • /PALANTIR/INC_01：增量数据流头表。包含增量设置，如增量字段、增量类型等。
  • /PALANTIR/INC_02：增量数据流项表。包含 Foundry 请求的历史增量值，以及这些请求是否成功抽取到 Foundry 中。
  • /PALANTIR/JOB：页面 ID 的未来作业
  • /PALANTIR/PAG_01：页头表

恢复重要的 Connector 表

SAP 系统迁移或维护完成后，可以重新安装 Connector。为了确保数据抽取正常继续，需要恢复 Connector 指针和系统设置。

  • 若要恢复表，请运行事务码 /n/palantir/restore。这将从备份中恢复所有 Connector 表及其内容。
  • 检查配置参数和表内容。
