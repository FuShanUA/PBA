将旧版 Foundry 报表转换为 Contour 或 Notepad

旧版 Foundry 报表可以转换为 Contour 仪表板或 Notepad 文档。

转换过程中不会删除原始报表，并且您可以对同一报表进行多次转换。除非需要编辑，否则无需转换旧版报表。

报表完成转换后，您可以选择将其中一个转换后的资源选作“指定替代项”，使其成为原始报表的权威转换版本。

选择 Contour 或 Notepad

Foundry 报表会根据报表内容推荐最合适的工具，但您也可以按照下文说明，自行选择将其转换为 Contour 仪表板或 Notepad 文档。

  • Contour 仪表板最适合由 Contour 分析驱动的内容。与报表相比，Contour 仪表板提供了更强的交互性，具备图表联动筛选等功能，并支持通过简单的拖拽界面边迭代分析边构建仪表板。此外，Contour 仪表板还支持全屏演示视图和 PDF 导出。在以下情况中，请选择 Contour 仪表板：
    • 报表仅包含 Contour 嵌入内容。
    • 报表使用了参数。
  • Notepad 文档最适合跨产品报表。借助 Notepad，您可以创建实时更新和静态（“时间点”）文档，对文档添加批注，在平台内共享文档，以及将文档导出为 PDF。在以下情况中，请选择 Notepad 文档：
    • 不需要通过参数或图表联动筛选提供的交互性。
    • 报表包含 Contour 仪表板不支持的嵌入内容，例如 Quiver 图表、Code Workbook 可视化或对象视图。这类报表仍可转换为 Contour 仪表板，但这些微件将被替换为指向 Foundry 中源资源的链接。

转换报表

请按照以下步骤转换旧版报表（示例图片中使用的所有数据均为演示数据）：

  1. 在 Foundry 报表中打开该报表。

  2. 在报表顶部的横幅中选择 转换报表。



  3. 按照报表转换对话框中的步骤操作，系统将引导您选择 Contour 或 Notepad 作为转换目标，并检查您的选择。



  4. 为转换后的资源选择名称和保存位置。

选择指定替代项

旧版 Foundry 报表至少完成一次向 Contour 或 Notepad 的转换后，您可以从原始报表顶部显示的横幅中访问转换后的资源。您还可以选择将其中一个资源设为原始报表的指定替代项。

选定指定替代项后，任何在报表应用中打开原始报表的用户，都会在顶部显示的横幅中看到指向新 Contour 仪表板或 Notepad 文档的链接。

如果您在选定指定替代项后改变主意，可以取消该资源的指定状态，并重新选择其他资源进行替代。
