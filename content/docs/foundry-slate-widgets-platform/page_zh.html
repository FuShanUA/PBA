对象卡片

对象卡片部件将 ObjectRID 作为其输入。它会按照在 Ontology 应用中定义的微型对象视图（卡片）进行渲染。对象卡片将链接到其对应的对象浏览器对象视图。

表格格式

将此部件与可重复容器结合使用，可以填充生成一个对象视图表格。

例如，假设你有一个返回 RID 数组的 Slate 函数。你可以将对象卡片部件插入到可重复容器中，以创建“表格视图”。你需要使用可重复容器的索引来索引对象部件，因此在“Object RID”中你需要指定：{{lookup f_function1 w_repeating_container.index}}。

这样便利用内置的查找函数，基于函数返回的 RID 和可重复容器的索引建立了一个卡片列表。

获取对象 RID

要获取 ObjectRID 输入，你有以下几种选择：

选项 1：Slate 提供了一个点击式用户界面，即《对象集合面板》，你可以在其中进行对象查询，而无需手动构建查询语句。

选项 2：你可以通过导航到特定对象的“对象视图”并选择“操作”下拉菜单来获取其 ObjectRID。在剪贴板中，它允许你复制 URL 链接。复制 URL 链接的最后一部分，即 ?objectId={ObjectRID} 之后的部分（例如，如果 URL 是 ...?objectId=ri.phonograph2-objects.main.object.09d2e0e9-dd3c-49b2-8b96-0cb1bf005c1d，那么你的 ObjectRID = ri.phonograph2-objects.main.object.09d2e0e9-dd3c-49b2-8b96-0cb1bf005c1d）。

选项 3：要通过 ObjectTypeId 和 ObjectPrimaryKey 获取 ObjectRID，你需要使用 Get Object by Locator 端点。虽然 ObjectTypeId 和 ObjectPrimaryKey 可以在其他地方用于参数化对象浏览器 URL 链接，但在 Slate 中它们需要被解析为 ObjectRID。你可以动态组合 ObjectTypeId 和 ObjectPrimaryKey，并将其传递给 Slate 查询以检索 ObjectRID（前提是这些端点已在你的实例上公开，并且你已将其设置为数据源）。

资源选择器

资源选择器部件允许你在 Compass 中选择文件。

此部件不会在你的页面上创建可视化元素。相反，它提供了一个用于打开资源选择器的动作，以及一些用于处理所选资源的事件。

事件

  • w_resource_selector_1.didClose： 当资源选择器关闭时会触发此事件。
  • w_resource_selector_1.selectedRid.changed： 在资源选择器中选中某个资源时会触发此事件。{{sl_event_value}} 会被设置为所选资源的 RID。

动作

动作名称描述
w_resource_selector_1.open触发此动作会打开资源选择器对话框。

时间序列

Slate 的时间序列部件提供了一种便捷的方式来可视化时间序列图表。你可以使用系列 ID 将多个系列添加到同一图表中。你还可以通过以下方式配置可视化：

  • 指定标题、单位和颜色，或者禁用十字准星或图例。
  • 动态定义时间范围以指定视图窗口。
  • 为数据开启“流式模式”。

操作

Slate 的 操作组件 允许 Slate 执行在操作中预配置的业务逻辑。将组件添加到页面后，你可以：

  • 选择你有权访问的 Foundry 操作，
  • 直接或通过 Slate 的 Handlebars 传入默认参数到操作表单，以及
  • 使用 Submit 按钮直接提交操作表单，或通过 Slate 的事件-动作对 w_widget.submit 间接提交。

此外，Slate 还提供了以下与操作表单相关的 Slate 事件：

  • 提交（w_widget.success/w_widget.failure）
  • 验证状态（w_widget.ValidationSuccess/w_widget.ValidationFailure），以及
  • 渲染变更（w_widget.transitioned 和 w_widget.cssClassesUpdated）。

最后，如果需要向用户隐藏 UI 并通过 Slate 事件触发提交，你还可以通过切换开关阻止操作表单显示。有关更多信息，请参阅《Slate 事件》。

在 Slate 中使用 Foundry 操作之前，需要先创建该操作。有关更多信息，请参阅《操作文档》。

日期预填

日期需采用 ISO 8601 格式；例如，YYYY-MM-DD（如 1990-01-12）即为有效的日期格式。

时间戳预填

时间戳需遵循 ISO 8601 格式：YYYY-MM-DD[T]HH:mm:ss.SSS[Z]（例如，1990-01-12T23:00:00.000Z）。
