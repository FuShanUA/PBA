产品组合

通过产品组合，用户可以在空间内组织项目。每个产品组合包含多个项目，且每个项目仅归属于一个产品组合。任何拥有空间访问权限的用户均可查看该空间下的产品组合，但若要查看产品组合内的具体项目，用户仍需单独获取相应权限。

创建产品组合

拥有空间内编辑者角色的用户可以在“产品组合”页面创建产品组合。每个产品组合必须关联一个空间，并具备唯一的名称。创建者可选择性地添加一段简短的文本描述，并通过图片或图标颜色配置产品组合的缩略图。缩略图应为正方形，系统会对其尺寸进行压缩。

编辑产品组合元数据

管理员和策展人可以通过右上角的“操作”菜单编辑产品组合的元数据。名称、描述和徽标均可编辑，但产品组合一经创建，便无法在不同空间之间迁移。

管理员和策展人还可以为产品组合添加 Markdown 格式的文档。所有具备该产品组合查看权限的用户均可阅读此文档。

策展产品组合

创建产品组合后，管理员可以通过“添加项目”对话框，将同一空间内的项目填充至该产品组合中。此对话框会展示与该产品组合同处一个空间的所有项目，包含那些已归属于其他产品组合的项目。由于一个项目只能归属于一个产品组合，将其移至另一产品组合会使其从原产品组合中移除。

选择要纳入该产品组合的项目后，用户可以选择更改每个项目的显示名称。此步骤为可选项。

其他产品组合策展人

通常情况下，只有具备空间内编辑者角色的用户才能管理其产品组合的内容。为扩展产品组合的策展权限，具备管理权限的用户可以打开产品组合的侧边栏，并编辑其策展人列表。这些用户将有权在该产品组合中添加或移除项目，以及编辑其描述和文档。

产品组合数据目录

在项目中被固定的资源会按项目或资源类型分组，显示在产品组合的数据目录中。管理员可以通过“添加内容”对话框搜索资源，从而推广产品组合各项目内的资源。

项目侧边栏

项目侧边栏会展示已添加至产品组合的项目列表。通过该侧边栏，用户可以打开各个具体项目，产品组合管理员则可以添加或移除项目。
