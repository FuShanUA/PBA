String cards

Back to: Index of cards

Cards in this section return and display strings.

The following cards accept object sets and return strings:

  • Unique property values
  • Property value select parameter

The following cards accept single objects and return a string:

  • Object property

The following cards accept transform tables and return strings:

  • Transform table column values

The following cards accept other strings and return strings:

  • Substring
  • Concatenate
  • Find and replace
  • To lower case
  • To upper case
  • Multipass attribute

The following cards accept time series and return a single string:

  • Enum value at time
  • Latest value for enum time series
  • Time series unit

Additional string cards include:

  • String parameter
  • Coalesce
  • Array join
  • Code function value
  • Visual function property select
  • Visual function string
  • Array start
  • Array end
  • Value at index
  • Date to string
