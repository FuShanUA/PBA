入门

要为您的应用程序启用本体论 MCP (OMCP)：

  1. 在开发者控制台中导航到您的应用程序。
  2. 在左侧菜单中选择 MCP 页面。
  3. 在右上角切换按钮以为您的应用程序启用 MCP 服务器。
  4. 根据您选择的代理框架，使用页面上列出的配置详细信息。

即使您的首选框架未列出，假设您的框架可以作为 MCP 客户端，配置可能类似。

启用后，您的应用程序将把其本体论资源作为 MCP 工具公开。外部 MCP 客户端可以使用设置面板中显示的连接详细信息连接到您的应用程序。

确保您的应用程序已配置适当的 应用限制，以控制通过 MCP 可访问的本体论资源。

在 MCP Hub 中发现本体论 MCP 服务器

本体论 MCP 服务器可通过 MCP Hub 应用程序进行发现。打开 MCP Hub 并导航到 本体论 MCP 选项卡，以查看您注册中配置的所有 MCP 服务器列表。选择任意一行，即可查看该服务器的完整配置详细信息。从那里，您还可以直接跳转到相应的开发者控制台应用程序，无需离开您的工作流程即可添加或修改资源。
