数据连接与集成

Foundry 提供了一套高度可配置的数据连接与集成工具，其能力远超传统的提取-转换-加载（ETL）或提取-加载-转换（ELT）方案。Foundry 旨在通过一系列丰富功能降低长期的数据集成成本，这些功能对数据团队起到了力量倍增器的作用。尽管基础云服务为核心流水线和实验环境提供了存储与计算资源，但要针对关键业务运营去管理、交付和校验数据集，仍需引入更多能力层。Foundry 的设计初衷，就是充当地球上最复杂环境下的数据集成主干。

连接数据

这一切始于一个可扩展的数据连接框架，该框架能够与各类源系统——结构化、非结构化或半结构化——建立连接，并支持所有关键的数据传输方式，如批处理、微批处理或流式传输。此功能与平台的数据转换及数据管理功能深度集成，涵盖数据版本的完整血缘关系、用于协同管理数据提取的细粒度安全机制，以及数据同步配置的分叉管理。

数据转换

在数据转换方面，Foundry 提供了一个可扩展、可伸缩的数据构建系统，利用多模态计算来生成输出数据集。Foundry 计算无关的“Build”框架提供了全面集成的安全机制与数据血缘关系，并支持混搭第三方计算运行时。Foundry 还内置了一套完整的数据转换编写、变更管理、数据质量、流水线调度和元数据内省功能，这些功能协同运作，为数据工程师打造了一个“任务控制中心”。

流水线管理

Foundry 的流水线管理功能将变更管理、数据质量和数据加载特性融为一体。

Pipeline Builder 应用支持快速、灵活、可伸缩地交付数据流水线，同时确保鲁棒性与安全。了解有关 Pipeline Builder 的更多信息。

数据工程师可以为生产流水线设定严密的发布流程，其中包含健康检查机制，确保只有完全合规的数据才会部署至生产环境。一旦发现问题，平台会对检测到的差异提供诊断分析。

诊断结果既可以在 Foundry 集成的分析与建模工具中查看，也可供任何通过 REST API 或其他接口访问输出结果的第三方工具使用。
