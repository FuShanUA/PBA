查找并使用数据

搜索

通过查找并使用数据搜索面板，可以轻松将数据和对象提取到 Fusion 中。选中 Fusion 表格右上角的查找并使用数据框，即可访问该搜索面板。

你可以直接在搜索输入区（“搜索单元格或关键词”）中输入值进行搜索，也可以点击“目标”图标后选择单元格值。

你还可以选择特定的对象类型或数据集进行搜索。如果所选数据集未建立索引，系统会提示你为其建立索引。

导入数据

找到需要导入的数据后，选择使用数据并打开导入数据面板，将其引入表格。需要注意的是，每次最多可导入 2,000 行。

导入数据面板会将搜索结果中使用的筛选条件添加为导入查询的筛选器。例如，如果你搜索 Red 并对一组在 color 属性或列上匹配的车辆选择使用数据，初始查询将以 color == red 的车辆作为起点。

在此处你可以添加更多筛选条件，并最终将结果数据以数组、下拉列表或多选下拉列表的形式添加到表格中。

数据集数组

以数据集数组形式导入，会在表格中添加一个 lookup_array 函数。数据集数组不包含源数据中的列名。有关如何使用查找功能，请参阅《查找数据集》。

提示

如需展开从 lookup 返回的数组结果，你可以按住 Shift 键并将该单元格向下拖动，将其分解到各个独立的单元格中。

数据集下拉列表与数据集多选下拉列表

将数据集下拉列表添加到表格中后，每个导入列的单独单元格内都会放置一个单选下拉列表。数据集多选下拉列表的功能与之类似，但允许选择多个值。随后，你可以利用该下拉列表来驱动表格的其他部分。

可以对数据集下拉列表单元格应用格式设置，包括合并或调整单元格大小、设置字号以及设置背景填充色。

导入对象

与数据集类似，如果对象不受受限视图支持，你可以使用筛选条件来细化导入表格的对象列表。筛选条件可以是具体值或单元格引用。单元格引用会根据单元格的值进行筛选，借此可以构建动态表格，使对象集能够根据其他值或选择进行更新。在此示例中，底层数据来源于《美国人口普查局》。

在下方示例中，City 对象类型具有一个筛选条件，即 State 等于单元格 B2 的值。当前该单元格的值为 Arizona，因此筛选出 State 为 Arizona 的城市。你可以更改该单元格的值，系统会随之更新筛选结果。

应用筛选条件后，你可以选择重新加载以查看生成的对象集。你可以将此对象集以对象下拉列表或对象表格的形式导入到表格中。默认情况下，单个对象下拉列表或对象表格最多可提取 200 个对象。

对象下拉列表

将对象以下拉列表形式添加到表格中后，会在单个单元格内放置一个下拉列表。你可以利用此下拉列表来驱动表格的其他部分。

对象下拉单元格将显示当前选定的对象及其对象图标。你可以选择该单元格以打开下拉菜单，并选择其他对象。

可以对对象下拉单元格应用格式设置，包括合并或调整单元格大小、设置字号以及设置背景填充色。

对象表

可以对对象表进行格式化以用于展示，或者在工作表的其他地方使用。

将对象下拉单元格与对象表结合使用，即可创建一个简单的仪表板：具体而言，利用对象下拉单元格选择对象，随后将这些对象作为对象表中的筛选条件。

Fusion 工作表不能同时包含对象和数据集导出同步。如果已包含对象，系统将阻止你添加新的数据集同步，反之亦然。
