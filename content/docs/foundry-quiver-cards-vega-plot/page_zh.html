Vega 图表

Quiver 允许用户使用 Vega 或 Vega-Lite 库创建完全可自定义和交互的可视化效果。查看 Vega ↗ 和 Vega-Lite ↗ 文档以获取灵感和示例。

Vega 是什么？

Vega 和 Vega-Lite 允许您以简洁的 JSON 规范形式创建、保存和共享交互式可视化设计，该规范描述了可视化的外观和行为。Vega 支持各种可视化设计，例如：

  • 数据加载 ↗ 和 转换 ↗
  • 比例尺 ↗
  • 地图投影 ↗
  • 坐标轴 ↗
  • 图例 ↗
  • 图形标记 ↗ 例如矩形、线条、绘图符号等。

Vega-Lite 是在 Vega 基础上构建的更高层次的语言，它提供了一种更简洁和方便的方式来创建常见的可视化效果。我们建议从 Vega-Lite 开始，因为与 Vega 相比，Vega-Lite 的语法更容易阅读、编写和调试。如果 Vega-Lite 的选项不足以满足需求，可以考虑使用 Vega。

配置 Vega 图表

要配置 Vega 图表，请将鼠标悬停在所需的卡片上，选择 可视化，然后在下拉菜单中选择（或搜索）Vega 图表。选择 Vega 图表 以打开配置菜单。

在编辑器菜单中，选择配置一个Vega图表或一个Vega-Lite图表。在选择模板下拉菜单中提供了常见可视化的模板，包括气泡图、箱形图、太阳图等，如下图所示。

选择完Vega图表模板后，您将被提示通过将它们映射到转换表数据源的列来填写与所选图表相关的参数。

填写完模板参数后，选择应用以生成您的Vega图表。请注意：

  • 任何未填写的字段将使用参数名称作为默认值。
  • 填写好的模板会覆盖编辑器中的当前值。

配置数据输入

Vega图表接受以下数据输入：

  • 转换表：任何转换表都可以用作Vega图表的输入。当从对象集添加Vega图表时，也会自动将一个支撑转换表添加到分析内容面板（但不在画布上）。

在其他应用程序中嵌入Vega图表

Vega编辑器包含两个有用的设置，用于通过Quiver仪表板在Workshop中嵌入Vega图表：

  • 自动调整图表： 当启用自动调整图表时，您的Vega图表将根据提供的数据自动适应卡片尺寸。
  • 默认样式： 向Vega规范中注入一些自动大小调整和样式配置，以使图表外观与Quiver卡片的风格相匹配。当禁用时，Vega可视化效果将与公共Vega网站上的相同。

使用AIP配置Vega图表

AIP可以通过您对所需图表或图表修改的自然语言描述来帮助生成Vega图表配置。用户只需描述他们希望创建或修改的图表，AIP将根据提示提供建议，从而简化JSON格式的Vega图表配置，这种格式容易出错。

描述所需的Vega图表

要使用AIP，请在Vega图表卡片的右上角选择AIP 配置。然后，提供一个提示，选择配置，并查看AIP生成的建议。要接受一个建议，请选择应用。如果您希望AIP提出其他建议，请编辑您的提示，然后选择重新配置。

在下面的示例中，用户向AIP提示：“在散点图上显示ph与咖啡因，并在x轴上13处和y轴上6处绘制一条红色线”。AIP使用该提示来建议包含两个感兴趣数值属性（咖啡因和pH值）的图表配置。

要接受AIP提出的更新，请选择应用。Vega图表将基于更新的Vega规范渲染可视化效果。

在现有图表基础上进行构建

AIP 可以识别并区分您提示中的抽象概念，并利用这些信息来帮助生成或修改Vega图表。例如，AIP可以使用对先前生成的图表特征的引用，使您可以“在现有图表基础上”进行构建。

在下面的图像中，用户提供了提示：“图表上的两条线将点分成四个象限。每个象限用不同的颜色表示。”即使当前的图表配置中没有定义或配置象限，AIP也能识别出由垂直和水平线定义的区域是象限。

AIP 准确地生成了符合用户提示的期望图表，这基于初始图表进行构建。

使用AIP Generate创建Vega图表

使用AIP生成的Quiver分析图可以包括Vega图表。为了指示AIP生成Vega图表而不是原生的Quiver可视化效果（如折线图或散点图），在您的提示中明确说明AIP应返回“Vega图表”。例如，一个准确的提示可以是：“使用Vega图表显示咖啡因与pH值的散点图。”

注意：AIP功能的可用性可能会发生变化，并且可能因客户而异。

Vega-Lite选择

为Vega-Lite图表配置选择

Vega-Lite 选择是一个强大的且高度可定制的功能，用于构建交互式可视化效果。Vega-Lite 库内置了对 选择 ↗ 的支持，Quiver 中的 Vega-Lite 图表可以配置为将选择输出为转换表。用户可以利用选择数据来参数化下游卡片、构建下钻工作流并继续分析。Vega-Lite 的选择行为与对象集图选择不同，需要额外的步骤进行设置。

Vega-Lite 允许用户通过 两种类型的选择 ↗ 与图表进行交互：

  • 点（Point） 选择：选择一个点，或按住 Shift 键选择画布上的多个点。
  • 区间（Interval） 选择：拖动以在画布上选择一个有界的矩形区域。

并非所有 Vega-Lite 图表类型都支持选择。有关更多详情，请参见下方的 常见问题部分。

要将图表选择数据输出为转换表，请执行以下步骤：

  1. 打开 Vega 图表卡片的配置面板，并滚动到 选择选项。然后，打开将输出点选择作为转换表和/或将输出区间选择作为转换表。

  2. 在Vega图表的JSON编辑器中，将一个选择参数添加到params字段。有关选择参数的更多信息，请参阅Vega图表文档。

    1. 该参数必须命名为quiverDefaultClick以用于点选择，或命名为quiverDefaultBrush以用于区间选择。

    2. 将type属性设置为point对于quiverDefaultClick，或者为interval对于quiverDefaultBrush。

    3. 在encodings属性中指定一个或多个编码 ↗。这些是您希望进行选择的字段，例如x、y或color。编码决定了如何选择值以及输出哪些信息。

  3. 一旦做出选择，就会出现包含选择数据的卡片页脚。选择输出选择，将选择数据输出为转换表。或者，使用Vega图表的下一步操作菜单并选择转换 > 新转换表 。

    • 点选择会以编码字段和值的表格形式输出，其中每一列对应一个字段，每一行表示一个选中的点。



    1. 区间选择如果字段是连续的，则以区间的范围（最小值、最大值）形式输出；如果字段是离散的，则以值数组的形式输出。



与对象集图表不同，对象集图表的选择是一个下钻操作，会输出过滤后的对象集，而 Vega 图表不能根据当前选择自动过滤输入数据。相反，Vega 图表会输出给定编码的所选值，这些输出可用于手动在输入表上构建过滤器。有关更多信息，请查看 构建下钻流程 部分。

请参考提供的Vega图表模板，以获取常见可视化中的选择设置的更多示例，例如气泡图和热力网格。这些模板可以在配置编辑器的 选择模板 下拉菜单中找到。

        "param": "quiverDefaultClick",

      }

  },

    "name": "quiverDefaultClick",

构建下钻流程

从Vega-Lite图表中选择的数据可以用来构建下钻流程，其中图表选择充当过滤器，用户可以根据上游选择继续对数据子集进行分析。以下步骤描述了如何构建下钻流程。

  1. 按照上述步骤设置选择参数，并将当前选择输出为转换表。



  2. 选择将列弹出为数组，以将属性转换为数组。





  1. 使用所选值作为筛选参数，对原始转换表的副本（或从中派生转换表的根对象集的副本）进行筛选。



  1. 现在，筛选后的表格将根据上游图表的选择动态更新。



常见问题与限制

是否可以配置任何类型的 Vega-Lite 图表进行选择？

并非所有 Vega-Lite 图表类型都支持选择。虽然 Vega 文档并未提供完整的支持图表类型列表，但以下是一个非详尽的列表：

  • 支持的： 条形图、散点图、热力网格图、折线图、饼图（仅支持点选择）、瀑布图（仅支持点选择）、地理坐标投影、GeoJSON地图（仅支持点选择）
  • 不支持： 箱型图、置信区间图

是否可以在单个图表规范中定义多个选择参数？

每个图表只能有一个点选择参数和一个区间选择参数，这些参数会将数据输出到Quiver。这是Quiver设置的限制；虽然Vega允许定义多个参数，只要它们有唯一的名称，但只有名为quiverDefaultClick和quiverDefaultBrush的参数会被输出为Quiver转换表。

即使已将其添加到选择参数中的编码列表中，我仍无法选择轴的一个子集。这是什么原因？

检查 Vega 规范中该轴上的值是否正在被聚合。Vega 不支持对由 Vega 聚合的字段进行选择。例如，如果 y 编码字段包含如下片段中的内联聚合，用户将无法选择 y 轴的子集。

"encodings": {
  "y": {
    "field": $C.field_y,
    "aggregate": "avg"
  },
  ...
}

要启用对 y 轴的选择，请先在 Quiver 中对数据进行聚合（例如使用透视表），然后再将其传递给 Vega。

当我尝试进行选择时，即使规范编译成功，为何会收到 Invalid data selection 错误？

请检查以下内容：

  • 确保所有选择参数都有非空的 encodings 字段。每个参数必须指定一个或多个编码来输出选择信息。
  • 如果是区间选择，请检查选择参数中定义的任何编码是否为非轴字段。Vega 不支持在非轴编码字段如 color、shape 等上进行区间选择。从编码数组中移除这些字段应能解决错误。
