按钮组

按钮组微件允许应用构建者在 Workshop 模块中添加按钮，用于触发《操作》和《Workshop 事件》、打开 URL 或启动导出任务。构建者在配置按钮组微件时，可以执行以下操作：

  • 在以下三种按钮类型中进行选择：
    • 内联按钮：提供单一选项
    • 菜单按钮：提供多个选项
    • 分体式按钮：包含一个主按钮以及一个附加选项菜单。
  • 为每个按钮配置 点击时 事件，以触发操作、触发一系列事件、打开 URL 或启动导出任务。
  • 通过设置每个按钮的颜色、图标、大小和填充选项来调整样式。
  • 基于布尔变量的值，配置条件禁用状态或条件可见性。

按钮配置选项

下方的截图展示了按钮组微件中单个按钮可用的配置选项：

  • 添加按钮：选择此选项将向该按钮组添加另一个按钮 / 菜单项。
  • 复制按钮：选择此选项将创建现有按钮配置的副本，便于构建者快速复制按钮设置。
  • 按钮文本：此参数用于设置给定按钮或菜单项的显示文本。
  • 按钮颜色：此配置用于控制给定按钮或菜单项的颜色。您可以选择带有关联颜色的预设“意图”，也可以指定自定义颜色。意图选项及其关联颜色包括无、主要（蓝色）、成功（绿色）、警告（琥珀色）和危险（红色）。您可以使用自定义选项从更丰富的颜色选项中进行挑选，包括使用十六进制代码设置颜色。
  • 左侧图标：此参数用于控制按钮 / 菜单项显示文本左侧显示的图标。设置为空白则不显示图标。
  • 右侧图标：此参数用于控制按钮 / 菜单项显示文本右侧显示的图标。设置为空白则不显示图标。
  • 描述：此参数将在用户将鼠标悬停在按钮上时作为工具提示显示。
  • 条件可见性：如果开启条件可见性，将提供以下选项：
    • 布尔变量：此变量将用于确定按钮的条件状态。
    • 状态为假时：此选项用于控制当所选变量值为假时，按钮应被禁用还是隐藏。
  • 点击时：此选项用于控制用户与按钮或菜单项交互时触发的动作。选项包括触发《操作》、一个或多个《Workshop 事件》、打开 URL 或启动导出任务。如果配置了多个 Workshop 事件，在其余事件运行之前，变量状态的更新可能尚未完成。下方提供了完整的点击时选项列表。
  • 显示与格式
    • 极简样式：开启后，此选项将移除按钮的边框。如果已应用颜色，按钮 / 菜单项的背景色和文本色将反转（例如，主要意图的按钮可以从蓝色背景和白色文本反转为白色背景和蓝色文本）。
    • 标签样式：开启后，此选项会将给定按钮调整为更窄的标签样式。
    • 大号样式：开启后，此选项将增大按钮的整体尺寸。
    • 在行和列布局中填充可用水平空间： 如果启用，此按钮组将填充其所在区块的水平空间。 - 场景 - 使用按钮组创建操作时，可以选择将该操作应用于场景或本体。 - 应用于场景： 启用此开关可将此操作应用于场景，而非主本体。 - 有关场景的更多信息，请参阅《场景文档》。 ## 点击时 按钮可以触发操作、布局事件、URL 以及不同类型的导出。下面详细介绍了每个选项。 ### 操作 操作允许用户以预定义的方式轻松创建、编辑、删除和链接对象，并可由 Workshop 事件触发。例如，应用构建者可以在模块中配置按钮组小部件，以触发“修改航班目的地”操作，允许用户编辑所选 `Flight` 对象上的 `Destination` 属性。 了解有关《在 Workshop 模块中配置和公开操作》的更多信息。 有关操作的更多一般性详细信息，请查阅《操作类型文档》。 #### 将事件与操作链接 将按钮配置为触发操作时，还可以配置在操作生命周期的特定时间点发生的 Workshop 事件。在小部件配置的底部，选择以下事件触发选项之一： - 操作提交开始时： 配置的事件在操作提交开始时发生。 - 操作提交成功完成时： 配置的事件在操作成功提交后发生。 这允许在单个按钮上链接操作和事件，从而实现诸如在操作完成后刷新数据、导航到另一个页面或更新变量等工作流。 ### 事件 请查阅《事件文档》，了解在 Workshop 中使用事件的完整详细信息。 ### URL URL 事件会触发从 Workshop 模块中打开指定的 URL。例如，应用构建者可以使用 URL 事件在 Foundry 中导航并打开对象资源管理器应用程序，或者指定一个外部网站在用户选择给定按钮组小部件时打开。 下面的截图展示了按钮组小部件中外部 URL 事件的示例定义： 定义外部 URL 时，请包含 `https://` 前缀。 ### 导出 导出事件将对象集变量作为输入，并触发将对象集中的对象导出到 Excel 或用户剪贴板。应用构建者可以选择配置文件名，并选择导出中应包含的属性集。 下面的截图展示了按钮组小部件中导出事件的示例定义： 请注意，如果包含《函数支持列》或链接对象列，导出文件格式将为 CSV 而非 Excel。 #### 函数支持导出 函数支持导出接受函数及其输入，并将输出下载为指定的文件类型。支持的文件类型包括 CSV、TXT、JSON、XML、PDF、DOCX 和 XLSX。以下是按钮组小部件中函数支持导出的示例定义： 函数输出必须是字符串，其编码由导出文件类型决定：
    • 文本格式（CSV、TXT、JSON、XML）：函数应返回表示文件内容的纯字符串。
    • 二进制格式（PDF、DOCX、XLSX）：函数应返回文件字节的 base64 编码字符串。在 Python 中，可以直接返回 bytes。请勿在返回的字符串中包含 MIME 类型前缀（例如 data:application/pdf;base64,），否则导出文件将无法下载。Workshop 会根据导出配置中选择的文件类型自动推断 MIME 类型。

    导出媒体

    导出媒体事件接收一个已导出项的列表，并触发与这些项关联的所有媒体的下载。要配置导出项，请选择对象集、媒体类型、媒体属性以及要下载的最大对象数量。您可以为下载的媒体配置自定义文件名，并支持替换当前时间戳、原始文件名以及非主键对象属性值。如果未提供自定义文件名，将使用原始文件名。
