Publish a template

Once you are done configuring your Notepad template, publish it to begin using it.

Publishing a template will save the current content, settings, and input parameters under a new version number. To publish a version, select Publish in the top right toolbar, optionally specify a name for the new version, and click the Publish Template button.

Template versioning ensures changes do not impact production workflows where the template is already in use. When adding templates to Workshop applications, you will need to explicitly reference the version number to be used. To accept template updates in production workflows, update your configuration to reference the new version number.

To inspect the history of published versions, click the pin icon in the top toolbar. This will open a table with all published versions as well as the dates and the users who published them.
