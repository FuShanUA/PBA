前提条件

要使用Palantir提供的语言模型，需要先在您的注册中启用AIP。

使用语言模型生成完成
在此示例中，我们将使用语言模型来确定输入数据集中的每条评论的情感。OpenAiGptChatLanguageModelInput在运行时提供一个OpenAiGptChatLanguageModel给转换器，然后可以用来为评论生成完成。


Copied!
1from transforms.api import transform, Input, Output
2from palantir_models.transforms import OpenAiGptChatLanguageModelInput
3from palantir_models.models import OpenAiGptChatLanguageModel
4from language_model_service_api.languagemodelservice_api_completion_v3 import GptChatCompletionRequest
5from language_model_service_api.languagemodelservice_api import ChatMessage, ChatMessageRole
6
7
8@transform(
9    reviews=Input("/path/to/reviews/dataset"),
10    model=OpenAiGptChatLanguageModelInput("ri.language-model-service..language-model.gpt-4_azure"),
11    output=Output("/output/path"),
12)
13def compute_sentiment(ctx, reviews, model: OpenAiGptChatLanguageModel, output):
14    def get_completions(review_content: str) -> str:
15        system_prompt = "Take the following review determine the sentiment of the review"
16        request = GptChatCompletionRequest(
17            [ChatMessage(ChatMessageRole.SYSTEM, system_prompt), ChatMessage(ChatMessageRole.USER, review_content)]
18        )
19        resp = model.create_chat_completion(request)
20        return resp.choices[0].message.content
21
22    reviews_df = reviews.pandas()
23    reviews_df['sentiment'] = reviews_df['review_content'].apply(get_completions)
24    out_df = ctx.spark_session.createDataFrame(reviews_df)
25    return output.write_dataframe(out_df)


嵌入
除了生成式语言模型外，Palantir还提供了一个嵌入模型。以下示例展示了如何使用palantir_models.transforms.GenericEmbeddingModelInput在相同的reviews数据集上计算嵌入。GenericEmbeddingModelInput在运行时提供一个GenericEmbeddingModel给转换器，可用于计算每条评论的嵌入。由于本体论向量属性需要，这些嵌入被显式地转换为浮点数。





Copied!
1from transforms.api import transform, Input, Output
2from language_model_service_api.languagemodelservice_api_embeddings_v3 import GenericEmbeddingsRequest
3from palantir_models.models import GenericEmbeddingModel
4from palantir_models.transforms import GenericEmbeddingModelInput
5from pyspark.sql.types import ArrayType, FloatType
6
7
8@transform(
9    reviews=Input("/path/to/reviews/dataset"),
10    embedding_model=GenericEmbeddingModelInput("ri.language-model-service..language-model.text-embedding-ada-002_azure"),
11    output=Output("/path/to/embedding/output")
12)
13def compute_embeddings(ctx, reviews, embedding_model: GenericEmbeddingModel, output):
14
15    def internal_create_embeddings(val: str):
16        return embedding_model.create_embeddings(GenericEmbeddingsRequest(inputs=[val])).embeddings[0]
17
18    reviews_df = reviews.pandas()
19    reviews_df['embedding'] = reviews_df['review_content'].apply(internal_create_embeddings)
20    spark_df = ctx.spark_session.createDataFrame(reviews_df)
21    out_df = spark_df.withColumn('embedding', spark_df['embedding'].cast(ArrayType(FloatType())))
22    return output.write_dataframe(out_df)


使用视觉语言模型提取PDF文档内容
对于使用视觉语言模型的PDF文档提取工作流，我们建议使用AIP 文档智能。AIP 文档智能提供了一个直观的界面，可在您的文档上测试不同的提取策略，比较结果，并在部署到生产环境之前评估提取质量。
部署配置后，会创建一个Python转换代码仓库，并自动配置您选择的提取方法、模型和任何自定义提示。您无需手动编写用于文档提取的转换代码。了解有关部署到Python转换的更多信息。
