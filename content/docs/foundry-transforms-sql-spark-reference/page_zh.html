Spark SQL 参考

本节介绍了编写 Spark SQL 数据转换与其它类型 SQL 查询之间的一些关键差异。它还包含可用的 Spark SQL 函数列表。

提示

在查找资源时，请记住，Spark SQL 实际上是基于 HiveQL 语法的。你可以在网上找到更多关于 HiveQL 的信息。

入门

基本查询格式

每个 SQL 数据转换查询都必须创建一个表。你的 SQL 查询的一般格式是： 

Copied!
1CREATE TABLE _____ AS SELECT _____

不要在语句末尾包含分号。 包含分号会导致错误。

注释语法

您可以在SQL代码中这样添加注释：

Copied!
1-- You can create comments using
2-- this syntax
3
4/* You can also create comments using
5   this syntax */

引用数据集

要引用一个数据集，请提供用反引号包围的数据集路径：

Copied!
1CREATE TABLE `/path/to/target/dataset` AS
2SELECT * FROM `/path/to/source/dataset`
3
4-- Alternative syntax
5
6CREATE TABLE `/path/to/target/dataset` AS (
7    SELECT * FROM `/path/to/source/dataset`
8)

请注意，数据集名称是区分大小写的。

引用列

要引用数据集中的特定列，请提供列名：

Copied!
1SELECT Name FROM `/path/to/source/dataset`

请注意，列名是区分大小写的。

派生列 & 别名

派生列是对你数据集列调用函数后的结果。你必须为任何派生列定义一个别名：

Copied!
1SELECT Sum(Val) AS Total FROM `/path/to/source/dataset`

以下查询将导致错误：

Copied!
1SELECT Sum(Val) FROM `/path/to/source/dataset`

请注意，别名名称是区分大小写的。

SQL 子句中的派生列

你不能在以下 SQL 子句中使用别名： WHERE 和 GROUP BY。因此，在 WHERE 或 GROUP BY 子句中，你必须引用实际的函数和数据集列：

Copied!
1SELECT Lower(Name) AS LowercaseName
2FROM `/path/to/source/dataset` WHERE Lower(Name) = "sara"
3
4SELECT Lower(Name) AS LowercaseName, Sum(Val) AS Total
5FROM `/path/to/source/dataset` GROUP BY Lower(Name)

你可以在以下 SQL 子句中使用别名： ORDER BY 和 HAVING。因此，以下任何查询都可以正常工作：

Copied!
1-- Both of the following queries are valid
2
3SELECT Lower(Name) AS LowercaseName, Sum(Val) AS Total
4FROM `/path/to/source/dataset` GROUP BY Lower(Name) ORDER BY Total
5
6SELECT Lower(Name) AS LowercaseName, Sum(Val) AS Total
7FROM `/path/to/source/dataset` GROUP BY Lower(Name) ORDER BY Sum(Val)
8
9-- Both of the following queries are valid
10
11SELECT Lower(Name) AS LowercaseName, Sum(Val) AS Total
12FROM `/path/to/source/dataset` GROUP BY Lower(Name) HAVING Total > 100
13
14SELECT Lower(Name) AS LowercaseName, Sum(Val) AS Total
15FROM `/path/to/source/dataset` GROUP BY Lower(Name) HAVING Sum(Val) > 100

类型转换

您可以将一个表达式进行强制类型转换，将其从一种类型转换为另一种类型。强制类型的语法是：

Copied!
1CAST(expr AS <TYPE>)

注意，expr 表示您要转换的表达式，而 <TYPE> 表示您希望将该表达式转换成的类型。如果 CAST(expr` `AS` `<TYPE>) 失败，它会返回 null。<TYPE> 的可用值包括：

  • boolean（布尔）
  • tinyint（小整数）
  • smallint（短整数）
  • int（整数）
  • bigint（大整数）
  • float（浮点）
  • double（双精度浮点）
  • decimal（十进制）
  • date（日期）
  • timestamp（时间戳）
  • binary（二进制）
  • string（字符串）

格式化日期

您可能需要重新格式化您的日期值或将字符串转换为日期格式。一个 date 的格式为 yyyy-MM-dd，并且不包含时间部分。您可以使用 CAST 函数以及可用的 日期时间函数 将字符串转换为日期。以下是一些快速示例：

Copied!
1-- Convert a string with the format 'yyyy-MM-dd' to a date
2CAST('2016-07-30' AS DATE)
3
4-- Convert a string with the format 'yyyy-MM-dd' to a timestamp (the timestamp will be based on midnight of the date provided)
5CAST('2016-07-30' AS TIMESTAMP)
6
7-- Convert a date string with the format 'ddMMyyyy' to a date
8TO_DATE(CAST(UNIX_TIMESTAMP('07302016', 'MMddyyyy') AS TIMESTAMP))
9
10-- Convert a timestamp string with the format 'ddMMyyyy HH:mm:ss' to just a date
11CAST('2016-07-30 11:29:27' AS DATE)
12
13-- Extract just the date from an ISO 8601 timestamp
14TO_DATE('2016-07-30T11:29:27.000+00:00')
15
16-- Extract just the date from a timestamp with the format 'yyyy-MM-dd HH:mm:ss'
17TO_DATE('2016-07-30 11:29:27')

请注意，如果字符串不是以日期/时间戳格式编写的，则不能将字符串转换为日期。因此，像 CAST('20160730' AS DATE) 这样的内容将返回 null。

WINDOW

描述： 给定一个指定时间戳的列，生成滚动或滑动的时间窗口。如果未提供slideDuration，则生成滚动时间窗口。如果提供了slideDuration，则通过将行分到一个或多个时间窗口中来生成滑动时间窗口。

窗口开始是包含的，窗口结束是不包含的，因此12:05将包含在窗口[12:05,12:10)中，但不在[12:00,12:05)中。窗口可以支持微秒精度。按月计算的窗口不被支持。

窗口从1970-01-01 00:00:00 UTC开始，除非提供了startTime。 此函数接受以下参数：

  • timeColumn：用于按时间进行窗口划分的列。时间列必须为TimestampType（时间戳类型）。
  • windowDuration：一个字符串，指定窗口的宽度，例如10分钟，1秒。请注意，持续时间是一个固定长度的时间，不会根据日历而随时间变化。
  • slideDuration：一个字符串，指定窗口的滑动间隔，例如1分钟。每个slideDuration将生成一个新的窗口。必须小于或等于windowDuration。此持续时间同样是绝对的，不会根据日历而变化。
  • startTime: 与1970-01-01 00:00:00 UTC的偏移量，用于开始窗口间隔。例如，为了获得每小时滚动窗口，从每小时的15分钟后开始（例如12:15-13:15, 13:15-14:15…），请提供startTime作为15分钟。

签名：

Copied!
1WINDOW(Column timeColumn, String windowDuration)WINDOW(Column timeColumn, String windowDuration, String slideDuration)WINDOW(Column timeColumn, String windowDuration, String slideDuration, String startTime)
