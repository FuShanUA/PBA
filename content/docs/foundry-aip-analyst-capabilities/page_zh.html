功能

AIP 分析师使用工具来搜索、分析并呈现您问题的答案。您可以通过从 工具 菜单中选择复选框来自定义可用的工具。

搜索

这些工具可以让 AIP 分析师通过模糊关键词匹配在您的本体论中查找相关资源。搜索范围设置位于 设置 菜单中，适用于这些工具。

  • 对象类型搜索：根据对象类型元数据（如显示名称、ID、描述或状态）识别相关对象类型。
  • 对象搜索：在完整的本体论或指定的对象类型中进行搜索，返回按对象类型分组的匹配对象。

查找

这些工具通过其标识符加载特定资源的信息。

  • 对象类型查询： 检索特定对象类型的全面元数据，包括属性和与其他对象类型的关联。
  • 对象查询： 从给定的对象类型中检索特定对象，并返回该对象的所有属性值。
  • 函数查询： 检索函数的签名和源代码预览。
  • 操作类型查询： 检索特定操作类型的元数据。
  • 数据集查询： 查找并预览数据集，包括模式信息和示例数据。
  • 支撑数据集查询： 检索给定对象类型的支撑数据集，使用户可以直接对本体论数据进行数据集级别的分析。
  • Workshop 查询： 检索 Workshop 模块所引用的对象类型、关联、函数和操作类型。
  • 便签查询： 检索便签文档的内容。
  • 媒体集项查询： 从媒体集中检索特定的媒体项。

查询与分析

这些工具生成的结果可以被 AIP 分析师用作进一步分析的构建模块，例如对象集合、聚合结果和 SQL 查询结果。

  • 对象集： 创建一个对象集，可通过应用过滤器、搜索或语义搜索等操作进行可选的筛选或转换。
  • 导入对象集： 将现有的对象集导入到当前分析上下文中。
  • 本体论聚合： 对对象集执行聚合操作，并可选择分组属性。支持的操作包括计数、求和、平均值、最小值、最大值、百分位数、基数、标准差和方差。
  • 本体论 SQL： 对对象集执行 SQL 查询，返回可用于复杂分析或进一步查询的表格数据。查询可以链接在一起，每个查询都可以引用前一个查询的结果。
  • 数据集 SQL： 对数据集执行 SQL 查询以检索和分析数据，返回可用于进一步分析或链接到后续查询的表格数据。支持引用多个数据集分支。

函数和操作

AIP 分析师可以在分析过程中执行函数和操作。

  • 函数执行： 执行函数以执行计算或数据转换。您可以直接在分析会话中提供输入并查看输出。
  • 操作类型执行： 执行操作以创建或修改对象。操作在执行前需要审批，如需可进行回退。

可视化

AIP 分析师可以使用多种可视化工具展示其输出结果：

  • Vega 图表： 从表格数据（如本体论聚合、SQL 结果或数据集查询）构建交互式图表。
  • 地图可视化： 在交互式地图上可视化对象的地理空间数据。

文件和媒体支持

您可以将文件上传到分析中，供 AIP 分析师处理：

  • 电子表格： Excel (.xlsx 和 .xls) 和 CSV 文件。
  • 文档： Word (.docx) 文件。嵌入的图片会与文本一起提取，因此代理可以解释文档中的图表、图示和截图，而不仅仅是文字内容。
  • 图像： JPEG、PNG、GIF 和 WebP。
  • PDF： 处理文本和视觉内容，因此代理可以解释页面上渲染的图表、图示和表格。

您还可以从媒体集合中附加媒体项作为分析上下文。

附加功能

  • 澄清问题：当查询存在歧义时，AIP Analyst 可以提出澄清性问题，在继续之前优化其分析方法。
  • 上下文清理：AIP Analyst 会自动隐藏对话早期的过时或不相关信息，使代理专注于与您当前问题相关的数据。
  • 管理工具：在分析过程中启用或禁用工具，使 AIP Analyst 能够专注于与当前问题相关的功能。您仍然可以从工具菜单手动配置工具。
