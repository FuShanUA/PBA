用户与用户组

用户

Foundry 的访问权限通过一个或多个身份提供程序进行管理，由其在用户登录时完成身份验证。身份提供程序还会提供用户及其属性和用户组的相关信息。Foundry 可与现有的身份提供程序无缝集成，允许通过现有系统实现端到端的访问管理。请参阅《身份验证与组织分配》，了解如何通过控制面板配置身份验证的更多详情。

用户是经过身份验证并有权访问 Foundry 的个体。用户通常由外部身份提供程序（例如 Active Directory 系统）进行定义。用户拥有属性，这是关于该用户的公开结构化信息。

了解有关《用户管理》的更多信息。

用户组

用户组是用户的集合。借助用户组，管理安全组并为多名用户定制 Foundry 用户体验将更加便捷。针对项目及资源的访问权限，通常直接授予用户组而非单个用户。《保障数据基座的安全》文档重点介绍了如何使用用户组来管理权限。

用户组通常由外部身份提供程序（例如客户的 Active Directory 系统）进行定义，并贯穿应用于整个 Foundry 平台。
