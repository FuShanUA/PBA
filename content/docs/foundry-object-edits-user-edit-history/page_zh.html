启用用户编辑历史

跟踪对Object Storage V2中索引的对象的用户编辑历史。通过Ontology Manager的数据源选项卡中的跟踪用户编辑历史切换开关来启用或禁用此功能，如下图所示。

要允许用户编辑，必须启用编辑切换开关。

常见问题和注意事项

  • 编辑历史反映了启用跟踪用户编辑历史后对对象所做的更改。在启用此功能之前所做的任何更改都不会被记录。
  • 一旦你启用了跟踪用户编辑历史 ，初始化需要几分钟。在此短时间内，用户无法对这些对象执行操作。
  • 如果您从对象存储V1迁移到对象存储V2，请确保在迁移过程中勾选preserve edit history，以便保留编辑历史记录。
  • 可以访问某个对象（具有相同主键的对象）当前状态的用户可以访问该对象的全部历史记录。这意味着如果一个对象被删除并重新创建，用户仍然可以看到删除操作之前发生的历史记录。

启用跟踪用户编辑历史后，可以将编辑历史小部件添加到Workshop模块或Workshop支持的对象视图中以显示编辑历史。

禁用编辑历史

禁用跟踪用户编辑历史 永久删除此对象类型的现有编辑历史记录。在保存本体论之前，用户将收到警告，并必须确认将删除编辑历史的声明。
