迁移现有项目

除了创建新用例项目外，用例应用还支持将现有 Foundry 项目迁移为用例支撑项目。

请按照以下步骤将现有项目迁移为用例。

  1. 首先，导航至需要迁移为用例的 Foundry 项目。在 Foundry 导航侧边栏中，选择 Projects & files > Projects 以查看可用项目。找到包含用例所需资源的项目，并打开该文件夹。

  2. 在项目概览页面，选择 Actions > Create use case。此操作将启动现有项目向新用例的迁移流程。

  1. 迁移完成后，将打开一个新页面，显示新用例的概览以及该项目中包含的所有资源。
