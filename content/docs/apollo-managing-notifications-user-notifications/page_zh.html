用户通知

本节将介绍如何接收资源的内置应用通知和电子邮件通知。

为资源启用通知

您可以订阅资源的内置应用和电子邮件通知。电子邮件通知会自动发送到您关联的电子邮件地址。要为资源启用通知，请执行以下操作：

  1. 导航到资源页面，例如环境、产品或团队页面。
  2. 在右上角找到操作图标，然后为您自己切换开启内置应用或电子邮件通知。



  3. 在设置中自定义此资源的通知偏好。这将决定您将收到哪些通知类型。



    选择您希望接收的通知类型并保存您的偏好。
  4. 要接收所有未来通知类型的提醒，请使用切换按钮订阅该资源的所有通知。



  5. 在资源的设置页面，团队管理员可以将联系人订阅到特定资源。这对于没有联系人团队的资源非常有用。



    无法在资源上创建新的联系人。要添加其他联系人，请打开该团队的Hub设置。

禁用通知

有以下两种方式可以取消对资源的通知订阅：

  1. 通知弹窗：打开每个资源右上角的菜单。取消订阅每个资源。
  2. 前往Hub设置并查看您已订阅的所有资源。
