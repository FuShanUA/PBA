将流水线添加至 Marketplace 产品

使用 Foundry DevOps 将您的 Pipeline Builder 流水线包含在 Marketplace 产品中，供其他用户安装和复用。了解如何创建您的首个产品。

支持的功能

除了以下情况外，Marketplace 产品支持所有 Pipeline Builder 功能：

  • 带有时间序列目标的流式流水线
  • 以下类型的参数：结构体类型常量、非由常量组成的复杂表达式、选项以及结构体定位器。

使用 Marketplace linter 检查 Marketplace 兼容性

在 Pipeline Builder 中，您可以使用 Marketplace linter 检查流水线是否与 Marketplace 兼容。要启用此功能，请导航至设置，并在您的流水线中选择启用 Marketplace 验证。此设置默认未启用。

启用后，流水线底部的流水线警告部分将显示任何阻止您的流水线在 Marketplace 中打包的错误。

如果不存在 Marketplace 不兼容的情况，错误/警告抽屉中将不会显示任何Marketplace 打包警告。请注意，其他类型的流水线错误或警告可能仍会出现。

将 Pipeline Builder 流水线添加至产品

要将 Pipeline Builder 流水线添加至产品，请先创建产品，然后添加输出。选择添加文件选项，从 Compass 文件系统中导航至该流水线并将其添加至您的产品中。

流水线参数

您可以使用流水线参数让安装者在安装时自定义流水线。例如，您可以使用 boolean 参数，根据安装者的输入选择遵循流水线的一个分支或另一个分支。有关支持的参数类型列表，请参阅支持的功能。当您打包带有参数的流水线时，该参数将作为流水线的依赖项以及安装者的输入显示，如下所示。

打包设置

要为安装者配置必需或可选的数据集和列，请导航至流水线输出面板 > 设置以访问打包设置。

默认情况下，Marketplace 安装需要所有列和输入数据集。如果存在任何非必需的列或输入数据集，您可以将它们标记为可选。如果用于流水线逻辑中，可选的输入数据集将默认为空，可选的列值将默认为 null。
