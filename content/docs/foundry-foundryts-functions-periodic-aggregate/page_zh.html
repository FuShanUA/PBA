foundryts.functions.periodic_aggregate

foundryts.functions.periodic_aggregate(aggregate, window, window_type='end', alignment_timestamp=0)

返回一个函数，该函数对单个时间序列的离散、周期性窗口进行值聚合。

周期性窗口将时间序列划分为固定持续时间的窗口。对于每个窗口，会将聚合函数应用于该窗口内的点。结果是一个时间序列，其值表示每个窗口的聚合结果。没有数据点的窗口不会包含在输出中。

此方法适用于对定期数据进行汇总，例如从连续的数据流中生成每小时、每天或每周的汇总数据。

支持的聚合函数：

聚合函数描述
min每个周期窗口中的最小值。
max 每个周期窗口内的最大值。
count每个周期窗口内的点数。
sum每个周期窗口内值的总和。
product每个周期窗口内值的乘积。
mean每个周期窗口内值的平均值。
standard_deviation每个周期窗口内值的标准差。
difference每个周期窗口中当前第一个点的值与最后一个点的值之间的差值，提供固定窗口内的相对变化。
percent_change每个周期窗口内第一个点的值到最后一个点的值的百分比变化，提供固定窗口内的相对变化率。
first每个周期窗口中的第一个值。
last每个周期窗口中的最后一个值。
  • 参数：

    • aggregate （字符串 ) – 要应用的聚合函数，请从上面的聚合函数表中选择一个有效选项。

    • window (int,datetime.timedelta,str) – 每个周期窗口的持续时间，例如 5ms，或 5e6。

    • window_type (str,可选) –

      要应用的窗口类型（默认为“end”）：

      start: 窗口在开始处包含，在结束处不包含。聚合的时间戳是窗口的开始时间。 end: 窗口在开始处不包含，在结束处包含。聚合的时间戳是窗口的结束时间。

    • alignment_timestamp (str|float|datetime.datetime,可选 ) – 用于对齐结果的时间戳，这样结果时间序列中的刻度将位于从对齐时间戳开始的窗口持续时间的整数倍处（默认值为0）。

  • 返回值： 一个函数，它接受单个时间序列为输入，并为每个周期性窗口计算指定的聚合值。
  • 返回类型： (FunctionNode) -> FunctionNode

Dataframe schema

列名类型描述
timestamppandas.Timestamp该点的时间戳
valueUnion[float, str]该点的值
另请参见

cumulative_aggregate(), rolling_aggregate()

注意

此函数仅适用于数值系列。
