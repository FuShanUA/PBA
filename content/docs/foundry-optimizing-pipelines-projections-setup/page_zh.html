设置投影

以下信息将指导您完成启用、配置和构建数据集投影的过程。

Noho 是一项管理数据集投影的服务。

为数据集启用投影

通过在数据集架构中配置 noho: true 来启用投影。

在通过转换写入数据集时，或者在详情选项卡中手动修改架构，都可以配置数据集的架构。

导航至“投影”选项卡

查看数据集时，如果其架构中配置了 noho: true 且您拥有编辑该数据集的权限，您将看到一个投影选项卡。

打开创建对话框

选择 添加新投影。

选择投影列

选择投影中要包含的列。

在多数情况下，选择 所有列 即可。不过，如果明确知道查询只会涉及部分列，也可以对此进行调整。

选择投影类型

选择投影的类型。

  • 对于 过滤优化投影，选择要过滤的列。
    • 顺序很重要，因为投影仅能加速对该列表前缀的查询。
  • 对于 连接优化投影，选择连接列和桶计数。
    • 仅会加速对这组确切列的连接操作。
    • 当与显式分桶的数据集或另一个连接优化的数据集进行连接时，桶计数必须保持一致。

创建投影

点击 创建投影 按钮。

投影现已存在，但暂无数据。投影旁边的红色警告图标即表示此状态。要在查询中使用该投影，需按后续步骤先完成构建。

设置构建

为了更好地控制资源使用，维护投影的内部构建任务不会自动调度，需手动设置。

首先，开启 在当前分支启用投影构建 开关。这允许在当前分支上运行构建。

接着，为构建配置调度计划。若要在其他分支上调度构建，需切换至该分支并重复此操作。

（可选）构建投影

若不想等待调度构建，可直接点击 构建 按钮手动构建投影。

接下来，等待构建完成。在投影数据更新到位前，可能会执行多次构建。投影构建状态 行旁出现绿色对号，即表示投影已完全更新到位。

投影现已更新完毕，后续将用于该数据集的读取操作。
