设置投影

以下信息将指导您完成启用、配置和构建数据集投影的过程。

Noho 是一个管理数据集投影的服务。

为您的数据集启用投影

通过配置 noho: true，可以在数据集的 schema 中启用投影。

您可以在从转换中编写数据集时配置数据集的 schema，或者通过手动修改 详细信息 选项卡中的 schema。

导航到投影选项卡

当查看数据集时，如果该数据集在模式中配置了noho: true，并且您有编辑数据集的权限，您将看到一个投影（Projection）选项卡。

打开创建对话框

选择Add new projection。

选择投影列

选择要包含在投影中的列。

大多数情况下，All columns 是合适的。但是，如果您知道查询只会选择部分列，则可以进行调整。

选择投影类型

选择投影的类型。

  • 对于 优化筛选的投影（Filter-optimized projections），选择要筛选的列。
    • 顺序很重要，因为投影只会加快此列表前缀上的查询。
  • 对于 优化连接的投影（Join-optimized projections），选择连接列和桶数量。
    • 只有在此确切的列集合上，连接才会被加速。
    • 当连接到显式分桶的数据集或另一个优化连接的数据集时，桶数量必须相等。

创建投影

点击 Create projection 按钮。

投影现在已存在，但不包含任何数据。这由投影旁边的红色警告图标表示。要将投影用于查询，必须首先按照下一步骤进行构建。

设置构建

为了给您更多对资源使用的控制，维护投影的内部构建不会自动安排；您需要显式地设置一个。

首先，切换开关 Enable projection builds on the current branch. 这允许在 当前 分支上运行构建。

然后，为构建配置一个计划。如果您想在不同的分支上安排构建，需要转到该分支并重复此过程。

(可选) 构建投影

如果您不想等待构建完成，可以通过选择构建按钮来显式构建投影。
