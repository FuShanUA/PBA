校验结果

Fusion 提供了一组校验公式。它们以按钮形式显示，点击后触发。如果校验失败，将显示一条错误信息说明原因。

校验函数最适合在串行操作（Actions）中使用，并结合 toast 操作来显示成功或失败的消息。详情请参阅《操作文档》。

本页概述了可用的校验公式。有关公式和参数的完整列表，请参阅《函数库》。

枚举

column_enum(...) 校验用于检查某列中的条目是否全部属于特定的值列表。这些列出的值可以手动输入到公式中（例如 array('red', 'blue', 'green')），也可以通过引用输入（例如 array_flatten(B3:B14)）

非空

column_not_null(...) 校验用于检查某列是否不存在空值。

数值

column_numeric(...) 校验用于检查某列是否仅包含数值。

表头

table_headers(...) 校验用于检查表中是否存在指定的列列表。

表键

table_key(...) 校验用于检查指定列列表中的条目是否唯一。

例如，对表的 name 和 age 列运行此校验，将检查 name 和 age 条目的组合是否唯一。如果表只有两行数据，如 ('David', 21), ('David', 24)，则会通过校验。但如果表中的行数据为 ('David', 21), ('David', 21)，则校验失败。请参阅以下示例数据截图作为说明。
