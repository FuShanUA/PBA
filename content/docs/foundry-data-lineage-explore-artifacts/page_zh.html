探索制品与本体实体

你可以在数据血缘中找到与数据集相关的 Foundry 制品和本体实体。通过数据血缘界面，你可以直接导航至这些资源，并查看它们如何融入你的本体。

查找相关制品

在数据血缘图中选择一个数据集，然后在右侧边栏选择 Related items 以展开 Related artifacts 面板。Related items 图标会显示一个徽标，标明与所选数据集相关的制品数量。在制品面板中，你可以查看贯穿 Foundry 平台的相关资源列表，包括 Contour 可视化和 Slate 应用。

点击资源旁边的节点图标可放大查看相关数据集，或点击资源以在新标签页的对应应用中将其打开。你可以对相关制品列表进行过滤以包含不同的项目类型，并按最早、最新、名称、路径或最后修改时间对列表进行排序。

查找本体实体

在血缘图中选择数据集，并在右侧边栏打开 View node properties 面板，即可找到由该数据集定义的对象类型。

在 About 标签页中，你可以看到使用所选数据集创建的所有对象类型。点击对象类型旁边的 Settings 图标，即可在新的本体管理器标签页中查看其配置。

你也可以使用右侧边栏的 Search Foundry 工具将对象类型添加到数据血缘中。通过基本或高级搜索找到对象类型，并从列表中选中以将其添加到你的图表中。随后，你可以查看与该对象类型相关的链接类型，并利用图表直观展示数据集与新增对象类型之间的关联。
