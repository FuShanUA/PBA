Google Data Catalog

Google Data Catalog 连接器是针对 Google Data Catalog 的 Palantir 提供的驱动程序。

要创建新的 Google Data Catalog 源，请按照 Palantir 提供的驱动程序的标准设置流程操作，然后参考以下章节进行 Google Data Catalog 的特定配置与网络设置。有关完整的属性参考，请查阅 Google Data Catalog 官方驱动程序文档 ↗。

配置

以下属性为必填或推荐配置。

属性是否必填？描述默认值
OrganizationId ↗必填关联至所要连接的 Google Cloud Platform 组织资源的 ID。—
ProjectId ↗必填关联至所要连接的 Google Cloud Platform 项目资源的 ID。—
AuthScheme ↗推荐连接至 Google Data Catalog 时使用的身份验证类型。OAuthJWT
InitiateOAuth ↗推荐指定获取或刷新 OAuth 访问令牌的流程，以便在已通过身份验证和授权的用户操作期间维持其访问权限。GETANDREFRESH
OAuthClientId ↗推荐指定创建自定义 OAuth 应用程序时分配的客户端 ID。（也称为使用者密钥）。此 ID 用于在 OAuth 授权服务器中注册自定义应用程序。—
OAuthClientSecret ↗推荐指定创建自定义 OAuth 应用程序时分配的客户端密钥。（也称为使用者密钥）。此密钥用于在 OAuth 授权服务器中注册自定义应用程序。—
OAuthJWTCert ↗推荐JWT 证书存储区。—
OAuthJWTCertType ↗推荐包含 JWT 证书的密钥存储区类型。GOOGLEJSONBLOB
OAuthJWTSubject ↗推荐应用程序正在请求委派访问权限的用户主体。—
