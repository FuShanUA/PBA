文档

您可以通过添加README文件，为Code Repositories中的项目向用户提供文档。Code Repositories中的README文件支持Markdown，以实现灵活、易于使用的格式和样式。

编辑或添加README.md文件到您的仓库中以开始使用。此页面包含有关可用于自定义README文件的其他功能的信息。

如果README文件不足以满足您的文档需求，您可以使用Code Repositories创建< a href="/docs/foundry/custom-docs/overview/">自定义文档，并可以作为平台内文档发布。

特性

Code Repositories为README提供了多种格式选项。

内联图片预览

您可以通过以下语法在Markdown文件的文本中内联显示来自您仓库的图像：

![File Name](/transforms-python/path/to/my/file.jpeg)

要将图像上传到代码仓库，您需要 将您的仓库本地克隆，将图像文件添加到本地仓库，然后将更改推送到服务器。

提及Foundry用户

要在Markdown文件中提及一个Foundry用户，请在其用户名前加上 @ 符号。这将创建对被提及用户的引用，并直接链接到他们的个人资料。

@username

引用Foundry资源

你可以通过将资源的资源ID直接粘贴到README的Markdown文件中来引用任何Foundry资源。以这种方式引用的资源会自动命名并链接到平台中的相应资源。

This repository will be deployed to ri.foundry.main.deployed-app.a00000aa-a000-000a-0000-000a0aa0a00a

在仓库中链接到文件

要创建对仓库内文件的链接，请使用 ___CODE_BLOCK_6 ___ 协议后跟文件路径；例如，___CODE_BLOCK_7 ___。以这种方式引用的文件在点击时会自动打开。这使您能够轻松引用和导航到仓库内的其他文件。

语法高亮

README文件支持代码块中的语言语法高亮，以提高代码的可读性。要使用语法高亮，请在开始代码块的分隔符后指定语言，如下所示：

Copied!
1def hello_world():
2    print("Hello, World!")

表格

您也可以使用标准的Markdown表格语法来创建表格：

| Header 1 | Header 2 |
| -------- | -------- |
| Cell 1   | Cell 2   |

链接

README中的URL和电子邮件地址会自动转换为可点击的链接。
