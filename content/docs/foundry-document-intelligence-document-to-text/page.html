Document-to-text media transformations

Document-to-text transformations provide a flexible and performant way to extract document content in multiple formats, including the ability to extract document layout structure, such as paragraphs, headers, and tables.

AIP Document Intelligence provides two document-to-text media transformation operations:

  • extractTextV2: Returns a list of strings containing extracted document text.
  • extractLayoutAwareTextV2: Returns a list of layout-aware text blocks across pages.

We recommend using the operations listed above, which replace the extractLayoutAwareContent and ocrOnPage operations.

You can reference an example operation signature below:

Operation signature parameters

  • config.mode: Controls how the document pages are interpreted.
    • ELECTRONIC: Treats all pages as electronic PDF files and extracts embedded/raw text. Use ELECTRONIC when you know your documents contain embedded text.
    • SCAN: Treats all pages as scanned images and performs Optical Character Recognition (OCR). Use SCAN when you know your documents are scanned images.
    • AUTO: Automatically decides per-page whether OCR is needed. Use only when you do not know ahead of time whether the PDF files are electronic, scanned, or mixed. AUTO mode runs with a small compute overhead.
  • config.format: Controls the output format.
    • TEXT
    • MARKDOWN
    • HTML
  • config.languages: Controls the languages to be detected, which is used for OCR in SCAN or AUTO mode.
    • language: The language code, such as KOR.
    • type: Always set to "language".
  • pageRange: Controls the pages to be extracted. In the operation signature above, the page range processes pages 0, 1, 2, 3, and 4. Use page ranges to improve build performance for larger documents. Instead of issuing one request per page, you can batch multiple pages together. A page range of roughly five to ten pages is an appropriate starting point, depending on document size, model limits, and rate limits.

Parameter considerations

  • You do not need to specify languages for electronic PDF files.
  • You do not need to specify languages if the document language is English.
  • For non-English scanned documents, specify the relevant OCR language.

Python examples

Reference the examples below to help you execute a document-to-text transformation using a Python transform or function.

1import polars as pl
2from concurrent.futures import ThreadPoolExecutor
3from transforms.api import Output, transform
4from transforms.mediasets import MediaSetInput
5from transforms.mediasets.utils._constants import MEDIA_ITEM_RID, MEDIA_REFERENCE, PATH
6
7
8THREAD_NUMBER = 20
9
10
11# @incremental(v2_semantics=True)  # Uncomment this line if incremental is needed.
12@transform.using(
13    output=Output(OUTPUT_DATASET_RID),
14    media_input=MediaSetInput(INPUT_MEDIA_SET_RID),
15)
16def extract(media_input, output):
17    media_refs = pl.from_pandas(
18        media_input.list_media_items_by_path_with_media_reference().pandas(),
19        schema_overrides={
20            MEDIA_ITEM_RID: pl.String,
21            MEDIA_REFERENCE: pl.String,
22            PATH: pl.String,
23        },
24    )
25
26    def process_batch(batch_df: pl.DataFrame) -> pl.DataFrame:
27        def create_page_tasks(row):
28            media_item_rid = row[MEDIA_ITEM_RID]
29            metadata = media_input.get_media_item_metadata(media_item_rid).document
30
31            if metadata is None:
32                raise ValueError(f"Media item {media_item_rid} is not a document")
33
34            if metadata.pages is None:
35                raise ValueError(f"Media item {media_item_rid} has no page count")
36
37            return [(row, page_num) for page_num in range(metadata.pages)]
38
39        def process_single_page(task):
40            row, page_num = task
41            media_item_rid = row[MEDIA_ITEM_RID]
42            media_reference = row[MEDIA_REFERENCE]
43
44            extraction_result = media_input.transform_media_item(
45                media_item_rid,
46                str(page_num),
47                {
48                    "type": "documentToText",
49                    "documentToText": {
50                        "operation": {
51                            "type": "extractLayoutAwareTextV2",
52                            "extractLayoutAwareTextV2": {
53                                "pageRange": {
54                                    "startPageInclusive": page_num,
55                                    "endPageExclusive": page_num + 1,
56                                },
57                                "config": {
58                                    "mode": "ELECTRONIC",
59                                    "format": "TEXT",
60                                },
61                            },
62                        },
63                    },
64                },
65            )
66
67            return {
68                "media_item_rid": media_item_rid,
69                "media_reference": media_reference,
70                "page_num": page_num,
71                "extraction_result": str(extraction_result.json()),
72            }
73
74        all_tasks = []
75        for row in batch_df.iter_rows(named=True):
76            all_tasks.extend(create_page_tasks(row))
77
78        with ThreadPoolExecutor(max_workers=THREAD_NUMBER) as executor:
79            results = list(executor.map(process_single_page, all_tasks))
80
81        return pl.DataFrame(results)
82
83    extracted_data = media_refs.lazy().map_batches(
84        process_batch,
85        schema={
86            "media_item_rid": pl.String,
87            "media_reference": pl.String,
88            "page_num": pl.Int64,
89            "extraction_result": pl.String,
90        },
91        streamable=True,
92    )
93
94    output.write_dataframe(extracted_data)

9def _create_transform_job(

12    fc = FoundryClient()

23def _is_transform_finished(media_set_rid: str, media_item_rid: str, job_id: str) -> bool:
24    fc = FoundryClient()
25    status = fc.media_sets.MediaSet.get_status(media_set_rid, media_item_rid, job_id, preview=True)
26    return status.status in ("SUCCESSFUL", "FAILED")

29def _get_transform_result(media_set_rid: str, media_item_rid: str, job_id: str) -> str:
30    fc = FoundryClient()
31    result = fc.media_sets.MediaSet.get_result(media_set_rid, media_item_rid, job_id, preview=True)
32    return result.decode("utf-8")

34
35def _run_transform_blocking(
36    media_set_rid: str, media_item_rid: str, transformation: models.DocumentToTextTransformation
37) -> str:
38    job_id = _create_transform_job(media_set_rid, media_item_rid, transformation)
39    while not _is_transform_finished(media_set_rid, media_item_rid, job_id):
40        sleep(0.5)
41    return _get_transform_result(media_set_rid, media_item_rid, job_id)
