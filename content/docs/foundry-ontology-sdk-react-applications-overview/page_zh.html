OSDK React 应用

开发者控制台支持应用构建者利用 React ↗ 打造完全可定制的用户界面，底层由本体软件开发工具包（OSDK）驱动。选择 React 构建应用，能够复用既有的开发经验与社区在这一技术上的深厚积累，从而快速交付一流的用户界面。以 Foundry 作为后端，可借助本体执行大规模查询与 Foundry 编辑的强大能力，结合细粒度的治理面，加速安全开发应用的进程，为组织运转提供动力。

本部分文档将引导您在 Palantir 平台中完成 React 应用的创建、开发与部署流程。

OSDK 文档

OSDK 与语言无关。除了提供用于 TypeScript 的 NPM（Node 包管理器）库外，我们同时也支持 Python 的 Pip 和 Conda、Java 的 Maven，以及适用于其他任意语言的 OpenAPI 规范。可通过以下链接查阅其他 OSDK 与开发者控制台文档：

示例与额外资源

可直接在平台的示例应用或开发者控制台主页探索 OSDK 参考示例。这些示例提供自学课程，演示了各种 OSDK 功能，助您开启构建专属应用的旅程：

  • Ontology SDK (OSDK) 入门：构建待办事项应用： 深入阅读这份分步教程，学习如何借助本体 SDK (OSDK) 的强大能力创建专属的待办事项应用。OSDK 允许您直接从开发环境访问本体的全部能力，并以 Foundry 作为后端来开发定制应用。这份详尽的指南将涵盖从搭建开发环境到部署最终产品的所有环节。

  • 结合 AIP Logic 的 Ontology SDK (OSDK)：构建由 AIP Logic 驱动的待办事项应用： AIP Logic 并非局限于 Foundry 内部，您可以随时随地调用它。深入阅读这份分步教程，学习如何利用 OSDK 的强大能力创建专属的待办事项应用。这份详尽的指南将涵盖从搭建开发环境到将应用投入生产环境的全过程，包括从 React 或 Python 应用中调用 AIP Logic 函数。

  • 使用 Ontology SDK (OSDK) 创建定制 Workshop 微件： 在 Workshop 中使用 Iframe 微件，将 Foundry 托管的 React Web 应用嵌入 Workshop，并实现双向状态共享。

每个示例均包含分步操作说明与代码示例，可按照个人节奏学习，通过实践掌握不同的 OSDK 特性与实现模式。
