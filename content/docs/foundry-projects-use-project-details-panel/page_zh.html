使用项目详情面板

在项目页面右侧选择相应的图标，即可打开资源详情面板。

文档

本部分提供了一个基于 Markdown 的富文本编辑器，可用于在项目或文件夹级别撰写文档，这与工作区内所有的文档部分保持一致。

活动

活动日志提供了贯穿整个项目的变更持续视图，且仅在项目级别可见。对于构建新项目或维护长期项目的团队而言，活动日志有助于更轻松地掌握近期动态与协作情况。请注意，活动日志仅保留最近一个月的活动记录。

可发现性

如有需要，您可以让其他组织的人员发现您的项目及其描述。此功能目前正在开发中。

访问

资源面板中的访问选项卡可用于管理项目内群组和用户的访问角色。

对于项目数据主人，此面板提供了添加所需标记、管理默认访问权限，以及通过向其他用户和群组授予角色来配置额外访问权限的界面。

对于具有查看者或编辑者角色的用户，访问面板会显示当前具备项目访问权限的群组概览。

详细了解《项目与角色如何组织工作并控制 Foundry 中的访问权限》。

通过使用工作区侧边栏中的“检查访问权限”面板或数据血缘关系工具，详细了解如何《检查某人对项目、文件夹或文件的权限》。

资源队列

资源队列选项卡可用于查看分配给该项目的资源队列。
