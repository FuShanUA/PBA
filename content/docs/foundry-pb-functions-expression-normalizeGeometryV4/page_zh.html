准备几何图形

支持：批处理、流处理

通过将几何字符串转换为有效的GeoJSON，为后续使用（例如索引到本体论）准备几何图形。多边形将被闭合和去重。跨国际日期变更线（由宽度 > 180度指示）的几何图形将在国际日期变更线的两侧拆分为多个要素。默认情况下，此操作将返回转换后的几何图形，如果字符串无法转换，则返回null。或者，在“显示错误”输出模式下，此操作将输出一个包含成功解析输出或描述性错误消息的结构。

表达式类别： 地理空间

声明的参数

  • 几何字符串: 要解析并转换为GeoJSON的几何字符串。
    表达式<String>
  • optional Output mode: Choose to output as a simple output where the output is the resulting geometry and errors are returned as null, or output a struct with two fields: an "ok" field with the resulting geometry if it can be converted; otherwise, an "error" field containing a descriptive error message.
    Enum<Simple, With errors>

Output type: Geometry | Struct<ok, error>
