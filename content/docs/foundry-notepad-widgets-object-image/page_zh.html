对象媒体预览

对象媒体预览 小部件允许您嵌入并预览来自 支持的媒体源 的图片、音频、视频或文档文件。

支持的媒体源

媒体引用

选择一个 媒体引用 类型的对象属性以预览底层媒体。

图像媒体引用可以调整大小。使用右下角的拖动手柄来调整图像大小。其他媒体类型不支持调整大小。

您可以配置应用于 导出 的文档媒体引用的打印设置。

  • 如果空间有限，在新页面上打印小部件（默认启用）：如果当前页面没有足够的空间，则在导出期间自动将小部件移动到新页面。
  • 打印时自动扩展小部件高度： 确保小部件扩展以显示整个文档，使所有内容在导出文件中可见。自动扩展最多限制为 15 个嵌入式页面。

其他媒体类型不支持打印设置。

媒体字符串

媒体字符串选项支持以下格式中字符串对象属性中的媒体引用：

  • 从媒体集：https://{my-foundry-url}/mio/api/media-set/{media set rid}/items/{media item rid}。
  • 从数据集：https://{my-foundry-url}/foundry-data-proxy/api/web/dataproxy/datasets/{dataset rid}/transactions/{transaction rid}/{filename} or https://{my-foundry-url}/foundry-data-proxy/api/web/dataproxy/datasets/{dataset rid}/views/{branch name}/{filename}。
  • 外部媒体 URL：请确保已在您的注册信息的 内容安全策略 设置中配置了访问权限。

附件

选择一个附件类型属性以预览附件媒体。有关配置附件属性的详细信息，请参阅 上传附件。

小部件属性

  • 对象选择: 您想要显示其属性的对象。
  • 要显示的属性: 包含或指向支持的媒体源的属性。

模板配置

  • 对象选择: 您想要显示其属性的对象。
