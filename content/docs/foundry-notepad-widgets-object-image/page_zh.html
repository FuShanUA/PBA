对象媒体预览

对象媒体预览微件允许您嵌入并预览来自受支持的媒体源的图像、音频、视频或文档文件。

受支持的媒体源

媒体引用

选择一个媒体引用类型的对象属性，以预览底层媒体。

图像媒体引用支持调整大小。拖动右下角的控制柄即可调整图像尺寸。其他媒体类型不支持调整大小。

您可以为文档媒体引用配置打印设置，这些设置将在导出时应用。

  • 空间不足时在新页面打印微件（默认启用）：如果在当前页面上没有足够的空间，则在导出期间自动将微件移至新页面。
  • 打印时自动扩展微件高度：确保微件扩展以显示整个文档，使所有内容在导出的文件中均可见。自动扩展最多限制为 15 个嵌入页面。

其他媒体类型不支持打印设置。

媒体字符串

媒体字符串选项支持在字符串对象属性中以下列格式引用的媒体：

  • 来自媒体集：https://{my-foundry-url}/mio/api/media-set/{media set rid}/items/{media item rid}。
  • 来自数据集：https://{my-foundry-url}/foundry-data-proxy/api/web/dataproxy/datasets/{dataset rid}/transactions/{transaction rid}/{filename} 或 https://{my-foundry-url}/foundry-data-proxy/api/web/dataproxy/datasets/{dataset rid}/views/{branch name}/{filename}。
  • 外部媒体 URL：确保已在您注册环境的内容安全策略设置中配置了访问权限。

附件

选择一个附件类型的属性以渲染附件媒体的预览。有关配置附件属性的详细信息，请参阅上传附件。

微件属性

  • 对象选择：您想要显示其属性的对象。
  • 要显示的属性：包含或指向受支持媒体源的属性。

模板配置

  • 对象选择：您想要显示其属性的对象。
