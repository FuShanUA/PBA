报告实体

Apollo 提供了在您的 Spoke 环境中跟踪已部署实体状态的能力。通过 添加实体 中描述的方法添加的实体称为“管理”实体。

也可以通过 Apollo 跟踪和报告非管理实体。Apollo CLI 命令 add-unmanaged-service [MavenCoordinate] --environment=[env-id] --stack=[stack] --service=[service-id] 可以使用已部署软件的坐标运行，以将其作为非管理实体添加或更新现有的实体。然后，该实体将在 Apollo 控制中心显示为一个实体，并显示在运行 CLI 命令时的已部署版本。

由于实体未由Apollo进行管理，Apollo控制中心中的已部署版本不会自动更新以反映软件的新部署版本。我们建议建立一个可以扫描非托管实体并将它们的最新版本报告给Apollo的流程。此流程可以设置为按计划运行，并通过CLI报告最新版本。这将确保Apollo控制中心中显示的版本不会落后。
