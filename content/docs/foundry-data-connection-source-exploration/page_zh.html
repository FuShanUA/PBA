数据源探索

配置数据源同步时，您可能希望在数据正式接入 Foundry 之前，先探索数据源及其包含的数据以预览同步情况。根据数据源类型的不同，Data Connection 允许您通过探索页面预览同步的数据。

要访问探索页面，请按照以下步骤操作：

  1. 进入数据源页面，点击您要配置的数据源名称。
  2. 在同步页面的探索与创建同步组件中，选择探索链接。

基于表的数据

  1. 左侧面板：从源系统查找并将表和视图添加到图表中。可以使用自由文本搜索助手查找特定表，也可以浏览树形结构查找资源。

  2. 图表：探索表、视图及其相互关系。您也可以在图表中右键点击表来创建同步。选中具有关联关系的表时，外键会在可展开的列列表和连线上方高亮显示。

    • 请注意，图表并不总是适用于基于表的数据探索。例如，在探索基于表的 REST API 模型时，由于对象之间没有明确的关联关系，图表将不会显示。

  3. 表详情：预览所选表的样本数据。

  4. 右侧面板：查看已选的表，并为这些表创建同步。

基于文件的数据

  1. 左侧面板：探索并选择包含待同步内容的目录，将其接入 Foundry。

  2. 预览：查看将要同步的文件预览。可以添加过滤器来控制哪些文件将被导入 Foundry。

  3. 右侧面板：查看已选的表，并为这些表创建同步。
