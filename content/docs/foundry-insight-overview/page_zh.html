洞察

Insight 是一个点击式应用程序，用于探索和分析 本体论 数据。逐步构建分析路径以过滤对象、遍历对象链接、聚合结果并创建可视化效果。Insight 允许您使用现有的或创建新的对象集，这些对象集可以保存、共享并在整个平台上使用。

主要功能

  • 分析路径：通过在本体论数据上链接过滤器、链接和转换来构建多步骤分析。
  • 对象类型链接：在对象类型之间跟随链接以过滤或切换到连接的数据。
  • 聚合操作：通过 Group By 操作对对象进行分组并计算指标。
  • 可视化：将属性可视化为分布、指标或直方图。
  • 地图： 与地理属性（如 GeoPolygon 或 GeoShape 类型）进行交互。地图上应用的过滤器会记录在分析路径中。
  • 集合操作： 使用并集、交集和减法操作，将对象集合或多个分析路径的结果进行组合。
  • SQL 查询： 编写 SQL 查询以进行高级分析并预览结果数据。
  • 数据回写： 直接从您的分析中使用 Foundry 操作创建、更新和删除对象。
  • 共享： 通过 Compass 权限与其他用户共享 Insight 工作簿和生成的对象集合。

何时使用 Insight

我们建议您在以下情况下使用 Insight：

  • 您的数据已在本体论中建模： Insight 在对象类型及其关系上运行，当您的数据已经作为本体论对象集成时，它非常适合。
  • 您需要在不编写代码的情况下执行分析： 可视化分析路径使构建复杂查询变得简单，逐步支持简单的和复杂的下钻工作流程。
  • 您想探索对象之间的关系： Link 步骤可让您跟随链接，以在本体论中发现连接的对象。
  • 您想将数据写回本体论： Insight 支持从分析结果中创建、更新和删除对象。

Insight 与对象浏览器

对象浏览器是 Foundry 的核心应用，为本体论提供主页。它是一个用于发现和搜索本体论的场所。对象浏览器还提供了一个本体论分析工具，允许对对象类型进行分析。Insight 扩展了对象浏览器的分析功能，现代化了界面，并增加了平台上的最新功能。

如果以下情况，Insight 是最佳选择：

  • 希望基于您了解的本体论数据进行分析
  • 是一名分析师或调查员，需要直观的界面和工具来支持您的分析流程
  • 希望在本体论上获得 Contour 分析体验

如果以下情况，对象浏览器是最佳选择：

  • 对本体论了解不多，需要学习可用的数据
  • 需要在整个本体论中搜索对象或关键字
  • 想要查看单个对象
