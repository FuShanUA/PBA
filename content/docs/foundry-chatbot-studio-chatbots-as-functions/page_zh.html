聊天机器人作为函数

AIP 聊天机器人可以作为 函数 发布，使它们可以在平台中任何可以执行函数的地方使用。例如，构建者可以将 AIP 聊天机器人作为函数发布，以便在 AIP 评估 中对其进行评估，通过 自动化 来自动化聊天机器人工作流，或者在 代码仓库 中使用聊天机器人。您可以配置函数版本，在每次发布聊天机器人时发布，或在每次保存聊天机器人时发布。

函数输入

以下为作为函数发布的聊天机器人所接受的输入：

  • userInput: 聊天机器人将作出回应的用户输入字符串。

  • sessionRid (可选): 当前会话的字符串标识符，格式如下：

    ri.aip-agents..session.{uuid}
    • 要与聊天机器人启动新会话，请将 sessionRid 输入留空。不要提供一个空字符串；请完全省略此输入。

函数输出

以下为作为函数发布的聊天机器人的输出：

  • markdownResponse: 由聊天机器人生成的最终文本响应，使用markdown格式进行格式化。

  • sessionRid: 当前会话的字符串标识符，格式如下：

    ri.aip-agents..session.{uuid}
    • 此RID会在所有函数执行中返回。
    • 将此值提供给下一个函数执行以继续会话。

  • 所有用于应用状态的应用程序变量都将作为可选输出添加。

    • 只有在变量值被更新时才会输出这些变量的值。未被更新的变量将为空。

将聊天机器人作为函数发布

  1. 要在Chatbot Studio中将聊天机器人作为Function发布，请点击右上角的发布按钮右侧的发布设置图标。这将打开一个发布设置对话框。



  2. 打开从聊天机器人发布函数选项，这将显示发布的函数的额外配置选项。



  3. 在扩展的发布设置对话框中，选择一个本体论并填写函数详细信息，例如函数名称、API 名称和描述。可选地，您可以通过在版本配置下选择复选框，在保存聊天机器人时选择发布次要函数版本。



  4. 在对话框底部选择发布聊天机器人和函数。

发布后，会弹出一个通知，告知您聊天机器人和函数已发布，并提供一个链接以在本体论管理器中查看发布的函数。您也可以通过打开发布设置并在已发布的函数下选择它来查看该函数。

使用AIP评估评估聊天机器人

将聊天机器人作为函数发布后，就可以使用AIP评估评估套件进行评估。要从聊天机器人工作室创建一个评估套件，请确保您的聊天机器人已作为函数发布，然后打开左侧工具栏上的评估选项卡。在此处，您可以选择创建评估套件，这会提示您为套件命名并选择与聊天机器人相同的项目。

Note that the evaluation suite must be in the same Project as the chatbot. To open the evaluation suite in AIP Evals after creation, select the arrow icon to the right of the created suite.

Keep the following in mind when setting up your test cases in AIP Evals:

  • To start a new session with your chatbot, make sure that the sessionRid is set to null. If a session RID is provided, the chatbot will continue an existing session, which is likely not the intent during testing.

  • Object set variables must be set to null or contain a real value. Object set variables cannot be empty.

For more information on evaluation suites, refer to the AIP Evals documentation.
