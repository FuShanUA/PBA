用户选择

使用 用户选择 部件，通过单选或多选下拉菜单来选择用户。

配置选项

  • 标签： 为部件设置可选标签，在下拉菜单上方显示文本。
  • 占位符： 设置可选的占位符文本，在部件未选择任何内容的状态下显示。
  • 已选用户：
    • 若选择模式设为“单选”，部件将显示为单选下拉菜单。
      • 输出变量： 部件的输出变量，用于存储用户选择的选项。若选择模式设为“单选”，输出变量将为包含所选用户 ID 的字符串变量。
      • 允许清除： 切换以启用/禁用对已选下拉菜单选项的清除操作。
    • 若选择模式设为“多选”，部件将显示为多选下拉菜单。
      • 输出变量： 部件的输出变量，用于存储用户选择的一个或多个选项。若选择模式设为“多选”，输出变量将为包含所选用户 ID 的字符串数组变量。
    • 指定 Multipass 组 ID： 提供一个包含组 ID 的字符串数组变量，将下拉菜单中显示的用户筛选为指定组内的用户。用户必须对配置的组在组织层面拥有 View group membership 角色，否则将看到权限错误，详见下文。

Multipass 组 ID 权限

若提供了 Multipass 组 ID，用户需对该组所属的组织拥有 View group membership 权限，否则在使用此部件时会遇到权限错误。此权限可在 设置 > 平台设置 > 组织 中授予，只需选择目标组织，接着点击 组织权限 旁的 管理 即可。有关管理组的更多信息，请阅读相关文档。
