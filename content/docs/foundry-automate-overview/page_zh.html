自动化

自动化是一个完全向后兼容的产品，它取代了 对象监控，作为平台中所有业务自动化的单一入口。

自动化 是一个用于业务自动化的应用程序。使用自动化，您可以定义持续检查或按计划执行的条件，以及在满足指定条件时自动执行的效果。

条件可以是 基于时间的条件（“每周一上午9点触发”），基于对象数据的条件 基于 Foundry 本体论（“当添加具有优先级 high 的 Alert 对象时触发”），或者基于时间的和基于对象数据的条件的组合。

可用效果：

  • 提交 Foundry 操作
  • 触发 AIP 逻辑函数
  • 执行 Foundry 函数
  • 发送带附件的平台和电子邮件通知

使用案例

Automate 可用于各种不同的自动化工作流程，包括：

  • 定时报告发送和摘要: 指定时间并自动向预定义的收件人列表发送周报。从 Notepad 或 Notepad 模板 生成的 PDF 可以自动附加到电子邮件中。
  • 数据警报: 定义并监视对象集合，并在满足特定数据条件时提醒用户；例如，当开放问题对象的总数超过阈值时。
  • 工作流自动化: Automate 可用于对符合指定标准的对象数据自动执行操作。一些可以自动化的任务包括：
    • 检查数据异常，并将这些对象自动传递给具有修复问题逻辑的操作。
    • link -> 关联 要翻译的内容： 当预配置的事件和时间条件满足时，监视建议或潜在操作并自动应用它们。此类操作可能包括通过Webhooks向外部系统发出API调用，直接在外部系统中应用更改。
  • 监视搜索： 配置自动化以在保存的对象探索有新结果时通知您，或者当搜索的所有结果中的聚合标准被满足时通知您；例如，所有传感器对象的最大温度超过阈值。

访问 Automate

要访问 Automate，请在 Foundry 导航侧边栏中选择 Automate 应用程序图标。按照我们的文档 开始使用 Automate 中的步骤开始。

Automate 专为业务自动化设计。如果您正在寻找用于数据连接和管道构建的健康监控，请参阅 健康检查 文档。
