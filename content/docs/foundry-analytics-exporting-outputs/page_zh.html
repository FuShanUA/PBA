导出输出内容

Foundry 允许符合条件的用户导出分析结果，以便在平台外使用。

支持的导出方式包括导出为 PDF、将可视化图表导出为图片以及导出为 CSV 或 XLSX 格式。

导出 PDF

Notepad 支持将文档下载为 PDF，并允许自定义页面方向、页眉和页脚，以及各个嵌入内容（图表、表格等）的显示样式。

了解如何开始使用 Notepad。

导出数据

您可以在 Contour、Quiver 和 Code Workbook 中将分析结果导出为 CSV 和/或 XLSX 格式。

Contour

从 Contour 导出数据有两种方式，两者的导出上限均为 100,000 行：

  • 在您的分析路径中添加一个导出看板。

  • 或者，您可以在分析中选中相关看板并打开表格面板，然后选择所需的导出选项。

Quiver

要从 Quiver 分析中导出数据，请从任意卡片的操作菜单中选择下载为 CSV。

Code Workbook

要从 Code Workbook 导出数据，请从已保存为数据集的转换操作菜单中选择下载为 CSV。默认的导出上限为 100,000 行。请注意，未持久化的转换结果无法下载为 CSV 格式。了解有关 Code Workbook 中持久化和未持久化转换的更多信息。
