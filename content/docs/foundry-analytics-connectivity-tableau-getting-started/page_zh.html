快速入门

本指南将教你如何通过 Tableau 向 Foundry 进行身份验证、选择数据集，并开始构建你的第一个交互式仪表板。

在 Tableau 中选择 Foundry 作为数据源

  1. 启动 Tableau 并选择 连接到数据。
  2. 在 到服务器 下，点击 更多。
  3. 搜索并选择 Palantir Foundry。

输入你的 Foundry URL

在 Palantir Foundry 对话框的 服务器 字段中输入你的 Foundry URL。你可以通过登录 Foundry、复制 URL 并删除 https:// 前缀以及 .com 之后的所有内容来获取你的 Foundry URL。

例如，如果你的 Foundry 链接是 https://myfoundrylink.palantir.com/workspace/home，你应在 Tableau 提示符中输入 myfoundrylink.palantir.com。

向 Foundry 进行身份验证

向 Foundry 进行身份验证有三种选项：Foundry OAuth、Foundry 令牌 或 Foundry OAuth 客户端凭据。OAuth 是在 Tableau Desktop 中开始开发的最简单方式。

如果在发布时希望报表依赖于查看报表的用户的权限，应使用 OAuth 方法。这将在用户访问报表时提示其进行身份验证。如果报表对所有查看用户使用一组静态凭据即可满足需求，出于长期可维护性的考虑，建议使用客户端凭据方法。请注意，最初配置客户端凭据方法需要 Foundry 管理员进行操作。

请注意，一旦发布，就无法再从 Tableau Server 网页界面更改身份验证方法。如需更改，请使用所需的身份验证方法在 Tableau Desktop 中重新发布该报表。

Foundry OAuth 身份验证

  1. 在 身份验证 下拉菜单中，选择 Foundry OAuth。
  2. 在 OAuth URL 字段中，输入 https:// 并附带你在 服务器 字段中输入的相同 URL。
  3. 选择 登录，你将被重定向到浏览器中的登录窗口。
  4. 登录后，选择 允许 以授权 Tableau 使用你的 Foundry 账户进行连接。
  5. 一旦你被重定向到显示“Tableau 创建此窗口用于身份验证。现在可以安全地关闭它”消息的网页，请关闭该窗口并返回 Tableau Desktop。

如果在登录 Foundry 时收到错误，可能是你的组织尚未为 Tableau 启用 OAuth 登录选项。在这种情况下，请联系你的 Foundry 管理员寻求支持，并提供《启用 OAuth 客户端的相关说明》作为参考。

使用 OAuth 发布报表

如果在使用 OAuth 时打算将报表发布到 Tableau Server 并嵌入凭据，你首先需要在 Tableau Server 上配置凭据，方法是在 Tableau Server 中导航至 我的账户设置 > 数据源的已保存凭据。有关更多信息，请参阅 Tableau 关于《管理数据连接的已保存凭据 ↗》的文档。

Foundry 令牌身份验证

  1. 在 身份验证 下拉菜单中，选择 Foundry 令牌。
  2. 在 令牌 字段中，输入唯一的《用户生成的令牌》。
  3. 点击 登录。

Foundry 第三方应用程序客户端凭据（自 Tableau 连接器 2.6.0 版本起可用）

在发布仅需静态凭据的报表时，建议使用第三方应用程序客户端凭据。此类凭据没有过期时间。

Foundry 管理员需要配置一个第三方应用程序。请参阅《相关说明》来配置该应用程序。选择 机密客户端 选项，并确保启用了 客户端凭据授予。不要 启用 Ontology SDK。

为第三方应用的服务用户授予相应的权限。Tableau 中的数据访问将反映该服务用户的访问级别。

在 Tableau 中，在身份验证下拉菜单下，选择Foundry OAuth 客户端凭证。在OAuth 客户端 ID字段中输入第三方应用的客户端 ID，并在令牌字段中输入客户端密钥。然后，选择登录。

如果正在 Tableau Server 中操作，并尝试将已发布报告的身份验证方法更改为客户端凭证，可能会由于 Tableau Server 的已知问题而遇到报错。建议从 Tableau Desktop 重新发布该报告，以使用客户端凭证身份验证。

选择数据集

按照上述说明操作后，应会在左侧边栏的连接下看到列出的 Foundry 环境。

要开始使用特定数据集，请使用左侧边栏的导航选项。在数据库部分，从下拉菜单中选择一个 Foundry 项目。然后在表部分，使用搜索字段按名称或完整路径查找数据集。

加载数据集后，选择连接模式（实时或提取），并按常规方式继续创建 Tableau 仪表板。

高数据规模与 Tableau 数据提取

可以通过使用《Tableau 数据提取》来处理更高规模的数据，并确保查询在由 Foundry SQL Server 处理时符合“直接读取条件”。有关直接读取功能的详细信息，请参阅《架构》。
