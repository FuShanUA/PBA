删除 Artifact 仓库

删除 Artifact 仓库会移除该仓库内的所有 Artifact。这属于破坏性变更，会影响到所有使用已删除 Artifact 的用户。请谨慎操作。

要删除 Artifact 仓库，请先导航至包含该 Artifact 仓库的项目。然后右键点击该 Artifact 仓库，选择移至回收站。

要永久删除 Artifact 仓库，请导航至项目中的回收站选项卡。右键点击该 Artifact 仓库并选择彻底删除。此操作无法撤销。

Artifact 仓库是可以恢复的。首先，导航至项目中的回收站选项卡。然后，右键点击该 Artifact 仓库并选择恢复。如果看不到回收站选项卡，请确认你当前处于项目概览页，而不是项目内的某个文件夹中。
