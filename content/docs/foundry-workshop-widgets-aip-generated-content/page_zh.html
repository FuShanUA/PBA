AIP 生成内容

AIP 生成内容微件可以显示 AIP Logic 函数的响应，或者以可视化方式流式传输大语言模型（LLM）的响应（类似于将 LLM 响应流式传输至变量事件）。

配置选项

AIP 生成内容微件提供以下三种生成响应的选项：

  1. Logic： 该选项需要选择一个 AIP Logic 函数。

  2. Direct to LLM： 该选项允许在微件中实时显示 LLM 的响应。其包含以下配置选项：

    • Prompt： 提示词的字符串变量。
    • Clear output when prompt changes： 启用后，当提示词变量发生变化时，将清空 AIP 生成内容微件的输出。当用户直接与提示词交互时，此功能十分实用。
    • Temperature： 模型所使用的温度值，介于 0 和 1 之间的数字。较高的值（如 0.8）会使输出更具随机性，而较低的值（如 0.2）则会使输出更加集中和确定。
    • Model： 要使用的语言模型。支持四种 OpenAI 模型：GPT-3、GPT-4、GPT-4 32K 和 GPT-4-Turbo。

  3. LLM via prompt function： 该选项与 Direct to LLM 相同，但提示词是一个函数而非字符串。

其他配置选项

  • Output variable： 存储响应的字符串或对象集合变量。
  • Icon： 显示在微件顶部的图标；默认为 Palantir 的 AIP 图标。
  • Button label： 按钮内显示的标签；默认为“使用 AIP 生成结果”。
  • Title： 显示在微件中心的标题。
  • Loading state message： 在响应加载过程中显示的消息。
  • Show loading spinner： 启用后，加载时将显示旋转加载图标，而非加载状态消息。
