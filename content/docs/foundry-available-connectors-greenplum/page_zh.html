Greenplum

Greenplum 连接器是 Palantir 提供的用于 Greenplum 的驱动程序。

要创建新的 Greenplum 源，请遵循 Palantir 提供的驱动程序的标准设置流程，然后使用以下章节进行 Greenplum 特定的配置和网络设置。有关完整的属性参考，请参阅 Greenplum 官方驱动程序文档 ↗。

配置

以下属性为必填或推荐选项。

属性是否必填？描述默认值
Password ↗必填指定进行身份验证的用户账户的密码。—
Server ↗必填服务器的主机名或 IP 地址。{serverAddress}
User ↗必填指定进行身份验证的 Greenplum 用户账户的用户 ID。—
UseSSL ↗必填此字段用于设置是否启用 SSL。TRUE
Database ↗推荐Greenplum 数据库的名称。—
Port ↗推荐Greenplum 服务器的端口号。5432

网络

下表列出了为保障源成功运行而必须允许其访问的域名。

对于每个域名，需添加相应的出站策略。如果源部署在本地且无法从 Foundry 直接访问，请改用智能体代理出站策略；同时，智能体宿主机本身也必须能够访问所列出的域名。有关详细信息，请参阅将智能体用作代理。

域名是否必需
<Server>:<Port>始终必需
<KerberosKDC>:88如果 AuthScheme=Negotiate
<KerberosServiceKDC>:88如果 AuthScheme=Negotiate 且 Kerberos 拓扑使用多个域
