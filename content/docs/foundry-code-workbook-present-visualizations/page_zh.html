呈现可视化图表

在 Code Workbook 中创建可视化图表后，你可能需要将其展示给协作者，而非仅限于 Code Workbook 环境内使用。

将可视化图表添加至 Notepad

Code Workbook 生成的可视化图表可展示于 Notepad 文档中。你可以将 Foundry 平台内的各类资源嵌入 Notepad 文档，涵盖 Contour 仪表板、Quiver 画布及 Code Workbook 可视化图表，并支持通过文本、表格和图片对文档进行批注。你还可以创建 Notepad 模板，使 Workshop 应用中的用户能够基于特定蓝图生成新的 Notepad 文档。了解有关 Notepad 的更多信息。

通过右键点击图上的节点并选择 复制到 Notepad，即可将 Code Workbook 可视化图表添加至 Notepad 文档。此时，该图片将复制至剪贴板，可将其粘贴至任意 Notepad 文档。

将可视化图表添加至报告

Foundry 报告可用性

自 2022 年 10 月 3 日起，Foundry 报告不再默认对新客户开放。如果你的使用场景需要访问 Foundry 报告，请联系你的 Palantir 代表。对于现有用户，依然支持使用 Foundry 报告来创建包含 Code Workbook 所生成可视化图表的仪表板。

Code Workbook 生成的可视化图表可添加至 Foundry 报告，用于展示与协作。你可以通过打开“可视化图表”选项卡并点击 添加至报告，或者右键点击图上的节点并选择 添加至报告，将可视化图表添加至报告。
