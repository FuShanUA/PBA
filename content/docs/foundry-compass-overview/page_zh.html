指南针

指南针 是 Palantir 平台的文件系统。您可以使用指南针来组织和管理您的项目、资源和文件夹。

平台中最基本的单元是 资源，它们类似于传统文件系统中的 文件。您可以在 项目 中存储资源，这些项目是协作空间，用于组织特定目的的人、资源和文件夹。文件 页面允许您在一个地方浏览、共享、保护并组织您的资源和项目。

要查找您的资源，请在工作区导航侧边栏中选择 文件。在页面顶部，您可以使用标签在 投资组合、项目、我的文件（仅对您可见）和 与您共享 之间切换。

Below the tabs, quick filter cards allow you to filter the view by portfolios, projects, or Promoted items. If you need access to a project, select Request access to submit an access request.

You can further refine the file list using the Filters panel in the left sidebar. Filter options include resource type, status, portfolio, project, organization, and tag. Select a project to access the project dashboard, or create a new project.

When you open a project dashboard, you can view the following areas: Files, Autosaved, References (file and external), Trash, and Sensitive Data Scanner.

  • Files: A collection of all resources within a project. Pinned resources appear at the top for quick access.
  • Autosaved: Resources created within the project that were automatically saved without a designated location.
  • References: A collection of resources that flow into the project, including file references and external references.
  • Trash: Resources deleted from the project that are available for recovery or permanent deletion.
  • 敏感数据扫描器： 一个视图，供操作用户查看 个人身份信息（PII） 检测结果。
