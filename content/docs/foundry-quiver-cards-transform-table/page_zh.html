转换表

转换表 是一种用于对对象或从对象派生的表格数据进行批量分析的容器。转换表允许用户根据输入对象（包括时间序列）的属性派生列，并为对象数据提供关联和分组功能。

由于这些操作的计算强度较大，转换表最多允许 50,000 行。

转换表可以让您：

  • 关联并聚合 对象；
  • 从对象属性中派生新列；
  • 通过创建或编辑值来丰富数据；
  • 批量对时间序列 列执行转换；以及
  • 以多种方式绘制 操作的输出。

添加转换

转换表搜索窗口，通过转换表中的添加转换按钮可访问，显示所有可用的转换，按转换操作类型分组，例如编辑列、过滤、时间序列操作或空值/错误处理。虽然大多数转换会添加相应类型的列，但有些转换会改变表格的结构（例如当与其他对象类型连接或对表格进行分类以聚合行时）。

也可以直接从 下一步操作菜单 中通过选择转换类别和一个转换来添加转换表。此操作将在分析中添加一个转换表卡片，并将所选转换应用于该表。

也可以对单个 支持的输入 应用转换

在转换表之外，仅对该值进行一次操作，而不是在表中作为批量操作。

要将转换卡片添加到分析中，只需使用顶部菜单栏上的搜索窗口进行搜索。

转换表输入

转换表可以接受各种类型的输入，包括对象集合、分类图表、时间序列图表、时间序列图、物化数据、透视表和其他转换表。用户也可以手动输入数据来构建一个转换表。

要将转换表添加到您的分析中，请在顶部的表格菜单中选择转换表。这将打开编辑器面板，在此您可以选择适用于您分析的现有输入。

输入：对象集合

对象集合是转换表最常见的输入。

您可以通过从下一步操作菜单中选择转换 > 转换表，从分析中的任何对象集合创建一个转换表。

请注意，每个转换表最多有50,000行（无论是从开始还是从连接），因此您只能创建包含少于50,000个对象的转换表。

当从对象集合创建转换表时，它将为集合中的每个对象有一行，一个代表每个对象唯一ID的主键列，以及每个对象的显著属性的列（如本体论中定义的），或者如果没有显著属性，则为对象上的前几个属性（最多20个）。

通过点击属性来添加或删除属性列，以及对象集合中的任何关联传感器对象。 表格右下角的按钮。将列拖动以在表格中重新排序。

您可以为转换表的每个转换步骤设置不同的列配置。有关如何在下面的章节中格式化转换表，请参阅 格式化转换表。

输入：分类图表

条形图、折线图和散点图都可以作为转换表的输入。使用这些输入将在您的图表上显示类别和值，并不会引入底层数据。要引入底层数据，您应该使用创建图表的对象集。您可以通过选择图表，点击 下一步操作 菜单，然后选择 表 > 转换表 来从分类图表创建一个转换表。

输入：时间序列图

时间序列图可以作为表格的输入，将时间序列数据转换为表格格式，以便进行操作、编辑或增强。

要从时间序列图创建一个转换表，请从图表中选择一个特定的时间序列图。然后，在时间序列图底部的下一步操作菜单中选择表格 > 转换表。

然后，在转换编辑器面板中选择可用的范围选项：

  • 使用输入时间序列的完整时间范围（默认）。
  • 指定要导入的时间范围，可以通过手动输入起始和结束时间戳，或者通过引用一个范围参数来完成。

转换表格由于性能原因最多限制为50,000行；转换表格会将数据拉取到前端进行操作，而不是推送到后端服务。因此，时间序列数据可能需要进行采样（分桶）以适应需求。

您可以在数据选项中选择如何将时间序列数据转换为表格格式：

  • 在“已采样”设置中，表格将显示每个分桶的边界时间戳和值（最早和最新，以及最小和最大），平均值和点数。您可以调整采样分桶的数量。
  • 在“未采样”设置中，表格将显示原始未采样的数据（“tick数据”），并包含三列：主键、时间戳和值。

时间戳系列数据默认以UTC显示。可以在转换表格编辑器的列设置中将时间戳时区更改为用户的时区或不同的静态时区。

输入：时间序列图表

时间序列图表可以作为表格的输入，将图表中的每个时间序列图作为表格中的一行。

要从时间序列图表创建转换表，请选择时间序列图表（而不是特定的时间序列图）。然后在时间序列图表底部的下一步操作菜单中选择表格 > 从图表的时间序列创建表格。

输入：透视表和其他转换表

可以将旋转后的数据作为转换表的输入，使您能够使用公式对透视表列进行操作。与图表一样，从透视表创建转换表时不会引入其输入数据，而是直接使用旋转后的数据本身。 转换表也可以作为另一个转换表的输入。这在您希望将转换逻辑拆分为“逻辑步骤”块，或者将数据转换和数据格式化分开时非常有用。 您可以通过在表格卡片底部的下一步操作菜单中选择表格来从另一张表格创建转换表。

输入：手动输入转换表

用户可以手动输入最多5000行数据以创建新的转换表。手动输入转换表卡具有类似电子表格的用户界面，并支持五种数据类型：字符串（String）、数字（Number）、布尔值（Boolean）、时间（Time）和时间序列（Time Series） 用户可以以与其他转换表相同的方式与手动输入转换表进行交互并应用操作。

要在Quiver画布中添加新的手动输入转换表，请在分析标题中选择手动输入，或在搜索卡片对话框中搜索该表。

手动输入表格工作流程示例

  • 使用表格值对分析进行参数化：将手动输入转换表与行和列选择器结合使用，以在不同行之间切换。表格中所选行的值可以在后续分析中作为动态参数使用。

  • 从外部源导入小数据集以补充分析和本体论： 用户可以将电子表格中的元数据复制粘贴到新表中，通过在对象集下方的下一步操作菜单中搜索连接到转换表，将其连接到现有对象集。

从已连接的对象集和手动输入转换表中，配置连接到转换表转换，以指定连接条件。

  • 分析小规模的时间序列数据，无需设置时间序列同步：复制和粘贴表格形式的时间序列数据，并使用表格时间序列卡片将表格转换为时间序列图，以继续使用 Quiver 的完整时间序列操作功能 时间序列操作。

默认情况下，手动输入的转换表格会为每一行生成一个随机唯一的主键，但用户可以选择将某一列设置为表格的主键。主键会向下游传播，并在其他卡片中作为唯一标识符提供，例如转换表格行选择器。

对于大于5000行的时间序列数据，或者要在Foundry中重用时间序列数据，请设置一个时间序列同步以提高性能和可重用性。

输入：本体论 SQL（Ontology SQL）

本体论 SQL（Ontology SQL）的结果可以作为转换表的输入，将SQL查询结果转换为表格格式，以便进一步处理、丰富或可视化。

要从本体论 SQL 卡创建转换表，请选择本体论 SQL 卡，然后在卡底部的 下一步操作（Next Actions） 菜单中选择 转换 > 转换表（Convert > Transform tables）。

当本体论 SQL 卡被转换为转换表时，列对应于 SQL 查询返回的列。然后您可以对结果数据应用任何转换表操作，例如过滤、分组、派生新列或图表。

转换表输出

转换表的输出有四个主要类别：时间序列、图表、表格和值。

输出：时间序列

从转换表中输出时间序列数据有几种方法：

  • 当悬停在时间序列表格单元格上时，会显示一个“弹出”按钮。点击它将在一个新的时间序列图表中绘制时间序列。
  • 通过在下一步操作中选择时间序列 > 分组时间序列图，可以在单个时间序列图表中绘制时间序列列。
    • 表格底部的菜单。选择输入列和页面大小，以配置叠加的时间序列数量。
  • 将表格格式的时间序列数据（时间戳和值）转换为时间序列图，以使用Quiver中可用的完整时间序列可视化和转换功能。在表格底部的下一步操作菜单中选择时间序列 > 表格时间序列。从转换表格中选择时间戳（日期/时间）和值（数字）列，并可选地设置单位。请注意，如果表格数据中每个时间戳包含多个值点，仅该时间戳的第一个点会被绘制。

输出：图表

您可以从转换表数据创建折线图、分类散点图、条形图或Vega图。这些图表可用于Quiver中任何需要图表作为输入的功能，例如分类公式图或叠加图。

输出：表格

您可以基于现有的转换表创建一个新的 转换表，以提供一个可以单独格式化和组织的视图，而不会影响底层数据。

输出：透视转换表

您可以从转换表创建一个 透视转换表 来显示聚合数据。

输出：值

您可以对转换表中的列进行聚合，并将这些列作为数字或数组输出到指标卡片上。这些指标可以用于Quiver中任何需要数字或数组作为输入的功能。

计算新列

要添加一个计算列，请选择 添加转换，然后选择与您想要创建的数据类型相对应的选项卡：数值、日期/时间、字符串、布尔值、数组或时间序列。

有关可用于创建列的完整转换列表，请参阅 转换表转换索引。

连接

连接到关联对象

当转换表将一个对象作为输入时，您可以使用 本体论（Ontology） 将关联对象连接到该表中。这种转换称为 连接到关联对象（Join to Linked Objects），可以在转换菜单的 表格（Tables） 选项卡下找到。

当您将一个对象集连接到其他关联对象时，系统会提示您选择 链接类型（link type），这是您的起始对象与传入对象之间的连接。结果表将拥有一个新的连接主键；这是您的起始集主键和传入集主键的组合。新的连接主键将把传入对象的属性添加到每一行中，从而增加列的数量。

如果您起始的对象只连接到一个传入对象（即“一对一”或“多对一”的关联类型），行数不会增加。如果每个起始对象有多个传入对象（换句话说，是“一对多”的关联类型），则表中的行数会增加。

在下面的示例中，我们从 Companies 对象集开始转换表格。最初这个表格有 505 行（每对象一行）。然后我们添加了一个“关联对象”的 Join 转换，以添加关联的 Stock Event 对象。现在，表格有 11,149 行，主键是两个对象主键的组合，Stock Event 对象的列也被添加了进来。

关联

关联 转换可以在转换菜单的表选项卡下找到。类似于SQL的连接操作，它允许您在指定匹配条件（由下面的图片中的绿色框指示）的情况下，将两个转换表进行合并。

执行连接操作：

  1. 选择要连接的表。
  2. 设置连接类型（左连接、内连接或全连接）。
  3. 配置匹配条件：匹配样式（全部、任意）以及一个或多个来自输入表和连接表的列对进行比较。
  4. 选择要保留的输入表中的列。
  5. 选择要添加到输入表的连接表中的列。如果输入表中已存在与要添加的列同名的列，则会添加连接列的前缀。

交叉连接

交叉连接转换允许您通过执行 笛卡尔积 ↗ 连接来组合两个转换表。交叉连接会为第一个表和第二个表中每一行的配对组合生成一行。它不会基于特定列进行连接。

分组

分组是指在一些预定义的窗口或桶上对一组值进行聚合的操作。您可以在转换表中以两种方式实现这一点：

  • 按...分组
  • 先按...分组再连接（也称为连接分组）

按...分组

在 按...分组 中，您创建一个 类别 对于您“分组依据”列中的每个属性。对于每个类别，所有相关的基于时间的、数值型和时间序列的列都会变成值数组（也称为组）。可以在这些数组（组）上执行数组转换和聚合操作。

例如，下表显示了一组具有不同海拔值的全球气象站。我们可以按这些气象站所在的国家进行分组，以创建每个国家的海拔值数组。

如果我们想对这些海拔值进行聚合，例如计算每个国家的平均海拔，我们可以使用数值分组聚合转换，这将在表格中添加一列，返回我们选择的输入列的平均值（在这里，elevation_array）。

连接分组依据

时间序列

转换表旨在实现对时间序列的批量分析。这意味着现在可以同时对多个时间序列应用诸如过滤器、导数或累积聚合等转换。有关批量时间序列分析的完整指南，请参阅 批量分析时间序列。

如需了解包含的各个时间序列转换的更多信息，请参考 转换表转换索引。

根据要转换的数据类别，有三种方法可以将时间序列列添加到转换表中；这些方法包括：

  • 从一组时间序列对象的对象集中显示时间序列折线图（小图表）
  • 从根对象中添加传感器对象
  • 从一组时间戳和值创建时间序列

从时间序列对象集合中显示一个时间序列折线图（小折线图）

默认情况下（出于性能原因），除非您显式添加，否则不会显示时间序列数据的小折线图。如果您创建了一个包含时间序列或传感器对象的转换表，可以通过添加一个“时间序列对象”转换，并选择时间序列对象的主键来包含它们。

从根对象中添加一个传感器对象

在本体论中与它们连接了时间序列对象的已标记为根对象的对象，可以通过添加一个关联传感器列来显示其关联的时间序列。为此，请选择< strong >关联时间序列传感器并选择父对象的主键。

从一组时间戳和值创建时间序列

格式化转换表

转换表的列可以自定义宽度和高度，列中的具体值也可以进行格式化。所有的格式化操作都在显示选项卡中进行。

列格式化

所有列，无论是数字列还是其他类型的列，都可以静态设置列宽和最小高度。

可以通过添加 重命名列 转换从转换菜单中。

值的条件格式化

此外，数值列具有值和条件格式选项，包含以下控制项：

  1. D3/视觉格式: 这些数字格式选项与Quiver中用于格式化指标卡片或其他数值数据的选项类似。 了解更多关于D3格式化的信息。 ↗
  2. 对齐方式: 这些选项控制单元格中数字的放置；提供左对齐、居中对齐和右对齐设置。
  3. 条件着色: 这些选项允许根据阈值对值进行条件着色。 阈值可以静态设置，也可以配置为在你的Quiver分析中提取任何其他数值。 你可以选择对单元格的文字或背景进行着色。

格式化时间序列列

在转换表中的时间序列列会显示相关时间序列属性的折线图。默认情况下，折线图显示该系列中数据的完整范围。这可以从编辑器的显示选项卡中更改：

  1. 选择感兴趣的时间序列列的列标题中的向下箭头图标。这将打开一个列操作菜单。
  2. 选择配置显示设置选项。这是一个快速操作，会将转换表编辑器打开到显示选项卡中的相关配置部分。
  3. 通过从菜单中选择一个选项来更改折线图的视图范围。

您可以按照相同的步骤为对象集表中的折线图配置视图范围。

查看范围选项

有四种方式可以设置折线图的查看范围：

  • 全部数据范围： 默认选项。显示时间序列中的所有数据。对于大型时间序列，这可能会影响性能。
  • 固定范围： 将查看范围设置为静态的开始和结束日期，例如 2025-5-21 到 2025-06-04。
  • 相对范围： 将查看范围设置为相对于今天的范围。例如，从 2 weeks ago 到 Now。默认情况下，范围的结束日期设置为 Now，但可以通过在右上角选择 设置为相对时间 切换按钮来输入特定时间。
  • 初始默认时间轴范围： 将查看范围设置为初始化默认时间轴所使用的范围。此值由 全局设置 菜单控制。

常见问题

我可以使用转换表来创建按时间分组的时序吗？

是的。通过转换表可以创建按时间分组的时序（从条形图创建时序，然后将其分割为几个时序）。您可以使用“按时间序列分组”功能，利用一组日期或时间以及一组值来创建按时间分组的时序。

有关更多信息，请参见 转换表中的时序。

我可以格式化转换表吗？

是的，您可以使用参数或手动输入值，以便根据用户定义的逻辑规则对表格中的单元格进行格式化。

有关更多信息，请参阅 格式化转换表 的文档。

转换表的规模限制是什么？

出于性能原因，转换表有 50,000 行的限制，因为转换表会将数据拉取到前端进行操作，而不是推送到后端服务。

我可以在转换表中绘制数值吗？

是的。转换表有四种输出：时间序列、表格、图表和数值（指标卡片）。 有关更多信息，请参阅 转换表输出 的文档。

对象集是转换表唯一允许的输入吗？

对象集是转换表唯一允许的输入吗？

不。虽然在转换表中通常使用对象，但您也可以使用转换表对时间序列数据、图表数据或类似透视表或其他转换表的表格进行操作。

请注意，当处理图表数据或透视表时，行将失去与底层对象的连接，您只能对聚合数据进行操作，而无法返回到底层数据。这使得可以在透视表上执行列级操作，并且能够以表格形式创建和操作图表数据。

有关更多信息，请参阅 转换表输入 的文档。
