Troubleshooting

This page contains tips for troubleshooting issues that you may encounter while using the Palantir extension for Visual Studio Code.

Code Workspaces errors

VS Code in the Palantir platform uses the Code Workspaces infrastructure; refer to the Code Workspaces troubleshooting page for more information.

VS Code Workspace fails to load or displays a blank screen

If VS Code fails to load or only displays a blank screen, there might be an issue with the browser cache. In this case, perform a hard reload of your browser page to resolve your issue.

  • Hold Cmd (macOS) or Ctrl (Windows), then select Reload.
  • Use Cmd+R (macOS) or Ctrl+R (Windows) to reload.
  • In Chrome, open the Developer Tools panel by navigating to View > Developer > Developer Tools or using the F12 key. Then, right-click the Reload button to the left of the search bar and choose Empty Cache and Hard Reload.
