Foundry DevOps

Foundry DevOps 能够依托您组织的本体、AI 模型、数据管道、数据连接或端到端用例，帮助您快速开发并部署基于数据的工作流包，所有这些工作流均在 Foundry 中构建。

Foundry DevOps 的核心功能包括：

  • 将任意 Foundry 资源组合作为产品进行灵活打包。
  • 通过自动化版本与依赖管理，确保模块化组件在产品扩展时实现无缝同步。
  • 利用发布通道标记新产品版本，确保版本仅推送到符合条件的安装实例中。
  • 对安装实例集群进行管理。

Foundry DevOps 适用于以下用例：

  • 产品分发：在 Foundry Marketplace 中发布您的产品，供组织内其他安装者或更广泛的 Foundry 社区使用。
  • 构建生态系统：结合各生态系统参与者（客户、子公司、部门等）的输入数据，为他们分别安装您的产品。
  • 发布管理：创建与开发环境对应的安装实例，并为升级和发布通道自定义维护窗口。
  • 引导新用例：一次性安装您的产品，作为您最新自定义用例的起点。
