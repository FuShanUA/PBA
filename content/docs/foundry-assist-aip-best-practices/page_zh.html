AIP Assist 最佳实践

AIP Assist 是一个基于大语言模型的辅助工具，熟悉 Palantir 的产品文档及相关信息。您可以通过与 AIP Assist 交互来获得有关 Foundry 平台的帮助和支持。AIP Assist 具有对您所处的应用程序的上下文意识，但无法访问您正在处理的任何数据或元数据。

与 AIP Assist 互动的最佳实践

以下是一些提升您使用 AIP Assist 经验的建议：

  1. 具体明确： 当提出问题时，尽量做到具体明确。提供关于您询问的产品或功能的详细信息。这将有助于 AIP Assist 更好地理解您的问题，并为您提供更准确的回答。
  2. 提出清晰的问题： 避免在一句话中提出多个问题。相反，一次只提出一个问题以避免混淆。
  3. 提出后续问题：如果您不理解AIP Assist的回复或需要更多信息，请提出更多问题以澄清查询。尝试重新表述您的问题，看看是否能获得更准确的回复。
  4. 使用完整句子：与AIP Assist交互时请使用完整句子，以提供更多关于您希望回答的具体内容的上下文。
  5. 提供示例：提供您所寻找内容的示例。当展示具体的格式要求时，AIP Assist的响应效果更好。

给AIP Assist的问题示例

通过提出带有具体细节的清晰、专注的问题，您可以从AIP中获得最佳结果。以下示例问题更有可能从AIP Assist处获得有用且有帮助的回复：

  • 专注问题：如何解决错误信息“无法求解环境规范”？
  • 专注问题：你能解释一下如何为Contour配置时区设置吗？
  • 专注问题：你能提供一个在对象上编写聚合函数的示例吗？
  • 聚焦问题： 如何在Slate中配置CSS样式？
  • 聚焦问题： 你能写一个Python转换程序，处理名为“id”的列，提取连字符前面的两个字符字符串吗？例如，“AB-02”应返回“AB”，“cd-1”应返回“cd”，而“123-MA”不应匹配。

以下示例问题由于缺乏具体性、清晰度或细节，可能无法从AIP Assist获得有用的答案。

  • 非聚焦问题： 这个东西不能工作，哪里出问题了？
  • 非聚焦问题： 这个按钮是做什么的？
  • 非聚焦问题： 我怎么使用这个东西？
  • 非聚焦问题： 我怎样才能让它看起来更好？
  • 非聚焦问题： Python 提取两个字符字符串

注意：AIP的功能可用性可能会发生变化，并且可能因客户而异。
