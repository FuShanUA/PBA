变量转换

变量转换允许 Workshop 中的构建者对变量应用常用操作，并通过引用前序操作将多个操作串联起来。

转换类型

常规操作

  • 字符串拼接： 返回给定静态文本和/或变量值的拼接字符串。如果输入的是数组变量，该操作会将数组内的元素拼接并转换为字符串。构建者可以选择指定一个单独的输入项，用于在元素之间进行填充。
  • 条件分支 (If/else) 语句： 执行条件判断语句，返回预先定义的输出值或变量。
  • 类型转换操作： 在不同的基本数据类型之间转换变量。支持以下类型转换：
    • 字符串 → 数值：传入仅包含数字字符的字符串类型，将其转换为数值类型。
    • 字符串 → 日期：传入包含有效日期格式的字符串类型，并在 Parser 字段中选择对应的日期格式。例如，如果传入值为 06/26/24 的字符串变量，请选择 M/dd/yyyy 作为对应的解析格式，将其转换为日期类型。
    • 字符串 → 时间戳：传入包含有效时间戳格式的字符串类型，并在 Parser 字段中选择对应的时间戳格式。转换输出时间戳值所使用的时区，可以通过用户的本地时区定义，通过下拉选项静态设置，或者通过字符串引用或变量动态设置。例如，如果传入值为 2024 06 26 12:50 AM 的字符串变量，请选择 yyyy M dd hh:mm aa 作为对应的解析格式，将其转换为时间戳类型。
    • 字符串 → 地理点 (GeoPoint)：传入格式为 [纬度], [经度] 组合的字符串类型，将其转换为地理点类型。例如：40.782142,-73.96596。
    • 字符串 → 地理形状 (GeoShape)：传入采用 JSON 格式化、包含形状类型和坐标数组的字符串类型，将其转换为地理形状类型。例如：{"type":"Polygon","coordinates":[[[-73.9580917968885,40.80044406792189],[-73.98186087075722,40.76811683550534],[-73.97304465743831,40.76434392565562],[-73.94930767833077,40.79687062648023],[-73.9580917968885,40.80044406792189]]]}。
    • 时间戳 → 日期：传入时间戳类型，将其转换为日期类型。转换输出日期值所使用的时区，可以通过用户的本地时区定义，通过下拉选项静态设置，或者通过字符串引用或变量动态设置。输入的时间戳将根据指定时区的绝对时间戳转换为日期。
    • 日期 → 时间戳：传入日期类型，将其转换为时间戳类型。转换输出日期值所使用的时区，可以通过用户的本地时区定义，通过下拉选项静态设置，或者通过字符串引用或变量动态设置。输入的日期将转换为表示指定时区当天开始时刻的时间戳。
    • 任意类型 -> 字符串：可以使用字符串拼接操作将任何基本数据类型转换为字符串。

对象集转换

  • 是否为空： 对给定对象集中是否存在对象执行布尔值检查。
  • 是否不为空： 对给定对象集中是否存在对象执行布尔值检查。
  • 对象属性： 给定单个对象及该对象类型的指定属性，返回对应的属性值。
  • 对象集聚合： 对给定对象集的指定属性执行聚合操作。支持的聚合方式包括最小值、最大值、求和、平均值和基数。
  • 对象 RID： 返回给定对象的对象 RID。

数学操作

  • 加法： 返回给定数值或变量之和。
  • 减法： 返回给定数值或变量之差。
  • 乘法： 返回给定数值或变量之积。
  • 除法： 返回给定数值或变量之商。
  • 绝对值： 返回给定数值或变量的绝对值。
  • 取反： 返回给定数值或变量的相反数。
  • 向上取整 (Ceil)： 将给定数值或变量按指定精度向上取整后返回。
  • 向下取整 (Floor)： 将给定数值或变量按指定精度向下取整后返回。
  • 四舍五入：返回给定数值或变量按指定精度四舍五入后的值。
  • 最大值：从数值、日期或时间戳的集合或变量中返回最大值。
  • 最小值：从数值、日期或时间戳的集合或变量中返回最小值。

日期/时间运算

  • 相对日期：根据给定数值或变量计算日期，指定加减的天数、周数、月数或年数，以及日期值或变量。
  • 相对时间：根据给定数值计算时间，指定加减的秒数、分钟数、小时数、天数、周数、月数或年数，以及时间值或变量。
  • 日期之间：返回两个给定日期值或变量之间的数值差。返回的差值可按天、周、月或年计算。
  • 时间之间：返回两个给定时间值或变量之间的数值差。返回的差值可按秒、分钟、小时、天、周、月或年计算。
  • 当前日期：返回当前日期。

日期比较

  • 晚于或等于：执行布尔检查，判断第一个给定日期值或变量是否等于或晚于第二个给定日期值或变量。
  • 晚于：执行布尔检查，判断第一个给定日期值或变量是否晚于第二个给定日期值或变量。
  • 早于或等于：执行布尔检查，判断第一个给定日期值或变量是否等于或早于第二个给定日期值或变量。
  • 早于：执行布尔检查，判断第一个给定日期值或变量是否早于第二个给定日期值或变量。
  • 相等：执行布尔检查，比较两个给定日期值或变量是否相等。

时间比较

  • 晚于或等于：执行布尔检查，判断第一个给定时间值或变量是否等于或晚于第二个给定时间值或变量。
  • 晚于：执行布尔检查，判断第一个给定时间值或变量是否晚于第二个给定时间值或变量。
  • 早于或等于：执行布尔检查，判断第一个给定时间值或变量是否等于或早于第二个给定时间值或变量。
  • 早于：执行布尔检查，判断第一个给定时间值或变量是否早于第二个给定时间值或变量。
  • 相等：执行布尔检查，比较两个给定时间值或变量是否相等。

数值比较

  • 等于：执行布尔检查，比较给定数值或变量是否相等。
  • 不等于：执行布尔检查，比较给定数值或变量是否不相等。
  • 小于：执行布尔检查，判断第一个给定数值或变量是否小于第二个给定数值或变量。
  • 小于或等于：执行布尔检查，判断第一个给定数值或变量是否小于或等于第二个给定数值或变量。
  • 大于：执行布尔检查，判断第一个给定数值或变量是否大于第二个给定数值或变量。
  • 大于或等于：执行布尔检查，判断第一个给定数值或变量是否大于或等于第二个给定数值或变量。

字符串比较

  • 是：执行布尔检查，比较给定字符串值或变量是否相等。
  • 不是：执行布尔检查，比较给定字符串值或变量是否不相等。
  • 包含：执行布尔检查，判断第二个给定字符串值或变量是否是第一个给定字符串值或变量的子字符串。
  • 不包含：执行布尔检查，判断第二个给定字符串值或变量是否是第一个给定字符串值或变量的子字符串。
  • 开头为：执行布尔校验，判断给定的第二个字符串值或变量是否为第一个给定字符串值或变量的前缀。
  • 结尾为：执行布尔校验，判断给定的第二个字符串值或变量是否为第一个给定字符串值或变量的后缀。

布尔比较

  • 为真：执行布尔校验，判断给定的布尔变量是否为真。
  • 为假 (NOT)：执行布尔校验，判断给定的布尔变量是否为假。
  • 为空：执行布尔校验，判断给定的变量是否为空。
  • 不为空：执行布尔校验，判断给定的变量是否不为空。

地理空间

  • 从地理位置生成 Geohash：将给定的地理位置转换为字符串形式的 Geohash 值。
  • 从地理位置获取纬度：返回给定地理位置的数值型纬度值。
  • 从地理位置获取经度：返回给定地理位置的数值型经度值。
  • 从地理位置生成 MGRS：将给定的地理位置转换为字符串形式的 MGRS 值。

数组操作

  • 组合：返回包含所有给定数组值的数组。
  • 交集：返回仅包含给定数组间共有值的数组。
  • 更新指定位置元素：返回更新了给定数组中特定元素后的数组。构建者可通过指定代表元素在数组内位置的索引来指定要更新的元素。
  • 获取指定位置元素：返回与给定数组中指定索引处元素相对应的值，且类型与数组类型匹配。
  • 长度：返回代表给定数组内元素数量的数值。

数组校验

  • 包含：执行布尔校验，检查给定数组中是否存在指定的值。
  • 不包含：执行布尔校验，检查给定数组中是否不存在指定的值。
  • 为其子集：执行布尔校验，判断给定的数组是否为另一个给定数组的子集。
  • 为空：执行布尔校验，判断给定的数组是否为空。
    • 注意：如果选择了数组变量引用作为值，此转换将评估并显示为“为空”。
  • 不为空：执行布尔校验，判断给定的数组是否不为空。
    • 注意：如果选择了数组变量引用作为值，此转换将评估并显示为“不为空”。

结构体转换

  • 提取结构体字段：给定结构体和字段 ID，返回对应的字段值。详情请参阅《从结构体中提取字段》。
