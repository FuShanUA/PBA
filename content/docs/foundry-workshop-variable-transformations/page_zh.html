变量转换

变量转换允许Workshop中的构建者对变量应用常见操作，并通过引用之前的操作来串联操作。

转换类型

通用操作

  • 字符串拼接: 返回给定静态文本和/或变量值的字符串拼接。如果输入的是数组变量，该操作会将数组内的元素连接并转换为字符串。构建者可以选择性地指定一个单独的输入，以添加到各个元素之间。
  • if/else 语句: 运行条件语句以返回预定义的输出值或变量。
  • 类型转换操作: 在不同基本类型之间进行变量转换。支持以下类型转换：
    • 字符串 → 数值：传入仅包含数字字符的字符串类型以转换为数值类型。
    • 字符串 → 日期：传入包含有效格式日期值的字符串类型，并在Parser字段中选择对应的日期格式。例如，如果传入值为06/26/24的字符串变量，请选择M/dd/yyyy作为对应的解析格式以转换为日期类型。
    • 字符串 → 时间戳：传入包含有效格式时间戳值的字符串类型，并在Parser字段中选择对应的时间戳格式。在转换输出的时间戳值时，使用的时区可以使用用户的本地时区、通过下拉菜单中的选项静态设置，或者通过字符串引用或变量动态设置。例如，如果传入值为2024 06 26 12:50 AM的字符串变量，请选择yyyy M dd hh:mm aa作为对应解析格式以转换为时间戳类型。
    • 字符串 → 地理点位：传入格式为[latitude], [longitude]对的字符串类型以转换为地理点位类型。例如：40.782142,-73.96596。
    • 字符串 → 地理形状：传入一个以JSON格式化的字符串类型，包含形状类型和坐标数组，以转换为地理形状类型。例如：{"type":"Polygon","coordinates":[[[-73.9580917968885,40.80044406792189],[-73.98186087075722,40.76811683550534],[-73.97304465743831,40.76434392565562],[-73.94930767833077,40.79687062648023],[-73.9580917968885,40.80044406792189]]]}。
    • 时间戳 → 日期：传入一个时间戳类型以转换为日期类型。在转换输出的日期值时使用的时区可以通过用户本地时区、通过下拉菜单中的选项静态设置，或通过字符串引用或变量动态设置。输入的时间戳将根据指定时区的绝对时间戳转换为日期。
    • 日期 → 时间戳：传入一个日期类型以转换为时间戳类型。在转换输出的日期值时使用的时区可以通过用户本地时区、通过下拉菜单中的选项静态设置，或通过字符串引用或变量动态设置。输入的日期将根据指定时区的开始时间转换为时间戳。
    • 任意类型 → 字符串：可以使用 String concatenation 操作将任何基本类型转换为字符串。

对象集转换

  • 为空： 对给定的对象集中的对象是否存在进行布尔检查。
  • 不为空： 对给定的对象集中的对象是否存在进行布尔检查。
  • 对象属性： 给定一个对象和该对象类型的指定属性，返回属性值。
  • 对象集聚合： 对给定对象集的指定属性进行聚合。支持的聚合包括最小值、最大值、总和、平均值和基数。
  • 对象RID： 返回给定对象的对象RID。

数学运算

  • 加法： 返回给定数值或变量的总和。
  • 减法： 返回给定数值或变量的差。
  • 乘法： 返回给定数值或变量的乘积。
  • 除法： 返回给定数值或变量的商。
  • 绝对值： 返回给定数值或变量的绝对值。
  • 取反： 返回给定数值或变量的相反数。
  • 向上取整（Ceil）： 返回给定数值或变量到指定精度的向上取整值。
  • 向下取整（Floor）： 返回给定数值或变量到指定精度的向下取整值。
  • 四舍五入： 返回给定数值或变量到指定精度的四舍五入值。
  • 最大值： 从一组数值、日期或时间戳值或变量中返回最大值。
  • 最小值： 从一组数值、日期或时间戳值或变量中返回最小值。

日期/时间计算

  • 相对日期： 给定一个数值或变量，返回计算后的日期，指定要添加或减去的天数、周数、月数或年数，以及一个日期值或变量。
  • 相对时间： 给定一个数值，返回计算后的时间，指定要添加或减去的秒数、分钟数、小时数、天数、周数、月数或年数，以及一个时间值或变量。
  • 两个日期之间： 返回两个给定日期值或变量之间的数值差。返回的差值可以按天、周、月或年计算。
  • 两个时间之间： 返回两个给定时间值或变量之间的数值差。返回的差值可以按秒、分钟、小时、天、周、月或年计算。
  • 当前日期： 返回当前日期。

日期比较

  • 在或之后： 在给定的第一个日期值或变量是否在第二个给定的日期值或变量之后进行布尔检查。
  • 在之后： 在给定的第一个日期值或变量是否在第二个给定的日期值或变量之后进行布尔检查。
  • 在或之前： 在给定的第一个日期值或变量是否在第二个给定的日期值或变量之前或之时进行布尔检查。
  • 在之前： 在给定的第一个日期值或变量是否在第二个给定的日期值或变量之前进行布尔检查。
  • 相等： 进行两个给定的日期值或变量的相等性比较的布尔检查。

时间比较

  • 在或之后： 在给定的第一个时间值或变量是否在第二个给定的时间值或变量之后或之时进行布尔检查。
  • 在...之后： 对第一个给定的时间值或变量是否在第二个给定的时间值或变量之后进行布尔检查。
  • 在...或之前： 对第一个给定的时间值或变量是否在第二个给定的时间值或变量或之前进行布尔检查。
  • 在...之前： 对第一个给定的时间值或变量是否在第二个给定的时间值或变量之前进行布尔检查。
  • 等于： 对两个给定的时间值或变量进行相等性比较的布尔检查。

数值比较

  • 等于： 对给定的数值或变量进行相等性比较的布尔检查。
  • 不等于： 对给定的数值或变量进行非等性比较的布尔检查。
  • 小于： 对第一个给定的数值或变量是否小于第二个给定的数值或变量进行布尔检查。
  • 小于等于： 对第一个给定的数值或变量是否小于或等于第二个给定的数值或变量进行布尔检查。
  • 大于： 对第一个给定的数值或变量是否大于第二个给定的数值或变量进行布尔检查。
  • 大于等于： 对第一个给定的数值或变量是否大于或等于第二个给定的数值或变量进行布尔检查。

字符串比较

  • 等于： 运行一个布尔检查，比较给定的字符串值或变量是否相等。
  • 不等于： 运行一个布尔检查，比较给定的字符串值或变量是否不相等。
  • 包含： 在给定的字符串值或变量上运行一个布尔检查，以确定第二个给定的字符串值或变量是否是第一个字符串值或变量的子串。
  • 不包含： 在给定的字符串值或变量上运行一个布尔检查，以确定第二个给定的字符串值或变量是否是第一个字符串值或变量的子串。
  • 以...开头： 在给定的字符串值或变量上运行一个布尔检查，以确定第二个给定的字符串值或变量是否是第一个字符串值或变量的前缀。
  • 以...结尾： 在给定的字符串值或变量上运行一个布尔检查，以确定第二个给定的字符串值或变量是否是第一个字符串值或变量的后缀。

布尔比较

  • 为真： 在给定的布尔变量上运行一个布尔检查，以确定该变量是否为真。
  • 为假（NOT）： 在给定的布尔变量上运行一个布尔检查，以确定该变量是否为假。
  • 为 null： 运行一个布尔检查，以确定给定变量是否为 null。
  • 不为 null： 运行一个布尔检查，以确定给定变量是否不为 null。

地理空间

  • 从地理点生成 Geohash： 将给定的地理点转换为字符串形式的 Geohash 值。
  • 从地理点获取纬度： 返回给定地理点的数值纬度。
  • 从地理点获取经度： 返回给定地理点的数值经度。
  • 从地理点生成 MGRS： 将给定的地理点转换为字符串形式的 MGRS 值。

数组操作

  • 组合： 返回包含所有给定数组值的数组。
  • 交集： 返回仅包含给定数组中公共值的数组。
  • 更新元素位置： 返回一个数组，其中给定数组中的特定元素已被更新。构建者可以通过指定表示数组中元素位置的索引来指定要更新的元素。
  • 获取元素位置： 返回与给定数组中指定索引处的元素相对应的值，并且与数组类型匹配。
  • 长度： 返回表示给定数组中元素数量的数值。

数组检查

  • 包含： 在给定数组中运行布尔检查，以确定是否包含给定值。
  • 不包含： 在给定数组中运行布尔检查，以确定是否不包含给定值。
  • 是子集： 运行布尔检查，以确定给定数组是否是另一个给定数组的子集。
  • 是否为 null： 运行布尔检查，以确定给定数组是否为 null。 注意：如果为值选择了数组变量引用，此转换将评估并显示为“为空”。
  • 不为 null： 对给定数组是否不为 null 进行布尔检查。
    • 注意：如果为值选择了数组变量引用，此转换将评估并显示为“不为空”。

    • 结构体转换

    • 提取结构体字段： 给定一个结构体和字段 ID，返回结构体字段的值。请参阅 从结构体中提取字段 以获取更多详细信息。
