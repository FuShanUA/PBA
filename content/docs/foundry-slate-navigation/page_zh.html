导航

Slate 的编辑模式包含四个主要区域：

  1. 操作栏： 这里会显示应用名称、操作下拉菜单、退出至查看模式的按钮，以及用于打开各种编辑面板的按钮。
  2. 组件列表： 组件列表会列出应用中的所有组件。如果应用中包含工具栏，该列表会分为工具栏组件和画布组件。
  3. 画布： 应用的主要工作区。可以在这里重新排列组件并测试布局选项。通过右上角的下拉菜单更改屏幕尺寸，可以预览应用在不同屏幕上的显示效果。
  4. 组件编辑器： 从组件列表或画布中选中某个组件后，可以通过组件编辑器对其进行配置。

此外，还有一些在画布前方弹出的面板：

  • 查询编辑器
  • 函数编辑器
  • 平台编辑器
    • 对象集
    • 对象上下文
    • Foundry 函数
  • 事件编辑器
  • 依赖关系图
  • 样式编辑器
  • 变量编辑器

编辑模式下的全局搜索功能可通过 Windows 上的 Ctrl+K 或 macOS 上的 Cmd+K 快捷键调出，支持搜索并跳转至 Slate 的查询、函数、对象、变量和组件。全局搜索还会保留搜索历史，以便优先展示最近的结果。点击搜索结果即可打开相应的 Slate 编辑器面板。

组件编辑器

组件编辑器包含三个选项卡，用于对选中的组件进行编辑配置。

属性选项卡： 主要的编辑选项卡，用于更改组件的属性。可用的选项因组件类型而异。

布局选项卡： 用于设置组件的位置和尺寸，并应用自定义样式。

JSON 选项卡： 如果属性选项卡中的可用设置无法实现所需的更改，可以在此选项卡中直接编辑组件的原始 JSON 配置。每个组件都以包含最常用属性的模板代码起始，在属性选项卡中修改的任何字段也会同步添加到JSON选项卡中。

未通过属性选项卡暴露的状态由 Slate 在内部管理，因此在JSON选项卡中修改 Slate 的默认值时需仔细检查，以避免出现意外行为。

事件选项卡： 部分组件具有关联事件，可以在此处进行配置。
