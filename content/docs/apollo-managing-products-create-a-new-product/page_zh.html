创建一个新的产品

首次创建产品时，请导航到 产品目录，然后从 操作 下拉菜单中选择 创建新产品。

步骤 1：选择产品 ID

您将被提示选择一个不可变的产品 ID。工件 ID 和组 ID 组合起来形成一个唯一的 产品 ID，Apollo 使用该 ID 来标识您的产品。了解更多关于 在 Apollo 中定义产品 ID 的信息。

选择 创建新产品 进入下一步。

步骤 2：设置角色

接下来，您必须为新创建的产品分配一些角色。在创建产品后的五分钟内，您必须设置初始权限。我们建议将“产品管理员”角色分配给您所属的团队，以便您可以进行进一步更改。从 权限 选项卡 中进行的任何全局角色分配都将被继承并在本步骤中显示。

在 添加更多团队... 文本框中输入团队名称以选择一个团队，然后使用下拉菜单配置他们获得的角色。

我们建议将“产品发布创建者”角色分配给用户最少的团队。这可以最小化能够为您的产品创建产品发布的用户数量。

选择 设置初始角色 以进入下一步。

第3步：创建产品发布

现在您的产品已创建完成，并且一些初始角色已被分配，您可以开始创建发布版本。只有拥有“产品发布创建者”或“产品管理员”角色的团队才具有创建产品发布版本的权限。

选择创建产品发布以创建您的新产品。

创建产品发布有两种方式：

  • 通过CI管道
  • 通过您的Apollo中心
