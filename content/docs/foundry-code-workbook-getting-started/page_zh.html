入门

设置数据集

本教程使用一个公开可用的数据集（下载 titanic_dataset.csv），其中包含泰坦尼克号乘客的信息。

该数据集包括泰坦尼克号乘客的姓名、年龄、性别和其他识别信息。导航到 Foundry 并打开您的个人项目。创建一个名为 Code Workbook Tutorial 的文件夹，并将泰坦尼克号数据集上传到那里，将其命名为 titanic_dataset。

设置代码工作区

点击项目的新建按钮并选择代码工作区以创建一个新的工作区。

导入数据集

点击 导入数据集 开始。在出现的对话框中，搜索 titanic_dataset。选择在设置过程中创建的文件，该文件应该位于 /user/Code Workbook Tutorial/titanic_dataset。

一旦确定了所需的数据集，点击文件然后点击“选择”以将数据集添加到你的图中。

使用 Python 转换数据

现在 titanic_dataset 已被导入到工作簿中，我们可以使用代码和可重用逻辑片段对其进行转换。将鼠标悬停在 titanic_dataset 上并点击 + 符号以添加下游转换。这会显示一个包含各种转换选项的下拉菜单 - 选择 Python。

一个Python代码节点现在出现在图表上，连接线显示它是titanic_dataset的子节点。通过点击逻辑选项卡顶部的文本框，为这个转换指定一个别名，titanic_filtered。

默认情况下，新创建的转换不会作为数据集保存在Foundry中。您可以通过点击保存为数据集切换按钮，选择将转换的结果保存为数据集。了解更多关于将转换保存为数据集的信息。 保存为数据集的转换有两个名称：别名和Foundry数据集的名称。

使用Pandas数据框

如果您更习惯使用Pandas语法，可以在Python节点中使用Pandas。让我们更新titanic_filtered以使用Pandas。

首先，我们需要更改titanic_dataset的输入类型。点击逻辑面板中的“输入”选项卡并展开侧边栏。您会看到输入类型被设置为Spark dataframe。点击下拉菜单并选择Pandas dataframe，以将titanic_dataset的输入类型更改为Pandas dataframe。

接下来，让我们更新代码以使用Pandas dataframe。我们将执行相同的过滤操作。

Copied!
1def titanic_filtered(titanic_dataset):
2    output_df = titanic_dataset[(titanic_dataset['Survived'] == 1) & (titanic_dataset['Sex'] == 'female')]
3    return output_df

此代码将输出一个包含女性泰坦尼克号幸存者的Pandas dataframe。

使用控制台

控制台为Code Workbook提供REPL（读取-求值-打印循环），允许对图中的任何转换或输入数据集进行快速、临时分析。为了在您首选的语言中快速迭代，您的工作簿中为每种启用的语言都提供了控制台。

打开页面右侧的控制台。选择Python控制台：

你可以通过在 Python 中执行命令快速地对数据进行试验。你也可以通过高亮代码并使用 快捷键 Cmd+Shift+Enter (macOS) 或 Ctrl+Shift+Enter (Windows)，将转换中的代码发送到控制台运行。

首先，你必须在 Python 控制台中导入以下 PySpark SQL 函数：

Copied!
1import pyspark.sql.functions as F

然后，确定女性泰坦尼克号幸存者的最大年龄：

Copied!
1titanic_filtered.select(F.max('Age')).show()

你也可以使用 SQL 控制台计算相同的统计信息：
