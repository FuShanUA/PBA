设置数据集

本教程使用公开可用的数据集（下载 titanic_dataset.csv），其中包含泰坦尼克号乘客的相关信息。

该数据集包含泰坦尼克号乘客的姓名、年龄、性别及其他身份识别信息。进入 Foundry 并打开您的个人项目。创建一个名为 Code Workbook Tutorial 的文件夹，并将泰坦尼克号数据集上传至该文件夹，将其命名为 titanic_dataset。

设置代码工作簿

点击项目中的新建按钮并选择代码工作簿以创建一个新的工作簿。

导入数据集

点击导入数据集开始操作。在弹出的对话框中，搜索 titanic_dataset。选择您在设置过程中创建的文件，该文件应位于 /user/Code Workbook Tutorial/titanic_dataset 路径下。

找到目标数据集后，点击该文件，然后点击选择，将数据集添加到您的图表中。

使用 Python 转换数据

既然 titanic_dataset 已导入工作簿，我们可以使用代码和可复用的逻辑片段对其进行转换。将鼠标悬停在 titanic_dataset 上并点击 + 号，即可添加下游转换。此举会弹出一个下拉菜单，显示各种转换选项——请选择 Python。

此时图表中会出现一个 Python 代码节点，并以连接线表明它是 titanic_dataset 的子节点。点击逻辑面板顶部的文本框，为该转换指定别名 titanic_filtered。

默认情况下，Foundry 中新建的转换不会保存为数据集。您可以通过点击另存为数据集开关，将转换结果保存为数据集。了解有关将转换保存为数据集的更多信息。保存为数据集的转换有两个名称：别名和 Foundry 数据集的名称。
