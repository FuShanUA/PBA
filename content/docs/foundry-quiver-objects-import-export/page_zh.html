导出对象集

在 Quiver 中创建的任何对象集均可导出并保存，以便日后在 Quiver 或其他 Foundry 应用中使用。要导出对象集，请点击卡片右上角的 图标打开更多操作菜单。然后，在以下两个选项中选择其一：

  • 导出对象集：基于对象集筛选条件进行导出。符合条件的新对象将自动加入该集合。
  • 导出静态对象集：基于当前返回的对象进行导出。若对象集较小，此方式有助于提升性能。

接下来，选择保存的名称与位置。

对象集导出后，你可以将其导入新的 Quiver 分析中，或在 Foundry 的其他应用（如 Object Explorer）中打开。只需在对象集的导出标签页中点击带超链接的对象集名称，或直接在保存该对象集的文件夹中选中它即可完成操作。

导出对象集后，如果该对象集在 Quiver 中被更新，系统将提示你更新已保存的对象集。

在 Foundry 应用中打开对象集

你可以将 Quiver 中创建的任何对象集直接在其他 Foundry 应用中打开，无需导出或持久化保存对象集，从而简化跨应用工作流。要在其他应用中打开对象集，请在对象集卡片右上角的更多操作菜单（）中选择以下选项之一：

  • 在 Object Explorer 中打开：基于该对象集开启新的探索。
  • 在更多应用中打开：在 Foundry 中开发并发布为 Carbon 模块的应用中打开该对象集。这些模块包括 Foundry 地图、Workshop 模块、Vertex 模块以及 Quiver 仪表板。
