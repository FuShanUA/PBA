DataFrame 上的 .groupBy() 方法接收任意数量的列作为参数，并基于这些列执行聚合操作。或者，若要对整个 DataFrame 进行聚合，则无需传入任何列。

分组后的 DataFrame 上的 .agg() 方法可以接收任意数量的聚合函数。


```python
aggregated_df = df.groupBy('state').agg(
	F.max('city_population').alias('largest_city_in_state'),
	F.avg('city_population').alias('average_population_in_state')
)
```

> 默认情况下，聚合操作生成的列名格式为 `aggregation_name(target_column)`。然而，Foundry 中的列名不能包含括号或其他非字母数字字符。请改为将每个聚合结果指定一个具体的别名。

## 透视表

PySpark 中的透视表与普通的分组聚合操作非常相似。

```python
pivoted_df = df.groupBy('equipment').pivot('sensor').mean('value')
```
