将操作类型添加到市场产品

使用 Foundry DevOps 将您的操作类型包含在 市场产品 中，以便其他用户安装和重用。 了解如何创建您的第一个产品。

支持的功能

大多数操作类型功能都受支持，但不包括引用 具有不受支持功能的对象类型 的操作。在准备您的操作类型进行打包时，请确保您的操作类型 安全与提交标准 不引用用户；请更新任何用户引用以改为引用组。

将操作类型添加到产品中

要将一个操作类型添加到产品中，首先 创建一个产品，然后如下选择 操作类型 内容类型。

然后您将被提示选择一个操作类型。
