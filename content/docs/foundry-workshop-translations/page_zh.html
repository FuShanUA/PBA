翻译、本地化 (l10n) 与国际化 (i18n)

应用构建者现在可以通过“翻译”功能，为 Workshop 应用中使用的受支持字符串类型提供翻译。该功能允许通过手动方式或在 AIP 的辅助下，将 Workshop 应用本地化 (l10n) 和国际化 (i18n) 为多种语言，前提是已启用相关权限。如果 Workshop 应用已通过此功能翻译为某种语言，查看者在其浏览器的区域设置下将看到该模块的翻译视图。

要开启此功能，请在编辑模式下导航至 设置 选项卡，并在 高级功能 部分打开 翻译 开关。启用后，将出现一个新的 翻译 选项卡，应用构建者可在此为其模块配置翻译。

可翻译内容

Workshop 应用中的以下内容可进行翻译：

  • 模块标题的 标题
  • 区块标题的 标题 和 选项卡
  • 来自组件的静态文本，例如：
    • 选项卡组件的 标签
    • 按钮组组件的 文本 和 描述
    • 指标卡片组件的 标签 和 描述
    • Markdown 组件的 内容
  • 对象表格和对象列表组件的 列标题
  • 透视表组件的 聚合标题
  • 筛选器列表组件的 区块标题

翻译方法

Workshop 应用中有效字符串的翻译，可由应用构建者手动完成，也可在启用自定义工作流的权限下使用 AIP 自动生成。

手动翻译

应用构建者可手动翻译模块中的字符串。选择目标语言后，模块中检测到的每个可翻译字符串都将显示一个输入框，供手动输入译文。添加翻译后，可直接在模块中预览以便立即查看。

使用 AIP 自动翻译

也可使用 AIP 翻译模块中的字符串。使用 AIP 翻译字符串需要配置《自定义工作流的 AIP 功能》。

为了向 AIP 提供执行字符串翻译所需的上下文，必须定义模块的起始语言并设置目标语言。随后，可使用带有紫色 AIP 图标的 翻译 选项，利用 AIP 自动翻译模块中检测到的所有符合翻译条件的字符串。可直接在模块中预览翻译以便查看，并在必要时进行手动编辑。

对翻译满意后，可将这些翻译标记为完成，这实质上是将这些字符串标记为已供应用构建者审阅。模块中检测到的任何新增或修改的字符串（例如，添加新按钮或编辑区块标题时）都将显示在 待翻译 部分中，从而与已翻译和审阅过的字符串区分开来。

预览翻译

在编辑模式下，导航至 翻译 选项卡并选择所选的已配置语言，即可直接在模块中预览翻译后的文本。
