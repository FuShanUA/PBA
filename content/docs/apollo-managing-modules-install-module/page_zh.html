安装模块

导航到软件目录，然后选择您要安装的模块。然后从操作下拉菜单中选择安装模块。

您也可以从环境页面的操作下拉菜单中选择安装模块。

首先，选择要安装的模块以及要安装该模块的环境。可选地，为这个特定的安装定义一个自定义显示名称。如果您计划在同一环境中多次安装同一个模块，这将很有用。然后选择下一步。

现在，您可以为该模块的安装定义任何特定配置。如果存在任何 变量，您可以在本步骤中定义其值。模块要求的机密信息也会在此列出。选择 编辑机密信息 以提供所需的值，并 满足机密信息要求。

您可以选择 预览模块内容 来查看模块中每个实体的配置。

选择 安装 以在环境中创建一个 变更请求 来安装该模块。

模块安装后，您可以在环境页面的 实体 选项卡中查看所创建的实体。 模块 列指明了该实体所属的模块。

在模块安装期间满足秘密要求

在模块安装过程中，您将被提示输入 模块定义 所需的任何秘密。如果这些秘密已经存在，则在此步骤中不需要额外的工作。

使用模块管理现有实体

要开始使用模块管理现有实体，您可以将模块 安装 到环境中。模块将接管对现有实体的管理，而不是创建新的实体。

您应验证模块中声明的实体与环境中的实体具有相同的实体ID。

例如，如果一个模块包含31个实体，其中30个已经在环境中安装，那么您可以将该模块安装到环境中。Apollo将安装未被安装的1个实体，并且还将开始使用该模块来管理这30个现有的实体。

解除模块关联

从环境中解除模块关联会将模块安装从环境中移除。在模块解除关联后，曾经属于该模块的实体仍然保留在环境中，但将不再由该模块进行管理。要从环境中解除模块关联，请导航至模块页面的安装选项卡，然后选择您想要解除关联的模块安装。接着从下拉菜单中选择解除模块。
