为 Marketplace 产品添加健康检查

使用 Foundry DevOps 将您的数据健康检查纳入 Marketplace 产品，供其他用户安装和复用。了解如何创建您的首个产品。

支持的功能

为 Marketplace 产品添加健康检查时，支持以下功能：

  • 针对数据集的健康检查。
  • 除以下情况外，均支持健康检查配置验证：
    • 数据集验证（例如，涉及次要数据集验证的健康检查）。
    • 路径配置验证（例如，带有源路径验证的健康检查）。
    • 配置为创建 Foundry issue 的健康检查。
  • 监控视图和检查组会自动添加为输入项，因此在安装过程中，打包在某个组内的所有健康检查都将被添加到提供的输入组中。
  • 如果打包了包含数据期望的转换逻辑的数据集转换，则数据期望健康检查将被自动添加。由于这些检查是转换逻辑的一部分，打包者无法手动添加或移除它们。

为产品添加数据健康检查

要为产品添加数据健康检查，首先需创建一个产品，然后向您的产品中添加数据集转换或 Pipeline Builder 管道。随后，您可以使用建议按钮，打包任何与您的数据集转换或管道输出数据集相关联的检查。
