字符串选择器

字符串选择器组件可用于以多种选择形式展示字符串选项，包括下拉菜单、单选按钮或一组复选框。

配置选项

  • 标签
    • 为组件设置可选标签。此文本将显示在组件顶部。
  • 选项生成
    • 静态：通过添加选择器选项按钮手动输入并重新排序选项值。
    • 动态：选择现有的字符串数组变量或创建新变量，用于为组件生成选项。
  • 选择
    • 可将组件设置为允许单选或多选。
    • 选中值：组件的输出变量，用于存储用户选中的选项。若选择设置为单选，输出变量将为字符串变量；若选择设置为多选，输出变量将为字符串数组变量。
    • 选择展示
      • 若选择设置为单选，组件可展示为下拉菜单或单选按钮。
        • 下拉菜单
          • 禁用清除下拉选项：可开启此选项以禁止清除已选中的下拉项。
          • 占位符：默认情况下，未进行任何选择时，组件上将显示“选择一个选项...”作为占位文本。可通过自定义选项设定自定义占位符内容。
        • 单选按钮
          • 单选按钮布局：选项布局可设置为垂直、水平或指定列数的网格排列。
      • 若选择设置为多选，组件可展示为下拉菜单或复选框。
        • 下拉菜单
          • 允许创建新选项：可开启此选项以允许用户创建新选项并添加至下拉菜单中。用户创建的任何选项将以斜体显示。
          • 占位符：默认情况下，未进行任何选择时，组件上将显示“搜索选项...”作为占位文本。可通过“自定义”选项设定自定义占位符内容。
        • 复选框
          • 复选框布局：选项布局可设置为垂直、水平或指定列数的网格排列。
