导出

Notepad 文档专为打印和导出为 PDF 或 DOCX 格式而设计，支持调整页面方向、定义页眉和页脚，以及针对组件的打印特定设置。

Beta

Notepad 中的 DOCX 导出功能目前处于 beta 开发阶段。在活跃开发期间，功能可能会发生变更。

Notepad 目前支持以下组件的 DOCX 导出：

  • Clip
  • Code Workbook 图表
  • Contour 图表
  • 文件引用 (Compass)
  • 图片
  • 对象卡片
  • 对象媒体预览
  • 对象属性
  • 分页符
  • Quiver 分析图表
  • Quiver 仪表板图表
  • Solution Designer 图表
  • 表格
  • Vertex 图表
  • 地图

导出文档

如需导出文档，请使用右上角的 文件 菜单，选择 导出，然后点击 导出为 PDF 或 导出为 DOCX。文档渲染完成后，Notepad 会以选定的文件格式保存文档。

渲染超时

导出时的后台渲染时间有上限。如果文档包含大量需要长时间计算的图表，可能会达到时间限制，导致图表无法完整加载。遇到这种情况，请使用下文介绍的 以打印模式打开 选项。

或者，你也可以通过 以打印模式打开 按钮，以利于打印的视图打开文档。页面及所有图表完整加载后，你可以使用 Ctrl+P (Windows) / Cmd + P (macOS) 快捷键，或点击右上角的打印按钮，将页面保存为 PDF 或进行打印。

请务必根据文档的 页面方向，将打印布局设为 纵向 或 横向。确保页面尺寸与你在 Notepad 中选择的 页面尺寸 一致。选择 默认 边距选项可保证内容对齐。

特定页面设置

Notepad 提供了额外的页面设置，方便你在导出前对文档进行微调。如需配置页面设置，请在 Notepad 左侧面板中选择 页面设置。

页面尺寸

Notepad 文档的默认页面尺寸为 Letter（8.5 x 11 英寸）。你可以使用 页面尺寸 下拉菜单选择其他尺寸。

可用的页面尺寸选项包括：

  • Letter
  • Legal
  • Tabloid
  • A3
  • A4
  • A5

页面方向

默认情况下，文档以纵向模式显示。若要将文档切换为横向模式，请在 页面方向 下选择 纵向 或 横向。

页眉和页脚

页眉和页脚仅在文档打印或导出时可见。如需启用页眉或页脚，请开启 页眉 或 页脚 选项。

分页符组件

使用 分页符组件 可以插入自定义分页符。分页符仅在打印时显示。

组件打印配置

部分组件支持特定的打印配置选项。如果有可用选项，你可以通过选择组件、打开 组件属性 面板并选择 打印配置 来进行访问。

选项包括 缩放、在新页打印 和 展开滚动条。

缩放

导出文档可能会限制可用的水平空间（即使在横向模式下也是如此），这对于某些可视化效果并不理想。例如，屏幕上可以水平滚动的宽表就需要进行调整才能适应页面。通过 缩放级别 配置，您可以对显示的内容进行缩小来实现这一点。

在新页面打印

默认情况下，如果空间有限，微件会在新页面上打印。对于显示表格或跨度超过一页的微件而言，这可能并非理想行为。在这些情况下，您可以禁用在新页面上打印的功能。

展开滚动条

使用 打印时自动展开表格高度 功能可以在打印时完全展开滚动条，确保垂直滚动的部分能够完全展开。目前该功能仅支持 Contour 表格以及 Quiver 对象表和数据透视表。
