查看数据

假设我们想看看哪些员工拥有的电话号码最多。我们将提取感兴趣的数据集（拥有多个号码的员工），并调用 .take(3) 以列表形式获取前 3 行数据。另外，也可以使用 .collect() 将 DataFrame 的所有行以列表形式获取回来。

警告

将过多数据拉取到 Python 环境中很容易导致内存耗尽。请仅对少量数据使用 collect()。


```python
def multiple_numbers(phone_numbers):
    df = phone_numbers.groupBy('employee_id').agg(
        F.count('phone_number').alias('numbers')
    ).where('numbers' > 1).sort(F.col('numbers').desc())

    print(df.take(3))
```
