在 Applications Portal 中策展应用

Applications Portal 是一个用于发现和访问 Foundry 中所有应用的工具。这包括 (1) Foundry 核心平台应用和 (2) 管理员向 Applications Portal 推广的受信任自定义应用。

前往 Applications Portal

选择左侧边栏中的 Applications Portal 图标即可将其打开。您可以查看在 Workshop、Slate 或 Carbon 中构建且已推广的任何应用。您还可以快速访问所有 Foundry 平台应用。

您可以将最常用的应用“固定”到左侧边栏，以便快速访问。在 Applications Portal 中或编辑应用时，选择应用名称旁边的星号图标即可。固定后，Foundry 边栏中将出现名为 Promoted Apps 的区域，其中列出了您收藏的已推广应用。

Applications Portal 的权限基于文件系统权限。如果您拥有 Discoverer 角色，系统会提示您申请访问权限。

Applications Portal 模式

Applications Portal 有两种使用模式：

  • 基础模式（即装即用）：一种用于发现和了解所有 Foundry 平台应用的工具。无需任何管理工作。即使没有任何已推广的自定义应用，Applications Portal 也是左侧边栏中的一项核心工具。
  • 高级模式（基于策展）：由管理员、Palantir 代表或客户治理团队将所有受信任的自定义应用推广至 Applications Portal。

对于多租户注册环境上的每个注册项或租户，仅存在一个 Applications Portal。用户能够查看的已推广应用仅限于属于其组织的空间。如果您拥有对某个共享空间的访问权限，您也会看到来自该空间的已推广应用。

Applications Portal 中的已推广应用

已推广应用是指被管理员标记为受信任且具备生产环境就绪状态的应用。每个已推广应用都包含以下必填的元数据，这些数据将显示在 Applications Portal 和左侧边栏中：

  • 名称：可以与资源名称不同
  • 图标
  • 描述（可选）
  • 应用数据主人
  • 缩略图
  • 集合与标签：您可以使用它们将已推广应用分类到不同板块，并在 Applications Portal 中应用筛选条件。集合为必填项，标签为选填项。

已推广应用会获得代表受信任内容的紫色对号标记，这与数据目录中的项目类似。

向 Applications Portal 推广应用的权限

管理员可以将 Workshop 模块、Slate 应用、Carbon 工作区或外部 Web 链接推广至 Applications Portal。向 Applications Portal 推广应用的权限在《控制面板》中授予。您必须拥有“组织管理员”或“用户体验管理员”角色。同时，您还必须是应用的编辑者或数据主人才能对其进行推广。

对于对应用及其空间至少拥有 Discoverer 角色的所有用户，已推广应用都会显示在其 Applications Portal 中。

如何向 Applications Portal 推广应用

应用推广界面同时存在于 Applications Portal 以及 Workshop 和 Slate 的编辑模式中（参见下方示例）。

取消推广已推广的应用同样需使用推广界面，只需选择左下角的 Unpromote 按钮即可完成。

您可以更改推广项所引用的资源，从而以受控的方式发布新应用。

Applications Portal 中的分类

你可以在应用门户中使用集合和标签对应用进行分类，并应用过滤器以便仅查看特定应用。

创建集合与标签或将其添加到已发布应用的权限，与将应用发布至应用门户的权限一致，均取决于控制面板中“组织管理员”和“用户体验管理员”的角色权限。

集合

集合在应用门户的左侧边栏中作为分区标题显示。它们在应用门户中作为顶级分类，用于精心规划你的使用体验。集合在数据目录中创建和管理。创建集合后，你可以从数据目录和文件系统中向其添加已发布的应用。

应用门户仅显示已关联发布应用的集合，且前提是你拥有查看/发现这些应用的权限。如果某个集合未关联任何已发布应用，或者你无权访问该集合中的任何应用，该集合将不予显示。

标签

标签在已发布的应用卡片上显示为标签，可从应用门户的右上角进行筛选。你可以从《平台设置》的“标签”部分创建和管理标签。标签创建完成后，即可在发布界面及文件系统中将其添加到已发布的应用上。

应用门户仅将已关联发布应用的标签作为筛选器显示，且前提是用户拥有查看/发现这些应用的权限。如果某个标签未添加到任何已发布应用，或者用户无权访问带有该标签的任何应用，该标签将不予显示。

在应用门户中管理平台应用

Foundry 平台应用包括 Quiver、Contour、Data Connection、Pipeline Builder 等工具。你可以在控制面板的“应用访问”选项卡下配置选项，向用户显示或隐藏平台应用。借此，你可以选择仅在应用门户及 Foundry 的其余部分中展示特定的应用子集。
