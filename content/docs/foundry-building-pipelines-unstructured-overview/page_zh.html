非结构化数据上的管道

如《数据集》概述所述，Foundry 中的非结构化数据与表格数据一样，以文件集合的形式存储在数据集中。

以下功能在结构化数据和非结构化数据的管道中工作方式完全一致：

  • 管道可以实现增量处理，以优化计算性能。
  • 可以为管道编写单元测试。
  • 计算输出数据集使用构建和调度完成。
  • Foundry 的管道安全功能可提供稳健的端到端安全保证。

与表格数据管道的一些差异包括：

  • 文档中的大多数指导和示例代码侧重于处理数据帧，而非结构化数据使用的输入类型并非数据帧。
  • 必须使用底层文件系统 API 来读写非结构化数据集中的文件。
  • 由于非结构化数据集没有模式，某些专注于验证表格数据集行和列的功能将不可用。
  • 可以使用 Spark 并行处理非结构化文件，但这些 API 比数据帧处理的 API 更底层，也更复杂。

要开始使用非结构化数据管道，请参阅 Python 和 Java 转换文档的相关部分：

  • 《Python 转换：读写非结构化文件》
  • 《Java 转换：读写非结构化文件》
