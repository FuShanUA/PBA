Palantir MCP

Palantir MCP 是 Model Context Protocol ↗ 的实现。Palantir MCP 使 AI IDE 和 AI 代理能够在 Palantir 平台内自主设计、构建、编辑和审查端到端的应用程序，涵盖从数据集成到本体论配置和应用开发的所有内容。此外，您还可以使用 Palantir MCP 允许外部 AI 系统查询文档、元数据和数据，并在平台上执行高级任务。开发者可以使用 Palantir MCP 自动完成辅助任务，同时专注于他们正在构建的系统。

在开始使用 Palantir MCP 之前，请查看我们的安装指南和其他资源：

  1. 按照 安装说明 将 Palantir MCP 安装到您的 IDE 中。或者，使用 VS Code 工作区中提供的 AI 开发工具 来访问 Palantir MCP。
  2. 查看 Palantir MCP 入门指南。
  3. 查看我们的 安全性和数据流文档。

当何时应使用Palantir MCP？

Palantir MCP 为开发者提供了两个主要优势：

  1. 上下文： Palantir MCP 为LLM代理提供上下文，以在内部Palantir库中导航并理解Foundry架构。
  2. 工具： Palantir MCP 为LLM代理提供工具，以探索您的本体论和Foundry项目并执行操作。

本体论MCP用于消费

Palantir MCP 专为本体论构建者设计，可以修改本体论类型，但不能编写本体论数据。相比之下，本体论 MCP（OMCP） 使本体论消费者可以通过外部 AI 代理安全地读写您的本体论数据。本体论 MCP 将您的应用程序的对象类型、操作类型和查询函数作为 MCP 工具公开。这些 MCP 工具可以被 Copilot Studio 或 Gemini Enterprise 等外部系统使用来执行操作和写入数据，同时通过应用限制来限制代理可以执行的操作。了解更多关于在 开发者控制台文档 中启用本体论 MCP 的信息。

构建OSDK应用程序

The Palantir MCP provides tools to take actions in Foundry. Specifically, the MCP can search your ontology, safely modify the ontology, and update your Developer Console application. For example, you can ask it to Find me the object/links/functions to {do something}, Create this object-type/link-type and integrate it with my application, or Apply this proposal to my Developer Console application.

Build and iterate on Python transforms

The MCP provides tools to run Python transforms. These tools allow agents to fix transforms iteratively. The agent runs the tool preview_transform and, on failure, attempts to fix errors and re-run until preview_transform succeeds.
