修改参数

在编辑模式下设置默认参数值

参数的默认值是指打开报告时该参数将使用的值。

要编辑参数的默认值：

  1. （如有需要）切换到编辑模式。
  2. 在参数栏中，选择参数旁边的齿轮图标。
  3. 找到默认值字段，并根据需要修改该值。
  4. 选择保存。



新的默认值将替换可能已存在的任何当前值。

在查看模式下修改参数值

通过参数的输入字段修改参数值，将使用当前值覆盖默认值。再次打开报告时，当前参数值通常会恢复为默认值。不过，通过将当前值存储在 URL 中，可以保留这些值；详见下文。

要修改报告中的参数值：

  1. （如有需要）切换到查看模式。
  2. 在参数栏中，为一个或多个参数输入新值。
  3. 点击应用，以新的参数值重新加载报告。（这确保您可以在开始可能耗费大量资源的重新加载之前，修改多个参数值。）



实用提示

重新打开报告时保留当前参数值

您可能希望刷新报告并保留当前值，或者分享一份带有非默认值的报告。报告功能支持这一操作，其方式是在您修改参数值时，将其附加到报告 URL 中。

例如，打开报告时，其初始 URL 可能如下：

/workspace/report/ri.report.main.report.123

将 Month 参数的值修改为 July（覆盖该参数的默认值）后，URL 将自动更新为：

/workspace/report/ri.report.main.report.123?Month=July

此时，刷新页面将为 Month 参数预填 July 值。在刷新大型、复杂的报告时，这可以节省时间。

要在打开报告时显示默认参数值，请移除 URL 末尾的参数（例如 ?Month=July&Year=2019）。

复制包含当前参数值的链接

点击参数栏中的复制链接按钮，即可复制一个可共享的 URL，该 URL 将使用您当前正在使用的完全相同的参数值来加载报告。
