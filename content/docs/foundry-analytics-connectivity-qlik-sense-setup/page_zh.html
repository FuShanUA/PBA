服务器设置

现在，您可以从 Qlik Sense 访问 Palantir Foundry 数据集，并利用它们构建交互式仪表板。要在 Qlik Sense 中配合使用 Foundry，您必须在 Qlik Sense 服务器上安装 Foundry ODBC 驱动程序。

请按照以下指南完成设置。这些步骤必须由具有 Qlik Sense 服务器访问权限的人员来完成。

步骤 1：安装 ODBC 驱动程序

前往《下载页面：ODBC 驱动程序》下载驱动程序，并将其安装到 Qlik Sense 服务器上。

步骤 2：配置 Foundry DSN

Qlik Sense 要求预先配置好 ODBC DSN。

  1. 在 Qlik Sense 服务器上，打开 Windows ODBC 数据源管理工具，为 FoundrySqlDriver 创建一个新的系统 DSN。
  2. 将您的 Foundry URL 设置为服务器地址。
  3. 选择“Additional Properties”（附加属性），添加一个名为 UserAgent 的新属性，并将其值设为 QLIKSENSE。这将启用针对 Qlik Sense 优化的设置。
  4. 点击 OK（确定）保存 DSN。

我们建议将 DSN 中的访问令牌字段留空，等到在 Qlik Sense 中创建连接时再进行设置。请记住，如果您在 DSN 中设置了令牌，那么在 Qlik 中任何有权访问该 DSN 的用户，都将能够查看该令牌授权访问的数据。

UserAgent 属性

设置 UserAgent 属性非常重要。若不设置，数据将无法正确加载。

（可选）步骤 3：预配置 Qlik Sense 连接

如果用户能够在 Qlik Sense 中自行创建连接，则可跳过此步骤。否则，您需要预先配置 Foundry 连接并向用户授予访问权限。有关如何创建连接，请参阅《入门指南》。
