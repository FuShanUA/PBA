R TransformOutput 对象

对 Foundry 数据集进行低级写入操作的接口。

write.spark.df(df, partition_cols=NULL, bucket_cols=空, bucket_count=空, sort_by=空)

  • 将给定的 DataFrame ↗ 写入输出数据集。

    参数
    • df (pyspark.sql.DataFrame) – 要写入的 PySpark 数据框。
    • partition_cols (List[str], 可选) - 写入数据时使用的列分区。
    • bucket_cols (List[str], 可选) - 用于对数据进行分桶的列。如果指定了 bucket_count，则必须指定此参数。
    • bucket_count (int, 可选) – 分桶的数量。如果指定了 bucket_cols，则必须指定此参数。
    • sort_by (List[str], 可选) - 用于对分桶数据进行排序的列。

write.data.frame(rdf)

  • 将给定的 R data.frame ↗ 写入输出数据集。

fileSystem()

  • 返回一个 FileSystem 对象用于直接访问FoundryFS。

R FileSystem 对象

ls(glob=NULL, regex='.*', show_hidden=FALSE)

  • 根据数据集根目录，列出与给定模式（glob或regex）匹配的所有文件。

    参数
    • glob（字符串，可选）– 一个 unix 文件匹配模式。也支持 globstar。
    • regex（字符串，可选）– 要用来匹配文件名的正则表达式模式。
    • show_hidden（布尔值，可选）– 包含以‘.’或‘_’开头的隐藏文件。
    返回 R 陣列，其中包含名為元組 (path, size, modified) 的 FileStatus - 逻辑路径、文件大小（字節）、修改時間戳（自 1970 年 1 月 1 日 UTC 起的毫秒數）

open(path, open='r', disk_optimal=FALSE, encoding=default)

  • 以指定模式打開 FoundryFS 文件。

    參數
    • path (str) – 數據集中的文件的邏輯路徑。 (遠程路徑)
    • open (str) - 描述要打開連接的 模式。
    • disk_optimal (bool, 可選) – 控制 FoundryFileSystem 如何處理文件 I/O。
    • encoding (str, 可選) - 默認為 R 語言預設值（UTF-8）。
    返回值一個 R 連接對象

get_path(path, open='r', disk_optimal=FALSE, encoding=default)

  • 对于给定的 FoundryFS（远程）路径，返回本地临时路径。

    参数
    • path (str) – 数据集中的文件的逻辑路径。(远程路径)
    • open (str) - 用于打开连接的 模式 的描述。
    • disk_optimal (bool, 可选) – 控制 FoundryFileSystem 如何处理文件 i/o。
    • encoding (str, 可选) - 默认为 R 语言默认值（UTF-8）。
    返回值str

upload(local_path, remote_path)

  • 将文件从本地路径上传到远程路径。仅写入。

    参数
    • local_path（str）– 要上传的文件的本地路径。
    • remote_path（str）- 文件在数据集中的逻辑路径。
    返回值无

高级主题：disk_optimal 设置

在 FileSystem 方法 open() 和 get_path() 中，disk_optimal 参数控制如何处理文件输入和输出（i/o）。

默认情况下，在 open() 和 get_path() 中，disk_optimal 被设置为 FALSE。在这种模式下，文件在被访问之前保证会被下载。

如果您选择将 disk_optimal 设置为 TRUE，则文件会在代码执行时同时下载。必须通过 fifo() 打开临时本地路径才能正确读取。请注意，并非所有库都支持读取这种类型的文件。

您可以选择在读取的文件非常大时将 disk_optimal 设置为 TRUE。

例如，假设我们有一个非常大的 txt 文件，并且只想读取前 10 行。使用以下代码仅打印前 10 行，而无需读取整个文件。

Copied!
1disk_optimal_example<- function(large_txt_file) {
2    fs <- large_txt_file$fileSystem()
3
4    ## Open a connection with fifo()
5    ## The text file is titled large_txt_file.txt
6    conn <- fs$open("large_txt_file.txt", "r", disk_optimal = TRUE)
7
8    A <- readLines(conn, n = 10)
9    print(A)
10    return(NULL)    
11}

如果您想使用 R TransformOutput 写入文件然后再读取它，disk_optimal 必须设置为 false。
