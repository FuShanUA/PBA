本体论构建

Palantir 本体论 是组织的操作层。本体论位于集成到 Palantir 平台的数字资产之上（数据集、虚拟表 和 模型），并将它们与现实世界中的对应物连接起来，从工厂、设备和产品等实物资产到客户订单或财务交易等概念。在许多情况下，本体论作为组织的数字孪生体，包含实现各种用例所需的语义元素（对象、属性、链接）和动态元素（操作、函数、动态安全）。

操作类型和函数

组织的动态——在遵守组织控制和治理的同时实现变更——通过使用操作类型和函数在本体论中定义。操作类型使您能够从组织中的操作员获取数据，或者协调连接到您现有系统的决策过程；而函数则提供了一种方法来编写和演变任意复杂度的业务逻辑。

接口

An interface is an Ontology type that describes the shape of an object type and its capabilities. Interfaces provide object type polymorphism, allowing for consistent modeling of and interaction with object types that share a common shape.
