PDF 查看器

PDF 查看器组件支持关键词检索与文本高亮、匹配时自动滚动等基础功能，并具备显示、创建及交互操作文本和区域批注等高级功能，从而实现用户标记工作流。

该功能提供多项增强特性，包括但不限于以下内容：

  • 接收字符串变量，以实现对整个文档的关键词匹配。
  • 内置搜索功能，可根据用户提供的关键词进行匹配。
  • 匹配项高亮显示（当前匹配项显示为绿色；其他匹配项显示为紫色）并展示命中次数。
  • 自动定位至首个结果及在不同命中项之间切换。
  • 支持《媒体引用（媒体集）》。
  • 展示现有的批注对象。
  • 通过配置的操作创建批注。
  • 自定义批注交互行为，包括选中时触发的事件、属性展示提示框以及自定义颜色高亮。

配置选项

该组件配置支持多种媒体输入源，并允许输入页码和文本变量。通过输入页码变量，可引导用户跳转至 PDF 内的指定页面。配置的文本输入内容将自动填入搜索栏，并以此作为关键词执行检索。即使未对组件进行配置，您依然可以在 PDF 视图中手动输入任意搜索词。若要开始在 PDF 文档中配置批注，请勾选启用批注选项，并参阅配置批注章节。

  • 配置 PDF 数据源选项：

    • 选择所需的媒体源以向组件加载 PDF。下拉菜单将列出当前可用的受支持媒体源。目前支持的媒体源包括 Compass、数据集、媒体引用和附件。选定媒体源后，即可对该媒体源进行配置。可用的媒体源示例包括：
      • Compass：使用 Palantir 文件系统作为 PDF 的媒体源。选择此选项时，需提供以下内容：
        • Blobster RID：一个字符串变量，用于指定目标 PDF 的 Blobster RID。
      • 数据集：使用数据集作为 PDF 的媒体源。选择此选项时，需提供以下内容：
        • 数据集 RID：一个字符串变量，用于指定包含目标 PDF 的数据集 RID。
        • 文件路径：一个字符串变量，用于指定目标 PDF 在数据集内的文件路径。
        • 分支：一个字符串变量，用于指定数据集的 branchID。若未指定分支，将默认使用 master 分支。
      • 媒体引用：定义一个包含单一对象的对象集，并为 PDF 选择《媒体引用》类型的属性。
        • 包含单一对象的对象集：一个用于指定目标 PDF 的对象集变量。所选对象集必须仅包含一个对象。
        • 属性：从所选对象集中选取目标 PDF 所需的媒体引用类型属性。
      • 附件：定义一个包含单一对象的对象集，并选择附件属性作为 PDF 的媒体源。选择此选项时，需提供以下内容：
        • 包含单一对象的对象集：一个用于指定目标 PDF 的对象集变量。所选对象集必须仅包含一个对象。
        • 属性：从所选对象集中选取目标 PDF 所需的附件属性。

  • 配置输入变量：

    • 自定义页码：通过切换开关来启用或禁用数值变量。该变量用于捕获用户当前所在的页码，或用于改变组件当前显示的页面。
    • 自定义初始搜索：通过切换开关来启用或禁用字符串变量。该变量用于预填充组件内执行搜索时使用的关键词。

  • 配置批注：

    • 展示现有批注：每个图层可配置一个批注对象集。若要启用批注功能，必须至少添加并配置一个图层。每个图层包含以下可配置字段：
      • 批注图层名称：用于设置批注图层的名称。请注意，该名称字段仅展示在组件的配置面板内部。
      • 对象集：输入需要在该图层内展示的批注对象集。
      • **页面属性：** 选择用于指定标注对象所在 PDF 页码的整数对象属性。 * **边界框属性：** 选择用于指定标注对象边界框坐标的字符串数组对象属性。数组中的每个字符串元素应为序列化的 JSON 对象，表示边界框的左上角 (x1, y1) 和右下角 (x2, y2) 坐标。 * 之所以需要使用数组，是因为标注可以包含多个边界框。例如，三行文本需要三个边界框，每行各对应一个。有效的边界框属性可能类似于 `[ "{"x1":0.0, "y1":2.5, "x2":50.0, "y2":10.1}", "{"x1":2.0, "y1":12, "x2":35.0, "y2":14.2}" ]`。 * **选定的标注：** 选择一个对象集，既用作初始选定的标注对象的输入，也用作包含文档中当前选定标注的输出。 * **选中事件：** 配置 Workshop 事件，以便在选中标注时触发。 * **在提示框中显示的属性：** 添加属性，当在此图层中选中标注时，这些属性将显示在提示框中。 * **编辑或删除现有标注（通过操作）：** 配置显示在标注对象提示框中的标注图层操作。 * **操作图标：** 选择一个图标来表示提示框中的此操作。若未设置图标，默认使用铅笔图标。 * **操作标签：** 设置操作的显示名称，将显示在提示框中操作图标的悬停提示和操作表单标题上。 * **针对标注的操作：** 设置可从提示框触发的操作。可使用 `Hovered object` 变量引用悬停的对象。有关操作的更多信息，请查阅我们的操作类型文档。 * **高亮颜色：** * **静态：** 设置一种静态颜色，用于展示此图层内的所有标注。 * **颜色属性：** 设置一个字符串对象属性，用于指定此图层内标注的高亮颜色。该字符串属性必须采用十六进制颜色代码格式（例如 "#FBD065"）。如果该字符串属性未包含有效的十六进制代码，高亮颜色将默认为 GOLD5 (#FBD065)。 * **自定义规则：** 指定条件格式规则，以控制此图层内标注的高亮颜色。若未满足任何已配置的条件格式规则，标注的高亮颜色将默认为 GOLD5 (#FBD065)。 * **创建标注（通过操作或事件）：** 配置在新建文本和/或区域选区时运行的操作或事件。 * **操作或事件图标：** 选择一个图标，在新建选区时于操作提示框中代表此操作。 * **操作或事件标签：** 此标签将在鼠标悬停于操作提示框中的操作上时作为提示显示。若未为此操作设置图标，此标签将显示在操作提示框中。 * **用于选区类型：** 指定此操作应在文本选区、区域选区或两者的操作提示框中显示。 * **操作或事件：** 设置可从操作提示框触发的操作或事件。 * 配置操作参数时，可使用特殊参数值引用当前文本或区域选区中的数据： * **页码：** 当前文本或区域选区所在文档的页码。此选项可用于整数参数。 * **选区坐标：** 表示当前文本或区域选区边界框的字符串数组。此选项可用于字符串数组参数。 * **高亮文本：** 文本选区中选定的字符串内容。请注意，对于区域选区，该值为空字符串。此选项可用于字符串参数。 * **滚动至标注：** 切换以启用或禁用自动滚动至标注对象集中的某个对象。借此可自动滚动至您选择的标注。 **配置更多选项：** * **输出用户选定的文本：** 切换以启用或禁用将用户的文本选区捕获至字符串变量中作为微件输出。 * **输出用户选定的坐标：** 切换以启用或禁用将用户的选区坐标捕获至字符串数组变量中作为微件输出。 * **输出用户选定的页码：** 切换以启用或禁用将用户进行选区所在页面的页码捕获至数值变量中作为微件输出。 * **启用 PDF 下载：** 切换以启用或禁用用户下载微件中所显示 PDF 的功能。 配置标注图层后，如需导出包含标注的 PDF，只能通过 PDF 查看器部件内置的下载功能完成。不支持从部件外部的自定义按钮触发包含标注的 PDF 导出。
