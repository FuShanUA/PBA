时间戳差值

支持于：Batch, Faster, Streaming

返回两个时间戳之间在指定时间单位下的差值。

表达式分类： 日期时间

声明的参数

  • 结束时间： 作为被减数的结束日期或时间。
    Expression<Date | Timestamp>
  • 开始时间： 作为减数的开始日期或时间。
    Expression<Date | Timestamp>
  • 单位： 时间单位。
    Enum<Days, Hours, Milliseconds, Minutes, Months, Quarters, Seconds, Weeks, Years>

输出类型： Long

示例

示例 1：基础用例

参数值：

  • 结束时间： 2022-10-01T10:00:00Z
  • 开始时间： 2022-10-01T09:00:00Z
  • 单位： HOURS

输出： 1

参数值：

  • 结束时间： End
  • 开始时间： Start
  • 单位： HOURS
