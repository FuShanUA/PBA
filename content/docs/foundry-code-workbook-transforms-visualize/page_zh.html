可视化数据

在 Code Workbook 中，您可以使用开源的可视化库来显示您的数据的可视化结果。这些可视化结果之后可以与他人共享，例如在 Notepad 文档 中。

Python 可视化

在 Python 中，Code Workbook 支持使用 Matplotlib、Seaborn 和 Plotly 进行可视化。

使用 Matplotlib 和 Seaborn

当使用 Matplotlib 时，对 matplotlib.pyplot.show() 的调用会将生成的图表图像保存到转换输出中并返回给用户界面，从而允许创建自定义图表。和任何可视化一样，您可以通过右键点击图中的转换项并选择 下载图片。

以下是一个使用 Matplotlib 渲染可视化的转换示例：

使用 Matplotlib 绘制不同语言和字体的图表

要使用 Matplotlib 绘制使用非罗马字符的语言（如日语或韩语）的标签和文本，或在使用非默认字体时，您必须明确指定希望 Matplotlib 在渲染图像时使用的字体系列。有关更多信息，请参阅 默认安装的可用字体列表。

以下是为 Matplotlib 指定字体的示例：

def japanese_korean_matplotlib_example():
    import matplotlib.mlab as mlab
    import matplotlib.pyplot as plt
    from matplotlib import rcParams
    # Set font family to the Noto Sans CJK font pack
    rcParams['font.family'] = 'Noto Sans CJK JP'
    # create data
    x = [10,20,30,40,50]
    y = [30,30,30,30,30]

    # plot lines
    plt.plot(x, y, label = "ライン１")
    plt.plot(y, x, label = "선2")
    plt.xlabel("X-軸")
    plt.ylabel("Y-축")
    plt.legend()
    plt.title("日本語ラベル図の例 // 한글 라벨 도표의 예시")
    plt.show()

Matplotlib 2.* 字体指定

对于 Matplotlib 2.*，.ttc 字体文件不会被 Matplotlib 自动检测到。请升级到 3.* 或直接将字体文件路径添加到 Matplotlib 的字体管理器中。

Copied!
1from matplotlib import rcParams
2import matplotlib.font_manager as fm
3fm.fontManager.ttflist += fm.createFontList(["/usr/share/fonts/opentype/noto/NotoSansCJK-Regular.ttc"])
4rcParams['font.family'] = 'Noto Sans CJK JP'</pre>

使用 Plotly

Plotly 是一个可视化库，允许您创建交互式图像。要使用 Plotly，请首先确保它已包含在您的环境中。

调用 fig.show() 会导致生成的图表图像作为转换输出的一部分被保存并返回到用户界面。以下是一个使用 Plotly 来渲染可视化效果的转换示例，使用了 Plotly Express。Plotly Express 预先加载了 iris 数据集。

Copied!
1def plotly_example():
2    import plotly.express as px
3    df = px.data.iris()
4    fig = px.scatter(df, x = "sepal_width", y = "sepal_length", color = "species")
5    fig.show()

运行转换后，Plotly 可视化结果将出现在可视化标签页中。我们建议以全屏模式查看可视化内容。您可以使用诸如缩放图表、在图表上进行选择等功能。

这是一个更复杂的示例，它生成一个动画可视化效果。

Copied!

Plotly 限制

以下注意事项适用于 Python 和 R。

  • 在控制台中，Plotly 可视化将被转换为图像并以 PNG 格式显示。它们将不是交互式的。要查看交互式可视化，请在转换中编写代码。
  • 创建 Plotly 可视化时，由于浏览器性能下降，不建议使用超过 20,000 个点的可视化。如果创建具有大量点的散点图，请使用 scattergl 以获得更好的性能。

Matplotlib 限制

以下 Matplotlib 的限制适用于 Python。

  • Matplotlib 不是线程安全的。 ↗
    • 当运行多个节点时，Spark 会并行运行这些计算。结果可能以 Matplotlib 运行时异常或在错误节点中创建的可视化形式表现出意外行为。
  • 在单个代码笔记本中使用多个Matplotlib可视化效果时，必须锁定每个节点。您可以使用线程安全的装饰器@synchronous_node_execution来锁定每个节点，如下所示。
Copied!
1import matplotlib.mlab as mlab
2import matplotlib.pyplot as plt
3from matplotlib import rcParams
4@synchronous_node_execution
5def thread_safe_node():
6    # Set font family to the Noto Sans CJK font pack
7    rcParams['font.family'] = 'Noto Sans CJK JP'
8    # create data
9    x = [10,20,30,40,50]
10    y = [30,30,30,30,30]
11
12    # plot lines
13    plt.plot(x, y, label = "ライン１")
14    plt.plot(y, x, label = "선2")
15    plt.xlabel("X-軸")
16    plt.ylabel("Y-축")
17    plt.legend()
18    plt.title("日本語ラベル図の例 // 한글 라벨 도표의 예시")
19    plt.show()
