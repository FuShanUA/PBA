Use raster data

Foundry enables large-scale processing of raster data in a first-class manner with media sets. File-level transforms are also supported for custom image pre-processing, or for processing any file formats not currently supported by media sets.

Learn more about the different types of geospatial data.

Data formats and sources

Supported media set raster file formats:

  • TIFF / GeoTIFF
  • NITF
  • JPEG2000

Other raster file formats that can only be processed at a file level:

  • PNG
  • JPEG

Use raster data with media sets

  1. Import supported raster file formats as media sets. You can either create a new media set via data connection or local machine uploads. Learn the different ways to import media.

  2. 导入后，您可以通过媒体引用使用媒体集。在 Pipeline Builder 中选择媒体引用，或在 转换 中选择。

  3. 在 本体论 中创建一个对象类型，可以是在 Pipeline Builder 中作为管道输出，也可以是在 本体论管理器 中直接创建。确保该对象类型具有媒体引用属性，并且该属性为媒体引用类型，同时在 能力 中声明了支持的媒体集，以便平台知道如何引用您的媒体项。有关更多信息，请查阅我们关于 在本体论中使用媒体 的文档。

一旦您有了一个对象类型，就可以将其导入到地图应用中进行查看。不支持在地图上显示大于 67 兆字节（MB）的栅格文件。

在转换中使用栅格数据

对于当前未被媒体集支持的栅格文件类型，您仍然可以在转换中处理它们。您可以在下面的章节中查看推荐的库和代码示例。

转换也适用于预处理受支持的媒体集文件类型。例如，您可以使用转换通过 MediaSetInput 来更新图像大小，并使用 MediaSetOutput 输出一个媒体集，如下例所示。 了解如何编写媒体集批量管道的更多信息。
