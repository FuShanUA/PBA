Time series cards

Back to: Index of cards

Operations in this section include transforming existing time series and visualizing time series data on charts. Many of these transformations can be saved as derived series for use outside of Quiver.

The following cards accept time series and/or event sets and return time series:

  • Align series to event set
  • Align to date
  • Coalesce time series
  • Combine time series
  • Cumulative aggregate
  • Derivative
  • DSP filter
  • Event indicator series
  • Event statistics
  • Extract time series bound
  • Filter time series
  • Integral
  • Linear aggregation
  • Periodic aggregate
  • Relative time series
  • Rolling aggregate
  • Sample
  • Segment statistics
  • Shift time series
  • Time series forecast
  • Time series formula
  • Time series regression

The following cards accept an object and return time series:

  • Linked series aggregation
  • Numeric series formula
  • Time series property

The following cards accept a table or time series sync and return time series:

  • Tabular time series
  • Time series sync

The following cards accept time series and return boundaries:

  • Bollinger bands
  • Reference profile bounds
  • Time series bounds

The following cards return grouped time series:

  • Event comparison plot - this accepts a time series and event set as input
  • Grouped time series - this accepts a table as input

The following cards accept time series and return a single value (number, string, or date):

  • Enum value at time
  • Latest value for enum time series
  • Time series numeric aggregation
  • Time series unit
  • Value at time

The following cards accept time series and return an object:

  • Backing object from time series

The following cards accept variable inputs and return a time series:

  • Code function time series
