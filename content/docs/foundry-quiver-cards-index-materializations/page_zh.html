物化卡片

返回：卡片索引

本节的卡片支持对本体 物化 进行分析。

物化 是 Quiver 中的一种 数据类型，提供了一种大规模转换、可视化和分析本体数据的方法，能够突破转换表中数据连接和转换时存在的 5 万行限制。

具体而言，物化卡片支持以下操作：

  • 对对象数据执行连接（左/右/内/全连接）
  • 派生新列、筛选或聚合对象数据
  • 使用 分类图表 绘制数据
  • 转换为转换表并使用 Vega 图表

对于以对象集为输入的大规模分析，可以使用物化卡片，Quiver 会在后台无缝转换对象集。从对象集出发，可以使用物化后续操作下的任意卡片，或者通过对象集物化卡片进行显式转换。

要查看支撑某张物化卡片的后端数据集原语，只需将鼠标悬停在数据源图标上即可。

可使用以下卡片：

  • 表达式
  • 筛选物化
  • 连接物化
  • 物化 SQL
  • 数值聚合（物化）
  • 对象集物化
  • 集合运算（物化）
  • 唯一列值（物化）
  • 基于物化的分类图表
