Oracle Fusion Cloud Financials

Oracle Fusion Cloud Financials 连接器是 Palantir 提供的用于 Oracle Fusion Cloud Financials 的驱动程序。

要创建新的 Oracle Fusion Cloud Financials 源，请遵循 Palantir 提供的驱动的标准设置流程，然后使用以下章节进行 Oracle Fusion Cloud Financials 的特定配置和网络设置。有关完整的属性参考，请参阅官方 Oracle Fusion Cloud Financials 驱动文档 ↗。

配置

以下属性为必填或推荐配置。

属性是否必填？描述默认值
Password ↗必填指定身份验证用户的密码。—
URL ↗必填要连接的 Oracle Fusion Cloud Financials 账户的 URL。通常，这是您的 Oracle Cloud 服务的 URL。https://servername.fa.us2.oraclecloud.com
User ↗必填指定身份验证用户的用户 ID。—

网络连接

下表列出了为使源成功运行而需要能够访问的域。

对于每个域，添加相应的出站策略。如果源托管在本地并且无法从 Foundry 直接访问，请改用智能体代理出站策略；智能体主机本身也必须能够访问所列出的域。有关详细信息，请参阅使用智能体作为代理。
