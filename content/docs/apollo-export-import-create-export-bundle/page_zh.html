按照这些步骤操作后，您可以使用适用于您网关的适当传输流程，将包从源中心传输到您的 目标中心。

前提条件

创建一个导出管道

导出管道 定义了将在包中从源中心导出的产品发布和相关元数据。您可以配置数据选择规则，以定义哪些产品发布应包含在包中。您还可以设置构建计划，以便定期构建该管道的包。

了解如何创建导出管道的更多信息。

创建一个捆绑包

将鼠标悬停在左侧边栏的应用程序图标上，然后选择导出。

从所有管道表格中选择一个管道。然后选择构建管道以创建一个新的捆绑包。

您可以在所有管道表格中选择管道名称，然后从捆绑包选项卡中选择您构建的捆绑包以查看其内容。

现在，您可以通过选择下载捆绑包来下载该捆绑包。从目标中心，您可以导入该捆绑包以解压其内容并应用相关更改。

了解有关将捆绑包导入到您的目标中心的更多信息。

如果你为你的导出管道（Export Pipeline）配置了构建计划，Apollo 将会按照预定的周期为你生成捆绑包（Bundles）。这些捆绑包将在导出管道的捆绑包选项卡中可供查看和下载。
