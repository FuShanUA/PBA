Pipeline Builder 中的 AIP 功能

Pipeline Builder 提供了一系列 AIP 辅助功能，旨在帮助您更好地理解、构建和管理数据管道。借助这些功能，您只需一句提示即可生成全新的数据转换逻辑，解释管道开发过程中的各个步骤，智能推荐节点名称与描述，创建并编辑正则表达式，以及进行时间戳类型转换。

Pipeline Builder 辅助功能

生成

使用生成功能，可以根据用户输入的提示创建全新的数据转换逻辑。AIP 能够调用 Pipeline Builder 中可用的全套数据转换算子，针对您的具体需求推荐最合适的方案。该功能利用元数据来生成逻辑，同时提供透明的推理过程和建议依据，且不会暴露底层数据。最终，AIP 生成的转换会像常规数据转换一样保存到您的管道逻辑中，实现与现有工作流的无缝集成。

要使用生成功能，请选择 Pipeline Builder 画布顶部中央带有两颗星号的紫色 AIP 图标。按照弹窗中的提示，在画布上选择作为输入的节点。按住 cmd (MacOS) 或 ctrl (Windows) 并点击节点，可将其添加至当前选择或从中移除。

接下来，输入供 AIP 评估的提示内容，然后选择生成开始运行。

运行结束后，系统会返回一个或多个以紫色高亮显示的转换节点，并附带所生成转换的描述说明。

若要继续进行转换操作，请选择返回生成。选择下一组节点并输入新的提示，即可继续构建您的数据管道。

选择输入框可以查看最近的提示历史记录。您可以选择某条历史记录来重新运行该提示，系统会自动选中最初的输入节点。

解释

使用解释功能，可以在开发过程的每一步动态获取管道的描述说明。这有助于让协作者随时了解当前状态，为新的审批人提供有价值的上下文信息，或者以极低的维护成本促进新团队成员之间的知识传递。

要使用解释功能，请选择 Pipeline Builder 画布顶部中央标有 AIP 的紫色灯泡图标。按照弹窗中的提示，在画布上选择作为输入的数据表。按住并点击节点，可将其添加至当前选择或从中移除。您的选择必须至少包含一个转换节点，并且必须是具有单一输出的连通节点集合。

按住并点击节点，可将其添加至当前选择或从中移除。您的选择必须至少包含一个转换节点，并且必须是具有单一输出的连通节点集合。

最后，选择解释 xx 个节点即可生成转换说明。

您也可以解释单个节点，只需将其选中，然后从右侧弹出的菜单中选择紫色的解释按钮即可。

此外，您还可以选择解释某个节点内的任意一组转换操作。

转换辅助

转换辅助功能可帮助您充分利用管道构建器中提供的众多转换选项。借助 AIP 的力量，正则表达式辅助器可以根据您的输入生成正则表达式并更新现有表达式。此外，通过时间戳格式化器，您可以快速将转换看板中的值从 string 转换为 timestamp。

生成提案描述

在 Pipeline Builder 中，您还可以借助 AIP，在提交提案时生成提案描述，从而快速说明您的更改。

为此，首先选择右上角的提案。这将跳转至提案创建视图。

现在选择紫色的生成按钮。AIP 将针对您分支上的管道更改撰写一段提案描述。

在本例中，存在多项更改，包括对部分管道设置的更改。AIP 会为您逐一描述所有更改。

在选择生成之前，您还可以在文本框中输入额外的上下文信息、突出特定更改，甚至起草初步的提案描述。AIP 将利用这些信息来完善提案描述。生成的内容将添加在您的文本下方，并有清晰的分隔线。

若要重新生成或修改已提供的描述，请在选择生成前确保文本框中仅保留相关文本。为获得最佳效果，应删除所有其他文本，包括之前生成的描述。

用于自定义工作流的 AIP 功能

要使用以下功能，用户必须由平台管理员《授予用于自定义工作流的 AIP 功能权限》。

文本到向量

您可以通过《文本到向量》表达式，使用 Text embedding ada-002 嵌入模型将文本作为向量嵌入，只需向其提供字符串即可转换为嵌入向量。这些向量旨在捕获单词或短语的语义含义，从而实现高级文本分析与操作。

文本嵌入的计算成本较高，可能会导致预览和构建速度变慢。

使用 LLM 节点

《使用 LLM 节点》功能提供了一种便捷的方法，可大规模地对您的数据执行大型语言模型（LLM），以便在各种数据转换之间无缝集成 LLM 处理逻辑，且无需编写代码即可将 LLM 简化集成到您的管道中。

注意：AIP 功能的可用性可能会发生变化，不同客户之间的可用情况可能有所差异。
