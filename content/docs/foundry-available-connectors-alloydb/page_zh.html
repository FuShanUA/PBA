AlloyDB

AlloyDB 连接器是 Palantir 提供的用于 AlloyDB 的驱动程序。

要创建新的 AlloyDB 源，请遵循 Palantir 提供驱动的标准设置流程，然后使用以下章节进行 AlloyDB 特有的配置和网络设置。有关完整的属性参考，请参阅官方 AlloyDB 驱动程序文档 ↗。

配置

以下属性为必填或推荐配置。

属性是否必填？描述默认值
AuthScheme ↗必填用于身份验证的方案。接受的值为 Password、OAuth。Password
Password ↗必填指定进行身份验证的用户账户密码。—
Server ↗必填服务器的主机名或 IP 地址。{serverAddress}
User ↗必填指定进行身份验证的 AlloyDB 用户账户的用户 ID。—
UseSSL ↗必填此字段设置是否启用 SSL。TRUE
Database ↗推荐AlloyDB 数据库的名称。—
Port ↗推荐AlloyDB 服务器的端口号。5432

网络配置

下表列出了为成功运行数据源而需要能够访问的域名。

对于每个域名，请添加相应的出站策略。如果源托管在本地且无法从 Foundry 直接访问，请改用智能体代理出站策略；智能体主机本身也必须能够访问所列出的域名。有关详细信息，请参阅将智能体用作代理。

域名是否必填
<Server>:<Port>始终必填。服务器和端口连接属性（默认 Port=5432）
