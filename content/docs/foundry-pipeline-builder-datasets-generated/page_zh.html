自动生成输入数据

在 Pipeline Builder 中创建手动输入的数据集时，您可以生成模拟数据来填充数据集。

为此，请在手动输入的数据集内部，选择生成模拟数据。

在左侧，选择要为其生成数据的列。然后，在自动填充内容字段中，选择该列应包含的数据类别。要查看模拟数据的示例，请选择示例。

要指定生成的行数，您可以在选择行数字段中输入 1 到 1000 之间的数字。

如果要预览手动输入的表格，请选择预览。这将在底部面板中打开一个名为预览生成数据的新标签页。

完成列配置后，选择位于右侧的生成表格。

默认情况下，生成表格将覆盖手动输入表格中任何先前存在的数据。

如果要保留某些列的值，可以使用特定列旁边的锁定选项。当某列被锁定时，即使重新生成表格，该列中的值也将保持不变。

您还可以使用另一个数据集来生成外键。这会将该数据集与另一个数据集中的列关联起来。

在 Pipeline Builder 中可以自动填充的其他数据示例如下：

  • 城市：生成随机城市名。
  • 公司：虚构的公司名称。
  • 常量：始终生成相同的值。
  • 国家代码：两个字母的国家代码。
  • 电子邮件：有效但虚构的电子邮件地址。
  • 名字：包含虚构名字的字符串。
  • 外键：将此数据集与另一个数据集中的列关联起来。
  • 全名：名字和姓氏组合成的单个字符串。
  • 姓氏：虚构的姓氏。
  • 列表：从预定义列表中选取加权随机值。
  • 空值：始终生成空值。
  • 其中之一：从加权生成器列表中随机选择。
  • 街道地址：随机街道地址。
  • 模板：使用辅助函数从模板生成随机字符串。
  • 美国州名：来自美国的随机州名。
  • 美国州代码：来自美国的随机两个字母的州名缩写。
  • 美国邮政编码：位于指定州内的邮政编码。
  • UUID：通用唯一标识符。

使用 LLM 生成数据

如果现有的自动填充类别都无法满足需求，您可以使用大语言模型（LLM）根据自定义的列描述来生成数据。

要选择 LLM 选项，请在自动填充内容下，选择使用 LLM 生成。

在列提示词中，输入要生成的数据的清晰描述。您可以通过输入 /[列名称] 来引用其他列。

强烈建议您提供示例单元格值，以帮助 LLM 理解您期望的数据类型和格式。您可以在示例单元格值中为 LLM 提供单元格值示例以供参考。

请注意，对于 LLM 生成的数据，您最多可以预览 10 行。

生成大量行或使用复杂的提示词可能需要长达一分钟的时间。
