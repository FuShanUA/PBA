分析工具栏

无论您是在画布模式还是图谱模式下进行分析，您都可以使用相同的分析工作区工具栏，该工具栏包含以下组件：

  • 分析工作区页眉
  • 分析顶部栏
  • 后续操作菜单
  • 搜索栏
  • 侧边栏
    • 分析内容面板
    • 参数面板
    • 可视化函数面板
    • 仪表板面板
    • 设置面板

分析工作区页眉

如上图标记为 1 所示，分析工作区页眉从左到右提供以下功能：

  • 撤销 / 重做： 这些按钮允许您在分析中撤销或重做操作。
  • 分析历史 ()：访问分析或仪表板之前的保存版本。
  • 保存： 保存您的分析。
  • 分享： 分享您的分析。
  • 详情面板： 用于管理访问权限、协作者和其他分析元数据。

分析顶部栏

如上图标记为 2 所示，分析顶部栏允许您向分析中添加数据和卡片。

  1. 数据卡片： 打开搜索栏以向分析中添加对象或时间序列。
  2. 搜索卡片： 打开搜索栏，允许您搜索并交互所有可用的卡片。

后续操作菜单

如上下图标记为 3 所示，后续操作菜单是一个工具栏，当您将鼠标悬停在画布或图谱上的卡片或时间序列图上时会出现。通过后续操作菜单，您可以找到并执行与所选卡片相关的常用操作。操作按类别进行分组，例如过滤、可视化、计算、连接、转换和转换数据。

请注意，后续操作菜单中的特定选项集将根据所选卡片的输出类型而有所不同。

使用搜索选项 () 跨所有类别搜索操作或在特定类别内进行搜索。结果将包含类别内的匹配项，并突出显示来自其他类别的匹配项。

适用于对象和对象集卡片，使用库选项 () 来发现现有的 Foundry 资源，例如本体操作和函数、已发布的 Quiver 仪表板以及可以将所选对象类型作为输入的 Quiver 可视化函数。选择一个构件会将预先配置了该构件作为输入的相关 Quiver 卡片添加进去。例如，当选择一个本体操作时，会将一个预先配置了所选操作的 Quiver 操作按钮添加到分析中。

使用所选卡片作为输入打开 AIP Generate，以使用自然语言生成分析。

搜索栏

如下图标记为 4 所示，搜索栏提供了对 Quiver 支持的所有卡片的单一访问入口。

通过选择分析顶部栏上的任意项来打开搜索栏。或者，您可以通过选择以下选项来打开搜索栏：

  • 在空分析中的 + 添加数据到分析。
  • 在空分析内容面板中的 + 添加数据。
  • 分析内容面板中的 新建卡片（悬停在画布名称上）。

卡片按以下选项卡分组：

  • 对象类型： 添加一个或多个对象集。此选项卡具有左侧菜单，其中包含所有配置了时间序列的对象以及在本体中配置的本体组。
  • 对象： 添加一个或多个特定对象。此选项卡具有左侧菜单，其中包含可用的对象类型。
  • 时间序列：添加一个或多个时间序列对象。此选项卡左侧的菜单显示了配置了时间序列属性的可用对象。
  • 卡片：添加卡片以与数据交互。此选项卡左侧的菜单根据每张卡片的分析目标对其进行分类。数据类型标签可以帮助你识别每张卡片支持的输入和输出。
  • 函数：搜索所有可用的函数并添加它们，分为时间序列函数和对象函数。

侧边栏

如本页顶部的图片中标为 5 所示，选择左侧的一个图标即可打开侧边栏。共有五个侧边栏：

  • 分析内容面板
  • 参数面板
  • 可视化函数面板
  • 仪表板面板
  • 设置面板

分析内容面板

分析内容面板可用于整理你的分析内容。在该面板中可以直接完成多项功能，包括：

  • 查找与整理内容
    • 通过输入全局 ID 或标题来搜索特定卡片
    • 按文本和数据类型进行筛选
    • 拖放内容以在不同部分之间移动（或在画布模式下重新排列画布上的内容）

在画布模式下，每张卡片都会按其在画布上出现的顺序列出。可用的操作包括：

  • 配置：打开编辑器以配置卡片
  • 复制：在同一分析中创建卡片的副本
  • 重命名：编辑卡片标题
  • 在新分析中打开：将卡片及其上游依赖项复制到新的分析中
  • 在图中查看依赖关系：切换至依赖关系视图的图模式
  • 隐藏：在画布上隐藏卡片
  • 删除：从分析中移除卡片。如果卡片存在下游依赖项，你将看到一个对话框，其中包含处理该卡片的各种选项。

将鼠标悬停在列表中的某张卡片上时，会出现更多操作图标 ()，通过该图标可以找到卡片操作。

在图模式下，卡片会根据其所属的颜色组进行列出。你还可以根据数据类型或按画布/仪表板/可视化函数来筛选图表。可从分析内容面板执行的操作包括：

  • 配置：打开编辑器以配置卡片
  • 重命名：编辑卡片标题
  • 在新分析中打开：将卡片及其上游依赖项复制到新的分析中
  • 在图中查看依赖关系：切换至依赖关系视图的图模式
  • 隐藏：在图表上隐藏卡片
  • 删除：从分析中移除卡片。如果卡片存在下游依赖项，你将看到一个对话框，其中包含处理该卡片的各种选项。
