可视化函数

可视化函数是使用 Quiver 的点击式界面构建的可重用逻辑块。这些函数可以在相同或其他 Quiver 分析中被多次使用。可视化函数在 Foundry 中作为独立资源发布，可与其他用户共享，他们也可以在自己的 Quiver 分析中使用这些函数。

可视化函数支持以下操作：

  • 无需编写代码，通过点击式界面构建可重用逻辑，
  • 构建可重用函数共享库，与同事协作，
  • 在一个位置轻松更新多个分析中的通用逻辑，降低手动出错或逻辑不一致的风险，
  • 简化大型分析，提升加载和导航性能。

输入

一个可视化函数包含零个或多个输入。输入是函数的参数，用户在使用该函数时可以进行配置。支持的输入类型包括：指标、对象集、单个对象或时间序列。请注意，可视化（分类或非分类）和表格不受支持的输入类型。

输出

一个可视化函数具有一个输出。函数的输出即该函数返回的结果。支持的输出类型包括：指标、对象集、单个对象或时间序列。

请注意，可视化（分类或非分类）和表格不受支持的输出类型。

包含函数输出的函数卡片，可以用作 Quiver 分析中其他卡片的输入。

逻辑

Quiver 函数的逻辑是指在分析图表中，位于输入和输出之间的所有卡片。
