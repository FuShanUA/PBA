警报调试页面

警报调试页面为特定资源上的监控规则触发提供了详细的诊断视图。要访问警报调试页面，请导航到 故障排除警报 选项卡中的 数据健康 应用程序，并选择一个警报的 查看详细信息。

警报调试页面目前支持函数和操作类型资源。计划增加对其他资源类型的支援。

页眉

页面页眉标识了监控规则并提供快速操作：

  • 监控名称： 与该规则创建所依据的模板名称一起显示。
  • 状态行： 格式为on <resource> since <time>，表示规则触发时所针对的资源以及规则当前状态持续的时间。
  • 静音： 暂时静音此监控规则的通知。当静音处于活动状态时，在标题中会出现指示器。
  • 编辑监控： 在规则配置视图中打开监控规则以进行编辑。

使用返回箭头返回到排查警报选项卡。

指标

指标部分显示被监控指标的当前状态：

  • 当前值 / 阈值： 被监控指标的最新值以及触发警报的阈值。最近一次评估的时间戳会出现在数值旁边。
  • 窗口持续时间： 对于具有窗口条件的监控规则（例如，统计固定时间间隔内失败次数的规则），计算条件所依据的评估窗口。
  • 状态徽章： 显示规则当前状态的徽章。当条件不再满足时，徽章显示 Passing。当规则触发时，徽章会显示 Failing 并与配置的严重程度级别（Low、Medium 或 High）配对。

监控描述出现在指标值下方。

将鼠标悬停在条件标签上以查看规则上配置的所有严重程度条件的详细信息。

标准操作程序

当监控规则有 标准操作程序（SOP） 时 附加的SOP卡片显示了与相关便签资源的链接。该卡片提供了对为该规则记录的故障排除步骤和常见解决方案的快速访问。

一个监控规则最多可以显示两个SOP，每个都作为其相关便签资源的链接显示：

  • 监控规则SOP： 直接附加到监控规则的SOP。您可以在管理监控选项卡中的数据健康应用程序中添加、替换或删除它。
  • 默认模板SOP： 如果设置了的话，由监控模板定义的SOP。此SOP由服务提供，并且是只读的。

图表

图表部分显示随时间变化的监控指标值。图表包括正在跟踪的指标描述。描述因资源类型（函数或操作类型）而异。

使用图表上方的日期范围选择器来调整时间窗口，或直接与图表进行交互。

警报历史

警报历史部分显示了过去30天内该规则的监控状态转换时间线。每个条目显示状态徽章和转换的时间戳。向下滚动以加载更多历史记录。

深入探究

深入探究 部分提供进一步调查受监控资源的链接：

  • 工作流谱系： 在工作流谱系中打开资源运行历史。工作流谱系基于触发条件的指标应用预过滤器，以突出显示最相关的执行。有关每种指标类型所应用的过滤器的详细信息，请参阅 从警报导航到资源谱系。
  • 本体论管理器： 在本体论管理器中打开目标资源概览。
  • 代码仓库： 对于函数警报，会打开代码仓库中的源仓库。此链接仅在已知函数的源仓库时显示。

不支持的资源类型

警报调试页面仅对函数和操作类型资源提供详细的诊断信息。如果您打开其他资源类型的监控规则页面，它将显示一个空状态，说明详细的诊断功能尚未可用。 排查警报 选项卡将继续为这些规则提供高层次的警报信息。
