为用例配置权限

用例权限仅用于管理用例元数据的权限。

元数据

用例元数据的权限通过角色面板进行管理。在用例概览页面中，选择左侧边栏或概览页面右上角的...选项，然后选择管理权限。

你可以通过角色面板为用例添加或移除用户及用户组，并为其分配 Use Case Owner（用例数据主人）或 Use Case Editor（用例编辑者）角色。

用例数据主人：拥有 Use Case Owner 权限的用户既可以管理用例权限，也可以编辑用例元数据。

用例编辑者：拥有 Use Case Editor 权限的用户只能编辑用例元数据。

默认情况下，用例元数据对组织内的所有人可见，即使用户无法查看或编辑用例资源。

资源

用例所含资源的权限，由资源本身或支撑该用例的项目权限来管理。如需详细了解如何管理项目和本体资源权限，请参阅相关文档。
