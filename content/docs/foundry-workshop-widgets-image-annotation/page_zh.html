图像标注

图像标注微件通过在感兴趣区域周围绘制矩形来对图像进行标注。

配置选项

以下截图展示了新添加的图像标注微件的初始状态及其初始配置面板：

  • 输入数据
    • 图像来源：可以通过媒体 URL 或媒体引用来显示图像。目前该微件支持以下图像文件类型：
      • 媒体 URL：选择包含有效媒体 URL 的字符串变量以渲染媒体预览。如果从数据集中引用媒体 URL，该 URL 应为以下任意一种格式：.png、.jpg、.jpeg、.bmp 或 .webp，
        • https://{my-foundry-url}/foundry-data-proxy/api/web/dataproxy/datasets/{dataset rid}/transactions/{transaction rid}/{filename}
        • https://{my-foundry-url}/foundry-data-proxy/api/web/dataproxy/datasets/{dataset rid}/views/{branch name}/{filename}。
        • 否则，如果是引用外部媒体 URL，请在您的注册项的内容安全策略设置中配置访问权限。
      • 媒体引用：定义包含单个对象的对象集，并选择媒体引用类型的属性，以渲染该对象的媒体预览。
    • 当前边界框选择：一个字符串变量，用于跟踪以“x1, y1, x2, y2”格式存储的活动边界框选择坐标，其中 (x1, y1) 是边界框左上角的像素坐标，(x2, y2) 是边界框右下角的像素坐标。
    • 活动标注对象：表示当前选中标注的对象。
  • 标注
    • 标注对象：表示要叠加到图像上的标注的对象集。目前每张图像最多支持 1000 个标注。
    • 边界框属性：标注对象集中表示点向量的字符串属性。
    • 颜色属性：标注对象集中表示以十六进制代码绘制颜色的字符串属性。此字段为可选。
    • 创建标注时：允许模块构建者配置当用户创建标注时触发的 Workshop 事件。
