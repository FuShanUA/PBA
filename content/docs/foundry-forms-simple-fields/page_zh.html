简单字段

Foundry Forms 不再是 Foundry 上数据输入或写回工作流的推荐方法。相反，使用 Foundry 本体论来构建用户输入工作流，将相关数据结构表示为对象类型，并配置与操作的写回交互。在 表单概述 文档中了解更多信息。

在表单中，简单字段会向受访者请求基本输入。此页面讨论了 Foundry Forms 中可用的不同类型的简单字段，不包括 附件 字段。

提示

提示 字段用于向受访者突出重要内容并支持Markdown格式化。用户可以配置以下选项：

  • 创建引用其他字段值的内容。
  • 设置标题、背景颜色和图标。
  • 添加已上传到Foundry或其他从Foundry可访问的网站的图片。

提示字段的值永远不会写入支撑对象类型或电子表格。如果您需要使用不同字段中写入的值来模板化某些文本，可以使用模板字段。

复选框

复选框字段以复选框形式显示一组选项。用户可以配置以下选项：

  • 提供可能的Values列表以及它们的显示Labels。
  • 设置Default values。
  • 将复选框显示为水平或垂直方向。

日期选择器

日期选择器 字段允许受访者输入日期或时间戳。用户可以配置以下选项：

  • 设置一个 Time Precision（天、分钟、秒或毫秒）。
  • 设置一个 Recorded formt（默认情况下，日期为 YYYY-MM-DD，时间戳为 YYYY-MM-DDTHH:mm:ss.SSSZZ）。
  • 强制设定最小/最大日期。
  • 禁用过去/未来的日期。
  • 在填写时预填充该字段的日期。

默认的最小和最大日期是当前日期的前后各25年，但可以通过更改其配置值来设置更远的过去或未来日期。例如，您可以通过将最小日期设置为1900年，从而允许选择超过25年前的出生日期。

在日期选择器字段中，根据用户的时区，日期始终由用户选择并显示给用户。然而，当将日期值写入支撑电子表格或对象类型时，它们始终以UTC格式写入，以便在不同时区之间兼容。

下拉菜单

下拉菜单字段以下拉形式显示一组选项。用户可以配置以下选项：

  • 提供一个可能的 Values 列表以及它们显示的 Labels 。
  • 设置 Default values 。
  • 允许 多选 。
  • 如果只有一种可用值，则预填充该字段。
  • 允许创建除了给定选项之外的值。
  • 设置 Placeholder 。
  • 使用 代码编辑器 来设置 noResultsText: string 。

    • 单选按钮

    单选按钮 字段以单选按钮的形式显示一组选项。用户可以配置以下选项：

    • 提供一个可能的 Values 列表，以及它们显示的 Labels。
    • 设置一个 Default value。
    • 水平或垂直显示单选按钮。

    资源选择器

    资源选择器 字段允许响应者从 Foundry 内部选择一个资源。用户可以指定是否允许响应者选择文件夹、数据集或任何资源类型。

    滑块

    滑块字段允许受访者选择一个数字。用户可以配置以下选项：

    • 强制设置最小值/最大值。
    • 设置一个Step size和Label step size。
    • 设置一个Unit label（例如，kg或lbs）。
    • 指定所选值的标签方式。
    • 使用 代码编辑器：
      • 设置initialValue: double。
      • 设置labelPrecision: double。
      • 设置showTrackFill: boolean。

    文本

    文本字段允许受访者输入一行文本。用户可以配置以下选项：

    • 设置一个Default value。
    • 设置一个Placeholder。

    文本区域

    文本区域

    文本区域字段允许受访者输入多行文本。用户可以配置以下选项：

    • 设置一个 Default value。
    • 设置一个 Placeholder。
    • 设置默认行数和最大行数。

    URL（网址）

    URL（网址）字段允许受访者输入超链接。用户可以配置以下选项：

    • 设置一个 Default value。
    • 设置一个 Placeholder。
    • 当格式无效时显示警告，并自定义显示的消息。
