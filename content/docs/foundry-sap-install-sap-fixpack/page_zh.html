安装修复包

修复包仅适用于指定的 SP 级别。

安装

面向 SAP 应用程序的 Palantir Foundry Connector 2.0（简称“Connector”）SPxx 修复包以 SAP 请求的形式发布，通过 SAP 传输管理系统进行传输。请按照以下步骤将请求导入 SAP 系统。

  1. 下载安装包。
  2. 将 Connector SPxx 修复包解压到一个文件夹中。文件列表如下（请求编号可能因所安装的版本而异）：
  • Connector 文件
    • K900xxx.D04
    • R900xxx.D04
  • 远程智能体文件
    • K900xxx.D02
    • R900xxx.D02
  1. 以“R”开头的文件名为数据文件；以“K”开头的文件名为协同文件。请将文件相应地复制到 SAP 应用服务器上下方列出的对应文件夹中：
    /usr/sap/trans/cofiles
    /usr/sap/trans/data
  1. 使用具有 STMS（SAP 传输管理系统）使用权限的用户登录 SAP 系统。
  2. 运行 STMS 事务。
  3. 选择传输的目标系统。
  4. 从工具栏菜单中，选择 Extras > Other Requests > Add，然后输入请求编号。请求编号的格式为：D04K9000xx。前三位数字代表请求文件的扩展名；其余数字为解压文件中所见的 K 文件名。
  5. 该请求会显示在导入队列中。通过选择请求编号来选中该请求，然后在工具栏上选择 Import。
  6. 转到 Options 选项卡，选择 Leave Transport Request in Queue for Later Import 和 Ignore Invalid Component Version。接着，启动传输。
  7. 选择工具栏上的日志按钮，检查导入日志。
  8. 检查是否有任何错误信息。成功的导入应当没有任何错误地完成。
