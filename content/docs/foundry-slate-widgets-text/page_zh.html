  • 可点击

  • 图片
  • 链接
  • 列表

图片组件

图片组件是文本组件的简单扩展。该组件利用 HTML 图片标签来创建图片。你需要先将图片上传至 Foundry，或者将其托管在 Slate 可访问的其他位置，然后将图片标签的 src 属性设置为该图片的 URL。对于托管在 Foundry 中的图片，其 URL 格式为 https://<FOUNDRY_URL>/blobster/api/salt/<RID>。你可以通过在 Foundry 中打开图片、右键点击图片并复制图片地址来获取此 URL。请按照以下示例配置你的图片组件：

<img src="https://<FOUNDRY_URL>/blobster/api/salt/<RID>">

卡片组件

卡片组件是文本组件的扩展。该组件利用 HTML div 元素来创建卡片，并通过 each 语句填充内容。向应用添加卡片组件时，该组件会预置一些骨架代码，你可以替换这些代码以自定义卡片。

  • #each : 集合是用于迭代并生成对应数量卡片的列表对象。你还需要取消文本末尾 /each 的注释。
  • item key: 在双花括号中设置为 this，以支持卡片选择功能。
  • Header: 作为每张卡片标题显示的文本。若要从查询中选择项目，可以在 Handlebars 语法中使用 lookup query.value @index 语法。
  • Content: 每张卡片中位于标题之后的文本。可以像标题一样从查询中提取。

文本组件

文本组件是 Slate 中用途最广的组件之一，它也是许多其他组件的基础，例如 HTML、图像、卡片和列表组件。这些组件本质上就是预设了部分默认代码的文本组件，以便为你提供正确的选项。

Text 组件具有两种模式：Markdown 模式和 HTML 模式。Markdown 是一种轻量级标记语言，最终会渲染为 HTML，因此在 HTML 模式下能实现的操作，在 Markdown 模式下同样可以完成。Markdown 模式让你能够利用更为简洁且极具可读性的 Markdown 语法。例如，若要将某段文本设置为一级标题（H1），只需在其前方添加一个 # 符号即可：

有关其他 Markdown 语法，请参阅《Markdown 文档 ↗》。

属性

属性描述类型必填修改方式
markupLanguage指定组件的标记语言，可为《Markdown 速查表 ↗》或 HTML。string是直接编辑
selectedValues在选中模式下，所有已选中项目的选定值。string[]是用户交互
selectionType决定选择的类型 - “无”、“单选”或“多选”。string是直接编辑
selectionRequired指定是否可以取消选择所有值。启用后，此标志会阻止用户取消选择组件中最后选定的值。如果组件初始状态下没有任何选定的值，则只有在用户进行初次选择后才会阻止取消选择。boolean是直接编辑
text要渲染并展示给用户的内容。string是直接编辑
tooltipText在 <sl-tooltip> 元素的工具提示中显示的文本。支持 HTML。string是直接编辑
tooltipsEnabled指定是否启用工具提示。工具提示将应用于 <sl-tooltip> 标签的内容。boolean是直接编辑
clickEvents此文本组件公开的点击事件名称列表。boolean是直接编辑
