文本

文本小部件类别包括以下小部件：

  • 卡片（Card）
  • 可点击（Clickable）
  • HTML
  • 图像（Image）
  • 链接（Link）
  • 列表（List）
  • 文本（Text）

图像小部件

图像（Image） 小部件是Text小部件的简单扩展。此小部件使用HTML图像标签来创建图像。您需要先将图像上传到Foundry或在Slate可访问的其他位置托管，然后将图像标签src属性设置为图像URL。对于托管在Foundry中的图像，图像URL是https://<FOUNDRY_URL>/blobster/api/salt/<RID>。您可以通过在Foundry中打开图像，右键单击图像，然后复制图像地址来访问此URL。请按照下面的示例进行图像小部件配置：

<img src="https://<FOUNDRY_URL>/blobster/api/salt/<RID>">

卡片小部件

卡片 小部件是 Text 小部件的扩展。该小部件使用 HTML div 来创建卡片，并使用 each 语句来填充它们。当您将卡片小部件添加到应用程序中时，该小部件会预填一些骨架代码，您可以替换这些代码以自定义您的卡片。

  • #each : 集合是用于创建卡片数量的对象列表。您还需要取消对文本末尾的/each的注释。
  • 项键：设置为双大括号中的this，以允许选择卡片。
  • 标题：这是每个卡片的标题上显示的文本。要从查询中选择项目，可以在handlebars内使用语法lookup query.value @index。
  • 内容：这是每个卡片标题之后的文本。它可以从查询中获取，就像标题一样。

文本小部件

文本小部件是Slate中最多功能的小部件之一。它也是许多其他小部件的基础，如HTML、图像、卡片和列表。这些小部件只是带有默认代码的文本小部件，以提供正确的选项。

文本小部件有两种模式：Markdown模式和HTML模式。Markdown是一种轻量级标记语言，可以渲染为HTML，因此您可以在HTML模式中执行的任何操作都可以在Markdown模式中执行。Markdown模式让您能够利用更简单、非常易读的Markdown语法。例如，要将一些文本设置为标题（H1），只需在其前添加一个#符号：
