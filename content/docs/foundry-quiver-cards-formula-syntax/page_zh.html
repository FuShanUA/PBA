公式语法

Quiver 允许您使用公式绘制自定义的数学表达式并指定过滤条件。我们在公式中支持以下结构：

  • 值
  • 输出类型
  • 运算符
  • 内置函数
  • 控制流
  • 语言参考

值

  • 数字：除了标准数字如 6，3.14159，-128，Quiver 还支持科学记数法：1.602e-19。
  • 图表引用：每个图表在创建时都会被分配一个参数名称，您可以使用 $ 符号来引用图表：$A 指的是参数名为 A 的图表，而 $A + $B 计算两个图表的总和。
  • 布尔值： 使用逻辑运算符时，当使用布尔值 true 和 false。
  • 时间参考：在公式中使用函数 secondsSince("time") 和 millisecondsSince("time") 来参考时间。时间项需要以 ISO-8601 格式 ↗ 格式表示。例如，secondsSince("1970-01-01T00:00:00+00:00") 会计算自 UTC 时间 1970 年 1 月 1 日以来的秒数。
  • NaN：表示未定义或不可表示的值，比如 null 值。

输出类型

公式只能输出数字或字符串，不能输出布尔值。此外，输出必须是 同质 的，这意味着一个公式在所有评估中必须始终只输出数字或只输出字符串。

如果您需要输出类似布尔值的值，可以考虑以下解决方法：

  • 对于数值输出：使用 1 表示 true，使用 0 表示 false。例如：$A > 0 ? 1 : 0
  • 对于字符串输出：使用 "true" 和 "false" 字符串。例如：$A > 0 ? "true" : "false"

操作符

  • 数值操作符： Quiver 支持 +、-、*、/ 和取余操作符 %：10 % 7 的结果是 3。你也可以使用括号 () 来强制优先级。这些操作符的结果是双精度数字。
    • 你不能使用 ^ 进行幂运算。请参见下面 内置函数 中的 pow 函数来执行幂运算。
  • 比较操作符： Quiver 支持 == 表示相等，!= 表示不相等，以及 >、<、>=、<= 表示比较。这些操作符的结果是布尔值。
  • 逻辑操作符： 你可以使用逻辑与（&&）和逻辑或（||）操作符来组合布尔值。Quiver 还支持逻辑非操作符（!），该操作符会反转布尔值。
  • 位操作符： 你可以在数值上执行位与（&）、位或（|）、位异或（^）和位非（~）。你还可以执行位移操作：<< 表示左移，>> 表示保留符号的算术右移，>>> 表示零填充的逻辑右移。

控制流

Quiver 支持三元运算符 a ? b : c 来表示“如果 a 则 b 否则 c”。例如，在每个时间点，如果 $A 的值大于 0，则 $A > 0 ? $B * 10 : $C * 10 取 $B * 10 的值；否则，它取 $C * 10 的值。

您也可以使用 NaN 值来在输入系列缺少值时返回替代值。例如，在每个时间点，如果 @M != NaN ? @M : @K 有数值，则取 @M 的值；否则，取 @K 的值。

您可以使用 return 和 skip 语句来“提前”停止，并返回一个值或表示不应返回任何值（例如，过滤掉一个点）。

语言参考

基本类型和赋值

基本类型和赋值

变量使用 var 关键字进行声明。= 符号用于给变量赋值。变量可以是数字（双精度）、布尔值或字符串。

var a = 6;
var b = 3.14159;
var c = 1.602e-19; // Exponential notation is supported
var d = 1.602E-19; // Exponents are not case-sensitive

var boolean_example = true;
var boolean_example = false; // re-assign to false

var string_example = "high";

表达式

数值

你可以将字面量和变量组合成更复杂的表达式。

基本的数学运算符为 +, -, *, /. 要计算余数，使用 %.

var a = 2 + 2; // a has value 4
a = 3 * 7; // a has value 21
a = 3.5 / 7; // a has value 0.5
a = 10 % 7; // a has value 3

运算符优先级如预期一样。你可以使用括号 ( ) 来将表达式组合在一起。

var a = 5 + 3 * 2; // a has value 11
var b = (5 + 3) * 2; // b has value 16

你也可以使用上述五种运算符的赋值版本。

var a = 10;
a *= 2; // a now has value 20
a += 3; // a now has value 23
a -= 5; // a now have value 18
a /= 3; // a now has value 6
a %= 5; // a now has value 1

布尔值

您可以使用比较运算符 <、<=、>、>= 来比较数字。这些运算符会产生一个布尔值（true 或 false）结果。

可以使用 == 检查相等性，使用 != 检查不等性。

然后可以使用逻辑运算符 && 和 || 组合这些表达式。

您可以使用 ! 运算符来反转布尔结果（true 变成 false，false 变成 true）。

字符串

可以使用 == 检查相等性，使用 != 检查不等性。

var a = "high";
var b = "low";

// compare values
var c = a == a; // true
var d = a != b; // true
