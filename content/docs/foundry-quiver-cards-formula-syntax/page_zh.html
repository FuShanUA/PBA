公式语法

Quiver 允许您通过公式绘制自定义数学表达式并设定过滤条件。我们在公式中支持以下结构：

  • 值
  • 输出类型
  • 运算符
  • 内置函数
  • 控制流
  • 语言参考

值

  • 数字： 除了 6、3.14159、-128 等标准数字外，Quiver 还支持科学计数法：1.602e-19。
  • 图表引用： 每个图表在创建时都会分配一个参数名，您可以使用 $ 符号来引用图表：$A 引用参数名为 A 的图表，而 $A + $B 则计算两个图表之和。
  • 布尔值： 在使用逻辑运算符时，请使用布尔值 true 和 false。
  • 时间引用： 使用 secondsSince("time") 和 millisecondsSince("time") 函数在公式中引用时间。时间参数需采用 ISO-8601 格式 ↗。例如，secondsSince("1970-01-01T00:00:00+00:00") 会计算自 UTC 时间 1970 年 1 月 1 日以来的秒数。
  • NaN： 表示未定义或无法表示的值，例如空值。

输出类型

公式只能输出数字或字符串，不能输出布尔值。此外，输出必须同质，即公式在所有计算中必须始终只输出数字或只输出字符串。

如需输出类布尔值，可参考以下替代方案：

  • 数字输出： 用 1 表示 true，用 0 表示 false。例如：$A > 0 ? 1 : 0
  • 字符串输出： 使用 "true" 和 "false" 字符串。例如：$A > 0 ? "true" : "false"

运算符

  • 算术运算符： Quiver 支持 +、-、*、/ 以及取余运算符 %：10 % 7 的结果为 3。您还可以使用括号 () 来强制改变运算优先级。这些运算符的结果均为双精度浮点数。
    • 不能使用 ^ 进行幂运算。如需进行幂运算，请参阅下文内置函数中的 pow 函数。
  • 比较运算符： Quiver 支持相等判断 ==、不等判断 !=，以及大小比较 >、<、>=、<=。这些运算符的结果为布尔值。
  • 逻辑运算符： 可以使用逻辑与 (&&) 和逻辑或 (||) 运算符来组合布尔值。Quiver 还支持逻辑非运算符 (!)，用于反转布尔值。
  • 位运算符： 可以对数值执行按位与 (&)、按位或 (|)、按位异或 (^) 和按位非 (~) 操作。还可以进行按位移位：<< 表示左移，>> 表示保留符号的算术右移，>>> 表示补 0 的逻辑右移。

控制流

Quiver 支持三元运算符 a ? b : c 来表达“如果 a 则 b 否则 c”。例如，在每个时间点上，如果 $A 的值在该时刻大于 0，则 $A > 0 ? $B * 10 : $C * 10 取 $B * 10 的值；否则取 $C * 10 的值。

您还可以使用 NaN 值在输入序列缺失值时返回替代值。例如，在每个时间点上，如果 @M 有数值，则 @M != NaN ? @M : @K 取 @M 的值；否则取 @K 的值。

你可以使用 return 和 skip 语句来“提前”停止，并返回一个值或指示不应返回任何值（例如，过滤掉某个点）。

语言参考

基本类型与赋值

使用 var 关键字声明变量。= 符号用于为变量赋值。变量的类型可以是数字（双精度）、布尔值或字符串。

var a = 6;
var b = 3.14159;
var c = 1.602e-19; // 支持指数表示法
var d = 1.602E-19; // 指数部分不区分大小写

var boolean_example = true;
var boolean_example = false; // 重新赋值为 false

var string_example = "high";

表达式

数值

你可以将字面量和变量组合成更复杂的表达式。

基本的数学运算符包括 +、-、*、/。计算余数请使用 %。

var a = 2 + 2; // a 的值为 4
a = 3 * 7; // a 的值为 21
a = 3.5 / 7; // a 的值为 0.5
a = 10 % 7; // a 的值为 3

运算符的优先级与预期一致。你可以使用圆括号 ( ) 对表达式进行分组。

var a = 5 + 3 * 2; // a 的值为 11
var b = (5 + 3) * 2; // b 的值为 16

你也可以使用上述五种运算符的赋值复合形式。

var a = 10;
a *= 2; // a 的值现在为 20
a += 3; // a 的值现在为 23
a -= 5; // a 的值现在为 18
a /= 3; // a 的值现在为 6
a %= 5; // a 的值现在为 1

布尔值

可以使用比较运算符 <、<=、>、>= 来比较数字。这些运算符会生成布尔值（true 或 false）结果。

使用 == 检查相等性，使用 != 检查不等性。

随后，可以使用逻辑运算符 && 和 || 组合这些表达式。

你可以使用 ! 运算符对布尔值结果取反（true 变为 false，false 变为 true）。

字符串

使用 == 检查相等性，使用 != 检查不等性。

var a = "high";
var b = "low";

// 比较值
var c = a == a; // true
var d = a != b; // true
