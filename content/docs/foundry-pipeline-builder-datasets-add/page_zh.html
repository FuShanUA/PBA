添加数据集

开始构建管道前，通过以下四种方法之一将数据添加到图谱中：

  • 数据连接应用程序
  • 从 Foundry 中选择数据集或媒体集
  • 手动上传数据
  • 在管道文件中手动输入数据

从数据连接向管道构建器添加数据

若要访问数据源中的数据，请前往 Foundry 导航侧边栏中的数据连接应用程序。找到需要集成的数据源，然后点击开始构建管道。为新管道选择一个位置，接着点击保存。系统将创建一条新管道，与该数据源相连的所有同步任务都会导入到管道构建器图谱中。

不能将新管道保存到个人文件目录下。建议建立推荐的项目结构，以便在开发流程起步阶段就理顺数据安全与数据治理工作。

从 Foundry 向管道构建器添加数据

若要导入已存放在 Foundry 文件系统中的数据集或媒体集，请进入管道构建器应用程序，并在图谱空间中心选择添加 Foundry 数据。搜索并选中可用的数据集，然后选择添加数据。

如需添加多个数据集或媒体集，可逐一添加并选择添加到所选；全部选定后，再选择添加数据。

从计算机向管道构建器上传数据

也可以从本地计算机上传数据集或媒体集文件。选择从计算机上传并选中待添加文件，或者直接将文件拖放到图谱中。

在管道构建器中手动输入数据

也可以通过定义数据表并手动填充数据的方式创建输入数据集。

选择列名与列类型，定义新表的 Schema，随后手动向表内添加值。手动输入的表支持随时修改。

下表列出了手动录入表中几种最常见的列类型：

列类型格式
String所有字符
Timestampmm/dd/yyy hh:mm:ss；可使用其他时间戳
格式
Datemm/dd/yyyy
Boolean0 → false，非 0 → true
Binary所有字符，以 base64 形式呈现

在查看大型手动录入表的拟议变更时，加载时间可能较长。作为替代方案，可以并排查看原始表。

后续步骤

将数据集或媒体集添加到管道构建器后，可以更改其计算模式，或者选择转换数据及添加输出。
