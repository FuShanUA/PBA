创建自定义发布通道

除了Apollo中的默认发布通道，您还可以选择创建自定义发布通道，以获得更多的灵活性和对产品发布的控制。

要创建自定义发布通道，请导航到发布通道页面上的设置与配置选项卡。选择创建发布通道。

将打开一个对话框，用于输入有关发布通道的信息，例如名称和描述。

标签要求

要求选项卡定义了发布通道的标签要求。

了解有关如何为产品发布添加标签的更多信息。

Apollo 对于向发布通道进行自动和手动提升都强制执行标签要求。

要添加标签要求，请选择添加标签要求，然后选择一个标签ID。您可以选择开启需要值以强制要求标签ID和值的组合，然后输入所需的值。

标签要求仅在发布通道准入时执行，这意味着如果产品的标签发生变化，该产品发布不会被从发布通道中移除。

内置的发布通道（RELEASE、RELEASE_CANDIDATE、DEV）不支持标签要求，因为根据其版本类型，发布会自动添加到这些通道中。

发布通道权限

Apollo 使用 基于角色的访问控制 (RBAC) 管理发布通道权限。

了解有关不同 发布通道的角色 的更多信息，以及如何 为发布通道分配角色。
