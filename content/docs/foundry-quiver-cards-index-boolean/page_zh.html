布尔卡片

返回：卡片索引

本节中的卡片用于返回和显示布尔值。

以下卡片接受字符串并返回布尔值：

  • 包含
  • 不包含
  • 以此开头
  • 以此结尾
  • 是
  • 不是
  • 字符串转布尔值

以下卡片接受数字并返回布尔值：

  • 等于
  • 不等于
  • 大于
  • 大于或等于
  • 小于
  • 小于或等于

以下卡片接受日期/时间并返回布尔值：

  • 早于
  • 在当天或早于
  • 晚于
  • 在当天或晚于
  • 在当天
  • 不在当天
  • 介于之间（含边界）

其他布尔卡片包括：

  • 布尔参数
  • 合并
  • 转换表格列值
  • 可视化函数布尔值
  • 代码函数值
  • 数组开头
  • 数组结尾
  • 数组包含
  • 索引处的值
