使用 AIP 分析员

本页介绍了构成 AIP 分析员会话的功能和概念。

上下文

上下文是 AIP 分析员在分析过程中可以访问的信息，例如您之前的对话、代理的工具结果以及您添加的任何 Foundry 资源。

您可以使用输入字段中的 + 按钮手动将资源添加到上下文中。您也可以直接将 Foundry 资源拖放到聊天输入区域以将其作为上下文添加。从其他浏览器标签或 Foundry 内部拖动资源 URL 会自动解析该资源并将其加载到您的分析中。支持的资源包括数据集、对象集、便签文档、Workshop 模块、函数等。您还可以直接将 Foundry 资源标识符（RIDs）粘贴到输入中以导入它们。

你可以使用 大纲 手动管理上下文，或者让 AIP Analyst 使用 上下文清理 工具来处理。

设置

设置 菜单包括分析设置和用户偏好。分析设置会影响当前的分析，并与 分析资源 一起保存。使用 默认 选项卡配置适用于新分析的值。

分析设置

使用分析设置来控制搜索范围、语义搜索行为以及 AIP Analyst 可以作为上下文包含的个人资源。

搜索范围

使用这些设置来限制 AIP Analyst 可以搜索的内容：

  • 本体论: 将搜索限制为单一本体论。
  • 对象类型组: 将搜索限制为所选本体论内的特定对象类型组。
  • 指南针项目： 限制搜索范围，仅包括特定指南针项目内的对象类型。
  • 对象类型状态： 按状态筛选可发现的对象类型：认可、活跃、实验性、已弃用 或 示例。
  • 对象类型可见性： 按可见性筛选可发现的对象类型：显著、正常 或 隐藏。

当设置这些限制时，对象类型搜索 和 对象搜索 工具只会返回符合要求的结果。对于拥有数百或数千个对象类型的本体论，应用这些过滤器可以提高性能。

语义搜索

  • 对象集语义搜索相似度阈值： 设置对象集工具使用的语义搜索过滤器的默认相似度阈值，取值范围为 0.0 到 1.0。数值越高，返回结果越少但更相关。

上下文来源

  • 包含最近资源： 在对话中包含您最近查看的资源的相关信息。
  • 包含收藏资源： 在对话中包含您收藏的资源的相关信息。

用户偏好

用户偏好适用于当前用户的分析：

  • 主题： 设置颜色主题。
  • 通知：如果启用，AIP 分析师可以在后台发送通知，使您能够提问并在分析需要您的注意时收到通知。

标签页与分支

分析路径可以在任何一点分叉，创建一个仅包含先前上下文的新标签页，并允许用户从相同的起始状态探索多个分析路径。可以使用标签页标题中的 + 按钮创建空的分析路径。即使不在焦点中，标签页也会继续运行。

大纲

分析大纲提供了会话的结构化摘要，显示您的问题、手动添加的上下文以及代理使用的工具。您的消息以圆圈图标显示，而工具调用则以对应其功能的图标标记。

您可以使用大纲来：

  • 快速浏览之前的分析步骤。
  • 查看每个工具调用的标记使用情况。
  • 通过选择悬停在大纲项上时出现的眼睛图标，隐藏特定工具的结果。

图

为了提高对AIP分析师输出的信心，您可以审查图视图来跟踪分析过程，该视图是一个有向图，显示了每一步分析的来源和逻辑路径。图视图允许您：

  • 追踪代理是如何得出其结论的，并检查每一步的逻辑。
  • 审计分析过程中应用的数据转换，并确保结果的可重复性。
  • 验证结果是否基于实际数据而不是幻觉。使用AIP分析师，您可以查看每一步中使用的本体论数据，并确保所有结论都是准确的。

音频输入

您可以直接从聊天界面录制音频输入，实现语音驱动分析。

导出为PDF

您可以通过从会话标题中选择导出为PDF选项，将分析会话导出为PDF。导出对话框允许您选择要包含的上下文项、展开或折叠工具结果，并在打印前预览文档。此功能使用浏览器的打印对话框，使您可以将输出保存为PDF文件。
