将 Foundry 规则添加到 Marketplace 产品

使用 Foundry DevOps 将您的 Foundry 规则工作流包含在 Marketplace 产品 中，并允许其他用户安装和重用它们。 了解如何创建您的第一个产品。

支持的功能

所有 Foundry 规则功能均受支持。

将 Foundry 规则工作流添加到产品中

要将一个 Foundry 规则工作流添加到一个产品中，首先 创建一个产品，然后选择 Workshop 应用程序 内容类型，接着选择您的 Foundry 规则编写应用程序，如下所示。

添加您的Workshop应用程序后，在您产品的输入部分中转到Foundry规则工作流部分，并包含您的工作流。

一旦您的工作流被包含，将会有额外的对象类型和操作类型作为您产品的输入。您可能希望将Rule和Proposal对象类型以及所有生成的操作类型都包含到您的产品中。

当将产品的安装模式设置为 Production 时，请确保在本体论管理器应用程序的 Datasources 选项卡中为 Rule 和 Proposal 对象类型启用 Only allow edits via actions。如果没有此步骤，用户在尝试创建提案时将遇到 Actions:PermissionDenied 错误。
