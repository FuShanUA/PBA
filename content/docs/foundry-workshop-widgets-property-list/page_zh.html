属性列表

属性列表微件用于展示单个指定对象的属性列表。

配置选项

  • 输入对象集
    • 该输入变量决定了微件中展示的对象数据。
    • 如果对象集包含多个对象，微件仅展示第一个对象。
  • 从场景加载数据
    • 开启后，支持从选定的场景加载对象数据。
  • 布局
    • 调整微件中所展示属性的位置。属性值可以展示在对应的属性类型标签旁边或下方。
  • 属性配置
    • 选择要在微件中展示的属性，并指定展示的列数。
  • 展示安全标记
    • 开启后，属性安全标记将以浓缩的灰色胶囊形式呈现，选中时会展开窗口视图。
  • 隐藏空值属性
    • 开启后，空值属性将从列表中隐藏。

不支持的属性类型

为提升性能，部分大型属性（如 Geoshape 和 Vector）默认不予加载。在视图模式下，用户可点击不受支持属性旁的加载按钮，按需查看其值。配置过程中，不受支持的属性会带有警告图标和提示框。

行内编辑

属性列表微件支持属性值的行内编辑。如需为某属性启用行内编辑，请在 Ontology Manager 中为该属性配置行内操作。完成行内操作配置后，用户即可直接在属性列表微件中编辑属性值。

如果属性列表微件中某属性的行内编辑功能无法正常使用，请检查是否已在 Ontology Manager 中为该属性配置了行内操作。
