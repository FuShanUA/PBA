设置

下文将探讨如何实现 Power BI® 连接器，以便从 Power Query 界面访问 Foundry 资源。如果您要查找有关用于数据集成的 Microsoft Power BI® XMLA 连接器的信息，请参阅我们的《数据连接文档》。

现在，您可以从 Power BI® 访问 Palantir Foundry 数据集，并利用它们来构建 Power BI® 报告和可视化图表。要在 Power BI® Desktop 中使用 Foundry，您的本地计算机上必须同时安装 Foundry 连接器和 Foundry ODBC 驱动程序。请按照以下指南为 Power BI® Desktop 完成安装。

第 1 步：验证是否已安装连接器

如果您使用的是 2020 年 6 月或更高版本的 Power BI®，那么您的 Power BI® 中应该已经安装了 Palantir Foundry 连接器。您可以通过打开 Power BI®，点击“获取数据”，然后在“联机服务”列表中查找“Palantir Foundry”来进行验证。

如果在列表中看到了 Palantir Foundry，请继续执行第 2 步以安装 ODBC 驱动程序。

如果在列表中未看到 Palantir Foundry，请联系您的 Palantir 代表以了解后续步骤。您可能需要升级到最新版本的 Power BI®。

第 2 步：安装 Palantir Foundry ODBC 驱动程序

要完成 Foundry Power BI® 集成的设置，您需要安装一个名为 ODBC 驱动程序的附加组件。请前往《下载页面：ODBC 驱动程序》下载并安装该驱动程序。

如果在安装过程中遇到任何问题，请联系您的 Palantir 代表。

或者，如果您无法安装 ODBC 驱动程序，可以按照说明使用《基于 REST 的 Palantir Foundry Power BI® 连接器》，该连接器只需互联网连接即可将 Palantir Foundry 连接到 Power BI®。请注意，此连接器存在更多限制，且性能预计不如使用 ODBC 驱动程序的内置 Palantir Foundry 连接器。因此，建议尽可能使用内置连接器。

第 3 步：开始构建报告

安装 ODBC 驱动程序后，您可以按照《Power BI®：入门指南》中的说明，开始构建由 Foundry 数据提供支持的第一个报告。

Power BI® 和 Power BI® 徽标是微软集团公司的商标。
