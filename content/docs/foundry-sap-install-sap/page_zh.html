安装 SAP 插件

面向 SAP 应用的 Palantir Foundry Connector 2.0（即“连接器”）由 Palantir 与 Diskover Limited ↗ 联合开发，是一款基于 ABAP 的数据抽取接口，用于将 SAP 系统中的数据集成至 Palantir Foundry。该连接器运行于 SAP ABAP 平台（前称 SAP NetWeaver Application Server）之上，通过 REST API 调用并基于 HTTPS 协议提供数据服务。

该连接器可用于将数据从 SAP 直接同步至 Foundry、探索 SAP 中的数据，以及通过远程函数将数据写入 SAP。

该连接器包含四个应用程序组件：

  • PALANTIR（必需）：Palantir 基础组件包含了 PALCONN（连接器）、PALAGENT（远程智能体）和 PALODATA（通过 OData 接入 SLT）所需的所有共享对象。
  • PALCONN：Palantir Foundry Connector 应用程序组件（PALCONN）包含了所有与连接器相关的对象和服务。该组件不适用于远程智能体，且无法安装于 SAP_BASIS（技术组件版本，适用于 SAP NetWeaver 和 SAP S/4HANA 系统）7.4 SP05 以下的版本。
  • PALAGENT（可选，适用于 SAP_BASIS 7.4 SP05 以下版本）：Palantir Foundry Remote Agent 应用程序组件（PALAGENT）包含了所有与远程智能体相关的对象和服务，用于从运行在 SAP_BASIS 7.4 SP05 以下版本的 SAP 应用程序中摄取数据。对于 SAP_BASIS 7.4 SP5 以下的系统，必须通过已安装 PALCONN 组件且运行 SAP_BASIS 7.4 SP5 及以上版本的系统进行远程访问。请查阅远程架构文档以获取更多详细信息。
  • PALODATA（可选，仅适用于 SLT 系统）：Palantir Foundry OData 应用程序组件（PALODATA）包含了通过 OData API 摄取 SLT 数据的相关对象和服务，也称为通过 OData 进行基于 ODP 的数据抽取 ↗。

在将其安装至生产环境之前，请先将插件安装至开发或沙箱系统，并进行全面测试。

安装场景

该连接器提供三种安装场景。

对于所有场景，在继续操作前，请确保已下载相关的安装包。

独立安装

该连接器可直接安装至源 SAP 系统中。此场景支持以下对象类型：

  • SAP 表和视图
  • InfoProvider
  • BEx 查询
  • SAP Business Content Extractor
  • SAP 函数 / BAPI
  • 基于 SAP ABAP 的 CDS 视图
  • 通过 SAP 函数回写（可选择使用 OAuth 2.0 进行用户归属标记）

针对此场景，请遵循以下指南：

  • 安装连接器

SLT 安装

对于连接器 2.34（SP34）及更高版本，必须安装额外的 PALODATA 组件才能连接至 SLT。

在此场景中，连接器将安装至 SAP SLT Replication Server 上。

此情况下仅支持 SLT 对象类型。

针对此场景，请遵循以下指南：

  • 安装连接器
  • 配置 SAP SLT

远程智能体安装

源 ERP 系统可能无法满足连接器的安装要求。在此场景中，应部署一个 SAP_BASIS 版本为 7.4 SP5 及以上的 SAP ABAP 平台，作为源 SAP 系统的网关。连接器应安装在该网关服务器上，通过 RFC 连接与源 ERP 系统进行通信，并基于 HTTPS 协议向 Foundry 提供数据服务。连接器的远程智能体应安装在源 ERP 系统上，以响应来自连接器的请求。

针对此场景，请遵循以下指南：

  • 使用主安装指南将连接器安装至网关服务器
  • 安装远程智能体

如果源 ERP 系统的 SAP_BASIS 版本低于 7.0 SP32，请遵循以下指南：

  • 安装适用于 ERP 4.6C/4.7 (620/640) 的远程智能体
