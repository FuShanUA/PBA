回写卡片

返回至：《卡片索引》

可以在 Quiver 分析画布、时间序列图和仪表板中添加操作按钮，以便用户将数据回写至本体；例如创建对象、更新现有对象的属性或修改对象链接。

  • 添加并配置操作按钮
  • 直接从时间序列图调用本体操作
  • 将选区边界值作为本体操作的输入
  • 自定义按钮显示

示例

此功能可用于增强各种工作流，包括：

示例 1：创建 Annotation 对象实例

在时间序列图中，可以创建一个对象来记录所观测现象的上下文信息，例如机械故障的原因或影响股票指数的市场宏观事件。X 轴选区的起始和结束时间值将作为输入传递给 创建标注 本体操作，进而生成一个标注事件。通过在图表上对多个事件进行标注，可以更清晰地展示当前的数据可视化内容。

示例 2：调整运营值

还可以调用本体操作，根据观测到的现象更新属性值。例如，下方的时间序列图展示了两条曲线，分别表示两类产品的预计每桶总成本：汽油（蓝色）和柴油（红色）。我们可以看到，今天汽油的实际每桶总成本（Y 轴，深蓝色点）在 6.35 到 8.45 美分/桶之间，远高于 6.01 美分/桶的最低预计总成本。为了降低生产成本，我们可以将 DRA 浓度从 15 ppm 调整为接近 6.5 ppm 的值。通过在 X 轴上选择 6 附近的范围，随后选择 调整 DRA 等级 本体操作，系统将自动使用选区边界值进行设置。

添加并配置操作按钮

若要向分析中添加操作按钮：

  1. 从分析顶部栏中选择 Search cards，打开搜索栏。
  2. 搜索 action，或导航至左侧的 Writeback 类别。
  3. 选择 Action button 卡片。

若要配置操作按钮卡片：

  1. 在参数面板中找到操作按钮。
  2. 选择 Configure 齿轮图标，打开操作按钮编辑器。
  3. 从下拉列表中选择现有的本体操作。如果该操作尚不存在，则需要在本体中创建新操作。

选择操作按钮后会显示一个表单，用于填写输入并提交操作。在操作按钮编辑器中，可以提供默认输入。

例如，在下图中，Event start 参数配置为默认值 2010 年 1 月 1 日。

选择操作按钮后，会将 2010 年 1 月 1 日显示为 Event start 参数的输入值。用户仍可在提交操作前编辑此值。

直接从时间序列图调用本体操作

可以直接在时间序列图的选择菜单中显示 Ontology Action 按钮。若要向时间序列图选择菜单添加操作按钮：

  1. 选择卡片标题上的 Configure 齿轮图标，打开时间序列图编辑器。
  2. 在操作按钮（Action Buttons）部分，打开下拉菜单，从分析中选择一个现有的操作按钮，或者新建一个。本体（Ontology）操作按钮随即会显示在选择菜单中。

将选区边界值作为本体操作（Ontology Actions）的输入

将本体操作按钮添加到时间序列图后，即可使用 x 轴或 y 轴选区的边界值对本操作的输入进行参数化处理。

若要将选区边界值作为本体操作的输入：

  1. 打开操作按钮编辑器，展开时间序列图表输入覆盖（Time Series Chart Input Override）部分。
  2. 根据 x 轴选区、y 轴选区或同时依据两者添加覆盖项。系统会根据数据类型将选区边界映射至对应的操作参数。例如，时间轴上的 x 轴选区可映射至日期/时间操作参数。

在以下示例中，选区的左右边界分别映射至事件开始（Event start）与事件结束（Event end）参数。从时间序列图选择菜单中点击该操作按钮，即可自动填充这些数值。

自定义按钮显示

在操作按钮编辑器的显示（Display）选项卡中，可自定义按钮外观，包括标签、图标、颜色与样式。
