预览转换

在代码仓库中使用预览工具，可以对输入数据集的有限样本运行代码，从而快速预览输出结果。预览功能会生成样本输出，而无需提交更改、运行检查或在 Foundry 中实体化任何数据集。预览能够加速开发周期，无需触发构建即可测试代码更改。

提示

预览适用于所有 Foundry 数据集，包括包含文件和模型的数据集。

运行预览

在代码仓库中，可以通过两个入口触发预览。

(1) 在代码编辑器选项面板中选择预览：

(2) 在辅助面板中选择预览：

预览执行完成后，将显示输出结果：

配置文件预览

预览可用于包含非结构化文件的数据集。首次对包含文件的数据集运行预览时，必须配置要在样本中使用的文件。

选定样本文件后，可通过从输入列表中选择相应输入来重新配置它们。保存配置后，预览将对所选文件样本执行代码。再次运行预览时，无需重新配置输入文件。预览执行完成后，您可以按行或文件形式查看样本输出。如果您拥有所需权限，还可以选择下载输出文件。

配置模型预览

模型资产

对于在 Foundry 中训练或由预训练文件支持的模型资产，无需额外配置即可支持预览。

容器支持的模型和外部托管的模型目前不支持预览。

预览在转换生成器中创建的转换

在转换生成器中创建的转换会共享函数名；为了更方便地选择预期进行预览的转换，请修改所生成转换的 __name__ 属性，以生成具有实际意义的名称。例如：
