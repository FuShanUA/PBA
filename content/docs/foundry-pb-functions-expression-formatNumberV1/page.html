Format number

Supported in: Batch, Faster, Streaming

Formats a number to a specific number of decimal places.

Expression categories: Cast, Numeric, String

Declared arguments

  • Decimal places: The number of decimal places.
    Literal<Integer>
  • Number: The number to format.
    Expression<Numeric>

Output type: String

Examples

Example 1: Base case

Description: Formats a number to 2 decimal places.

Argument values:

  • Decimal places: 2
  • Number: 1234.5678

Output: 1,234.57

Example 2: Base case

Description: Formats a byte number to 1 decimal place.

Argument values:

  • Decimal places: 1
  • Number: 42

Output: 42.0

Example 3: Base case

Output: 1,234.6
