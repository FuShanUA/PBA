数据集预览

数据集预览应用提供了指定数据集的各类详细信息，包括元数据、构建历史、健康状态等。流式数据集还支持更多功能，例如查看流式作业信息和指标，以便排查并调试流性能。

下方截图展示了数据集预览应用的界面。带有编号的各个区块将在后续章节中详细说明。

1. 数据集头部

页头标识了所选数据集，并提供了名称、显示名称（如有）、位置和所选分支等基本信息。页头还支持分享、移动、重命名等文件相关操作。

2. 信息面板

信息面板提供了数据集的元数据以及一些基础管理操作。该面板分为三个部分：

  • 关于： 包含数据集创建和更新时间、创建者和最后更新者、表大小、用于创建该数据的工具和输入数据集、标签等信息。在更新者部分下还提供了编辑架构视图，可推断 CSV 和 JSON 文件的架构。用户在此还可以应用额外的解析选项，以丢弃不规则行、更改编码，或添加文件路径、行字节偏移量、导入时间戳或行号等附加列。对于其他文件类型，可在详情标签页下的架构部分进行架构编辑。
  • 列： 包含数据集中各列的信息，包括数据类型、描述和数据统计信息（空值百分比、分布和样本）。
  • 调度： 包含有关已配置的构建调度的信息，这些调度将运行以更新数据集。

3. 标签页视图

预览

预览标签页以表格形式提供了数据样本，并允许与完整数据集进行轻度交互。有关预览表格的更多信息，请参阅 4. 预览表格。

历史记录

历史记录标签页视图提供了历史作业（构建）信息。页面右侧的摘要视图展示了随时间变化的作业状态汇总信息。

在左侧面板中，作业列表显示了各自的状态和持续时间。选中某个作业后，右侧会显示详细的作业视图，包含进度、规格、构建日志、文件和生成的架构等详细作业信息。

流式数据集

对于流式数据集，只有将视图设置为归档时才会显示历史记录标签页。该标签页将并列显示归档事务和流式作业。

从历史事务创建分支

你可以使用历史记录标签页，针对未被保留策略删除的数据历史事务创建分支。从左侧面板中选择先前的事务，然后选择省略号 (...) 图标以创建分支。

详情

详情视图提供了有关数据集的更多技术信息，以及一些管理操作：

  • 架构： 提供有关表架构（列规格）的完整信息，并允许你编辑架构（如适用）。
  • 文件： 显示组成数据集的文件列表，并允许你下载这些文件。
  • 作业规格： 显示包含数据集构建所需基本信息的作业规格。
  • 同步： 呈现数据同步至不同数据库的状态与详细信息。针对某些同步类型，还可以应用附加设置。
  • 自定义元数据： 允许您向数据集添加自定义信息字段。在此部分添加的字段将显示在概览页面的信息面板中。
  • 资源使用指标： 提供数据集随时间变化的磁盘和 Spark 使用情况的图表与信息。
  • 最近运行详情（仅限流）： 显示有关最近一次流运行的详细信息。

流（仅限流式数据集）

当数据集为流式数据集时，流选项卡将显示流式作业的当前与历史信息。通过更改时间段，您可以探索在该时间段内进行数据流式传输的作业日志与详情。

指标（仅限流式数据集）

当数据集为流式数据集时，指标选项卡会显示用于分析和排查流式作业性能问题的图表及相关交互功能。该页面包含一个下拉菜单，用于选择指标以可视化趋势，并在其中高亮显示推荐的调试指标。指标图表可展开以进行详细查看，并显示阈值和带有调试链接的警告指示器。

排查与调试模式

您可以使用页面右上角的开关启用专用的排查模式。该模式提供逐步指引，帮助您调试流中断问题。从右侧面板中选择内联指标标签，即可高亮显示相应的图表，轻松定位问题根源。

健康度

健康度选项卡提供了用于监控数据健康度的工具。该页面显示针对特定资源的健康度检查、按特定监控视图分组的资源监控规则，以及影响该资源的相关计划构建。选择任意行即可查看健康度检查和监控规则的历史报告。

流式数据集

在流式数据集中，健康度选项卡仅在视图设置为归档时才会出现。随后，检查将针对归档数据集而非流进行。

比较

使用比较选项卡来对比两个不同的数据集。选择该选项卡并选择一个要比较的数据集。比较选项卡可用于以下几种场景：

  • 比较两个独立的数据集以了解它们之间的差异。
  • 将数据集与同一数据集的旧事务进行比较，以观察其随时间的变化。
  • 将数据集的原始版本与不同分支进行比较，以了解合并该分支对数据集产生的影响。
流式数据集

在流式数据集中，比较选项卡仅在视图设置为归档时才会出现。随后，您可以将归档数据集与其他非流式数据集进行比较。

4. 预览表

使用预览表可以了解数据结构，并快速探索数据集中的值。

默认情况下，预览表仅显示部分数据样本，具体的行数会显示在预览表的标题栏中。但是，对数据执行的任何操作（如筛选或排序）都会应用于整个数据集，并增加预览样本的大小。根据行数的不同，您可能无法在预览中看到完整的数据集。

预览表提供以下几项实用功能：

  • 选择列菜单，可对该列数据进行排序、筛选和生成图表。
  • 选择单个单元格，可从预览中排除或仅包含所选值。
  • 报告和查看各个列的问题。
  • 搜索特定的列名。
流式数据集

流式数据集的数据预览表提供了少量最近流式传输的数据行样本。设置为实时更新时，它会自动更新。排序、筛选和生成图表功能仅在页面设置为归档时可用，并且仅反映归档数据集的状态。

手动上传文件

在数据集预览中，您可以将以下类型的文件直接上传至数据集：.csv、.tsv、.xls、.xlsm 和 .xlsx。

对于 .csv 和 .tsv 文件，Foundry 会尝试推断新文件的架构。如果新文件的文件名和架构与之前上传的文件相同，您可以更新现有数据集中的数据。如果文件名与之前上传的文件不同，您可以追加数据到现有数据集。

以下步骤适用于上传所有文件类型：

  1. 导航到您首选的文件夹并创建数据集。

  1. 将文件拖放到数据集预览窗口中。

5. 操作

所有操作下拉菜单提供了对 Foundry 工具和操作的快速访问，允许您分析、探索、转换和管理数据。某些操作（如 Contour 中的分析和构建）会显示在操作菜单之外，以便快速访问。
