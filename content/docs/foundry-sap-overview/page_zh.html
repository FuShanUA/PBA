SAP

将 Foundry 连接到 SAP 包含两个步骤：

  1. 在 SAP 端安装高级业务应用编程 (ABAP) 插件。
  2. 在 Foundry 端配置与你的 SAP 架构相匹配的源：
    • SAP ERP：SAP ECC、S/4HANA（本地部署和云私有版）以及其他 ERP 核心组件系统。
    • SAP SLT：通过 SAP 景观转换 (SLT) 复制服务器的连接，包括实时流式同步。

关于 SAP 商业数据仓库 (BW) 和现有集成，请参阅《SAP（自定义源）》。

本页的其余部分将介绍连接器插件，并概述端到端的高级设置流程。

Palantir Foundry Connector 2.0 for SAP Applications（“连接器”）是一款 SAP 认证插件，由 Palantir 与 Diskover Limited ↗ 合作开发。

该连接器会安全地从 SAP 系统（S/4HANA、ECC、Business Warehouse、SLT Replication Server）中采集数据和元数据，并将其集成到 Foundry 平台中。连接器安装在 SAP 应用层，遵循标准的 SAP 安全策略，并使用原生的 SAP 应用逻辑进行数据访问。

  • 连接器安装：该连接器是一款 SAP 认证插件，通过 SAINT（SAP 插件安装工具）进行安装。

  • 连接器通过 HTTPS 公开访问：该连接器通过 SICF 运行 Web 服务。这使得 Foundry 数据连接器能够通过 HTTPS 请求底层的 ERP 或 BW 数据。

  • 在 Foundry 中定义数据传输逻辑：数据传输的表、对象、过滤器和调度均在 Foundry 中定义，并由 Foundry 数据连接协调器执行。

  • 应用层访问：Palantir Foundry 会将请求发送至 SAP ABAP 平台（前身为 SAP NetWeaver Application Server），而不受 SAP 底层运行的是何种数据库的影响。不存在直接的数据库访问。所有信息访问均从应用层进行。

  • SAP 标准安全：Foundry 将使用在 SAP 中受到安全和授权的 SAP 用户来调用连接器。因此，所有 SAP 标准安全流程和策略均适用。无需额外维护数据流安全。

  • SAP 应用逻辑访问：表、函数、BW InfoProviders 和 BEx 查询均可作为向 Foundry 提取数据的可用对象。

  • 系统负载和可扩展性：连接器在开始提取前会检查系统的数据负载。如果不满足特定条件，提取操作将被中止。连接器与 SAP 的可扩展性完全保持一致。连接器可以共存于同一应用服务器中，或者选择在专为数据处理保留的独立应用服务器中运行。

设置指南

先决条件

  • SAP_BASIS 7.4 SP5 或更高版本（技术组件版本，适用于 SAP NetWeaver 和 SAP S/4HANA 系统）
  • SAP_BASIS 7.5（无最低 SP 级别要求）

这些是最低版本要求。连接器支持所有更高版本的 SAP_BASIS，包括基于 SAP_BASIS 8.x 的 SAP S/4HANA 版本（例如 SAP_BASIS 816）。

如果你计划通过 OData 使用 SLT，OData 组件（PALODATA）要求 SAP_BASIS 7.50 SP09 或更高版本。

如果你的主要 SAP 系统运行的 SAP_BASIS 版本低于 7.4 SP5，请参阅《远程代理安装》了解如何设置远程连接的详细信息。

入门指南

以下是连接组织 SAP 系统到 Foundry 的高级步骤。

  1. 与您的 SAP Basis 团队协作，决定哪种连接模式最适合您的 SAP 架构环境。阅读有关三种连接模式的更多内容。

  2. 在您的组织网络内创建一个空间，用于安装智能体。

  3. 确保智能体具备必要的网络出站权限，以便与 Foundry 通信。

    智能体用于通信的 SAP 服务器端口，是 ICM（Internet Communication Manager，SAP ABAP 平台的组件之一）的默认 HTTPS 端口。该端口号可以通过在 SAP 系统中运行事务代码 SMICM 来获取。

  4. 从 Foundry 下载连接器附加组件安装包。

  5. 请负责的团队（例如您的 SAP Basis 团队）在相应的 SAP 系统上安装适用于 SAP 应用程序的 Palantir Foundry Connector 2.0 附加组件。

  6. 通过数据连接（Data Connection）下载智能体软件并进行安装。

  7. 在 Foundry 中设置一个数据源以连接到 SAP 系统。Foundry 侧的配置取决于哪种连接器与您的 SAP 架构环境相匹配：

    • SAP ERP：用于连接 SAP ECC、S/4HANA（本地部署版和云私有版）以及其他 ERP Central Component 系统。
    • SAP SLT：用于通过 SAP Landscape Transformation (SLT) Replication Server 进行连接，包括实时流式同步。

    有关 SAP Business Warehouse (BW) 及现有集成的信息，请参阅 SAP（自定义数据源）。
