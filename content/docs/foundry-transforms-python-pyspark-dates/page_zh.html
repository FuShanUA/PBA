日期和时间戳

1. 添加或减去日期

向日期添加或减去天数

  • F.date_add(start, days)
  • F.date_sub(start, days)

向日期添加月份

  • F.add_months(start, months)

获取两个日期之间的天数或月份数

  • F.datediff(end, start)
  • F.months_between(date1, date2)

获取当月的最后一天

  • F.last_day(date)

获取下一个星期几的日期

  • F.next_day(date, dayOfWeek)

2. 日期值

获取年、月、日、分钟、秒

  • F.year(column)
  • F.month(column)
  • F.dayofmonth(column)
  • F.hour(column)
  • F.minute(column)
  • F.second(column)

从日期中获取业务季度

  • F.quarter(column)

从日期中获取天数或周数

  • F.dayofyear(column)
  • F.weekofyear(column)

日期和时间格式语法

从字符串转换

  • F.to_date(column, format=None)
  • F.to_timestamp(column, format=None)
  • F.to_utc_timestamp(timestamp, tz)
  • F.unix_timestamp(timestamp=None, format='yyyy-MM-dd HH:mm:ss')

转换为字符串

  • F.date_format(date, format)
  • F.from_unixtime(timestamp, format='yyyy-MM-dd HH:mm:ss')
  • F.from_utc_timestamp(timestamp, tz)

从 long 转换到 timestamp

一些系统将时间戳存储为 long 数据类型，以毫秒为单位。PySpark SQL 将时间戳存储为秒。我们必须将 long 时间戳版本除以 1000，才能正确地将其转换为 timestamp：

Copied!
1casted_timestamp = (F.col('timestamp') / 1000).cast("timestamp")
2df = df.withColumn("timestamp", casted_timestamp)
3# 1531860192661 =>  Tuesday, July 17, 2018 8:43:12 PM

我们可以使用 F.from_unixtime(timestamp) 以提高清晰度：

Copied!
1timestamp = F.from_unixtime(F.col('timestamp') / 1000)
2df = df.withColumn("timestamp", timestamp)

当从 long 转换为 timestamp 时，我们会失去一个级别的粒度。SQL 无法存储百分比或秒的小数。

截断

  • F.trunc(date, format)
  • F.date_trunc(format, timestamp)
