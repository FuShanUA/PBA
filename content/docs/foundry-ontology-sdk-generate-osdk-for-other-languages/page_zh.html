为其他语言生成 OSDK

开发者控制台内置了对 Java、Python 和 TypeScript 的支持，可通过 pip 和 Conda 进行代码生成，但不仅限于这些语言。开发者控制台还支持以行业标准的 OpenAPI 格式 ↗ 导出 API。你可以使用开源代码生成器，基于下载的 OpenAPI 规范生成几乎适用于任何语言的客户端。

导出 OpenAPI 规范

在开发者控制台应用中导航至 Application API 页面，并打开 SDK generation 选项卡。然后，选择 Other languages 并点击 Export as OpenAPI。

由于导出的文件将包含开发者控制台应用中所含资源的名称与描述，请确保这些字段不包含敏感信息。

使用其他语言生成客户端和服务端

导出 OpenAPI 文件后，你可以使用开源生成器生成客户端和服务端。可以在 OpenAPI 生成器网站 ↗ 上找到 OpenAPI 生成器列表。
