创建建模目标

“建模目标”应用为团队提供了一个直观的工作空间，便于团队在模型整个生命周期中进行管理。创建建模目标，可将模型、元数据和数据源与模型的审查、发布和部署集中在一起，从而简化建模项目。

在 Foundry 侧边栏中选择 Modeling Objectives 应用，点击 New objective 即可创建建模目标。

创建建模目标后，您可以选择目标保存位置。建议将建模目标保存在集中位置，确保所有相关贡献者、利益相关者和消费者都能访问该项目。

配置建模目标

建模目标可通过多种方式进行编辑和配置。本部分提供有关建模目标可用配置选项的更多信息。

编辑描述

创建建模目标后，点击 Edit description 按钮即可添加描述。描述字段支持 Markdown ↗；建议将此空间用作该目标及其目标的 README。

编辑目标元数据

编辑目标元数据有多种可用选项，包括：

  • 建模目标检查：确保模型在投入运行前通过预定义的质量检查。
  • 模型提交元数据：使建模目标数据主人能够确保提交至该目标的所有模型均具备所需的元数据。
  • 模型评估：允许配置建模目标，自动生成并为所有模型提交构建推理和指标管道。从而实现软件中的系统化测试与评估计划。
