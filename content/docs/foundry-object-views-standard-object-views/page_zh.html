标准对象视图

当你在本体论中创建并配置一个 对象类型 时，Foundry 会自动生成一个标准的对象视图，以提供其所有对象的标准化表示，确保其他用户能够全面了解其架构和链接。标准对象视图通过突出显示显著属性来匹配对象类型的配置，如果该属性的 基础类型 是时间序列、媒体引用或地理空间属性，则以专用表格或其他可视化格式显示。常规属性显示在普通表格中，隐藏属性则不可见。

虽然所有对象类型都提供标准对象视图，但你可以创建 配置视图，根据对象类型在本体论中的功能，为任何对象类型提供上下文化和精选的对象视图体验。当你创建一个配置的对象视图时，Foundry 会将其作为用户的默认视图；然而，用户可以选择选择标准视图以查看对象数据的标准格式。

突出属性显示

Foundry 在标准对象视图的顶部显示 突出属性，以提供有关对象最重要信息的即时上下文。

显示行为

被标记为突出的属性会根据其类型获得增强的视觉效果：

  • 媒体引用属性: 使用专用的媒体查看器显示所有支持的媒体类型。
  • 时间序列属性: 显示为交互式图表，展示时间数据模式。
  • 地理空间属性: 拥有突出的geohash、geoshape或geotemporal系列引用（GTSR）属性的对象将在 地图 上显示。那些 表示对象随时间变化的纬度和经度 的时间序列属性也会在地图上显示。
  • 其他属性类型: 所有其他突出属性均以较大的卡片式格式显示，高于显示其余标准属性的表格。

Linked objects component

The Linked objects component enables you to traverse across linked objects directly within the standard Object View.

Use the Linked objects component of a standard Object View to:

  • View linked objects grouped by link type.
  • Preview properties of linked objects inline without leaving the current view.
  • Open a subset of linked objects in a new tab for further exploration.
  • Preview a selected linked object in the side panel of the standard Object View.

Panel standard Object View display behavior

You can also interact with all of a standard Object View's functionality in its panel form factor.
