逻辑

应用程序在向用户展示数据或以其他方式可视化之前，往往需要对数据进行修改或丰富。同理，应用程序要按预期运行，可能也需要额外的信息。例如，你可能希望指定某个组件必须拥有适当的配置才能成功运行，或者希望根据应用程序的状态来显示特定信息。Slate 提供了多种基础组件，用于管理和操作状态与数据。

函数（Functions）是能够读取 Handlebars 并返回任意类型输出的 JavaScript 代码片段。函数可以处理来自查询的数据，接收组件或变量的状态以构造新值，或者为组件准备类。如需了解更多信息，请参阅 Slate 中的函数。

Handlebars 用于将值从一个组件传递给另一个组件，例如在组件中使用函数的输出，或在事件中使用查询的输出。Handlebars 允许你通过双花括号 {{ }} 访问当前流经应用程序的所有信息。如需了解更多信息，请参阅在 Slate 中使用 Handlebars。

变量（Variables）用于存储值，以保存特定状态、存储用户输入或设置默认值。你可以通过 URL 参数设置变量的值，或者通过事件更新它们。如需了解更多信息，请参阅 Slate 中的变量。

事件（Events）由单个事件和用户操作组成。事件用于触发应用程序中的活动；例如，你可以配置一个事件，使其在选中按钮时提交查询，或者在关闭对话框后显示提示信息。事件和操作负责处理 Slate 内部的各种自动化交互。如需了解更多信息，请参阅 Slate 中的事件与操作。
