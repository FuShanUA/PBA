调度器中的 AIP 功能

在创建具有特定时间触发的数据集构建调度时，可使用 AIP 生成调度配置。在新建调度视图侧边栏中输入调度触发提示词，即可针对复杂触发器生成正确的 cron 格式。

使用 AIP 功能

若要在调度器中使用 AIP Assist，请在 Pipeline Builder 中打开您的图谱，然后按照以下步骤操作：

  1. 右键单击数据集节点，选择管理调度...，然后选择创建新调度以进入数据血缘应用。

  2. 在新建调度视图中，找到何时构建并选择在特定时间。

  3. 最后，选择由 AIP 双星图标指示的建议，然后在提示词中输入您所需的调度描述，随后点击紫色箭头图标。

如需接受建议，请选择保存。如需拒绝，请选择建议并输入新的提示词，或者手动配置您的 cron 作业。

注意：AIP 功能的可用性可能会发生变化，且不同客户之间可能存在差异。
