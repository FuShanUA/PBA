创建一个演练

配置

  1. 导航到演练应用程序并选择新建演练。您将被提示为演练命名并选择保存位置。

  1. 接下来，进入演练编辑器以配置演练内容。

  1. 填写名称字段以为您提供演练的标题。
  2. 在描述字段中输入简短的描述。

注意：名称和描述都会出现在演练首页，并可以帮助用户确定要完成的适当演练。

  1. 可选地，通过选择自定义图标或上传自定义图像来自定义演练图标。
  2. 可选地，启用文档切换以显示一个专用的概述面板，用于使用Markdown显示长格式的演练详细信息。

添加步骤

指南中的主要元素称为步骤。每个步骤应包括用户应执行的操作的详细描述。您可以选择性地包含相关资源的链接，或在设置期间在文档配置中添加文档。这将在指南过程中通过了解更多选项进行发现。我们建议仅向给定步骤添加一个资源以简化最终用户的体验。要预览指南，请随时在右上角选择预览。

  1. 在步骤部分中，选择点击添加新步骤。屏幕右侧将出现一个新的面板用于步骤配置。
  2. 为该步骤提供名称和描述。
  3. 通过选择添加资源下拉菜单并选择要添加的资源类型来添加相关资源。添加的资源将在同一窗口中打开。

Publish your walkthrough

  1. Select the Preview option to test the rendering and readability of your content.
  2. Select Save.
  3. When you are ready to make your walkthrough discoverable to users with access, select Publish.

Walkthrough translations

To translate a walkthrough, select the language dropdown menu.

You can manually input translations, or use AIP to generate suggested translations for each description in the walkthrough.
