Slate

Slate 让应用开发者（构建者）能够通过拖拽界面构建动态、响应式的应用，并支持自定义设计，从而缩短开发周期、降低成本。借助极少的编码工作量，开发者即可快速搭建出可投入运作的应用程序、交互式仪表板以及具备专属样式的自定义落地页。

你可以使用 CSS 自定义应用程序的样式，为用户打造独特的使用体验，使其契合你的企业标识，或者用于尝试全新设计。在 Slate 中，所见皆可调整——从画布的背景色，到按钮的字体与圆角，无一不可修改。

借助 Slate，构建者还能构建可接入公共互联网的应用程序，全程免于配置服务器、DNS 或身份验证；这使得没有 Foundry 账号的用户也能在受验证逻辑及其他安全机制约束的前提下，向 Foundry 上传或提交数据。

你可以无缝调用本体，使用对象数据、函数与操作，并整合来自 Foundry 外部数据源（例如数据库和 API）的数据。

如果你初次接触 Slate，建议先熟悉界面，并了解如何管理 Slate 应用程序。
