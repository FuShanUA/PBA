Change Types

There are many types of changes that can be made through Apollo through in-context actions in the user interface. Each of the actions below can be accessed through the Config tab on an Environment and open varying types of change requests.

Change requests for an Installation's configuration are common. Service operators can see line-by-line "diff" views provided for change request details for all configuration changes.

Settings and permissions changes are another source of change requests within the Apollo platform.

Changes to an existing setting for an Environment, Product, or other concept supported by Apollo can be approved by the administrator for that specific setting. For changes that introduce a new setting, like creating a new Release Channel, there is a group of per-context admins who must approve the setting addition.
