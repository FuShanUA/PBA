合并数据

在数据管道构建器中，对数据进行转换和结构化处理的另一种方式是应用合并操作。合并操作会将两个数据集组合在一起，包含每个数据集的所有行。在数据管道构建器中，合并操作会保留所有行，包括重复行。

选择数据集

要合并两个数据集，请在工作区中选择第一个数据集节点，然后点击 合并。

首先选中的数据集作为左侧数据集。选择另一个数据集节点作为右侧数据集。点击开始进入合并输出预览页面。

预览合并结果

在预览面板中，点击 创建合并，然后查看输出数据集预览。

合并操作要求所有输入必须具有相同的模式。如果输入模式不完全匹配，合并操作将显示错误消息并列出缺失的列。

要解决此问题，请移除对缺失列的引用或检查您的输入。

应用合并

创建合并操作完成后，点击 应用 将其添加到工作流中。您会看到合并节点已连接到图中的两个合并数据集。我们将新的合并节点命名为 Union，它是原始 Correct columns 和 Vendor Cut 2 - demo data 数据集的直接输出。

点击合并节点并选择 编辑，即可对其重命名或进行编辑。

拖动节点上的白色或灰色圆圈可更改连接并移除图中的链接。点击合并节点上的灰色椭圆形可移除多个连接。

请注意，合并操作会保留左右两个数据集的所有行，包括重复行。如需移除重复行，请在合并输出中添加 去除重复项 转换操作。
