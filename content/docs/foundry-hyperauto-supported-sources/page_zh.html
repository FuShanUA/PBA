HyperAuto V2 支持的数据源

本页面介绍了 HyperAuto V2 支持的《数据连接》源类型。

SAP

源连接

HyperAuto V2 支持来自《Foundry SAP 数据源》的数据。 支持 SAP ECC 和 S/4HANA，不论是否配置相应的 SLT 复制服务器（流式处理需要 SLT）。要支持该数据源，必须满足以下最低要求：

  • SAP 托管源 (magritte-sap-source) 版本：1.25.0
  • 《Palantir Foundry SAP 连接器》版本：SP26 (2.26.0)

有关如何在 Foundry 中设置 SAP 的更多信息，请查阅《SAP 插件安装指南》。

基于文件夹的数据

HyperAuto V2 也支持基于 SAP 数据静态切片运行，无需直接连接。在这种情况下，您需要将以下数据字典表作为数据集上传：

  • DD02L (SAP 表)
  • DD02T (SAP DD：SAP 表文本)
  • DD03L (表字段)
  • DD04T (R-3 DD：数据元素文本)
  • DD05S (外键字段)
  • DD08L (R-3 DD：关系定义)

随后需通过 HyperAuto 处理的所有表，都必须作为 Foundry 数据集上传至同一文件夹中。

有关 HyperAuto 如何处理静态数据的更多信息，请查阅《基于文件夹的 SAP 管道文档》。
