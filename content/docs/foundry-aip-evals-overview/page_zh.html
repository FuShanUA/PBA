AIP 评估

AIP 评估是一个测试环境，用于评估您 AIP 逻辑函数、AIP 聊天机器人函数 或 代码编写的函数 的性能。它特别设计用于帮助您处理 LLM 不确定性的特性。AIP 评估允许您创建测试用例，定义评估函数以衡量性能，并将结果与您函数的先前版本进行比较。它使您可以建立必要的信心，将基于 LLM 的函数投入生产或对现有实现进行更改。

您可以使用 AIP 评估来：

  • 创建测试用例并定义评估标准。
  • 调试、迭代和改进函数和提示。
  • 比较不同模型在您的函数上的性能。
  • 检查多次运行之间的差异。

AIP 评估还可作为 AI FDE 中的一个集成工具，允许您通过对话命令创建和运行评估套件。

核心概念

评估套件：用于基准测试函数性能的测试用例集合、目标函数和评估函数。

目标函数：正在被评估的函数。一个套件可以配置为同时测试 多个目标函数。

评估函数：在将目标函数的实际输出与预期输出进行比较或评估时所使用的方法。

测试用例：在评估套件运行期间传递给评估函数的定义好的输入集和预期输出集。

指标：评估函数的结果。每个测试用例都会生成指标，并可以在运行之间进行整体或单独比较。

开始使用，请创建一个 逻辑函数的评估套件，或者创建一个 通用函数的评估套件，并了解更多关于 评估运行配置 的信息。
