在代码仓库中预览自定义文档

为了帮助您编写和编辑自定义文档，您可以使用代码仓库预览文档仓库中的 Markdown 文件。Markdown 预览与源 Markdown 代码在代码编辑器中并排显示，如下图所示。

如何开启 Markdown 预览

要在代码仓库中查看您的 Markdown 文件的渲染预览，您可以按照以下步骤开启 Markdown 预览。

  • 右键点击您打开的 Markdown 文件标签。
  • 选择 垂直拆分 或 水平拆分 以在两个面板中查看内容。
  • 在您希望以 Markdown 渲染的面板中开启 切换渲染 Markdown。

已渲染Markdown预览与发布后的自定义文档之间的差异

请注意，使用代码仓库（Code Repositories）进行Markdown预览时，Palantir自定义文档的所有功能可能无法完全呈现，因为Markdown预览使用的是“原生”Markdown，不包含Palantir特定的扩展。Markdown预览旨在帮助自定义文档的创建过程，但不会显示最终发布文档的1:1复制内容；Palantir特有的格式将在发布过程中稍后应用。

特别是以下内容在发布前不会以显示格式呈现：

  • Palantir自定义语法 - 由@标签表示，如@title或@description - 在Markdown预览中将显示为纯文本，但在发布过程中会稍后应用。
  • 导航元素，如在目录ordering.md文件中指定的，可能在Markdown预览中不可用。
  • HTML元素，如 提示框 在Markdown预览中的显示方式可能与最终发布的版本不同。
