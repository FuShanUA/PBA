函数

函数使代码作者能够编写可在操作环境中快速执行的逻辑，例如用于增强决策过程的仪表板和应用程序。此逻辑在隔离的服务器端环境中执行。

值得注意的是，函数对基于本体论编写逻辑提供了一流的支持。这包括支持读取各种对象类型的属性、遍历链接以及灵活地进行本体论编辑。

函数的常见用例包括：

  • 返回可用于 Workshop 的对象集或变量值。
  • 使用 Workshop 的函数支持列 在派生表列中显示转换后的值。
  • 聚合对象类型值以在 Workshop 图表 中显示。
  • 表达对本体论的复杂编辑，通过 函数支持的操作 更新许多对象。
  • 在后端运行逻辑，以便在 Slate 前端显示返回的信息。
  • 计算自定义指标或聚合，以在 Quiver 中显示。
  • 查询外部系统，通过 外部函数 丰富本体论中的对象。
  • 使用 Python 函数作为 Pipeline Builder 中的边车容器。

函数支持的语言有 TypeScript ↗ 和 Python ↗。

有关语言的功能支持信息以及如何选择语言或语言版本，请参阅 语言功能支持规范。
