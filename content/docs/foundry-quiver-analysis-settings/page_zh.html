设置

Quiver 提供了多种设置，用于配置你的分析中的显示和格式。这些设置仅适用于当前的分析，而不会影响你创建的其他分析。

要打开设置面板，请点击侧边栏 侧边面板 中的齿轮图标。

个人默认设置

Quiver 允许你保存你偏好的分析设置。这些个人默认设置将自动应用于你创建的任何新分析。你还可以选择将你的个人设置应用到现有的分析中。

将您的个人默认设置应用于现有分析将会覆盖现有的分析设置。这也将影响在该分析中工作的其他用户。

全局设置

  • 允许控制设置： 启用后，可以使用在分析中配置的布尔参数切换全局设置的开启和关闭。
  • 启用应用按钮： 启用一个应用按钮，该按钮将全局跟踪对分析中卡片的更新，并阻止它们传播到分析的其余部分，直到点击按钮。这对于许多更新预期在重新计算结果之前进行的分析/仪表板（例如更改多个参数时）可以显著提高性能。
  • 等待第一次应用： 如果使用“应用”按钮，设置此选项将防止分析在点击“应用”按钮前初始加载。当分析/仪表板的默认状态处理的数据规模过大，并且用户需要在查看分析前应用筛选条件时，这有助于提高性能。
  • 启用市场 linting： 启用后，市场模板功能将在分析中的所有仪表板上开启，并且当你与分析进行交互时，市场 验证会自动运行。错误和警告会在保存按钮旁边的验证指示器中显示。你可以从其仪表板设置中禁用单个仪表板的模板功能。了解有关将 Quiver 仪表板添加到市场产品的更多信息。
  • 显示模式： 控制画布是在浅色模式还是深色模式下显示。所选模式将应用于分析中的所有画布、仪表板和模板。请注意，当仪表板或模板被嵌入到 Quiver 以外的应用程序（例如 Workshop 或 Object View）中时，显示模式将继承自父应用程序。
  • 加载设置： 控制在打开画布时卡片的加载方式。选择“可见”设置可以提高大型分析的加载性能。
    • 所有: 画布中的所有项目都将被加载。
    • 可见: 仅加载画布中可见的项目。当它们变得可见时（例如，当向下滚动时）将加载其余项目。
    • 默认: 当前默认设置为“所有”。

分类图表覆盖

  • 覆盖默认的图表颜色/标签: 这些设置允许您覆盖分析中对象图表的分段和标签的颜色和显示名称。如果进行了更改，则本地图表显示设置优先于全局设置。

    默认情况下，同一图表中不同对象集的数据将以不同的颜色显示；同样，分段图表中的不同段也将以不同的颜色显示。在没有干预的情况下，这些颜色将被分配以确保类别之间的最大对比度。如果需要，您可以设置全局颜色覆盖，无论是由于颜色偏好，还是希望在整个分析过程中所有图表中的某些段具有相同颜色。

    请注意，要为条形图中的每个条形分配特定颜色，您首先需要根据您所分组的同一属性对系列进行分段。

时间序列提示信息设置

  • 显示图表提示信息：当此功能被禁用时，您在悬停于时间序列上时将不再看到显示时间序列名称、时间戳和值的提示信息。
  • 将提示信息值显示为范围/最小值/最大值/平均值： 默认情况下，Quiver 在每个时间序列图表中显示 1,000 个桶。此设置用于高频情况，其中多个底层时间序列点在一个桶中。
  • 在工具提示中显示时间：当此选项被禁用时，您悬停在时间序列上时将看不到时间戳值。
  • 在工具提示中显示时间序列标题：当此选项被禁用时，您悬停在时间序列上时将看不到时间序列名称。
  • 在工具提示中换行长文本：当此选项被启用时，工具提示中的长文本将会换行显示。
  • 显示时间序列悬停标记：当此选项被启用时，光标悬停的十字线也会在其他图表上显示。这对于在多个图表之间跟踪光标位置很有用。
  • 自动折叠图表选择菜单：当此选项被启用时，通过选择或拖动图表创建的时间序列图表选择菜单默认处于折叠状态，并可以选择展开它。

时间序列轴和图例

  • 启用全局流式传输： 在所有轴上激活流式传输，使时间序列下游的卡片在时间序列更新时能够刷新其值。请注意，这可能会对性能产生负面影响。如果您的使用场景不需要下游卡片反映流式更新，则在 时间轴配置 中启用 流式传输单个轴 设置，相比此设置将提供更好的性能。
  • 流式传输刷新率： 控制在启用了流式传输的轴上的时间序列图每隔多久轮询新数据。
  • 初始默认时间轴范围： 当创建默认时间轴时，例如，当首次将时间序列属性添加到分析中时，它将使用此值作为初始范围。有三种方式可以设置此范围：
    • 全部数据范围： 轴将初始化为时间序列图上的开始和结束日期。这可能会对大型时间序列的性能产生负面影响。这是在添加此全局设置之前创建的分析的默认设置。
    • 固定范围：轴将初始化为静态的开始和结束日期，例如，从2025-5-21到2025-06-04。
    • 相对范围：将视图范围设置为相对于今天的范围，例如，从2 weeks ago到Now。默认情况下，范围的结束日期设置为Now，但可以通过选择右上角的设置为相对时间切换来输入特定时间。新分析的默认设置是2 weeks ago到Now的相对范围。
  • 显示插值线：当启用此功能时，当外部插值设置为除NONE以外的任何值时，Quiver将显示虚线（标记前后）。
  • 在图例中显示隐藏的图表：当启用此功能时，隐藏的图表将在其图表的图例中以灰显状态显示。
  • 在图例中显示数值： 当此选项启用时，时间序列的最后一个点将在图表图例中其名称下显示。
  • 显示时间轴滑块： 当此选项启用时，在每个图表的x轴上会显示一个“滑块”。滑块用于独立控制图表的开始和结束显示。
  • 坐标轴中的缩写数字： 控制数字是否以字母缩写形式显示或完整显示（例如，120.0k 是 120,000 的缩写）。
  • 按数据分组图表： 当此选项启用时，具有相同根对象和数据提供者的时序图表将在图表编辑器和图例中被分组在一起。根据根对象显示额外的工具，使在不同根对象之间切换更加方便。
  • 默认叠加y轴： 当此选项启用时，默认情况下y轴将与图表数据叠加。当此选项禁用时，默认情况下y轴将使图表数据向一侧偏移。
  • 默认y轴位置： 控制y轴是在图表的左侧还是右侧显示。
  • 默认图例位置： 控件用于确定图例是在图表的左侧还是右侧显示。

时间序列日期/数字格式化

  • 启用UTC时间格式：默认情况下，Quiver使用您当前所在的时区（根据您的浏览器）显示时间戳。当此开关被打开时，时间戳将改用UTC显示。
  • 启用24小时制：默认情况下，Quiver使用AM/PM格式显示时间。当此开关被打开时，时间将以24小时制显示。请注意，这仅影响图表x轴上的时间显示，而不会影响图表上的悬停提示。
  • 显示的有效数字（1-15）：控制在图表悬停提示中显示的有效数字数量（默认为6）。

其他设置

视图模式设置

选择您偏好的视图模式，以便您的分析始终按照您的偏好打开。在分析的右上角选择视图模式设置切换按钮，以了解特定模式中的新功能并比较画布和图形功能。然后选择您偏好的模式。

如果您选择 自动选择 选项，当您添加的第一个数据与 时间序列 相关时，新分析将默认进入 画布模式。所有其他数据类型将默认进入 图形模式。
