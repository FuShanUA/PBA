在 Jupyter® 笔记本中使用 Palantir 提供的语言模型

前提条件

要使用 Palantir 提供的语言模型，必须首先在您的租户中启用 AIP。

Palantir 提供了一套语言和嵌入模型，可在 Jupyter® 笔记本中使用。这些模型可以通过 `palantir_models` 库来调用。该库提供了一系列类，用于绑定并与模型进行交互。

在代码工作区中设置 Palantir 提供的模型

要为您的笔记本添加语言模型支持，请打开代码工作区左侧的包搜索面板。搜索 `palantir_models`，然后选择“最新版本”。此操作会将安装命令复制到剪贴板，您可以将其粘贴到空单元格中并运行。

向您的笔记本添加 Palantir 提供的模型

要向您的笔记本添加语言模型，请打开代码工作区左侧的“模型”面板。如果您尚未导入模型，请点击“导入 Palantir 提供的模型”。如果您已经导入了模型，可以通过选择面板顶部的“+”图标来导入更多模型。

随后，面板将显示一个可供您使用的、可搜索的模型列表。模型分为两类列出：对话补全模型和嵌入模型。选择所需的模型将其导入您的代码工作区。

不同客户可用的模型可能有所不同。如需了解更多信息，请联系您的 Palantir 代表。

导入后，您的模型将显示在“模型”面板中。在“模型”面板中选择该模型，将展示一段演示该模型基本功能的代码片段。

要开始使用该模型，请点击代码片段将其复制，并粘贴到您笔记本中的任意单元格内。

使用语言模型生成补全内容

在本示例中，我们将使用 OpenAI 模型来回答问题。假设您已经导入了模型，您可以将下面的代码片段复制到任意单元格中以继续操作。



Copied!
1from language_model_service_api.languagemodelservice_api_completion_v3 import GptChatCompletionRequest
2from language_model_service_api.languagemodelservice_api import ChatMessage, ChatMessageRole
3from palantir_models.models import OpenAiGptChatLanguageModel

6response = model.create_chat_completion(GptChatCompletionRequest([ChatMessage(ChatMessageRole.USER, "why is the sky blue?")]))

向量嵌入

除了生成式语言模型外，Palantir 还提供了向量嵌入模型。以下示例展示了如何使用向量嵌入模型来计算一系列单词的向量嵌入，并通过绘制向量嵌入图来进行可视化。下方每个代码块都应视为独立的单元格。

首先，添加本示例所需的依赖项：

Copied!
1!mamba install -y "palantir_models>=0.1795.0" matplotlib numpy scikit-learn

随后，请确保已在模型面板中导入了向量嵌入模型。在本示例中，我们将使用 OpenAI 的 text-embedding-ada-002 模型。

为了生成所需的向量嵌入，我们从复制模型代码片段开始，并按如下所示进行了一些修改：



Copied!
1from language_model_service_api.languagemodelservice_api_embeddings_v3 import GenericEmbeddingsRequest
2from palantir_models.models import GenericEmbeddingModel
3
4fruits = [
5    "apple", "banana", "orange", "melon", "kiwi", "pear", "grape", "strawberry", "lemon", "lime",
6    "blueberry", "berry", "mango", "watermelon"
7]
8animals = [
9    "dog", "cat", "cow", "eagle", "mouse", "horse", "squirrel", "lion", "deer", "goose", "chicken", "pig"
10]
11words = fruits + animals
12
13model = GenericEmbeddingModel.get("text-embedding-ada-002")
14embeddings = model.create_embeddings(GenericEmbeddingsRequest(inputs=words)).embeddings
最后，我们可以使用 scikit-learn 和 Matplotlib 来可视化嵌入向量： 


已复制！
1from sklearn.decomposition import PCA
2import matplotlib.pyplot as plt
3import numpy as np
4
5pca = PCA(n_components=2, random_state=0)
6embeddings_2d = pca.fit_transform(np.array(embeddings))
7
8plt.figure(figsize=(8, 4), dpi=100)
9for i, word in enumerate(words):
10    x, y = embeddings_2d[i, 0], embeddings_2d[i, 1]
11    plt.scatter(x, y)
12    plt.annotate(word, xy=(x, y), xytext=(5, 2), textcoords='offset points', ha='right', va='bottom')
13plt.show()
