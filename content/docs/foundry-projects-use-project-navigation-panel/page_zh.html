使用项目导航面板

项目导航面板包含预览和项目工作区部分，提供用于编写文档和管理文件列表的工具。

封面页

封面页部分提供了一个基于 Markdown 的富文本编辑器，用于为项目编写文档。更多信息，请参阅《封面页》文档。

文件

文件标签页会显示项目内所有文件的集合。项目中最主要的文件将显示在该面板顶部的固定区域。

自动保存

从现有文件创建新文件时，系统会将其自动保存在项目的自动保存部分，而不是您的文件中。这些文件会带有 Autosaved 标签，任何有权访问该文件夹或项目的用户均可查看。自动保存文件有助于快速开展工作并减少项目杂乱。

手动保存文件后，它们将显示在文件部分。

文件权限

创建自动保存文件时，它会继承其最初创建所在文件夹的权限。创建该自动保存文件的用户即为该文件的数据主人。

在与其他用户共享自动保存文件之前，您需要先手动保存该文件。

请联系您的 Palantir 代表以帮助配置自动保存访问权限。

引用

由于项目既界定了关联工作的概念边界，也构成了用于应用和管理访问权限的安全边界，因此数据流转需要特别关注。

项目引用允许您管理跨项目边界的数据和代码包流转，并分为两个部分：文件引用和外部引用。

文件引用包含来自项目外部且在《代码仓库》、《代码工作簿》、《管道构建器》、《Fusion》或《等高线》中使用的任何文件。拥有 Editor 角色的用户可以向项目添加文件引用。右键单击文件并选择移除引用即可移除文件引用。详细了解为何需要《文件引用》。

外部引用通常是为了运行构建，而从各种数据分析应用程序自动添加到项目中的代码包。

回收站

每个项目都有其专属的回收站部分。如果项目不再需要某个文件，您可以将其移至回收站来执行删除。选中要删除的文件，然后选择回收站图标以移至回收站。

如果您改变主意，可以从回收站中恢复文件。在回收站标签页中，选中文件并点击恢复图标。

恢复文件会将其放回删除前的位置，并还原之前的权限。

使用X按钮可永久删除所选文件。

项目使用量

对于在项目中拥有 Owner 角色的用户，导航面板底部会提供一个链接，用于在《资源管理》中查看《项目使用量》。

访问图谱

访问图谱是一款可用于可视化项目安全设置的工具。借助于此，您可以查看项目上的标记、在此项目中拥有角色的群组，以及属于这些群组的用户。如果您的设置涉及大量嵌套群组，访问图谱将大有用处。
