分析

Foundry 为企业内的各类用户提供了与 Foundry 本体深度融合的分析能力。开箱即用方面，Foundry 内置了可视化点击操作与代码驱动的工具，支持表数据分析、自上而下的可视化分析、地理空间分析以及时序分析等。Foundry 的分析功能打破了传统“只读”模式的局限，支持将数据回写至本体中，在统一的安全、血缘关系与数据治理模型下产出极具价值的新洞察。

核心应用

Foundry 的核心分析应用包括：

  • Contour，一款自上而下的分析应用，支持大规模快速探索表数据，通过可视化转换派生新数据集并创建图表。
  • Quiver，一款多模态图表应用，支持对象驱动分析、时序驱动分析、点击式机器学习以及仪表板构建。
  • 《代码工作簿》，一款融合数据工程与数据科学特性的应用，支持快速构建基于 Python、R 或 SQL 的数据转换、训练机器学习模型等操作。
  • 《记事本》，一款集成解决方案，支持在格式化文本与媒体内容旁，嵌入来自 Foundry 各处的动态分析、可视化及业务构件。
  • Fusion，一款电子表格驱动应用，将表格化计算与 Foundry 本体及对象驱动查询系统的强大能力相结合。

分析连接

Foundry 还可与企业现有的分析工具进行深度集成。企业可以通过标准 API 和接口，将商业智能、可视化及其他分析应用接入 Foundry。集成选项包括：

  • 面向常用工具（如 Power BI® 和 Tableau）的开箱即用连接器。
  • 用于临时连接的 REST API。
  • 用于通用连接的 ODBC 和 JDBC 驱动程序。
  • 用于连接数据科学工具的 Python ↗ 和 R ↗ SDK。
  • 面向 Microsoft 报表生成器和 Excel 等工具的向导式集成，此类工具均利用标准接口实现对接。
