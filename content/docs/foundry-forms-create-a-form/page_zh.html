创建表单

对于 Foundry 上的数据录入或回写工作流，Foundry Forms 已不再是推荐的方法。相反，请使用 Foundry Ontology 构建用户输入工作流，将相关数据结构表示为对象类型，并通过 Actions 配置回写交互。在《表单概览》文档中了解更多信息。

要创建新表单，请先从 Foundry 导航侧边栏打开 Forms 应用程序。系统将提示您创建新表单或打开现有表单。或者，在文件系统视图中导航至某个项目，然后选择 + 新建 > 表单。

创建新表单后，系统会提示您选择响应目标：

  • Fusion 电子表格
  • 本体对象类型（如果有权限

您也可以现在创建表单，稍后再决定响应目标。

虽然这两个目标选项都会将数据写入数据集，但 Fusion 会在每次更改时直接写入电子表格。Foundry 中的对象存储要求用户安排构建在本体中配置的回写数据集。此决定可以稍后更改。

创建并配置新表单

请按照以下步骤创建一个简单的表单：

  1. 首先，导航到表单创建界面。



  2. 接下来，为新表单选择响应目标。



  3. 然后，选择 添加您的第一个字段。

  4. 从可用选项列表中选择所需的字段类型。



  5. 现在，使用表单右侧的可视化编辑器侧边栏，根据需要修改字段。

  6. 然后，选择现有字段上方或下方的 + 继续添加更多字段。

  7. 最后，选择右上角的绿色 保存 按钮以保存表单。



更改响应目标

要更改表单的响应目标，请按照以下步骤操作：

  1. 从屏幕右侧的 响应数据 选项卡中，选择 更改响应目标。

  2. 为表单选择新的响应目标。



  3. 将所有添加的字段映射到新目标中可用的属性。



  4. 选择绿色的 提交 按钮进行保存。

工作表和字段

表单由工作表和不同类型的字段组成：

  • 简单字段向受访者请求基本输入。
  • 数据支撑字段将表单链接到 Foundry 中已存在的数据。
  • 自动填充字段允许用户捕获有关表单的元数据，例如创建者和创建时间。
  • 附件字段是一种特殊的简单字段，用于通过文件补充响应内容。

修改工作表和字段

要修改某个项目：

  1. 打开可视化编辑器。可以通过以下任一方式完成：
    • 将鼠标悬停在该项目上，然后选择右上角的灰色设置齿轮；或者
    • 双击该项目（对于工作表，双击标题）。
  2. 浏览并修改可用的配置选项，这些选项按选项卡分组：
    • 属性
    • 验证器（仅适用于字段）
    • 转换
  3. 选择面板底部的绿色 保存 按钮。

要解锁其他配置选项，请使用代码编辑器。

重排工作表与字段

重排项目的方法：

  1. 将鼠标悬停在待重排的项目上。
  2. 选择左上角的任一箭头，将字段上移或下移一个位置。

字段左侧还提供了一个拖拽手柄。

移除工作表与字段

移除项目的方法：

  1. 将鼠标悬停在待删除的项目上。
  2. 选择右上角带圈的红色 x 图标。
  3. 再次选择同一图标以确认删除。

也可以通过可视化编辑器移除字段：

  1. 双击字段以打开可视化编辑器。
  2. 在 属性 标签页中，选择面板底部的 删除工作表/字段。

最后，还可以将项目拖拽到表单底部的 拖放到此处删除 按钮处进行移除。
