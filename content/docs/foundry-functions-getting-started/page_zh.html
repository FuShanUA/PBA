使用函数入门

在Foundry中，有三种语言选项可用于开始使用函数；TypeScript v1、TypeScript v2 和 Python。有关每种语言支持功能的更多信息，请查看 语言功能支持 规范。

尽管每种语言都有不同的支持功能集，但您将能够访问每种语言的相同基本平台功能，包括运行、测试和发布函数。本页概述了这些功能，以帮助您了解如何使用函数仓库，无论您将使用哪种语言。

如需有关特定语言的详细说明，请参阅下面的教程：

  • 使用TypeScript v1 函数入门
  • 使用TypeScript v2 函数入门
  • 使用Python函数入门

请查阅以下部分，了解有关函数仓库创建和使用的通用信息。

创建函数仓库

在创建函数仓库时，您将能够选择最适合您需求的语言。您可以通过选择+ 新建 > 仓库，直接从您选择的项目中初始化函数仓库，或者从代码仓库应用程序中通过选择右上角的+ 新建仓库来完成。一旦仓库被初始化，您可以添加并运行函数。

有关如何为特定语言创建函数仓库的详细说明，请参阅下面的教程部分：

  • 创建 TypeScript v1 函数仓库
  • 创建 TypeScript v2 函数仓库
  • 创建 Python 函数仓库

在实时预览中测试

函数的实时预览功能允许您在将函数提交到代码仓库之前对其进行测试。您可以在将函数添加到代码仓库后运行该函数。为此，请打开底部工具栏上的函数，然后选择实时预览。选择一个函数，输入输入值，然后选择运行以运行该函数。

实时预览运行在与已发布函数不同的运行时环境中。区别包括CPU资源、可用内存以及函数在超时前可以运行多长时间。
