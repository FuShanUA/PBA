遗留版

Code Workbook 目前处于开发生命周期的遗留阶段，预计不会再进行额外开发。我们仍提供全面支持。建议您探索其他应用和工具以满足实际用例需求。

Code Workbook 是一款应用程序，用户可以通过直观的图形界面，使用代码对数据进行分析和转换。

目标

Code Workbook 的设计秉持以下原则：

  • 迭代速度： 用户可以快速测试并优化转换与可视化逻辑，从而得出有效结果。
  • 低门槛： Code Workbook 的图形界面和支持复用预置逻辑的设计，旨在让不同技术水平的用户都能轻松上手。
  • 协作： 不同用户群体可以共享逻辑，并在 Code Workbook 中协同完成同一项分析。
  • 平台互操作性： 用户可以在整个平台中落地分析成果，例如将可视化结果添加至 Notepad、将可用于生产环境的管道提升至 Code Repositories，以及迭代机器学习模型。

功能

Code Workbook 的主要功能包括：

  • 交互式控制台，可快速迭代转换逻辑并进行临时数据探索。
  • 可视化支持，可使用常用软件包（Matplotlib、Plotly、Seaborn）创建详细且可交互的图像。
  • 模板，可通过简单界面复用复杂且特定领域的逻辑。
  • 多语言支持，支持多种代码和模板语言，方便用户为分析选择最合适的语言。
  • 分支，通过隔离不同用户的修改来促进协作。
  • 所见即所得的用户界面，便于自定义 Spark 环境及设置转换的输入类型，并查看节点间的关系。
