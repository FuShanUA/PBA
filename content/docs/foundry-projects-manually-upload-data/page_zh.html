手动上传数据

你可以将自己的数据源手动上传到项目中使用。所有数据均应针对你的使用场景获得相应许可（例如不包含个人数据），或者是虚拟数据。

首先在电脑上打开文件以核对内容。检查表头行数，以及各列是否包含数值或日期/时间数据。这些信息在后续操作中会派上用场。

进入平台后，通过工作区导航栏的文件前往目标文件夹，然后选择你的文件。

在主文件夹下，你可以创建一个名为 data 的文件夹来保持文件井然有序。选中该文件夹进入。此时路径应为 你的文件 > data 或 /<共享文件夹路径> > data。

接下来，你可以直接将文件拖拽到浏览器中上传至文件夹，或者选择+ 新建 > 上传文件...启动上传流程。

在上传过程中，选择文件的上传方式：

  • 作为媒体集上传（推荐）：将所选文件上传至新的媒体集。
  • 上传至无模式的数据集：用于存储任意文件的集合。
    • >作为原始文件上传，不修改扩展名

随后，配置上传参数并选择写入模式。

写入模式包括：

  • 无事务（更新立即生效）：写入操作按单个项目立即生效，失败仅限该项目，不支持快照，因为删除和覆盖均按单个项目执行。

  • 事务性（基于事务的保证，类似于数据集）：每个媒体集同一时间仅允许一个打开的事务，每个事务最多写入 10,000 个文件，构建失败时不提交任何写入操作，支持快照和增量构建。

设置完成后，选择上传完成操作。

在数据集预览应用中，你可以直接将 .csv、.tsv、.xls、.xlsm 和 .xlsx 文件上传至数据集。
