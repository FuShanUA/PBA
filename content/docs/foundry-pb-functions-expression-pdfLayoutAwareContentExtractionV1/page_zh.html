从PDF中提取布局感知内容

支持: 批处理、更快

从指定文档中提取内容，同时保留文档的布局。

表达式类别: 媒体

声明的参数

  • 要检测的语言: 要在输入文件中检测的语言。
    Set<Enum<南非语, 阿尔巴尼亚语, 阿姆哈拉语, 阿拉伯语, 亚美尼亚语, 阿萨姆语, 阿塞拜疆语, 阿塞拜疆语（西里尔字母）, 巴斯克语, 白俄罗斯语, 等等...>>
  • 媒体引用: 要从中提取内容的PDF。
    Expression<Media reference>
  • 输出格式: 输出的期望格式。可在基于文本的简单输出和包含所有详细信息（包括边界框）的结构化输出之间进行选择。
    Enum<完整提取, 文本和表格>
  • 可选 结束页: 页面范围的结束（包括在内）。如果没有提供值，则默认为最后一页。
    Expression<Integer>
  • 可选 错误处理: 确定管道对于无法处理的输入的行为。
    Enum<FAIL, NULL>
  • optional Start page: The start of the page range. If no value is provided, it will default to the first page.
    Expression<Integer>

Output type: Array<Array<Struct<block_index, block_id, page, block_type, content, bounding_box, languages<String>, confidence>>> | Array<String>
