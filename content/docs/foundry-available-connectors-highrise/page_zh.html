Highrise

Highrise 连接器是由 Palantir 提供、用于 Highrise 的驱动程序。

要创建新的 Highrise 源，请按照Palantir 提供的驱动程序的标准设置流程操作，然后参考以下章节进行 Highrise 专属配置和网络设置。有关完整的属性参考，请查阅官方 Highrise 驱动程序文档 ↗。

配置

以下属性为必填或建议填写项。

属性是否必填？描述默认值
AccountId ↗必填当前已认证用户的账户 ID。—
APIToken ↗建议当前已认证用户的 API Token。—
AuthScheme ↗建议选择使用 OAuth2 还是 APIToken 连接至 Highrise。APIToken
InitiateOAuth ↗建议指定获取或刷新 OAuth 访问令牌的流程，确保已认证且已授权的用户在工作期间能够维持访问权限。REFRESH
OAuthClientId ↗建议指定创建自定义 OAuth 应用程序时分配的客户端 ID。（亦称消费者密钥）。此 ID 用于在 OAuth 授权服务器中注册自定义应用程序。—
OAuthClientSecret ↗建议指定创建自定义 OAuth 应用程序时分配的客户端密钥。（亦称消费者密钥）。此密钥用于在 OAuth 授权服务器中注册自定义应用程序。—
