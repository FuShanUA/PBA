应用参考

本页面提供了在执行数据集成工作流时可能遇到的 Foundry 应用参考。

流水线构建器

流水线构建器是 Foundry 用于数据集成的主要应用。借助流水线构建器，您可以创建从数据源到最终输出的端到端流水线工作流。流水线构建器的用户可以在一个易于使用的应用中描述其工作流、转换数据、编辑模式并构建输出。

流水线构建器具有直观的点击式界面和强大的后端模型，使技术和非技术用户都能比在重度代码应用中更快地定义和部署流水线。经过精简的构建器界面允许用户在检查模式的同时应用数据转换，从而节省了构建时通常用于计算和检查的时间与成本。完全版本控制和可扩展性等附加功能，使流水线构建器成为安全协作的理想应用。

代码仓库

代码仓库是 Foundry 用于编写代码的主要界面，最常用于使用 Python、Java 和 SQL 创建数据流水线。代码仓库在 git 服务器之上提供了集成开发环境 (IDE)，支持对流水线逻辑进行协作和治理，并原生支持编写、测试和预览数据转换逻辑。代码仓库还可用于编写机器学习模型和本体函数。

如果您对数据科学和基于代码的分析感兴趣，代码工作簿可能更适合您的用例。了解有关代码工作簿、代码工作空间和代码仓库之间差异的更多信息。

数据血缘关系

数据血缘关系是一款展示数据如何在 Foundry 中流动的应用。您可以使用它来探索 Foundry 中的任何资源如何跨单个项目或用例的边界与其他资源相连。这包括对数据源、数据集、分析、本体对象和链接类型以及面向用户的应用的支持。除了探索连接之外，您还可以使用数据血缘关系查看数据预览、查看用于派生任何数据的逻辑，并管理计划的流水线。

数据连接

数据连接是用于将数据同步到 Foundry 并管理包括源凭据在内的相关资源的应用。初始设置完成后，数据连接使得探索数据源和同步新数据以用于用例开发变得简单，同时符合大规模管理源系统和用例所需的全套治理控制要求。

数据集预览

数据集预览是一款用于查看和理解数据集的应用。从任何其他应用打开数据集都会向您展示数据集的内容以及一系列上下文信息。这包括有关数据集权属、数据集随时间的更改情况、任何适用的健康检查以及更多详细信息。

数据健康

数据健康用于管理所有数据流水线中的数据质量。流水线维护者可以执行健康检查，以快速了解其流水线的性能和可靠性，还可以订阅监控视图上的警报，从而实现广泛的数据流水线维护工作流。

构建

构建应用程序 —— 前身称为 Job Tracker —— 允许您查看 Foundry 中发生的所有构建，并探索每个构建的详细信息，包括执行进度、调度安排以及过往的成功率和失败率。构建应用程序还支持访问底层的 Spark 执行引擎的细粒度信息，从而支持调试与优化工作流。
