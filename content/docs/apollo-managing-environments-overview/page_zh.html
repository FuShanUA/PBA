管理环境

本节将介绍与管理环境相关的流程。

您第一次导航到Apollo中的环境页面时，将进入“环境”概览标签页。

1. 实体

此部分提供当前环境中实体的升级状态概览。您可以查看已更新、处于宽限期或被阻止的实体数量。选择一种状态，可在实体标签页中查看具有该状态的实体列表。

2. README

此部分允许您为环境添加README。您可以在此部分提供有关环境的额外信息，例如其用途、位置或其他相关信息。了解有关 环境显示元数据 的更多信息。

3. 环境属性

此部分显示有关环境的一般信息：

  • 环境 RID： 环境的标识符，例如 ri.apollo.main.environment.aaaaaaaa-bbbb-cccc-dddd-eeeeeeeeeeee。
  • 最后报告状态：环境上次向Apollo报告的时间。

4. 计划

本节显示在环境中运行的最新计划。您可以查看每个计划的状态以及启动时间。选择一个计划以查看有关它的更多详细信息。

5. 模块

本节显示环境中安装的模块。您可以查看每个模块的版本。选择一个模块以查看有关它的更多详细信息。

6. 资产仓库

本节显示分配给环境及其连接状态的资产仓库。选择一个资产仓库以查看有关它的更多详细信息。

7. 代理连接

在此部分中，您可以查看部署在环境中的Apollo代理以及它们上次向Apollo报告的时间。

8. 标签

您可以查看已应用到此环境的所有标签。

9. 发布通道

本节显示环境所遵循的默认发布通道。请注意，环境中的某些实体可能遵循不同的发布通道。

10. 角色

本节显示已分配给该环境的团队的角色。

11. 联系人

本节显示环境的联系团队。如果该环境有其他联系团队，它们也会在此节中显示。

12. 维护时段

本节显示环境的维护时段。您可以查看停机时间和无停机时间的时段及其重叠部分。根据产品维护时段，某个实体可能具有不同的维护时段。
