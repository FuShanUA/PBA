定义并运行 Slate 函数

Slate 函数仅在 Slate 中可用，且不同于 Foundry 函数；Foundry 函数可在整个平台范围内访问。你可以使用 Slate 平台编辑器在 Slate 中访问 Foundry 函数。有关更多信息，请参阅《函数文档》。

函数编辑器允许你对 Handlebars 可访问的项目（如查询结果、全局变量和微件属性）定义并运行 JavaScript 函数。

函数无法访问 DOM 或 Slate 空间，且不会保存任何状态。它们仅可用于数据转换。

函数通常用于字符串解析等轻量级数据处理任务。函数支持异步语法（包括 async、await 关键字和 promise）。

函数按顺序运行，而非并行。Slate 从队列中逐个执行函数，因此每个函数必须在前一个函数完成后才能开始。即使函数使用异步调用，此规则同样适用：等待长时间运行操作的函数将阻塞队列，直到其完成，从而延迟其后跟随的所有函数。请避免依赖函数的并发运行。

文档级函数库

用户能够编写带参数的可重用 JavaScript 函数。这将有助于重构代码，并减少函数中代码的复制粘贴操作。你还可以使用 重新运行所有函数 按钮重新运行并更新依赖于某个函数库的所有函数。

可用的默认 JavaScript 库

为了增强函数的使用体验，Slate 默认内置了以下外部 JavaScript 库：Lodash ↗、Math.js ↗、Moment ↗、Numeral ↗ 和 es6-shim ↗。在编写函数时，请随意使用这些库。

除非强制要求所有用户使用支持这些特性的浏览器，否则请勿使用 ES6 语法特性。

示例

示例 1：添加两个微件属性

此函数将两个下拉微件的值相加，并在文本微件中显示结果。

打开函数编辑器并添加一个名为 addNumbers 的函数。添加以下 JavaScript 代码：

Copied!
1return {{lhs.selectedValue}} + {{rhs.selectedValue}};

然后，在文本微件中，通过在 Handlebars 中引用函数名 {{addNumbers}} 来显示返回值。

函数编辑器不接受任何形式的《辅助工具》。

示例 2：解析查询结果

该函数会解析查询结果中的每个元素。假设存在一个名为 asteroids 的查询，它返回以下 JSON：

已复制！
1{"id": [6, 7, 8],"name": ["Hebe", "Iris", "Flora"],"diameter": [185.18, 199.83, 135.89]}

以下函数返回的结果将名称与 ID 组合在一起，并新增了一个周长属性。


```javascript
var asteroids = {{asteroids}};
var formattedNames = [];
var circumference = [];

for (var i = 0; i < asteroids.name.length; i++) {
  formattedNames.push(formatName(
    asteroids.name[i], asteroids.designation[i]));
  circumference.push(asteroids.diameter[i] * 3.14);
}

return {
  name: formattedNames,
  diameter: asteroids.diameter,
  circumference: circumference
};

function formatName(name, designation) {
  return name + " (" + designation + ")";
}
```

5}
