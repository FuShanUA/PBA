定义和运行Slate函数

Slate函数仅在Slate中可用，与Foundry函数不同；Foundry函数可在整个平台上访问。您可以通过Slate平台编辑器在Slate中访问Foundry函数。有关更多信息，请参阅 函数文档。

函数编辑器 允许您在Handlebars可访问的项（如查询结果、全局变量和小部件属性）上定义和运行JavaScript函数。

函数无法访问DOM或Slate 空间，也不会保存任何状态。它们只能用于数据转换。

函数通常用于轻量级的数据处理任务，例如字符串解析。函数支持异步语法（包括 async、await 关键字和Promise）。

函数按顺序运行，而不是并行运行。Slate 会从队列中依次执行函数，因此每个函数必须在下一个函数开始之前完成。即使函数使用异步调用时也是如此：一个等待长时间操作的函数会阻塞队列直到它完成，从而延迟后续函数的执行。避免依赖函数同时运行。

文档级别的函数库

用户可以编写带有参数的可重用 JavaScript 函数。这将有助于代码重构，并减少函数中代码的复制和粘贴。您还可以使用 Re-run All Function 按钮重新运行和更新所有依赖于函数库的函数。

可用的默认 JavaScript 库 默认的 JavaScript 库

为了更好地使用函数，Slate 默认附带了以下外部 JavaScript 库： Lodash ↗、Math.js ↗、Moment ↗、Numeral ↗ 和 es6-shim ↗。在编写函数时可以自由使用这些库。

除非所有用户都被强制使用支持这些特性的浏览器，否则不要使用 ES6 语法特性。

示例

示例 1：添加两个 widget 属性

此函数将两个下拉框 widget 的值相加，并在文本 widget 中显示结果。

打开 Functions 编辑器并添加一个名为 addNumbers 的函数。添加以下 JavaScript：

Copied!
1return {{lhs.selectedValue}} + {{rhs.selectedValue}};

然后，在文本小部件中，通过在Handlebars中引用函数名称来显示返回值 {{addNumbers}}。

函数编辑器不接受任何类型的 辅助函数。

示例2：解析查询结果

此函数解析查询结果中的每个元素。假设有一个名为 asteroids 的查询，并且它返回以下 JSON：

Copied!
1{"id": [6, 7, 8],"name": ["Hebe", "Iris", "Flora"],"diameter": [185.18, 199.83, 135.89]}

以下函数返回一个组合了 name 和 id 的结果，并添加了一个新的 circumference 属性（周长）。

Copied!
1var asteroids = {{asteroids}};
2var formattedNames = [];
3var circumference = [];
4
5for (var i = 0; i < asteroids.name.length; i++) {
6  formattedNames.push(formatName(
7    asteroids.name[i], asteroids.designation[i]));
8  circumference.push(asteroids.diameter[i] * 3.14);
9}
10
11return {
12  name: formattedNames,
13  diameter: asteroids.diameter,
14  circumference: circumference
15};
16
17function formatName(name, designation) {
18  return name + " (" + designation + ")";
19}

生成的 JSON 看起来像这样：

Copied!
1{
2  "name": ["Hebe (6)", "Iris (7)", "Flora (8)"],
3  "diameter": [185.18, 199.83, 135.89],
4  "circumference": [581.4652,627.4662,426.6946]
5}

示例3：使用 set timeout 和 promises 创建暂停

此函数创建一个 setTimeout 每隔五秒执行一次，暂停Slate功能五秒后再执行。

Copied!
1return new Promise(resolve => {
2  setTimeout(() => {
3    resolve();
4  }, 5000);
5});

这个函数在暂停五秒后返回undefined。
