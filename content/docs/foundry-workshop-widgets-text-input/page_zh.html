文本输入

文本输入组件允许用户在表单字段中输入文本值。

配置选项

  • 标签
    • 为组件设置可选标签。该文本显示在组件顶部。
  • 字符串值
    • 组件的输出变量，用于存储用户输入的文本。
  • 占位符
    • 定义在用户尚未输入文本时，输入框中显示的占位符文本。
  • 格式
    • 将输入框格式设置为单行、多行文本区域或 Markdown 编辑器。
    • 单行
      • 回车触发事件：设置按下回车键时触发的事件
    • 文本区域
      • 初始高度：设置文本输入区域的初始高度
    • Markdown
      • 通过格式化工具栏提供富文本编辑体验。用户可以使用 Markdown 语法或工具栏控件来编写和格式化文本。编辑器支持在富文本视图和原始 Markdown 视图之间切换。详情请参阅下文的 Markdown 模式。

Markdown 模式

当 format 字段设置为 Markdown 时，文本输入组件会提供富文本编辑体验，该体验由 记事本 所使用的同一编辑器驱动。此模式适用于需要用户输入格式化文本的字段，例如描述、备注或评论。

功能特性

  • 格式化工具栏：无需了解 Markdown 语法，即可使用工具栏控件应用粗体、斜体、代码等格式。
  • 富文本与原始 Markdown 视图：在富文本视图（带格式化预览的内联编辑）和原始 Markdown 视图（带 Markdown 语法的纯文本）之间切换。
  • 占位符文本：配置在编辑器为空时显示的占位符文本。
  • 自动调整大小：使编辑器能够根据内容长度自动扩展。
  • 变量绑定：输出字符串变量将内容以 Markdown 字符串的形式存储，可供其他组件（如 Markdown 组件）调用以进行展示。

注意事项

  • Markdown 模式不支持回车触发事件（仅在单行模式下可用）。
  • 内容以 Markdown 格式化字符串的形式存储在绑定的字符串变量中。调用此值的其他组件应能够解析 Markdown 格式。
