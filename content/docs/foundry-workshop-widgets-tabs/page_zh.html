选项卡

选项卡（Tabs）组件用于展示可配置的选项卡，通过触发 Workshop 事件来实现模块各个页面和覆盖层的导航切换。

配置选项

以下为选项卡组件的配置设置说明：

  • 选项卡

    • 添加项： 添加新的选项卡。创建完成后，选中包含该选项卡的框即可进行配置。
      • 标签： 设置选项卡的标签。
      • 点击时
        • 添加事件： 添加在选中该选项卡时触发的 Workshop 事件。
      • 图标： 在选项卡标签左侧添加图标。
      • 徽标： 在选项卡标签右侧显示徽标。默认情况下，选项卡不显示徽标。若构建者选择在选项卡上显示徽标，可以选择通过字符串变量显示文本，或通过数字变量显示数值。
      • 条件可见性： 开启后，可通过设置的布尔变量值控制选项卡处于启用/禁用或可见/隐藏状态。
        • 布尔变量： 设置布尔变量，用于控制选项卡处于启用、禁用或隐藏状态。
        • 禁用： 若选中此项，当所设布尔变量值为 true 时选项卡启用，为 false 时选项卡禁用。
        • 隐藏： 若选中此项，当所设布尔变量值为 true 时选项卡可见，为 false 时选项卡隐藏。

  • 显示与格式

    • 设计： 选择应用于组件中所有已配置选项卡的样式预设。可选项包括轮廓（Outline）、块状（Block）、突出（Prominent）或分组（Grouped）。
    • 轮廓（Outline）
      • 尺寸： 将选项卡的显示值尺寸设置为默认或大号。
    • 块状（Block）
      • 极简样式： 开启后，对选项卡启用极简块状样式；此设置会淡化活动选项卡所使用的活动颜色（见下文）。
      • 尺寸： 设置选项卡显示值的尺寸。尺寸选项包括默认或大号。
    • 突出（Prominent）
      • 宽度： 设置组件的宽度。
      • 尺寸： 将选项卡的显示值尺寸设置为默认或大号。
    • 分组（Grouped）
      • 尺寸： 将选项卡的显示值尺寸设置为小号、默认或大号。

  • 活动颜色： * 设置活动选项卡的颜色。

  • 选项卡高度： 为选项卡组件设置预定义的区段标题高度。

    • 自定义： 输入自定义像素值，设置组件中选项卡的高度。
    • 自动： 将选项卡高度设置为随其容器高度自动调整。
      • 填充： 开启后，组件中选项卡的高度将设置为与组件高度一致。

  • 方向

    • 水平： 在组件中以水平方向显示选项卡。
    • 垂直： 在组件中以垂直方向显示选项卡。
      • 对齐方式
        • 左对齐： 将组件中的所有选项卡左对齐。
        • 居中对齐： 将组件中的所有选项卡居中对齐。

选中状态

选项卡组件本身不保存选中状态。相反，选中状态派生自为该组件配置的事件。若切换至选项卡和切换至页面事件未引发布局状态变更，则系统将根据这些事件判定选中的选项卡。通过在选项卡区段的变量驱动选项卡选择字段中，或在设置面板的变量驱动页面选择字段中将字符串变量设为选项卡或页面名称，即可配置默认选中的选项卡或页面。

请注意，将选项卡组件添加至模块并不会自动生成切换至选项卡或切换至页面事件。有关如何生成这些事件以供选项卡组件使用的更多信息，请参阅切换至选项卡和切换至页面。

无布局状态变更事件

不会改变布局状态的事件——即导致“无布局状态变更”的事件——是指配置为切换到与当前布局状态相匹配的选定标签页或页面的布局事件。例如，如果当前布局状态选定了 标签页 A，那么配置了 切换至标签页 A 事件的标签页将显示为选定状态。

局限性

  • 如果配置了多个布局事件，包含最多 “无布局状态变更”事件 的标签页将成为选定的标签页；如果数量相同，则优先选择最早出现的标签页。
  • 设置变量值 事件目前不用于检查选定的标签页状态。
    • 如果使用由变量支持的选定标签页，你可以使用 切换至标签页 事件，该事件在设置选定标签页状态的同时，还会更新选定标签页的变量状态。
    • 如果使用由变量支持的选定页面，除了 切换至页面 事件外，你还需要使用 设置变量值 事件，因为页面切换事件不会自动更新选定页面的变量值。
