标签页

标签页 小部件显示可配置的标签页，用于触发 Workshop 事件以在模块的页面和覆盖层之间导航。

配置选项

以下列出了 Tabs 小部件的配置设置：

  • Tabs（标签页）

    • 添加项： 添加一个新的标签页。创建后，选择包含该标签页的框以进行配置。
      • 标签： 设置标签页的标签。
      • 点击时
        • 添加事件： 添加一个当标签页被选中时触发的 Workshop 事件。
      • 图标： 添加一个显示在标签页标签左侧的图标。
      • 徽章： 在标签页标签的右侧显示一个徽章。默认情况下，标签页不会显示徽章。如果构建器选择在标签页上显示徽章，他们可以选择通过字符串变量显示文本或通过数字变量显示数值。
      • 条件可见性： 当打开此选项时，允许设置一个布尔变量的值来控制标签页何时启用/禁用或可见/隐藏。
        • 布尔变量： 设置一个布尔变量来控制标签页何时启用、禁用或隐藏。
        • 禁用： 如果选中，则当设置的布尔变量的值为 true 时，标签页将被启用；当设置的布尔变量的值为 false 时，标签页将被禁用。
        • 隐藏： 如果选中，则当设置的布尔变量的值为 true 时，标签页将可见；当设置的布尔变量的值为 false 时，标签页将被隐藏。

  • 显示与格式

    • 设计： 选择一个样式预设，应用于小部件中所有配置的标签页。可从 轮廓， 块、突出 或 分组。
    • 轮廓
      • 大小： 设置选项卡显示值的大小，可以是默认或大。
    • 块
      • 最小样式： 打开以在选项卡上启用最小的块样式；此设置会减轻用于活动选项卡的活动颜色（见下文）。
      • 大小： 设置选项卡显示值的尺寸。尺寸选项包括默认或大。
    • 突出
      • 宽度： 设置小部件的宽度。
      • 大小： 设置选项卡显示值的大小，可以是默认或大。
    • 分组
      • 大小： 设置选项卡显示值的大小，可以是小、默认或大。

  • 活动颜色： * 设置活动选项卡的选项卡颜色。

  • 选项卡高度： 为选项卡小部件设置预定义的标题栏高度。

    • 自定义：通过输入自定义像素值来设置小部件中选项卡的高度。
    • 自动：选择让选项卡的高度自动配置为容器的高度。
    • 方向

      • 水平：以水平方向显示小部件中的选项卡。
      • 垂直：以垂直方向显示小部件中的选项卡。
      • 对齐方式
        • 左对齐：将小部件中的所有选项卡左对齐。
        • 居中对齐：将小部件中的所有选项卡居中对齐。

    选择状态

    选项卡小部件不维护自己的选择状态。相反，选择状态是从为该小部件配置的事件中推导出来的。切换到选项卡和切换到页面事件，如果导致 没有布局状态变化，将用于确定所选选项卡。可以通过在选项卡部分的变量支持的选项卡选择字段或设置面板中的变量支持的页面选择字段中将字符串变量设置为选项卡或页面名称来配置默认选中的选项卡或页面。

    请注意，向模块中添加选项卡小部件不会生成切换到选项卡或切换到页面事件。有关如何为选项卡小部件生成这些事件的更多信息，请参见 切换到选项卡 和 切换到页面。

    没有布局状态变化事件

    不改变布局状态的事件——也就是说，它们导致“无布局状态变化”——被配置为切换到与当前布局状态匹配的选定选项卡或页面。例如，一个配置了Switch to Tab A事件的选项卡，如果当前布局状态中选中了Tab A，则会显示为已选中。

    限制

    • 如果配置了多个布局事件，则具有最多"无布局状态变化"事件的选项卡将成为所选选项卡，在平局的情况下优先选择最早的选项卡。
    • 设置变量值事件目前不用于检查所选选项卡状态。
      • 如果使用基于变量的所选选项卡，您可以使用切换到选项卡事件，这将更新所选选项卡变量状态，同时设置所选选项卡状态。
      • 如果使用基于变量的所选页面，您需要使用设置变量值 事件，除了< strong >切换到页面事件外，因为页面更改事件不会自动更新所选页面变量的值。
