创建项目

如果具备相应权限，可以通过进入项目落地页并选择右上角的+ 新建项目来创建新项目。

选择项目，打开新建项目面板。

为项目命名，添加可选的描述，并选择项目所在的路径（空间）。你也可以更改组织内用户的默认角色。

如需设置特定空间下初始选中的默认角色，请前往相应的空间设置，更改项目默认角色。

选择创建，进入新项目仪表盘。

添加文档

通过将名为 README.md 的 Markdown 文件拖拽至文件夹中，或在文件夹的操作菜单中选择添加描述，即可为任意文件夹添加文档。系统支持标准 Markdown ↗，但存在部分与安全相关的限制：

  • 禁用内联 HTML。
  • 除另行配置外，仅支持渲染上传至 Foundry 的图片文件。Foundry 托管图片的 Markdown 语法如下：![Alt text](link to image in Foundry)。

同时支持指向 Foundry 资源的链接。使用以下语法可在描述中自动添加带有图标和文件名的链接：[optional link text](rid)。

项目内已有的 .md 文件不会自动转换为就地渲染，即使其名称已正确命名为 README.md。需下载现有的 README.md，将其从文件夹中删除并重新上传，方可正常显示。
