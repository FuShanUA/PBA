转换数据

向工作区添加数据集后，您就可以在 Pipeline Builder 中开始转换和结构化数据。

选择数据集

要对数据集应用转换，请在工作区中选择一个数据集节点，然后点击转换。

搜索转换

在转换页面中，您可以按名称搜索转换类型，或从可用转换列表中进行浏览。如果您使用的是结构化（表格型）数据集，此字段会显示一份详尽的表转换列表。

对于 JSON 文件等半结构化数据集，搜索字段会包含文件转换，允许您将数据集解析为表格格式。

了解更多关于数据集的信息。

配置转换

在转换配置面板中填写所需信息，包括列、表达式或值。在下面的示例中，我们选择了 重命名列 转换，选择了要重命名的列，并为这些列输入了新的名称值。

复用现有表达式

Pipeline Builder 允许您在创建新表达式时复用现有表达式中的值。

在嵌套表达式中替换表达式时，新表达式默认会移除您之前设置的任何字段或值。如果要保留这些现有字段或值，请选择表达式旁边的复用值。这会将之前的值自动复制到您的新表达式中。

应用转换

完成转换表单后，点击应用将转换添加到工作流中。您会看到转换节点已连接到图中的原始数据集。我们将新转换命名为 清理设施数据，它是原始 设施数据集 的直接输出。

您可以点击转换节点并选择编辑，对转换进行重命名或编辑。

拖动节点上的白色输出圆圈可更改图中的连接。

转换视图

Pipeline Builder 提供两种查看转换的方式：传统的折叠面板渲染和伪代码渲染。设置视图偏好后，将全局更新您在 Palantir 平台上的个人视图。

折叠面板渲染

折叠面板渲染格式以紧凑的、类似面板的结构显示转换。此视图为传统格式，适合习惯于该布局的用户使用。

伪代码渲染

伪代码渲染格式以更简洁的方式显示转换，类似于代码，但不遵循任何特定编程语言的语法。

连接、联合和 LLM 节点不受伪代码渲染选项的影响。

对于熟悉代码或希望以文本形式呈现管道逻辑的用户而言，此选项尤为实用。伪代码会自动调整以适应屏幕，从而减少滚动操作。

启用伪代码渲染后，将无法在管道构建器中编辑代码。该格式仅供提供更为熟悉的视图。

可通过以下方式启用伪代码渲染选项：

  • 设置菜单：进入设置菜单。随后选择用户偏好设置，并在折叠转换样式下切换伪代码选项。

  • 在转换路径内：在任意转换路径中，选择 </> 图标。
