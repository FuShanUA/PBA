函数中的时间序列

函数支持对时间序列属性的操作。在本页中，我们将介绍如何在函数中设置和使用时间序列属性。

初始设置

要使用函数中的时间序列，您需要已经在本体论中存储了时间序列。您可以按照以下链接中的步骤开始操作：这里。

一旦您的时间序列已存储在本体论中，我们需要为我们的时间序列函数创建一个代码仓库。在此仓库中，我们首先导入本体类型，以便可以引用这些本体类型中存储的时间序列。

现在您就可以在函数中处理时间序列了。

在 Functions 中处理时间序列

要在 Functions 中访问时间序列，请首先在代码仓库中创建一个 基于对象的 Function。这些函数可以直接访问您的 Ontology 中的时间序列属性。一旦编写了 Function，有两种方式可以运行您的新 Function。您可以在 实时预览中测试您的 Function，或者 发布您的 Function 并开始在平台上的其他应用程序中使用它。

Python 函数

FoundryTS 库 不兼容 Functions。相反，Python OSDK 提供了一种通过 Ontology 属性访问时间序列数据的替代方法。

访问数据

返回时间序列

要从 Python 函数返回时间序列数据，请导入 timeseries_sdk 库，并使用 TimeSeries.serialize 方法将您的返回数据框架格式化为 list[bytes]。然后您可以使用 Quiver 中的代码函数时间序列卡片 进行可视化。

不支持分类时间序列输出。

12        ],
13        "value": [1.0, 2.0, 2.5]

17@function
18def normalized_altimeter_rate(aircraft: Aircraft) -> list[bytes]:
19    """
20    Read an aircraft's altimeter TSP, smooth it, compute its rate of change,
21    then normalize by the maximum absolute rate.
22    """
23    df = aircraft.altimeter.to_pandas(all_time=True)
24    if df.empty:
25        return TimeSeries.serialize(df)
26
27    df["value"] = df["value"].rolling(window=10, min_periods=1).mean()
28
29    time_diff_seconds = df["timestamp"].diff().dt.total_seconds()
30    df["value"] = (df["value"].diff() / time_diff_seconds).fillna(0)
31
32    max_abs_rate = df["value"].abs().max()
33    if max_abs_rate > 0:
34        df["value"] = df["value"] / max_abs_rate
35
36    return TimeSeries.serialize(df)

TypeScript 函数

以下部分提供了常见 TypeScript 函数操作的示例。

返回第一个或最后一个点

由于函数没有内置的时间序列点类型，您可以返回值或时间戳。例如，以下函数读取机器上的最新温度：

Copied!
1    @Function()
2    public async getLatestTemperature(machine: MachineRoot): Promise<Double | undefined> {
3        const latest = await machine.temperatureId?.getLastPointV2();
4        return latest?.value;
5    }

您可以通过以下函数类似地获取第一个温度读数：

Copied!
1    @Function()
2    public async getEarliestTemperature(machine: MachineRoot): Promise<Double | undefined> {
3        const earliest = await machine.temperatureId?.getFirstPointV2();
4        return earliest?.value;
5    }

在一系列数据上进行聚合

一个有用的聚合操作是计算一组点的平均值。考虑以下函数，它获取示例机器的平均温度：

Copied!
1    @Function()
2    public async getAverageTemperature(machine: MachineRoot): Promise<Double | undefined> {
3        const aggregation = await machine.temperatureId?.aggregate()
4            .overEntireRange()
5            .mean()
6            .compute();
7        return aggregation?.mean!;
8    }

求导

在上面的例子基础上，你也可以通过使用以下计算函数来检索同一机器上的温度平均变化：

Copied!
1    @Function()
2    public async getAverageTemperature(machine: MachineRoot): Promise<Double | undefined> {
3        const aggregation = await machine.temperatureId?.derivative()
4            .aggregate()
5            .overEntireRange()
6            .mean()
7            .compute();

指定时间范围

除了导数外，你还可以对时间序列应用其他转换。以下是一个如何将时间戳参数作为范围应用于时间序列的示例：

Copied!
1    @Function()
2    public async getAverageTemperatureOverRange(
3        machine: MachineRoot,
4        start: Timestamp,
5        end: Timestamp): Promise<Double | undefined>
6    {
7        const latest = await machine.temperatureId?.timeRange({min: start, max: end})
8            .aggregate()
9            .overEntireRange()
10            .mean()
11            .compute();
12        return latest?.mean;
13    }
