访问非结构化文件

除了操作具有已定义表格结构的 Foundry 数据集外，Code Workbook 还支持访问数据集中的非结构化文件。这对于分析和转换图像等其他类型的媒体数据、XML 或 JSON 等半结构化格式、GZ 或 ZIP 等压缩格式，以及 RDA 和 RDS 等 R 语言数据格式非常有用。

Python 中的非结构化文件

读取文件

在 Python 转换中，你可以通过将上游数据集作为 Python transform input 读取来访问文件。该 API 暴露了一个 FileSystem 对象，允许根据文件在 Foundry 数据集内的路径进行文件访问，从而屏蔽底层存储细节。了解有关 FileSystem 的更多信息。此外，还会暴露其他信息，包括分支和 RID（详见转换输入文档）。

可以通过输入辅助栏或输入选项卡来更改输入类型。

只有导入的数据集和持久化的数据集才能作为 Python 转换输入被读取。未保存为数据集的转换无法作为 Python 转换输入读取。

没有结构的数据集应自动作为转换输入读取。

示例：读取 ZIP 文件中的 CSV

例如，以下代码将读取 ZIP 文件中的 CSV，并将 CSV 内容作为 dataframe 返回。

1import tempfile
2import zipfile
3import shutil
4import io
5from pyspark.sql import Row
6
7# datasetOfZippedFiles is a dataset with a single zipped file that contains 3 CSVs with the same schema: ["id", name"].
8def sample(datasetOfZippedFiles):
9    df = datasetOfZippedFiles
10    fs = df.filesystem() # This is the FileSystem object.
11    MyRow = Row("id", "name")
12    def process_file(file_status):
13        with fs.open(file_status.path, 'rb') as f:
14            with tempfile.NamedTemporaryFile() as tmp:
15                shutil.copyfileobj(f, tmp)
16                tmp.flush()
17                with zipfile.ZipFile(tmp) as archive:
18                    for filename in archive.namelist():
19                        with archive.open(filename) as f2:
20                            br = io.BufferedReader(f2)
21                            tw = io.TextIOWrapper(br)
22                            tw.readline() # Skip the first line of each CSV
23                            for line in tw:
24                                yield MyRow(*line.split(","))
25    rdd = fs.files().rdd
26    rdd = rdd.flatMap(process_file)
27    df = rdd.toDF()
28    return df

写入文件

支持向输出文件系统写入数据。这在写入图像、PDF、文本文件等非表格数据格式时非常实用。

调用 Transforms.get_output() 可实例化一个 TransformOutput。了解有关 TransformOutput API 的更多信息。

只能在保存为数据集的节点中，通过 TransformOutput 来写入文件。无法在控制台中使用 TransformOutput 写入文件。

一旦实例化了 TransformOutput 并通过调用 filesystem() 或其他方法对其进行了操作，返回除 TransformOutput 对象以外的任何内容都将被忽略。

示例：写入文本文件或数据集

以下代码展示了如何写入文本文件：


```python
def write_text_file():
    output = Transforms.get_output()
    output_fs = output.filesystem()
    
    with output_fs.open('my text file.txt', 'w') as f:
        f.write("Hello world")
        f.close()
```
