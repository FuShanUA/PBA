AIP 逻辑

AIP 逻辑是一个无需编码的开发环境，用于构建、测试和发布由大型语言模型（LLMs）驱动的功能。AIP 逻辑使您能够构建功能丰富的 AI 驱动功能，利用本体论而无需通常由开发环境和 API 调用引入的复杂性。使用 Logic 的直观界面，应用构建者可以设计提示、测试、评估和监控、设置自动化等。

您可以使用 AIP 逻辑来自动化并支持您的关键任务，无论是将非结构化输入中的关键信息连接到您的本体论，解决日程冲突，通过找到最佳分配优化资产性能，对供应链中断做出反应，或更多。

Logic 功能也可以 自动化，这样 本体论编辑可以自动应用或暂存以供人工审核。

AIP Logic 提供了一个直观的界面，通过一个 Logic 函数来利用本体论和大语言模型（LLMs），该函数可以接收输入（如本体论对象或文本字符串），并返回输出（对象和/或字符串）或对本体论进行编辑。例如，下面这个由大语言模型驱动的函数会从本体论对象中获取输入数据，并将该数据与客户电子邮件交叉参考，根据以往的解决方案为给定问题推荐一个解决方案。

AIP Logic 基于与 Palantir 平台其余部分相同的严格 安全 模型，包括用户和 函数权限。这些平台安全控制仅允许 LLM 访问完成任务所需的资源。
