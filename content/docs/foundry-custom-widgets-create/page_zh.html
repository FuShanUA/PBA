创建小部件集

要创建一个新的小部件集，请按照以下说明操作：

  1. 从工作区导航栏中选择文件，并找到您所需的项目或文件夹。
  2. 在右上角选择新建 > 小部件集，以在当前项目或文件夹中创建一个新的小部件集。新的小部件集将继承该项目或文件夹的权限。
  1. 在创建新小部件集的向导中，配置您的小部件集名称，并选择您将在Palantir内还是外进行开发。在Palantir内开发是入门的最简单方式，因为它会自动引导一个代码仓库，并内置CI/CD来创建、测试和发布您的小部件集。或者，您可以将代码托管在自己的仓库（如GitHub）中，并使用我们的CLI管理版本控制并自行发布小部件集。

启用本体论 SDK（OSDK）

要从您的小部件集中通过代码访问本体论，请使用创建向导配置一个 OSDK（设置期间可选）。您可以在任何时间添加或配置您的小部件集的 OSDK。

如果您选择配置本体论 SDK，则还必须为您的小部件集启用本体论 API，这可以由拥有信息安全官角色的任何用户完成。否则，从 OSDK 发出的 API 调用将失败。有关更多详细信息，请参阅 在小部件集中使用本体论 SDK（OSDK） 文档。

预览小部件

您的小部件第一个版本发布后，刷新小部件集的概览页面以显示新的小部件。选择该小部件即可预览。

您无需等待发布即可预览您的小部件。使用开发模式，您可以在首次发布之前，在自定义小部件游乐场和代码工作区中预览未发布的的小部件。
