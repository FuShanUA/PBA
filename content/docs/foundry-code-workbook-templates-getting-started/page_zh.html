创建图表

由于titanic_dataset包含行（表示乘客）和列（表示乘客的信息），我们可以使用柱状图来可视化给定乘客属性（如性别或舱位等级）的乘客数量。例如，以下是按性别划分的乘客分布：

在这个示例中，我们将创建一个转换，生成所需的图表。创建一个名为bar_chart_of_row_counts的Python转换，并插入以下代码：

Copied!
1def bar_chart_of_row_counts(titanic_dataset):
2    import matplotlib.pyplot as plt
3    from pyspark.sql import functions as F
4    import numpy as np
5
6    input_df = titanic_dataset
7    categorical_column = "Sex"
8
9    # calculate the counts
10    total = input_df \
11        .groupBy(categorical_column) \
12        .agg(F.count("*").alias("count")) \
13        .orderBy("count")
14
15    # convert summarized dataset to pandas
16    total_pdf = total.toPandas()
17
18    # plotting code
19    fig = plt.figure()
20    ax = fig.add_subplot(111)
21
22    y_pos = np.arange(len(total_pdf[categorical_column]))
23    ax.set_yticks(y_pos)
24    ax.barh(y_pos,total_pdf["count"])
25    ax.set_yticklabels(total_pdf[categorical_column])
26    plt.xlabel("count")
27    plt.ylabel(categorical_column)
28
29    plt.tight_layout()
30    plt.show()
31
32    # return the aggregated dataframe to save it as a dataset
33    return total

运行此转换时，如上所示的图表将在工作簿中的转换节点中显示。您还可以通过将鼠标悬停在转换节点中的图表上并选择查看图片来进入全屏图像视图。此图像查看器也可以从内容侧边栏和可视化选项卡中访问。

创建模板

接下来，我们将把这个转换变成一个模板，以便它可以被通用化和重用。点击代码编辑器右上角的操作按钮，然后点击创建模板。

你现在将进入全屏编辑器中的模板创建视图。 在模板编辑器中，你可以编辑模板的名称、描述和参数。让我们将此模板命名为Bar Chart of Row Counts by Categorical Variable，并给出如下描述：Create a transform with a bar chart of the row counts of 1 categorical column in any input dataset.

任何输入数据集——在本例中只是titanic_dataset——会自动作为模板类型为dataset的参数添加。点击模板编辑器中的titanic_dataset以更改它。由于我们希望这个模板是通用的，让我们将参数名称从titanic_dataset更改为input_dataset并添加描述。

{{{input_dataset}}}的两个实例将在转换代码中被高亮显示。接下来，让我们对输入列进行参数化。要将代码主体中的变量指定为模板的输入参数，请点击转换顶部右上角的添加新参数，并在代码中突出显示适当的变量。如下面所示，突出显示字符串"Sex"：

这会将此部分代码作为类型为column的参数，源数据集为input_dataset。在模板编辑器中，编辑param1参数名称并将其重命名为selected_column。

在此示例代码中，我们在转换的顶部将列名定义为一个变量，这样我们只需对其进行一次参数化。当您对其他转换进行模板化时，可以使用添加按钮来添加更多相同参数的实例。

接下来，选择该模板是否默认保存为数据集。通过勾选保存为数据集复选框，当添加该模板时，默认会将其作为持久化的转换添加。如果未勾选保存为数据集，则默认会将该模板作为非持久化的转换应用。在这种情况下，由于我们希望在其他应用程序中使用输出结果，因此选择默认保存为数据集。

最后，点击创建模板 创建并保存模板的按钮。每次创建新模板时，您需要选择一个文件夹来保存它。在本示例中，您可以将模板保存在您的主文件夹中。

只有具有访问权限的用户才能发现和使用模板，因此您可以在仍在编辑模板时将其保存在主文件夹中，并在想要推广用于更广泛用途时将其移动到共享文件夹。模板也可以添加到 数据目录。

使用模板

创建并保存模板后，您可以以点击方式使用该模板。

要查看可用的模板，请在转换创建视图中点击浏览所有模板。

浏览模板支持通过名称、描述和标签搜索模板，或者根据收藏夹、最近使用的模板或文件结构进行浏览。要应用一个模板，请点击其名称并选择“选择”。让我们添加刚刚创建的模板。

将模板添加到图中后，您可以重命名转换并填写输入项。通过点击点击此处添加数据集并点击图中的input_dataset来分配titanic_dataset参数。现在您可以选择任何列作为selected_column参数，以该列为依据创建图表。

选择运行以计算转换。对于输出可视化结果的转换，默认在图中显示为图视图。您可以通过右键单击，选择编辑然后选择显示表格视图以将节点显示为表格。

编辑模板

如果您想更新支撑模板的代码，请点击操作然后点击编辑模板 进入代码编辑器并编辑模板。
