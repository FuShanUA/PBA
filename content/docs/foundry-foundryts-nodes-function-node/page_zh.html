foundryts.nodes.FunctionNode

类 foundryts.nodes.FunctionNode(children)

用于将一个或多个时间序列转换为新时间序列的惰性查询容器，该时间序列是所提供的转换函数的输出。

每个FunctionNode可以转换为另一个FunctionNode或计算为最终的 SummarizerNode。

您也可以通过 FunctionNode.to_pandas() 或 FunctionNode.to_dataframe() 将惰性 FunctionNode 解析为数据框，这将生成以数据框形式表示的转换后的时间序列。

示例

Copied!
1>>> series = F.points(
2...     (100, 0.0),
3...     (200, float("inf")),
4...     (300, 3.14159),
5...     (2147483647, 1.0),
6...     name="series"
7... )
8>>> series.to_pandas()
9                      timestamp    value
100 1970-01-01 00:00:00.000000100  0.00000
111 1970-01-01 00:00:00.000000200      inf
122 1970-01-01 00:00:00.000000300  3.14159
133 1970-01-01 00:00:02.147483647  1.00000
Copied!
1>>> scaled = series.scale(1.5) # scaled is a FunctionNode that is not evaluated yet
2# scaled can be chained to another FunctionNode operation resulting in another unevaluated FunctionNode
3>>> time_shifted = scaled.time_shift(1000)
4# converting time_shifted to a Pandas dataframe evaluates the lazy query with the output of the scaled and
5# time_shifted functions
6>>> time_shifted.to_pandas()
7                      timestamp     value
80 1970-01-01 00:00:00.000001100  0.000000
91 1970-01-01 00:00:00.000001200       inf
102 1970-01-01 00:00:00.000001300  4.712385
113 1970-01-01 00:00:02.147484647  1.500000

列()

返回一个字符串元组，表示通过将此节点评估为 pandas 数据框所产生的 pandas.DataFrame 的列名。

注意

嵌套对象的键将被展平为一个元组，其中嵌套的键用 . 连接。

  • 返回: 包含当前节点评估后得到的结果数据框中的列名的元组。
  • 返回类型: Tuple[str]

示例

Copied!
1>>> series_node = foundryts.functions.points(((100, 0.0), (200, 1.0))
2>>> series_node.columns()
3("timestamp", "value")
4>>> stats_node = series_node.statistics(start=0, end=100, window_size=None)
5>>> stats_node.columns()
6("count", "smallest_point.timestamp", "start_timestamp", "latest_point.timestamp", "mean",
7"earliest_point.timestamp", "largest_point.timestamp", "end_timestamp")

to_dataframe(fts=None)

Evaluates this node to a pyspark.sql.DataFrame.

PySpark DataFrames enable distributed data processing and parallelized transformations. They can be useful when working with dataframes with a large number of rows, for example loading all the points in a raw series or the result of a FunctionNode, or evaluating the results of multiple SummarizerNode or FunctionNode together.

  • Parameters:fts (foundryts.FoundryTS,optional) – FoundryTS session used to execute the query (a new session will be created if not provided).
  • Returns: Output of the node evaluated to a PySpark dataframe.
  • Return type:pyspark.sql.DataFrame

Examples

Copied!
1>>> series_node = F.points(
2...     (100, 0.0), (200, float("inf")), (300, 3.14159), (2147483647, 1.0), name="series"
3... )
4>>> series_node.to_dataframe().show()
5+-------------------------------+---------+
6| timestamp                     | value   |
7+-------------------------------+---------+
8| 1970-01-01 00:00:00.000000100 | 0.0     |
9| 1970-01-01 00:00:00.000000200 | Infinity|
10| 1970-01-01 00:00:00.000000300 | 3.14159 |
11| 1970-01-01 00:00:02.147483647 | 1.0     |
12+-------------------------------+---------+

to_pandas(fts=None)

Evaluates this node to a pandas.DataFrame.

This is useful for loading raw or transformed time series data into a pandas.DataFrame and performing transformations using operations provided by pandas.DataFrame.

  • Parameters:fts (foundryts.FoundryTS,optional ) – FoundryTS session used to execute the query (a new session will be created if not provided).
  • Returns: Output of the node evaluated to a Pandas dataframe.
  • Return type: pd.DataFrame

Examples

Copied!
1>>> series = F.points(
2...     (100, 0.0), (200, float("inf")), (300, 3.14159), (2147483647, 1.0), name="series"
3... )
4>>> series.to_pandas()
5                    timestamp    value
60 1970-01-01 00:00:00.000000100  0.00000
71 1970-01-01 00:00:00.000000200      inf
82 1970-01-01 00:00:00.000000300  3.14159
93 1970-01-01 00:00:02.147483647  1.00000

types()

Returns a tuple of types for the column of the pandas.DataFrame that would be produced by evaluating this node to a pandas dataframe.

  • Returns: Tuple containing types of the columns in the resulting dataframe which the current node gets evaluated to.
  • Return type: Tuple[Type]

Examples
