foundryts.nodes.FunctionNode

class foundryts.nodes.FunctionNode(children)

懒查询容器，用于将一个或多个时间序列转换为新的时间序列，该新序列作为所提供转换函数的输出结果。

每个 FunctionNode 可以转换为另一个 FunctionNode，或者计算为最终的SummarizerNode。

你也可以通过 FunctionNode.to_pandas() 或 FunctionNode.to_dataframe() 将懒加载的 FunctionNode 解析为 dataframe，这将以 dataframe 的形式输出转换后的时间序列。

示例



已复制！
1>>> series = F.points(
2...     (100, 0.0),
3...     (200, float("inf")),
4...     (300, 3.14159),
5...     (2147483647, 1.0),
6...     name="series"
7... )
8>>> series.to_pandas()
9                      timestamp    value
100 1970-01-01 00:00:00.000000100  0.00000
111 1970-01-01 00:00:00.000000200      inf
122 1970-01-01 00:00:00.000000300  3.14159
133 1970-01-01 00:00:02.147483647  1.00000


已复制！
1>>> scaled = series.scale(1.5) # scaled 是一个尚未求值的 FunctionNode
2# scaled 可以与另一个 FunctionNode 操作进行链式调用，从而生成另一个未求值的 FunctionNode
3>>> time_shifted = scaled.time_shift(1000)
4# 将 time_shifted 转换为 Pandas dataframe 会计算该延迟查询，并输出 scaled 和
5# time_shifted 函数的结果
6>>> time_shifted.to_pandas()
7                      timestamp     value
80 1970-01-01 00:00:00.000001100  0.000000
91 1970-01-01 00:00:00.000001200       inf
102 1970-01-01 00:00:00.000001300  4.712385
113 1970-01-01 00:00:02.147484647  1.500000

columns()

返回一个字符串元组，表示在将该节点求值为 pandas 数据帧时所生成的 pandas.DataFrame 的列名。

注意

嵌套对象的键将被展平为一个元组，其中嵌套的键会以 . 连接。

  • 返回： 包含当前节点求值后所得数据帧中各列名称的元组。
  • 返回类型： Tuple[str]

示例

已复制！
1>>> series_node = foundryts.functions.points(((100, 0.0), (200, 1.0))
2>>> series_node.columns()
3("timestamp", "value")
4>>> stats_node = series_node.statistics(start=0, end=100, window_size=None)
5>>> stats_node.columns()
6("count", "smallest_point.timestamp", "start_timestamp", "latest_point.timestamp", "mean",
7"earliest_point.timestamp", "largest_point.timestamp", "end_timestamp")

to_dataframe(fts=None)

将该节点求值为 pyspark.sql.DataFrame。

PySpark DataFrame 能够实现分布式数据处理和并行化转换。处理包含大量行的数据框时，它们非常有用，例如加载原始序列中的所有数据点或 FunctionNode 的结果，或者同时求值多个 SummarizerNode 或 FunctionNode 的结果。

  • 参数： fts (foundryts.FoundryTS , 可选) – 用于执行查询的 FoundryTS 会话（如果未提供，将创建一个新会话）。
  • 返回： 求值为 PySpark 数据框的节点输出。
  • 返回类型： pyspark.sql.DataFrame

示例



已复制！
1>>> series_node = F.points(
2...     (100, 0.0), (200, float("inf")), (300, 3.14159), (2147483647, 1.0), name="series"
3... )
4>>> series_node.to_dataframe().show()
5+-------------------------------+---------+
6| timestamp                     | value   |
7+-------------------------------+---------+
8| 1970-01-01 00:00:00.000000100 | 0.0     |
9| 1970-01-01 00:00:00.000000200 | Infinity|
10| 1970-01-01 00:00:00.000000300 | 3.14159 |
11| 1970-01-01 00:00:02.147483647 | 1.0     |
12+-------------------------------+---------+

to_pandas(fts=None)

将该节点求值并转换为 pandas.DataFrame。

这适用于将原始或转换后的时间序列数据加载到 pandas.DataFrame 中，并利用 pandas.DataFrame 提供的操作执行数据转换。

  • 参数： fts (foundryts.FoundryTS , 可选) – 用于执行查询的 FoundryTS 会话（若未提供，将创建一个新会话）。
  • 返回： 节点求值后输出的 Pandas 数据帧。
  • 返回类型： pd.DataFrame

示例



已复制！
1>>> series = F.points(
2...     (100, 0.0), (200, float("inf")), (300, 3.14159), (2147483647, 1.0), name="series"
3... )
4>>> series.to_pandas()
5                    timestamp    value
60 1970-01-01 00:00:00.000000100  0.00000
71 1970-01-01 00:00:00.000000200      inf
82 1970-01-01 00:00:00.000000300  3.14159
93 1970-01-01 00:00:02.147483647  1.00000
