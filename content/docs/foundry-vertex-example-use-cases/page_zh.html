测试版

Vertex中的绘图功能正处于 测试版 阶段，可能无法在您的注册中使用。在积极开发期间，功能可能会发生变化。要启用此功能，请联系平台管理员在控制面板中修改Vertex设置。

  • 创建一个动态的、集成的数字孪生体，使您能够测试最大化组织成果的条件。
  • 快速配置基于对象的系统图或任何类型的流程图，从物料加工和设备调谐到生产和供应链管理。
  • 利用您建模对象和流程之间的预定义关系，可视化材料流和事件在整个系统中的情况。

模拟、动态分析与决策优化

Vertex 允许您通过与 Foundry 中发布的任何模型、预测或业务逻辑进行集成，与您的数字孪生进行交互和查询。

  • 在您的整个网络中运行“假设分析”和模拟，以理解并比较影响您系统的决策、事件或潜在变化的含义。
  • 配置并运行模拟，以应用现实世界的情境，准确量化对建模宇宙中拟议变更的影响。
  • 将流程和系统连接到您的数字孪生中，突破运营和数据孤岛，推动组织内的跨学科创新。
