自定义文档的权限

您可以使用 Palantir 平台的安全功能来管理自定义文档的权限，并与他人共享。

查看自定义文档的权限

只有拥有文档代码仓库查看权限的用户才能看到该仓库发布的文档。换句话说，任何具有自定义文档仓库查看权限的用户都可以查看该文档的发布版本。

这使您能够将文档发布给有限的受众，或者在更广泛的发布之前与有限的用户组进行迭代。这些访问控制还可以减少跨组织或群组的不必要共享的风险。

您可以在 Palantir 平台文件系统中根据代码仓库的位置来管理这些权限。

创建自定义文档的权限

所有用户都可以使用文档模板创建仓库，但将仓库作为平台内自定义文档发布的能力需要由租户管理员启用发布功能。
