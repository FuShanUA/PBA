从 SAP 摄取 HANA 视图

要将 HANA 视图中的数据抽取到 Foundry 中，该 HANA 视图必须发布为外部视图。本文档介绍了具体的操作步骤。

前提条件

  • SAP HANA Studio 或 SAP HANA Tools for Eclipse
  • ABAP Development Tools for Eclipse

请注意，这些工具可在《SAP 开发工具》中获取。

外部视图

外部视图是 ABAP 字典中的一种特殊视图，用于在 ABAP 程序中定义 SAP HANA 视图。

外部视图只能使用 ABAP Development Tools (ADT) 创建，且仅在当前数据库为 SAP HANA 数据库时才可用。

激活外部视图时，系统会在 SAP HANA 数据库中创建一个与视图同名的别名，该别名指向 SAP HANA 视图。外部视图的字段名称可以设置为与 SAP HANA 视图的字段名称不同。这实现了 HANA 特有数据类型到 ABAP 字典预定义类型的映射。下表列出了目前支持的 HANA 特有数据类型，并标明了它们默认映射到的 ABAP 字典类型。

HANA 类型含义ABAP 字典中的类型
SMALLINT2 字节整数INT2
INTEGER4 字节整数INT4
BIGINT8 字节整数INT8
DECIMAL压缩数字DEC
SMALLDECIMAL压缩数字DEC
FLOAT二进制浮点数FLTP
NVARCHARUnicode 字符串CHAR
VARBINARY字节字符串RAW
BLOB字节字符串RAWSTRING
NCLOBUnicode 字符串STRING

可以使用基于 SAP GUI 的 ABAP Workbench 中的 ABAP 字典工具来显示外部视图，但无法对其进行编辑。

在 SAP HANA 视图之上创建外部视图

  1. 创建新的 ABAP 存储库对象，并选择字典视图。
  2. 在下一个屏幕中，选择外部视图选项，并为外部视图指定名称和描述。
  3. 接下来，您可以验证从 SAP HANA 视图到外部视图的列映射。
  4. 您可以使用 SE16 事务码查看外部视图的内容，从而对其进行测试。

在 Foundry 中摄取 HANA 视图

创建外部视图后，即可通过 SAP 连接器对其进行摄取，只需在配置同步时选择 HANA 视图对象类型即可。
