添加来自其他 Foundry 应用程序的内容

从 Contour 添加看板

若要展示单次 Contour 分析的结果，建议使用 Contour 看板模式来创建交互式看板。详细了解 Contour 看板模式。

分析的编辑者和查看者均可将看板添加至报告中。

将 Contour 看板添加至报告的步骤如下：

  1. 打开你的 Contour 分析。

  2. 滚动至目标看板，然后点击看板头部的更多操作按钮（...）。



  3. 点击发送至报告，随后从菜单中选择一份报告：

    • 从最近的报告部分选择一份最近的报告，或者
    • 点击新建报告以创建一份新报告，或者
    • 点击浏览文件以从 Foundry 的其他位置选择一份报告。



    看板成功添加至所选报告后，你会看到一条确认提示。



从 Code Workbook 添加看板

报告非常适合将来自不同工作流的内容汇聚到同一处。

为了在报告中查看并操作 Code Workbook 可视化内容，我们建议授予用户对底层 Code Workbook 的编辑权限。仅有查看权限的用户只能看到由具有编辑权限的用户先前计算出的输入数据与参数组合。

将 Code Workbook 看板添加至报告的步骤如下：

  1. 打开你的 Code Workbook。

  2. 找到目标看板，随后右键点击看板头部，或点击操作按钮（···）。



  3. 点击添加至报告，随后从菜单中选择一份报告：

    • 从最近的报告部分选择一份最近的报告，或者
    • 点击新建报告以创建一份新报告，或者
    • 点击浏览文件以从 Foundry 的其他位置选择一份报告。



    看板成功添加至所选报告后，你会看到一条确认提示。



从 Object Explorer 添加视图

报告非常适合将来自不同工作流的内容汇聚到同一处。

Object Explorer 提供多种类型的视图，每种视图均可添加至报告中：

  • 对象 - 特定对象的完整视图
  • 分区 - 对象视图内的某个分区
  • 列表 - 包含多个对象的表格列表

将对象视图添加至报告

将 Object Explorer 的对象视图添加至报告的步骤如下：

  1. 在 Object Explorer 中打开一个对象视图。

  2. 点击页面右上角的操作。

  3. 点击添加至报告。



  4. 从菜单中选择一份报告。

    • 从最近的报告部分选择一份最近的报告，或者
    • 点击新建报告以创建一份新报告，或者
    • 点击浏览文件以从 Foundry 的其他位置选择一份报告。



将对象视图分区添加至报告

在某些 Foundry 实例上，此工作流可能已被禁用。如果你未看到下文所述的“添加至报告”按钮，请联系你的 Palantir 代表。

将 Object Explorer 的对象视图添加至报告的步骤如下：

  1. 在对象浏览器中打开一个对象视图。 滚动至需添加到报告的版块，然后点击版块标题栏中的 **添加到报告**。



    从菜单中选择一份报告。 从 **最近报告** 版块中选择一份最近的报告，或者 点击 **新建报告** 创建新报告，或者 点击 **浏览文件** 从 Foundry 的其他位置选择报告。
