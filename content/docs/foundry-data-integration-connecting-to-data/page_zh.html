连接数据

要从 Foundry 中获取价值，第一步是将其连接到您所在组织的数据源。Foundry 的数据连接工具全面支持各类标准企业数据源，涵盖基于云的对象存储、文件系统，以及数据库和数据仓库。

您可以根据需要访问的数据类型，通过不同的 Foundry 应用程序以多种方式连接数据。

数据连接

连接到数据源以运行批处理、流式传输、媒体和 CDC（变更数据捕获）同步，并使用虚拟表。

数据连接框架旨在通过数据集事务管理的离散版本来长期管理数据。该框架能够实现跨时间数据版本的完整血缘关系，让您清楚了解哪些同步任务生成了特定数据集的哪些版本。在每次同步无法进行全量数据加载的情况下，它还支持仅同步所需数据。

数据连接中的细粒度安全机制支持跨不同团队进行数据同步的联邦管理。同步集合甚至单个数据同步，可以仅对特定团队可见或可编辑（通过基于角色或分类的访问控制来定义）。详细了解保护数据基础。

您可以独立于实际的同步定义来管理同步元数据。这使得新配置可以实现完全分支化，即新的同步在分支中进行沙盒测试后，才会影响任何下游转换作业。

HyperAuto

为了超越简单的数据同步解决方案，Palantir HyperAuto 实现了对软件定义数据集成（SDDI）的支持。该工具包不仅允许组织连接到常见的 ERP 和 CRM 系统，还能以空前的速度通过编程方式生成数据管道，对数据集进行清洗、规范化和协调，使其成为连贯的数据资产。随后，该数据资产可以输入到本体中，将数据转化为运营价值。

外部转换

使用 REST API 执行计划的同步和导出到外部系统。

如果要连接到外部源以创建同步和导出数据，建议使用代码仓库通过 REST API 编写外部 Python 转换。您还可以向转换中添加数据集输入和媒体集输出。
