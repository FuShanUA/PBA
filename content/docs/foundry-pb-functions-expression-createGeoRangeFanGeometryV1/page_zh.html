创建范围扇形几何图形

支持于：批处理、流处理

将近似范围扇形表示为多边形，该区域涵盖所有满足以下条件的点：其与原点的半正矢距离介于最小半径与最大半径之间，且从原点出发的方位角落在以指定方位角参数为中心的夹角范围内。范围扇形的左右两侧绘制为沿逼近地球表面的 WGS84 椭球面计算出的测地线。若范围跨度超过 180 度且同时穿过反子午线，或者最大半径超过地球周长的一半，则返回 null。

表达式分类： 地理空间

声明的参数

  • 方位角： 范围扇形相对于北极的方位角。
    Expression<DefiniteNumeric>
  • 最大半径长度： 椭圆最大半径的长度。必须大于最小半径且小于地球周长的一半。
    Expression<DefiniteNumeric>
  • 最大半径长度单位： 最大半径的单位。
    Enum<Centimeter, Data mile, Decameter, Decimeter, Foot, Hectometer, Inch, Kilometer, Meter, Mile, 以及更多 ...>
  • 最小半径长度： 椭圆最小半径的长度。必须小于最大半径。
    Expression<DefiniteNumeric>
  • 最小半径长度单位： 最小半径的单位。
    Enum<Centimeter, Data mile, Decameter, Decimeter, Foot, Hectometer, Inch, Kilometer, Meter, Mile, 以及更多 ...>
  • 原点： 范围扇形原点的经纬度。
    Expression<GeoPoint>
  • 范围： 范围扇形的夹角范围，以其方位角为中心。必须大于 0 度。目前尚不支持跨度超过 180 度且同时穿过反子午线的范围扇形，此类情况将返回 null。
    Expression<DefiniteNumeric>
  • 可选 方位角单位： 方位角的单位。默认为度。
    Enum<Degrees, Minutes, Radians, Seconds>
  • 可选 弧线点数： 用于近似绘制范围扇形两侧弧线的点数。
    Expression<Byte | Integer | Long | Short>
  • 可选 边缘点数： 用于近似绘制范围扇形边缘的点数。
    Expression<Byte | Integer | Long | Short>
  • 可选 范围夹角单位： 范围的单位。默认为度。
    Enum<Degrees, Minutes, Radians, Seconds>

输出类型： Geometry
