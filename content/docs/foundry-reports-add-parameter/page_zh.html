添加参数

在文本中添加参数

你可能希望在文本块中显示参数的当前值。操作步骤如下：

  1. 选中一段文本。

  2. 点击格式工具栏中的参数按钮，创建一个新的字符串参数。



  3. 将新参数合并到目标 Contour 参数中。了解有关合并参数的更多信息。



在查看模式下，合并后参数的当前值将直接显示在文本中（在编辑模式下，仅显示参数名称）。



从其他 Foundry 应用程序添加参数

部分 Foundry 应用程序允许编辑者通过添加参数来控制可视化中数据的筛选方式。将其他 Foundry 应用程序的小部件添加到报告中（例如：《从 Contour 添加看板》）时，系统会自动添加所有影响该小部件的参数。

从 Contour 添加参数

《了解有关 Contour 中参数的更多信息》。

  1. 打开 Contour 分析，然后创建或打开一个路径。

  2. （如需）切换至编辑模式。

  3. 创建一个参数（例如名为 $value 的字符串参数）。



  4. 在你的路径中添加一个筛选看板。输入要筛选的列名，然后在值字段中输入参数名称（例如 $value）。



  5. 在筛选看板下方创建一个可视化看板（例如图表、表格）。

  6. 将可视化看板添加到报告中（参见：《从 Contour 添加看板》））。

可视化看板将出现在你选中的报告中，且 $value 参数会显示在报告的参数栏中。更改参数值即可改变可视化看板中显示的数据。

从 Code Workbook 添加参数

与 Contour 类似，Code Workbook 也允许编辑者使用参数来筛选数据。

  • 参阅《在模板中使用参数》，了解如何在 Code Workbook 中创建参数
  • 参阅《从 Code Workbook 添加看板》，了解如何将带参数的内容从 Code Workbook 添加到报告中
