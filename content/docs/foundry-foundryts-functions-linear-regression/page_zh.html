foundryts.functions.linear_regression

foundryts.functions.linear_regression(include_intercept=True, time_unit='ns', start=None, end=None)

返回一个对单个时间序列执行线性回归的函数。

线性回归找到输入时间序列点的最佳拟合直线的参数。线性回归表示为y = Ax + B，其中A是线的斜率，B是y截距。返回的函数将提供参数A和B。

当您需要在时间序列数据中识别和量化线性趋势时，线性回归非常有用。

  • 参数：
    • time_unit（str,可选） – 系数的时间单位，必须是“s”、“ms”、“us”、“ns”之一（默认为“ns”）。
    • start（str,可选） – 时间序列的起始时间（例如“2020-01-01”）。如果未指定，则从第一个数据点开始。
    • end（str,可选） – 时间序列的结束时间（例如“2020-01-01”）。如果未指定，则到最后一个数据点结束。
    • include_intercept（bool,可选） – 是否包括截距项。默认值为True。

返回类型： function – 返回一个函数，该函数接受一个时间序列并返回线性回归的参数。

示例：

from foundry import functions
import pandas as pd

# Create a sample time series
df = pd.DataFrame({
    'timestamp': pd.date_range(start='2020-01-01', periods=100),
    'value': range(100)
})

# Apply linear regression
result = functions.linear_regression(time_unit='s', include_intercept=True)(df)

print(result)

输出：

{'slope': 1.0, 'intercept': 0.0}

这表示线性回归的斜率为1.0，截距为0.0。

  • 注意： 此函数适用于单变量线性回归。对于多变量回归，请使用其他方法。
str|int|datetime.datetime,可选) – 用于计算线性回归的时间序列的起始点（包含）。
  • end (str|int|datetime.datetime,可选) – 用于计算线性回归的时间序列的结束点（不包含）。
  • 返回值： 一个接受单个时间序列并使用线性回归返回时间序列中点的最佳拟合线参数的函数。
