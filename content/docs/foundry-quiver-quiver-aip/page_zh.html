AIP功能在Quiver中

Quiver的AIP功能旨在帮助您使用自然语言提示进行数据分析。除了在与Quiver相关的问题上从AIP Assist获得有用的回答外，用户还可以使用：

  • AIP Generate来添加和配置卡片，在保持用户参与的同时生成分析的下一步。
    • AIP如何使用本体论数据
  • AIP Configure通过描述数据或Quiver卡片应如何被操作来更改现有的卡片配置。
    • 配置时间序列图表

生成分析

你可以使用 AIP Generate 通过以下文本输入框从现有卡片或分析中的数据继续进行分析：继续分析 下一步操作的选项卡，这会提示你 Enter a query to continue the exploration。

在文本输入框中编写你想执行的分析操作，AIP Generate 将使用 Palantir 提供的大语言模型（LLM） 建议多种选项。

为了获得最佳效果，请尽可能详细地描述你的需求。你可以尝试将你的提示语句以问题形式表达，例如“销售趋势如何随时间变化？”或者采用指令语气，如“显示按月份分组的销售收益条形图”或“显示按广告渠道分组的行和按零售商分组的列的市场营销支出总和”，例如。

当准备就绪时，选择 AIP 提供的一个建议选项以生成并添加卡片到你的画布中，根据需要进行其他配置。

如果 AIP 认为提示需要多个步骤，那么 AIP 生成会在选项标题旁边包含一个 + 跟进 操作标签，你可以将光标悬停在该标签上以查看它将执行的额外步骤。

AIP Generate 在你选择包含 + 跟进 操作标签的选项时进入 Chain Of Thought 模式，使你能够基于初始提示查看其逻辑。在根据你的提示配置卡片后，AIP Generate 会自动输入 + 跟进 中包含的步骤以及之前步骤的历史记录。

选择 AIP Generate 的一个选项以继续，或选择 重置 返回到原始输入提示。

选择 Tab，从 AIP Generate 的文本输入框中将其添加到队列中，以便于迭代的最新提示。

AIP Generate 功能目前主要支持对象和对象集卡片。支持的卡片在其右侧带有 AIP 图标。

此外，您还可以从画布顶部的选项卡中选择并启动 AIP Generate，这使得 AIP 能够更加自主地进行操作，并在没有人工指导的情况下执行多步骤分析。

在查询框中输入您想要执行的下一个操作。AIP 会在您输入时提供提示补全建议。

准备就绪后，选择 生成。

要更详细地查看每张卡片，您可以使用鼠标滚轮手势放大或缩小图表，并通过选择左下角的按钮来拖动图表。

将光标悬停在卡片上，可以查看有关为何选择特定卡片以及卡片如何根据相应提示进行配置的有用解释。

要将生成的图表添加到分析画布中，请选择添加卡片。然后，您可以进一步修改或确定每张卡片的配置或可视化设置。

如何利用本体论数据

为了高效回答用户提示，AIP仅使用对象集数据的摘要版本，或“属性值提示”，如下所示：

  • 字符串属性类型：计算最多100个唯一值

  • 数值属性类型：最小值、最大值和平均值会被计算

这些摘要允许在进行定量描述（如高、低、中等）时确定阈值，同时也能纠正列名和属性值的拼写错误。

配置卡片

您可以使用 AIP 配置来更新卡片配置。

要访问 AIP 配置，请在 AIP 生成输入栏中选择 修改 按钮。

接下来，输入您希望对卡片所做的更改的描述，例如“将图例移动到底部并将样式更改为分组”或“将指标更改为销售总额，并将y轴标题更新为总销售额”。当您输入时，AIP会生成一系列卡片配置建议。每个建议显示与您提示中某一部分匹配的配置设置，您可以单独接受或拒绝这些设置。

准备就绪后，选择AIP的一个或多个建议，AIP会立即应用这些建议到卡片上。

如果您希望AIP进行其他配置，可以输入新的提示并重复该过程。

一些卡片，如Vega 图表，支持通过AIP Configure进行下游分析，但不支持 AIP Generate。

通过 AIP 配置，即使您不熟悉每个 Quiver 卡片提供的所有可视化设置，每个人现在也可以获得完美的可视化效果。

时间序列

当用于分析 时间序列 图表时，AIP 作用于图表中的单个时间序列，而不是图表容器。

AIP 配置在您选择要分析的图表中的一个时间序列后出现。

AIP 功能的可用性可能会发生变化，并且在不同注册中可能有所不同。
