语言模型类

palantir_models 提供了一组可在 Python 转换中使用的类。

<
类描述
OpenAiGptChatLanguageModelInput一种 FoundryInputParam，绑定到语言模型服务的 OpenAiGptChatLanguageModel API。在 Python 转换装饰器中使用。
GenericCompletionLanguageModelInput一种 FoundryInputParam，绑定到语言模型服务的 GenericCompletionLanguageModel API。在 Python 转换装饰器中使用。
GenericEmbeddingModelInput一种 FoundryInputParam，暴露了语言模型服务的 GenericEmbeddingsModel API。在 Python 转换装饰器中使用。
OpenAiGptChatLanguageModel提供访问 OpenAI 聊天补全模型的客户端。
GenericCompletionLanguageModel提供客户端，用于执行 AIP 中可用的任何聊天补全模型。
GenericEmbeddingModel提供客户端，用于执行 AIP 中可用的任何嵌入模型。
