XY图表

如果XY图表（Chart XY）小部件不支持所需的功能或格式，可以考虑使用Vega图表小部件。

XY图表小部件用于将对象（Object）可视化为交互式图表。配置XY图表小部件的模块构建者可以：

  • 以条形图、折线图和散点图显示数据。
  • 选择要可视化的属性，这些属性如何聚合（例如：计数、总和、平均值），以及是否以及如何对属性进行分段。
  • 使用函数（Function）支撑的图层来显示更高级的聚合类型和图表。
  • 设置图表标题、坐标轴、图例和数值格式的显示选项。
  • 在图表上启用选择和下游过滤功能。

图层配置选项

要向Chart XY小部件添加数据，必须配置图层。以下为图层提供的配置选项：

  • 标题
    • 在你的配置面板中设置当前图层的标题。
    • 注意：此标题对模块用户不可见，但旨在帮助构建者整理和管理使用多个图层的XY 图表配置。
  • 数据输入
    • 控制此图层的输入数据。
    • 对象集选项允许将Workshop对象集变量用作输入。
    • 函数聚合选项允许使用返回二维聚合或三维聚合的函数作为输入。
      • 注意：函数聚合图层仅会提供以下部分图层配置选项。
    • 时间序列集选项允许将Workshop时间序列集变量用作输入。有关时间序列集变量的更多信息，请参阅变量。此配置用于设置时间序列图，其中时间范围在X轴上，变量的时间序列值在Y轴上。
  • 图层类型
    • 选择显示的图表类型。当前选项包括柱状图、折线图和 散点图。如果数据输入是一个时间序列集，仅支持折线图选项。有关时间序列集变量的更多信息，请参阅 变量。
  • 面积选项
    • 为折线图提供三种可视化选项：
      • "折线"（显示简单的折线图），
      • "面积"（绘制折线图并在每条线下面填充颜色），以及
      • "堆叠"（类似于"面积"选项，但将分段图表值堆叠在一起）。
    • 面积选项仅适用于折线图。
  • 标签
    • 切换图表上数值标签的显示。
    • 此选项目前适用于柱状图和折线图。
  • X轴属性
    • 确定在图表上绘制的属性类型。
  • 柱状图/折线图/图表系列
    • 使用单系列/多系列
      • 允许为所选的X轴属性绘制一个或多个图表系列。
      • 例如，如果“警报类型”被选为X轴属性，多个系列将允许绘制每个“警报类型”的计数以及每个“警报类型”的“延迟小时数”的总和。
    • 系列聚合
      • 确定用于生成图表上每个值的聚合方法。
      • 默认情况下，此选项设置为“计数”。
      • 其他选项包括：“平均值”、“最小值”、“最大值”、“总和”和“近似唯一计数”。
    • 按...分段
      • 可选。
      • 使每个绘制的值可以按第二种属性类型进行分段。
      • 例如，如果“警报类型”被选为X轴属性，然后在柱状图中选择“飞机类型”作为按...分段选项，每个柱子将显示每个“警报类型”按每个“飞机类型”划分的对象数量。
    • 空值/缺失值的显示
      • 仅适用于折线图。
      • 控制折线图中空值或缺失值的显示方式。
      • 选项包括“空缺”（缺失值在绘制的线条中显示为空白间隙）、“忽略”（缺失值被忽略，绘制的线条则连接前一个和下一个可用值）或“零值”（缺失值被视为等同于“0”的值）。
    • 显示覆盖
      • 可选。
      • 覆盖当前系列的图例显示名称。
      • 对于分段图表，覆盖单个分段的图例显示名称。
    • 分段覆盖
      • 仅适用于柱状图。
      • 修改每个柱状图值的显示方式。
      • 选项包括：“堆叠”、“百分比”和“分组”。
  • 选择作为过滤器
    • 可选，且仅适用于由对象集支撑的图表。
    • 当设置时，允许通过输出对象集过滤器变量对该图表层进行选择和下游小部件过滤。
  • 删除图层
    • 允许删除当前图表图层。
  • 场景
    • 与情景对比
      • 启用此切换以选择要比较数据的情景数组变量。这将使用图表的“按段”轴，将表格中的数据与数组中情景的值进行比较。
      • 如果启用了此选项，则不能通过其他属性进行分段。
    • 有关情景的更多信息，请参阅 情景文档。

图表全局配置选项

除了上述描述的层配置选项外，主图表XY配置面板还包含一些图表全局配置选项：

  • 分类轴
    • 显示标题
      • 如果启用，将显示分类轴的标题，并允许覆盖该标题。
      • 默认情况下，此标题将显示图表系列中绘制的属性类型。
    • 启用数值格式
      • 如果启用，将为分类轴键中显示的数值提供配置选项。
      • 配置选项包括数值分组、显示的最小/最大小数位数、科学记数法等。
    • 按...排序
      • 控制图表中每个值的显示方式的排序逻辑。
      • 默认情况下，按字母顺序从 A 到 Z 对分类键进行排序。
  • 值轴
    • 使用多个值轴
      • 仅在已配置多个图表系列时可用，并允许根据每个系列配置值轴。当图表上的不同系列具有显著不同的值尺度时，这会很有帮助。
    • 显示标题
      • 如果启用，将显示值轴的标题，并允许覆盖该标题。
      • 默认情况下，此标题将显示图表系列中使用的聚合类型。
    • 启用数值格式化
      • 如果启用，将为值轴中显示的数值提供配置选项。
      • 配置选项包括数值分组、显示的最小/最大小数位数、科学计数法等。
    • 比例类型
      • 允许将值轴比例设置为“线性”（默认）或“对数”。
    • 最小边界
      • 默认情况下，根据显示的图表值自动计算最小边界。
      • 如果切换为“最小值”，模块构建器可以控制值轴上显示的最小值。
    • 最大边界
      • 默认情况下，根据显示的图表值自动计算最大边界。
      • 如果切换为“最大值”，模块构建器可以控制值轴上显示的最大值。
  • 图例
    • 显示图例
      • 切换图表系列标题和系列颜色的图例显示。
    • 定位选项
      • 当启用“显示图例”时，控制图例在图表中的位置。
  • 条形方向
    • 水平/垂直切换
      • 控制图表的方向。
      • 对于条形图，“水平”是默认值。
      • 对于折线图或散点图，只允许“垂直”。

函数聚合（函数支撑的图层）

配置一个函数支撑的图层需要编写一个返回TwoDimensionalAggregation或ThreeDimensionalAggregation的函数。

以下是一个完整的示例，用于返回一个TwoDimensionalAggregation，以便将一个时间序列除以另一个时间序列进行图表展示：

Copied!
1import { Function, TwoDimensionalAggregation, ThreeDimensionalAggregation,
2         IRange, Timestamp } from "@foundry/functions-api";
3import { ObjectSet, MyObjectType } from "@foundry/ontology-api"
4
5export class TimeseriesAggregations {
6
7    @Function()
8    public async percentOfTotal(objects:ObjectSet<MyObjectType>):
9                                Promise<TwoDimensionalAggregation<IRange<Timestamp>>> {
10        const numerators = await objects.groupBy(e => e.date.byDays())
11                                        .sum(e => e.value);
12        const denominators = await objects.groupBy(e => e.date.byDays())
13                                          .sum(e => e.total);
14
15        return this.divide(numerators, denominators);
16    }
17
18    private divide(numerators:TwoDimensionalAggregation<IRange<Timestamp>>,
19                              denominators: TwoDimensionalAggregation<IRange<Timestamp>>):
20                              TwoDimensionalAggregation<IRange<Timestamp>> {
21
22        const percentage = numerators.buckets.map((bucket, i) => {
23           const numerator = bucket.value;
24           const denominator = denominators.buckets[i].value;
25            if (denominator == 0) {
26                return { key: bucket.key, value: 0 };
27            }
28            return { key: bucket.key, value: numerator / denominator }
29        });
30
31        return { buckets: percentage };
32    }
33}

以下是一个完整的示例，返回一个ThreeDimensionalAggregation，它将为segmentBy()返回的每个值绘制单独的系列：

Copied!
1import { Function, TwoDimensionalAggregation, ThreeDimensionalAggregation,
2         IRange, Timestamp } from "@foundry/functions-api";
3import { ObjectSet, MyObjectType } from "@foundry/ontology-api"
4
5export class TimeseriesAggregations {
6
7    @Function()
8    public async percentOfTotalSegmented(objects:ObjectSet<MyObjectType>):
9                                         Promise<ThreeDimensionalAggregation<IRange<Timestamp>, string>> {
10        const numerators = await objects.groupBy(e => e.date.byDays())
11                                        .segmentBy(e => e.groupId.topValues())
12                                        .sum(e => e.value);
13        const denominators = await objects.groupBy(e => e.date.byDays())
14                                          .segmentBy(e => e.groupId.topValues())
15                                          .sum(e => e.total);
16
17        return this.divideThreeDimensional(numerators, denominators);
18    }
19
20    private divideThreeDimensional(numerators:ThreeDimensionalAggregation<IRange<Timestamp>, string>,
21                             denominators: ThreeDimensionalAggregation<IRange<Timestamp>, string>):
22                             ThreeDimensionalAggregation<IRange<Timestamp>, string> {
23
24        var percentage = numerators.buckets; //copy
25        for (let i = 0; i < numerators.buckets.length; i++) {
26            for (let j = 0; j < numerators.buckets[i].value.length; j++) {
27                percentage[i].value[j].value = numerators.buckets[i].value[j].value /
28                                               denominators.buckets[i].value[j].value;
29            }
30        }
31
32        return { buckets: percentage };
33    }
34}

有关更多示例，请参阅Foundry的函数文档中的对象集聚合和创建自定义聚合。
