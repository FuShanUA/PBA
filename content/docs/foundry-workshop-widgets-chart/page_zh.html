Chart XY

如果 Chart XY 部件无法实现所需的功能或格式化效果，请考虑使用 Vega Chart 部件。

Chart XY 部件用于将对象可视化为交互式图表。配置 Chart XY 部件的模块构建者可以执行以下操作：

  • 以柱状图、折线图和散点图形式展示数据。
  • 选择要可视化的属性、这些属性的聚合方式（例如计数、求和、平均值），以及是否对属性进行分段及如何分段。
  • 使用函数支持层来展示更高级的聚合类型和图表。
  • 设置图表标题、坐标轴、图例和数字格式的显示选项。
  • 在图表上启用选择和下游过滤功能。

层配置选项

配置层是向 Chart XY 部件添加数据的必经步骤。层提供以下配置选项：

  • 标题
    • 在配置面板中为当前层设置标题。
    • 注意：此标题对模块用户不可见，旨在帮助构建者组织和管理使用多个层的复杂 Chart XY 配置。
  • 数据输入
    • 控制此层的输入数据。
    • 对象集选项允许使用 Workshop 对象集变量作为输入。
    • 函数聚合选项允许将返回二维聚合或三维聚合的函数用作输入。
      • 注意：函数聚合图层仅支持以下部分图层配置选项。
    • 时间序列集合选项允许将 Workshop 的时间序列集合变量用作输入。有关时间序列集合变量的更多信息，请参阅变量。此选项会配置一个时间序列图表，X 轴显示时间范围，Y 轴显示变量的时间序列值。
  • 图层类型
    • 选择显示的图表类型。当前选项包括柱状图、折线图和散点图。如果数据输入是时间序列集合，则仅支持折线图选项。有关时间序列集合变量的更多信息，请参阅变量。
  • 区域选项
    • 为折线图提供三种可视化选项：
      • “折线”（显示简单的折线图），
      • “区域”（绘制折线图并为每条线下方的区域填充阴影），以及
      • “堆叠”（类似于“区域”选项，但将分段图表值彼此堆叠在一起）。
    • 区域选项仅适用于折线图。
  • 标签
    • 切换图表上值标签的显示。
    • 此选项目前适用于柱状图和折线图。
  • X 轴属性
    • 确定图表上绘制的属性类型。
  • 柱状/折线/图表系列
    • 使用单个/多个系列
      • 允许为选定的X 轴属性绘制一个或多个图表系列。
      • 例如，如果选择“告警类型”作为X 轴属性，多个系列将允许既绘制每个“告警类型”的计数，也绘制每个“告警类型”的“延迟小时数”总和。
    • 系列聚合
      • 确定用于生成图表上每个绘制值的聚合方法。
      • 默认情况下，此项设置为“计数”。
      • 其他选项包括：“平均值”、“最小值”、“最大值”、“总和”和“近似唯一计数”。
    • 分段依据
      • 可选。
      • 允许按次要属性类型对每个绘制值进行分段。
      • 例如，如果在柱状图中选择“警报类型”作为X 轴属性，然后选择“按...分段”选项为“飞行器类型”，则每个柱状条将显示按每种“飞行器类型”分段的各“警报类型”的对象计数。
    • 空值/缺失值的显示
      • 仅适用于折线图。
      • 控制折线图上空值或缺失值的显示方式。
      • 选项包括“间隙”（缺失值在绘制的线条中显示为空白间隙）、“忽略”（忽略缺失值，绘制的线条直接连接前一个和后一个可用值）或“零”（缺失值被视为等同于值“0”）。
    • 显示覆盖
      • 可选。
      • 覆盖当前系列的图例显示名称。
      • 对于分段图表，覆盖单个分段的图例显示名称。
    • 分段覆盖
      • 仅适用于柱状图。
      • 修改每个柱状图值的显示方式。
      • 选项包括：“堆叠”、“百分比”和“分组”。
  • 作为过滤器选择
    • 可选，且仅适用于以对象集为数据源的图表。
    • 设置后，允许通过输出的对象集过滤器变量对此图表层进行选择和下游组件过滤。
  • 删除图层
    • 允许删除当前图表图层。
  • 场景
    • 与场景对比
      • 启用此开关可选择要对比数据的场景数组变量。这将使用图表的“按...分段”轴，将表格中的数据与数组中各场景的值进行对比。
      • 如果启用此选项，则无法按其他属性进行分段。
    • 有关场景的更多信息，请参阅《场景文档》。

图表全局配置选项

除了上述针对图层的配置选项外，主要的 XY 图表配置面板还包含多项图表全局配置选项：

  • 类别轴
    • 显示标题
      • 启用后，将显示类别轴的标题，并允许覆盖该标题。
      • 默认情况下，此标题将显示在图表系列中绘制的属性类型。
    • 启用数字格式化
      • 启用后，可为分类轴键中显示的数值提供配置选项。
      • 配置选项包括数字分组、显示的最小/最大小数位数、科学计数法等。
    • 排序依据
      • 控制每个图表数值的显示排序逻辑。
      • 默认情况下，分类键按字母从 A 到 Z 排序。
  • 数值轴
    • 使用多个数值轴
      • 仅在配置了多个图表系列时可用，允许按系列配置数值轴。当图表中不同系列的数据量级差异较大时，此功能非常实用。
    • 显示标题
      • 启用后，将显示数值轴的标题，并允许覆盖该标题。
      • 默认情况下，该标题会显示图表系列中所使用的聚合类型。
    • 启用数字格式化
      • 启用后，可为数值轴上显示的数值提供配置选项。
      • 配置选项包括数字分组、显示的最小/最大小数位数、科学计数法等。
    • 刻度类型
      • 允许将数值轴刻度设置为“线性”（默认）或“对数”。
    • 最小边界
      • 默认情况下，根据显示的图表数值设为“自动计算最小边界”。
      • 若切换为“最小值”，模块构建者可控制数值轴上显示的最小值。
    • 最大边界
      • 默认情况下，根据显示的图表数值设为“自动计算最大边界”。
      • 若切换为“最大值”，模块构建者可控制数值轴上显示的最大值。
  • 图例
    • 显示图例
      • 切换图表系列标题及系列颜色的图例显示状态。
    • 位置选项
      • 启用“显示图例”后，可控制图例在图表中的位置。
  • 条形方向
    • 水平/垂直切换
      • 控制图表的方向。
      • 对于条形图，默认为“水平”。
      • 对于折线图或散点图，仅允许使用“垂直”。

1import { Function, TwoDimensionalAggregation, ThreeDimensionalAggregation,
2         IRange, Timestamp } from "@foundry/functions-api";
3import { ObjectSet, MyObjectType } from "@foundry/ontology-api"
4
5export class TimeseriesAggregations {
6
7    @Function()
8    public async percentOfTotalSegmented(objects:ObjectSet<MyObjectType>):
9                                         Promise<ThreeDimensionalAggregation<IRange<Timestamp>, string>> {
10        const numerators = await objects.groupBy(e => e.date.byDays())
11                                        .segmentBy(e => e.groupId.topValues())
12                                        .sum(e => e.value);
13        const denominators = await objects.groupBy(e => e.date.byDays())
14                                          .segmentBy(e => e.groupId.topValues())
15                                          .sum(e => e.total);
16
17        return this.divideThreeDimensional(numerators, denominators);
18    }
19
20    private divideThreeDimensional(numerators:ThreeDimensionalAggregation<IRange<Timestamp>, string>,
21                             denominators: ThreeDimensionalAggregation<IRange<Timestamp>, string>):
22                             ThreeDimensionalAggregation<IRange<Timestamp>, string> {
23
24        var percentage = numerators.buckets; //copy
25        for (let i = 0; i < numerators.buckets.length; i++) {
26            for (let j = 0; j < numerators.buckets[i].value.length; j++) {
27                percentage[i].value[j].value = numerators.buckets[i].value[j].value /
28                                               denominators.buckets[i].value[j].value;
29            }
30        }
31
32        return { buckets: percentage };
33    }
34}
