创建路径

在 Contour 中，你可以使用两种类型的数据集来开始新的分析路径：

  • 保存在平台上的数据集
  • 当前分析中某一路径的结果

从 Foundry 中的数据集开始分析

在 Foundry 中，每个用户都有一个名为“你的文件”的专属文件夹。用户可以在此文件夹中使用数据进行原型设计，并与其他用户共享结果。进入该文件夹后，用户即可创建自己的首个分析：

  1. 导航至“项目与文件”模块。
  2. 导航至“你的文件”。
  3. 选择绿色的“新建”操作菜单，并在下拉菜单中选择“分析”，以创建新的分析。
  4. 生成新分析后，选择“创建新路径”。
  5. 在相应文件夹中选择要分析的数据集。

使用结果数据集开始新路径

单个分析可以包含多条路径，将某一路径的分析结果用作新路径的起点往往很有价值。我们建议在第一条路径中完成对数据集的所有筛选操作。这样，你就可以在其他路径中利用生成的结果数据集来进行可视化展示，并进一步转换目标数据集。我们还建议将 Contour 分析中参数的使用集中到这条主筛选路径中。

每条路径的底部都有一个结果数据集，反映了使用该路径中的看板执行的所有转换操作。无需保存结果数据集，你就可以将其作为同一分析中新路径的起始数据集。

关于结果数据集的说明

保存结果数据集后，不仅可以在其他 Contour 分析中使用，还能在平台上的其他工具中使用。如果你是在同一个 Contour 分析的另一条路径中继续分析，则无需保存结果数据集。
