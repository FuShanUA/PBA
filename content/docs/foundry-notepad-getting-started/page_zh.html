入门指南

本教程将引导您创建新的 Notepad 文档，添加文本、图片和表格，然后将文档导出为 PDF 或 DOCX 格式。

创建 Notepad 文档

可以通过多种方式创建新的 Notepad 文档：

  • 将鼠标悬停在 Foundry 侧边栏的 Notepad 应用程序图标上，然后选择新建文档。
  • 在项目文件夹中选择+ 新建 > Notepad 文档。
  • 右键单击项目文件夹中的任意位置，然后选择新建 > Notepad 文档。

创建文档后，首先添加标题：输入标题文本，然后在顶部工具栏中选择大标题进行相应格式化。

接下来，向文档添加一些文本，并添加第二列以将图片紧贴放置在文本旁边。为此，首先在工具栏中选择添加列，然后拖放现有图片，或直接从剪贴板将图片粘贴到新列中。

最后，在文档底部插入表格。为此，请通过在段落中键入/快捷方式，或在工具栏中选择+ 部件并搜索表格，来打开部件插入菜单。

导出 Notepad 文档

要导出文档以与他人共享，请选择左上角的文件菜单，然后选择导出为 PDF或导出为 DOCX以触发文档渲染过程。Notepad 完成渲染后，Foundry 会自动下载文档并以相应的文件格式保存。

搜索 Notepad 文档

使用 Notepad 主页上的搜索文档栏，搜索您有权访问的任何文档。

要从当前查看的文档导航至 Notepad 主页，请选择文件 > 打开文档。
