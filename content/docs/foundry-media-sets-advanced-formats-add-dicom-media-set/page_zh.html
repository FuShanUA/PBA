添加 DICOM 媒体集

本指南将详细介绍如何将 DICOM (.dcm) 文件作为媒体集导入到 Foundry 中。

第一部分：导入 DICOM 文件

首先，您需要创建一个新的媒体集，并将 DICOM 文件添加到该媒体集中。

  1. 导航至将用于创建媒体集的文件夹。选择 新建 > 媒体集。



  2. 为您的媒体集输入名称。选择 DICOM 作为媒体类型，并选择 批处理 作为延迟模式。选择 创建媒体集 以创建 DICOM 媒体集。



  3. 接下来，将一个或多个 .dcm 文件添加到媒体集中。



DICOM 媒体集类型包含诸如 患者 ID 和 检查 ID 等元数据。

您可以选择一个 DICOM 文件，通过左右上下拖动来调整对比度和曝光度。

第二部分：创建对象类型

接下来，您需要创建一个新的管道，将媒体集转换为可在 Foundry 中使用的对象类型。

了解有关为媒体集创建管道的更多信息。

  1. 从 所有操作 下拉菜单中选择 创建新管道，以此创建一个管道。



  2. 媒体集将自动添加到管道中。选择 转换 以将媒体集转换为表格。



  3. 选择 将媒体集转换为表行，然后选择 应用。



    在生成的表格中，每一行代表媒体集中的一个 DICOM 文件。



  4. 从右侧面板的 管道输出 菜单中选择 添加管道输出，以此创建对象类型。



    选择 对象类型 选项。



  5. 为对象类型输入名称，例如 DICOM 媒体集。您可以选中 媒体项 Rid 属性右侧的三个点，然后选择 设为主键，将其设置为主键。



完成后，您可以保存并部署该管道。

管道部署完成后，您可以在对象资源管理器或本体管理器中查看该对象类型。

（可选）第三部分：创建 Workshop 模块

您可以通过选择 创建 Workshop 模块 来打开 Workshop。

Workshop 将自动生成实用的组件，例如对象表格和预览。

了解有关在 Workshop 中创建组件的更多信息。
