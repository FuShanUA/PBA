添加一个DICOM媒体集

本指南将介绍如何将DICOM（.dcm）文件导入Foundry作为< a href="/docs/foundry/data-integration/media-sets/">媒体集。

第一部分：导入DICOM文件

首先，您需要创建一个新的媒体集，并将DICOM文件添加到媒体集中。

  1. 导航到将创建媒体集的文件夹。选择 新建 > 媒体集。



  2. 为您的媒体集输入名称。选择 DICOM 作为媒体类型并选择 批量作为延迟。选择创建媒体集来创建DICOM媒体集。



  3. 接下来，将一个或多个.dcm文件添加到媒体集中。



DICOM媒体集类型包括诸如Patient ID和Study ID之类的元数据。

您可以选择一个DICOM文件，并左右、上下拖动以更改对比度和曝光。

第二部分：创建对象类型

接下来，您需要创建一个新的管道，将媒体集转换为可在Foundry中使用的对象类型。

了解有关为媒体集创建管道的更多信息。

  1. 通过从所有操作下拉菜单中选择创建新管道来创建一个管道。



  2. 媒体集将自动添加到管道中。选择转换以将媒体集转换为表格。



  3. 选择将媒体集转换为表格行，然后选择应用。



    在生成的表格中，每一行代表媒体集中的一个DICOM文件。



  4. 通过从管道输出中选择添加管道输出来创建一个对象类型。 右侧面板中的菜单。



    选择 对象类型（Object type） 选项。



  5. 为对象类型（Object type）输入一个名称，例如 DICOM media set。您可以通过在属性的右侧单击三个点，然后选择 设为主键（Set as primary key） 来将 Media Item Rid 属性设置为主键（Property）。



完成后，您可以 保存并部署管道（pipeline）。

管道（pipeline）部署后，您可以在对象（Object）资源管理器或本体论（Ontology）管理器中查看对象类型（Object type）。

（可选）第三部分：创建 Workshop 模块

您可以通过选择 创建 Workshop 模块（Create Workshop module） 打开 Workshop。 。

Workshop 将自动生成有用的小部件，例如对象表和预览。

了解有关在 Workshop 中创建小部件的更多信息。
