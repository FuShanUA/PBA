foundryts.objects.Object

class foundryts.objects.Object(object_type_id)

本体中的对象类型。

该类提供了一系列方法，用于创建对存储为本体对象中时间序列属性的时间序列的引用。

  • 参数： object_type_id (str) – 本体中对象类型的 ID。
注意

请确保为 object_type_id 传入的是 ID，因为平台上提供了三种对象类型引用：ID、API、RID。

示例

已复制！
1>>> aircraft_object_type = Object("aircraft") # 对象类型引用
2>>> airplane = airplane_object_type.id("aircraft-1") # 现在可以使用主键获取对象引用

id(object_primary_key_value)

使用对象的主键创建对该本体对象的引用。

  • 参数： object_primary_key_value (str) – 对象的主键，可以在定义该对象的数据集中或通过 ↗ 对象浏览器找到。
  • 返回： 对本体对象的引用，可用于通过 时间序列属性以及 FoundryObject.property() 访问时间序列属性
  • 返回类型： FoundryObject

示例

已复制！
1>>> aircraft_object_type = Object("aircraft")
2>>> airplane = airplane_object_type.id("aircraft-1") # 该对象引用可用于访问 TSP
