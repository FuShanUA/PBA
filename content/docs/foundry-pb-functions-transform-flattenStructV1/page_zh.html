结构体展平

支持于：Batch, Faster, Streaming

提取结构体中的所有字段，并将其转换为输出数据集中的列。

转换类别：Struct

声明的参数

  • 数据集： 包含结构体列的数据集。
    Table
  • 表达式： 解析为待展平结构体列的表达式。
    Expression<Struct>
  • 最大深度： 指定嵌套结构体展平深度的层级数值。
    Literal<Integer>
  • 可选 列前缀： 为展平过程中创建的所有列添加前缀。
    Literal<String>
  • 可选 分隔符： 用于分隔来自嵌套结构体的字段名称。
    Literal<String>

示例

示例 1：基础用例

参数值：

  • 数据集： ri.foundry.main.dataset.a
  • 表达式： raw
  • 最大深度： 2
  • 列前缀： new_
  • 分隔符： null

输入：

raw
{
 airline: {
 id: NA,
 name: new air,
},
 tail_no: NA-123,
}
{
 airline: {
 id: FA,
 name: foundry airways,
},
 tail_no: FA-123,
}

输出：

new_airline_namenew_airline_idnew_tail_noraw
new airNANA-123{
 airline: {
 id: NA,
 name: new air,
},
 tail_no: NA-123,
}
foundry airwaysFAFA-123{
 airline: {
 id: FA,
 name: foundry airways,
},
 tail_no: FA-123,
}
