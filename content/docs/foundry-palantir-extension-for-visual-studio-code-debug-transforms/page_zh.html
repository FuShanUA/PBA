调试转换

Palantir 的 Visual Studio Code 扩展支持在调试 Python 转换时检查变量。复杂的数据结构，如数据框（DataFrames）可以加载到扩展的 预览 面板中，使您能够在 初始化构建 之前验证转换的准确性并识别问题。

开始调试会话

要使用 Palantir 的 Visual Studio Code 扩展进行调试，请打开包含转换的 Python 文件。在代码中至少设置一个断点，以便在执行该代码时暂停执行。当达到断点时，Visual Studio Code 的 调试 面板将自动打开并显示当前作用域中的变量。

要开始调试会话，请在转换上方选择 调试。

当在调试会话中达到断点时，您可以通过以下两种方法使用预览面板检查受支持的变量：

  • 在您希望检查的变量上方选择🔎 检查当前值 …

  • 在 Visual Studio Code 的 调试 面板中右键单击变量，然后选择 Palantir：检查变量。

检查变量会将其加载到扩展程序的 预览 面板中。然后您可以查看变量的内容，并使用 预览 面板的功能来探索数据，例如查看列统计信息。
