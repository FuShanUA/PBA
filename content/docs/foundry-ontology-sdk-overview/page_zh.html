本体 SDK (OSDK)

本体软件开发工具包 (OSDK) 允许您直接从开发环境中调用本体的全部功能。OSDK 支持 TypeScript 环境下的 NPM (Node Package Manager)、Python 环境下的 Pip 或 Conda、Java 环境下的 Maven，以及适用于其他任意语言的 OpenAPI 规范。

将 Foundry 视为您的后端基础，便能利用本体执行大规模查询的强大能力与 Foundry 的数据回写功能，配合细粒度的治理面管控，从而加速安全开发进程，打造赋能企业组织的各类应用。

如需创建并管理 OSDK 应用，请使用《开发者控制台》。本章节涵盖 SDK 专属的参考文档。

OSDK 的核心优势

OSDK 旨在提供以下几项核心优势：

  • 加速开发进程： 借助 OSDK，您可以迅速投入开发由 Foundry 本体驱动的应用程序。OSDK 提供了便捷的方式来调用本体 API，让您只需极少的代码即可实现对本体的读取与回写。
  • 强类型安全： 为 OSDK 生成的函数与类型，仅基于您实际所需的本体子集。这些类型和函数直接从您的本体中生成，便于您在编辑器内直接查询和探索本体结构。
  • 集中化维护： 由于本体是在 Foundry 中进行集中构建与管理的，您可以专注投入应用开发，从而减轻构建数据底座通常所需的维护负担。
  • 设计内生安全： 除了依赖用户自身的数据权限外，OSDK 采用的令牌仅限应用于您希望程序访问的本体实体，实现权限最小化。

此外，面向前端开发的 TypeScript 绑定，为开发者快速构建基于 Foundry 的 React 应用提供了极大便利。

生成的代码会使用关于您本体的元数据，包括属性名称与描述。您可以直接在编辑器中查看这些元数据。
