外部缓存连接

支持于：流处理

将左侧与右侧数据集输入连接在一起，缓存两侧事件时间最新的记录以供后续连接使用。记录的处理时间将作为决胜条件。在时间连接场景下，若没有可匹配的值，则会乐观地输出结果。

转换类别：连接

声明的参数

  • 默认缓存时间单位：左侧和右侧缓存在数据被淘汰前所缓存的时间长度的默认单位。
    Enum<Days, Hours, Milliseconds, Minutes, Seconds, Weeks>
  • 默认缓存时间值：左侧和右侧缓存在数据被淘汰前所缓存的时间长度的默认值。
    Literal<Long>
  • 连接键：用于连接的左侧和右侧输入的列列表。
    List<Tuple<Column<Binary | Boolean | Byte | Double | Float | Integer | Long | Short | String | Timestamp>, Column<Binary | Boolean | Byte | Double | Float | Integer | Long | Short | String | Timestamp>>>
  • 左侧数据集：连接中使用的左侧数据集。
    Table
  • 右侧数据集：连接中使用的右侧数据集。
    Table
  • 可选 右侧缓存时间覆盖：右侧数据集的数据在淘汰前所缓存的时间值和单位。如果缓存时间设为 0，则右侧不进行缓存。若希望左侧也不缓存，请将默认缓存时间设为 0 并使用此参数进行覆盖。
    Tuple<Literal<Long>, Enum<Days, Hours, Milliseconds, Minutes, Seconds, Weeks>>
